====== Dactylo ======
 
Dactylo est un logiciel permettant d'apprendre à taper rapidement au clavier, et avec ses dix doigts. Créé par geenux, il est fortement inspiré du logiciel Dactylo de G. LE DU.
Il affiche un clavier virtuel avec mise en évidence de la touche à presser et donne le nom du doigt à utiliser.

{{:dactylo_screenshot.png?700|}}
 
===== Pré-requis =====
 
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  * Avoir [[:tutoriel:comment_installer_un_paquet|installé le paquet]] **[[apt>libqt4-gui|libqt4-gui]]**.
 
===== Installation =====

====Via le paquet DEB====
 
Il suffit d'[[:tutoriel:comment_installer_un_paquet#installer_un_paquet_isole|installer le paquet]] [[https://app.box.com/s/r4iqd5r9b6rrybhnufz7|dactylo]]
 
====Via le binaire====

  * Télécharger l'archive à l'adresse suivante : [[http://skateinbrest.free.fr/dactylo.tar.gz]]
  * Extraire le contenu de l'archive dans votre dossier personnel.

===== Utilisation =====
 
Si vous avez installé le logiciel via le paquet DEB, vous pouvez le lancer depuis le menu //Applications → Éducation → dactylo// ou via le [[:terminal]]  avec la [[:commande_shell|commande]] suivante :
<code>dactylo</code>
Si vous l'avez installé via le binaire, lancez-le via les [[:commande_shell|commandes]] suivantes :
  cd ~/dactylo
  ./dactylo

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.

Si vous avez installé le logiciel via le binaire, supprimez le dossier dactylo de votre dossier personnel.
