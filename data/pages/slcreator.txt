SLCreator est un GUI de DVD-Slideshow, permettant de prendre un ensemble de photos pour les faire défiler dans une vidéo. L'idée est d'ensuite intégrer cette vidéo sur un DVD de salon.

====== Procédure d'installation ======

===== Installation de dvd-slideshow =====
Tant que Edgy nous propose la version 0.7.2 de ce paquet, cela ne sert à rien de l'installer : c'est une version bogus qui ne fonctionne pas bien.

Aller sur le site du [[http://sourceforge.net/project/showfiles.php?group_id=100188|projet dvd-slideshow]] et cliquer sur le "package" **dvd-slideshow**. Puis charger le fichier le plus récent se terminant par *all.deb (ex: dvd-slideshow-0.8.0-1_all.deb). Firefox devrait vous proposer automatiquement d'installer ce paquet sur votre Ubuntu.

DVD Slideshow est inclus dans les dépots sous Jaunty. Pour l'installer, il suffit d'entrer cette commande dans un terminal
<code>sudo apt-get install dvd-slideshow</code>

===== Installation du paquet dvdauthor =====
Utiliser le gestionnaire de paquet pour installer **dvdauthor**.

===== Installation de SLCreator =====
Aller sur le site du [[http://sourceforge.net/project/showfiles.php?group_id=154121|projet SLCreator]], et cliquer sur le "package" **slcreator**. Puis charger le fichier le plus récent se terminant par *all.deb (ex: slcreator-0.7.1-1_all.deb). Firefox devrait vous proposer automatiquement d'installer ce paquet sur votre Ubuntu.
