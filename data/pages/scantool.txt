====== Scantool - logiciel de diagnostic automobile =====

===== Description =====
Scantool est un logiciel open-source conçu par les développeurs de ScanTool.net qui permet de réaliser le diagnostic d'une automobile.

Il s'utilise avec un "câble" OBD-II. Ce "câble" convertit un signal CAN-bus en RS-232 (port série). Les données reçues sont alors analysées par le logiciel.

===== Installation =====
Le paquet scantool est disponible dans le dépôt officiel d'ubuntu, il suffit de l'installer via synaptic, avec aptget ou un autre logiciel de gestion de paquets....

===== Fonctionnement =====
L'application a été initialement écrite pour DOS, c'est pourquoi la configuration interne utilise les ports COM1, COM2, ...

Pour contourner le problème sous Linux, la sélection du port COM1 tentera de se connecter à /dev/ttyUSB0 puis /dev/ttyS0, COM2 de se connecter à /dev/ttyUSB1 puis /dev/ttyS1, .

Lors de la connexion à un véhicule, la vitesse de transmission correcte devra être déterminée manuellement, par essai-erreur.

<note help>Si la connexion avec le périphérique OBD échoue, tenter de démarrer Scantool avec les droits administrateurs.</note>
