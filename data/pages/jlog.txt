====== JLOG ======

Jlog est un journal de trafic informatisé pour radioamateur sous Linux.
C'est un logiciel multiplate-forme fonctionnant sous java : disponible également pour windows et mac.
Disposant de multiple fonctions automatiques comme la recherche d'indicatif sur la base de donnés QRZ.com.
Les recherches dans le log sont assez rapide. ( inférieur à une seconde d’après le concepteur )
Vous pourrez gérer vos QSL, eQSL et sans oublier le fameux LOTW ! (Log Book Of The World), consulter le DX cluster et y envoyer vos spots.
Le logiciel dispose également d'un keyer CW intégré et d'un pilotage de vos transceivers par port série.







===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]] ;
  * Disposer d'une connexion à Internet configurée et activée ;
  * Avoir activé l'accès aux [[:depots#universe_et_multiverse|dépôts multiverse]] ;
  * Savoir utiliser une [[:console|console]]
  










===== Installation =====

Commencez par telecharger le fichier "install.bin" disponible à cette adresse :
http://jlog.org/v44/linux/install.bin

Nous allons maintenant installer JAVA.
Dans une console on tape :
  sudo apt-get install sun-java6-jre
il nous demande si on veut installer les paquets --> on repond "y" comme yes

une fois fini tapez :
  echo $PATH
il devrai il avoir quelque chose comme ceci qui apparait : /usr/java/ .
si ce n'est pas le cas, tapez :
  PATH=$PATH:/usr/java/j2re1.5.0_08/bin; export PATH
puis verifiez avec :
  echo $PATH
voila :) c'est tout bon (en principe)

une fois ca fait, il va falloir installer jlog proprement dit.

Pour cela il suffit de se rendre dans le repertoire où il est installé,
et de taper ceci :
  sh ./install.bin

Cela vous ouvre une fenetre d'installation.
Il suffit de suivre les instructions.

une fois le logiciel installé,
vous pouvez le lancer depuis votre repertoire perso avec la commande :
  sh runjLog

Biensur on peut rendre le fichier executable ou encore mieux : créer un lanceur.

Pour le rendre executable sans taper sh devant à chaques fois, il suffit de taper ceci:
  sudo chmod -x runjLog
