====== Cas d'utilisation ======

===== Utilisation courante =====

  * Connexion de l'utilisateur
  * Lancer le navigateur internet
  * Aller sur une page internet ( http://www.ubuntu-fr.org )
  * Copier du texte sur cette page
  * Ouvrir un traitement de texte
  * Coller le texte copié
  * Exporter le document au format PDF

===== Utilisation des fichiers =====
  * Créer sur le bureau un dossier **Perso**
  * Copier le fichier pdf créé précédemment dans ce dossier
  * Déplacer le fichier pdf créé précédemment dans ce dossier
  * Faire un raccourci vers le navigateur internet sur le bureau
  * Changer l'icône du raccourci
  * Donner la taille du fichier pdf
  * Donner la taille de l'espace disque disponible
  * Supprimer le fichier pdf
  * Restaurer le fichier de la corbeille

===== Personnalisation =====

  * Changer le fond d'écran
  * Changer le thème des fenêtres
  * Changer le thème d'icônes
  * Ajouter l'applet "Eyes"
  * Changer la paire d'yeux
  * Configurer l'applet date pour afficher la météo
  * Changer l'heure

===== Multimédia =====
  * Lire un fichier audio
  * Lire un fichier vidéo
  * Ouvrir une photo
  * Lire un fichier audio mp3
  * Lire une vidéo flash
  * Installer un greffon Firefox présent dans les dépôts

===== Administration =====
  * Installer le programme VLC
  * Lancer une vidéo avec VLC
  * CHanger l'association des fichiers vidéo pour les lancer avec VLC
  * Changer son mot de passe
  * Créer un nouvel utilisateur
  * Ouvrir une session avec ce nouvel utilisateur
  * Éteindre l'ordinateur
