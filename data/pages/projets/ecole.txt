====== Projet Ecole pour Ubunteros  ======




==== But ====

Le but de ce projet est de proposer des sessions d'apprentissage de tous niveaux, sur divers sujets (ayant bien entendu rapport avec les logiciels libres, en particulier Ubuntu).
Plus que de simples tutoriaux, les sessions auront pour but d'initier (ou perfectionner) les auditeurs sur un sujet choisi, par le biais d'exemples pratiques. Les auditeurs, à la fin de la session, devraient pouvoir partir avec un paquet Debian créé, un script python fonctionnel, un serveur LAMP correctement configuré, etc, ainsi que des pistes leur permettant de progresser par eux même suite à la session.

==== Les sessions ====

== Qui les mène ? ==
Chacun est susceptible de devenir professeur pour un temps. La seule condition est évidemment de connaître le sujet abordé.

== Où ? ==
Sur IRC, canal #ubuntu-fr-classroom, réseau FreeNode.

== Calendrier ==
Un calendrier devrait être mis en place, avec des [[http://wiki.ubuntu-fr.org/projets/ecole/sessions | sessions prévues]] à intervalles (plus ou moins) réguliers. Il faudra autant que possible varier les sujets abordés (ne pas parler de scripting python 3 sessions de suite), et alterner entre sessions pour débutants, et sessions pour initiés.

== Choix des sujets ==
Il faudra mettre en place un système ([[http://wiki.ubuntu-fr.org/projets/ecole/sessions | page wiki]] ? [[http://forum.ubuntu-fr.org/viewtopic.php?pid=627968 | thread du forum]] ?) permettant à tous les ubunteros de proposer

  * les sujets qu'ils voudraient voir traités ;
  * les sujets pour lesquels ils pourraient diriger une session.

== Après les sessions ==
Une "Wikifiction" des logs IRC sera faite et publiée, avec pour chaque session une FAQ résumant les questions ayant été posées par les "élèves" durant la session.
En attendant les logs "bruts" de sessions sont [[http://wiki.ubuntu-fr.org/projets/ecole/sessions | dispos ]] .
