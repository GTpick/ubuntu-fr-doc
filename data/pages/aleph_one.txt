====== Aleph One : la trilogie Marathon ======
Aleph One est le prolongement du moteur open source du jeu Marathon 2 de Bungie. Il permet de jouer à la [[wpfr>Marathon_(série)|trilogie Marathon]], qui connut une grande popularité sur MacOS dans les années 90. Disponible pour Mac OS X, Windows et Linux, Aleph One soutient Marathon 2 et Marathon Infinity en natif, et  Marathon à travers une conversion. De nombreux scénarios tiers et des cartes pour jeu multijoueur en ligne sont également disponibles.

{{ http://3.bp.blogspot.com/-_VGm9soIiEI/Tte8BrsB_sI/AAAAAAAABoo/OMqJUlnu96Y/s1600/Aleph+One+Linux.jpg?500 }}

=====Installation=====
==== Via le dépôt PlayDeb ====
Les trois jeux sont disponibles dans le dépôt [[PlayDeb]] à partir d'[[precise|Ubuntu 12.04 LTS]]. Après l'installation du dépôt, rendez-vous sur [[http://www.playdeb.net/updates/ubuntu/12.04/?q=marathon|cette fiche]] et cliquez sur le bouton « Install this now » de la version désirée.

==== Via les binaires officiels ====
Cette méthode d'installation est moins pratique. Si toutefois vous préférez cette méthode (ou pour l'installer sur une version antérieure d'Ubuntu), consultez [[https://alephone.lhowon.org/|le site officiel (en)]].

==== Compilation ====
  * https://github.com/Aleph-One-Marathon/alephone
===== Utilisation =====
Lancez l'application via le [[:unity#tableau_de_bord_dash|dash]] (Ubuntu 11.04 et ultérieures) ou via le [[:terminal]] (toutes versions d'Ubuntu) avec la [[:commande_shell|commande]] suivante :
  * Marathon :<code>marathon</code>
  * Marathon 2 :<code>marathon2</code>
  * Marathon Infinity :<code>marathon-infinity</code>


=====Désinstallation=====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
