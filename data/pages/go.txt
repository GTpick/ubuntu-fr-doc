====== Jeu de go sous Ubuntu======

===== Présentation =====
//Un jeu qui essaye de préserver des libertés depuis 4000 ans !//\\
Le jeu de go est jeu intuitif où deux joueurs essayent de conquérir un même espace.
C'est aussi un jeu millénaire, à la fois art et méthode d'apprentissage.
Les ordinateurs n'y sont pas doués, mais ils peuvent jouer.

//Aperçu de glgo//\\
{{http://jeuxlibres.net/files/screenshots/3c03f0ba7cf6977f136d694c2062adb5.png?640x480|Glgo}}

===== Jeux de GO sous Ubuntu =====

  * **[[apt>Kigo]]** : Un jeu de go intégré à KDE 4.
  * **Uligo** : petit programme pour s'entrainer. Propose des tsumegos (problèmes de vie et de mort). Il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt://uligo|uligo]]**, puis lancer le programme en tapant ''uligo'' dans un [[terminal]].
  * **qgo** : Qgo s'intègre bien à l'environnement KDE. Pour l'installer, il suffit d'[[:tutoriel:comment_installer_un_paquet|installer les paquets]] **[[apt://qgo|qgo]]** et **[[apt://gnugo|GNU Go]]** (([[:gnugo|GNU Go]]) est un moteur pour l'adversaire virtuel). Vous trouverez son [[raccourci-lanceur|lanceur]] dans le menu Applications->Jeux. Si vous voulez jouer contre l'ordinateur, allez dans le menu Fichier-Jouer avec un programme. L'adversaire informatique peut être un bon professeur, vous pouvez tester ses compétences [[http://gnugo.jeudego.org/cgi-bin/gnugo|ici]]
  * [[Cgoban]] : Cgoban permet de jouer contre d'ordinateur et sur un serveur type **IGS**. Pour l'installer, il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt://cgoban|cgoban]]**. Vous trouverez son [[raccourci-lanceur|lanceur]] dans le menu Applications->Jeux. 
  * [[Cgoban|Cgoban3]] : Cgoban3 est le client du serveur [[http://www.gokgs.com/|KGS]]. Il n'est pas disponible dans les dépôts et doit être installé depuis [[http://www.gokgs.com/download.jsp|le site]] (''sauvegarder / double-click sur cgoban.jnlp'') 
  * [[:Wbaduk]] (ex-Oro baduck): jeu de go fonctionnant sous Wine
  * [[:Glgo]] : nettement plus esthétique. Cette version gnome en 3D n'est pas encore intégrée aux paquets Ubuntu. Mais il semble qu'on puisse tout de même l'utiliser, voir la page [[:glgo|Glgo]].
  * **smartgo** : intéressant mais commercial, fonctionne bien avec [[:vmware]] (mais pas avec Wine).
  * [[:gnugo|GNU Go]] est un logiciel open-source qui vous permet de jouer au go. Pour l'installer, il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt://gnugo|GNU Go]]**.

===== Règle du jeu =====
//À la sauce ubuntu//
<note tip>[[http://jeudego.org/_php/dede.php|Initiation]]</note>

**Le jeu**
  * Les joueurs posent chacun leur tour une pierre sur les intersections du plateau, le goban. 
  * Ces pierres ne se déplacent pas, les emplacements libres (reliées par des traits) qui les entourent sont des libertés.
  * Ces libertés servent à s'unir pour former une chaîne.
  * Une liberté protégée par des chaînes est un souffle ou un œil.
  * Une chaîne qui a deux yeux est vivante, c'est-à-dire qu'il est impossible de la capturer.

**Le but du jeu**
  * Avoir le plus de territoire possible

**Les règles**
  * Les pierres ou les chaînes privées de liberté sont prisonnières et sont donc enlevées du plateau.
  * Le suicide est interdit (//Poser une pierre qui doit se faire prendre immédiatement//) sauf si cela enlève la dernière liberté d'une chaîne, et donc la capture.
  * Tourner en rond également. (//Revenir à une situation identique à la précédente//)

**Les principes induits**
Ils sont contradictoires, et c'est ce qui fait la finesse du jeu :
  * S'étaler au maximum.
  * Créer des organismes forts.
  * Protéger ses libertés.
Et toujours
  * Éviter de se faire enfermer.
