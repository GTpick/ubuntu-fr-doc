====== BRL-CAD ======

===== Présentation =====

Developpé depuis une vingtaine d'années, ce logiciel de [[cao|conception 3D]] multiplateforme (Windows, Mac, Linux) conçu par l'armée arméricaine, a été mis sous license GPL. 
Son Interface spartiate nécessitera de vous un investissement en terme de temps et de motivation, heureusement ce logiciel est pourvu d'une documentation étoffée (millier de pages) qui vous guideront dans vos premiers pas.

===== Description Technique =====

BRL-CAD contient une boite de dialogue interactive de construction de géométrie, un module de rendu et analyse de géométrie.
Il est distribué sous sa forme de code source.

==== Principe de fonctionnement ====

N'oubliez pas de vous documenter AVANT de vous lancer dans le logiciel. Ayez une documentation dès la première exécution.
Dans le terminal BRL-CAD, la commande ci-dessous correspond au lancement de l'interface graphique

  mged

===== Installation =====

==== Voie du dépôt Ubuntu====

Cette voie est hélas impraticable pour le moment.

==== Voie du téléchargement ====

Vous téléchargez le dernier paquet debian disponible sur [[http://sourceforge.net/project/showfiles.php?group_id=105292&package_id=113559| ce site]]

Sauvez le paquet .deb que vous ouvrez en double-cliquant dessus ou avec ouvrir dans votre navigateur.

Sur Edgy Eft 6.10, vous résolvez automatiquement les dépendances.  Si vous avez une version antérieure résolvez les dépendances manuellement (Pourquoi ne pas compléter cet article après ?).

**après avoir commencé la lecture de la documentation**, lancez l'application en mode texte à l'aide d'un terminal 

  /usr/bin/brlterm
