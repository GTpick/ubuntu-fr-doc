====== Lecteur multimédia : Xine ======

Xine est un moteur utilisé par divers lecteurs multimédia :
  * [[:Totem]] (avec totem-xine)
  * [[:Kaffeine]].
  * [[:Amarok]] (avec amarok-xine)
  * Gxine
  * Xine-ui
  * KMPlayer (peut utiliser Xine ou MPlayer)

=====Installation=====
[[:tutoriel:comment_installer_un_paquet|Installez le paquet]] **[[apt>xine-ui]]**.

===== Utilisation =====

Le moteur Xine est utilisé automatiquement par [[:totem]].

De manière générale, tous les fichiers (vidéo et audio) sont déjà assignés à Totem pour la lecture par défaut. De ce fait, il vous suffit de double-cliquer sur un fichier pour l'ouvrir dans Totem.

Si vous souhaitez le lancer simplement, faites une recherche dans [[unity#tableau_de_bord_dash|le tableau de bord Unity]] avec le mot clé **Vidéos**.

Pour lire un VCD ou un DVD, vous insérez le disque dans le lecteur et Totem sera appelé automatiquement. Dans le cas contraire, vous pouvez toujours le lancer manuellement via le tableau de bord. Dans la fenêtre de Totem, vous allez dans le menu //Film => Lire un disque//.

==== Codecs ====

Pour installer les codecs Microsoft (vidéo WMV et audio WMA), veuillez vous référer à la page traitant des [[:formats_non_libres|formats non libres]].

Pour profiter de codecs complémentaires (par exemple : pour les vidéos xvid) [[:tutoriel:comment_installer_un_paquet|Installez les paquets]] **[[apt>libxine1,libxine1-plugins,libxine1-ffmpeg]]**.

===== Problèmes connus =====

  * Si vous avez des problèmes de fluidité pour lire des DVD vidéos, cela provient surement du fait que les transferts DMA ne sont pas activés. Pour remédier à ce problème, allez sur [[:hdparm|cette page]].

  * Les problèmes de fluidité sur une machine moins puissante peuvent également provenir du post-traitement du son et/ou de l'image. Pensez donc à désactiver ces fonctionnalités très gourmandes en temps processeur. Pensez aussi à essayer les différents pilotes audio et vidéo à votre disposition (alsa, pulseaudio, xv, …), ne vous contentez pas des valeurs par défaut.

  * Si vous avez des problèmes pour lire des DVD protégés contre la copie ("Copy Protected"), par exemple, si votre lecteur se bloque et/ou le DVD n'est pas "monté" (il n’apparaît pas dans les raccourcis), vous pouvez essayer les manipulations décrites dans le [[http://forum.ubuntu-fr.org/viewtopic.php?id=58207|post suivant]]. Il s'agit de lancer un lecteur comme Gxine (installable depuis Synaptic ) //avant// d'insérer le DVD.
