====== Carte TV : Hauppauge_WinTV-NOVA-HD-S2 ======

La Hauppauge_WinTV-NOVA-HD-S2 est une carte [[http://fr.wikipedia.org/wiki/Peripheral_Component_Interconnect|PCI]] utilisant la norme [[http://fr.wikipedia.org/wiki/DVB-S|DVB-S]] permettant la réception des chaines diffusées par satellite.

Bien que désignée comme une **Hauppauge WinTV-HVR400** celle-ci est parfaitement reconnue.

===== Installation =====

[[:tutoriel:comment_installer_un_paquet|installez le paquet]] **[[apt://linux-firmware-nonfree|linux-firmware-nonfree]].**
puis redémarrez pour que le firmware soit pris en compte.

[[tutoriel:comment_installer_un_paquet|Installez le paquet]] **[[apt>linux-firmware-nonfree]]** puis redémarrez l'ordinateur.
<note important>Depuis [[xenial|Ubuntu 16.04]] le paquet linux-firmware-nonfree n'est plus disponible dans les dépôts.

De ce fait la carte ne peut fonctionner. Il vous faut donc télécharger et installer manuellement le paquet disponible sur cette [[http://packages.ubuntu.com/trusty-updates/linux-firmware-nonfree|page]]  </note>

===== Utilisation =====
Installez [[kaffeine|kaffeine]].
==== Selection du satellite ====
   *Lancez [[kaffeine|kaffeine]] et cliquez sur **DVB** / **configuration DVB** / **Périphérique DVB x:x**. ( Nom : connexant CX24116/cx24118 )
   *Sélectionnez le satellite ( Hotbird-13.0E , Astra-19.2E , etc ) en fonction de l'orientation de votre parabole. 
==== Recherche automatique des chaines ====
Cliquez sur **DVB** / **chaines** / **commencer l'analyse.**

===== Télécommande =====
Dans [[:environnements|l'environnement bureautique graphique]] **[[gnome|gnome]]**, les touches sont nativement **presque** toutes reconnues.
Il vous faudra tout de même faire quelques manipulations pour les attribuer dans [[kaffeine|kaffeine]]. ( configuration / configurer les raccourcis du clavier )
