====== Komodo Edit ======

Komodo Edit est un programme de [[programmation#le_developpement_web|conception de sites web]]. Ce logiciel vous permet de créer un site web en saisissant le HTML mais avec une assistance du logiciel pour éviter les erreurs communes (balises manquantes) ou prévisualiser le rendu. Komodo-edit n'est pas un logiciel libre.

{{ komodo_edit.png?400 }}

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  
===== Installation =====


  *Télécharger l'archive "Linux" correspondant à votre [[architecture matérielle]] (32 ou 64 bits) sur [[http://www.activestate.com/komodo-edit/downloads|cette page]].
  *Extraire l'archive
  *Exécuter le fichier install.sh qui se trouve dans le dossier.
  *Un message apparait dans le terminal, vous demandant le dossier ou vous souhaitez installer le logiciel. Si vous tapez sur la touche "Entrée", le logiciel s'installera par défaut dans un dossier ~/Komodo-Edit-numero_de_version dans votre Dossier personnel.
  *(optionnel) Vous pouvez faire un raccourci pour exécuter Komodo en tapant simplement **komodo** dans votre terminal. Pour cela, copiez-collez cette ligne en remplacent **chemin_vers_install** et **dossier_komodo**:
   <code>sudo ln -s "chemin_vers_install/dossier_komodo/bin/komodo" /usr/local/bin/komodo</code>

=====Configuration=====


=====Utilisation=====

Pour lancer le logiciel, il suffit de saisir la commande suivante dans un [[terminal]] :
  ~/dossier_d_installation/bin/komodo %F

Par exemple :
  ~/Komodo-Edit-6/bin/komodo %F

===== Désinstallation =====

Pour supprimer cette application, il suffit d'effacer le dossier Komodo créé durant l'installation. Vous pouvez aussi effacer vos préférences en supprimant le dossier ~/.komodo
