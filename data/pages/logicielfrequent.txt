===== L'interface d'Ubuntu : démarrer un logiciel, chercher un document, etc... =====

L'interface de bureau d'Ubuntu s'appelle Unity. Consultez [[:unity|cette page]] pour apprendre à l'utiliser.

=====Navigation sur internet=====
Le navigateur par défaut sur Ubuntu est [[Firefox]], que de nombreuses personnes utilisent aussi sous Windows. Si vous ne connaissez pas la différence entre "Connexion internet", "Navigateur internet", et "Messagerie internet", vous pouvez consulter [[internet|cette page]] qui s'adresse aux débutants.

{{firefox:firefox_15.10.png?300|Firefox sous Ubuntu 15.10}}

=====Explorateur de fichiers=====
 [[Nautilus]], le gestionnaire de fichiers par défaut sous Ubuntu, permet d'accéder aux fichiers de votre ordinateur. Il possède un système de "signets" très pratique et gère la navigation par onglets !


{{nautilus:nautilus_Ubuntu_15.10.png?300|Nautilus sous Ubuntu 15.10}}

===== Lecture de musique, vidéo =====
[[Totem]] est le lecteur multimédia installé par défaut sur Ubuntu. [[rhythmbox]] est le gestionnaire de musique par défaut sous [[maverick]]. Sous [[natty]], [[banshee]] est le gestionnaire de musique par défaut. Il est recommandé d'installer le lecteur [[VLC]] en complément.
__**Note**__ // : [[:formats_non-libres|cette page]] vous indiquera les paquets à installer pour activer la lecture de MP3, de DVD, Flash, Java, etc.//

{{totem_ubuntu_15.10.png?300|Videos sous Ubuntu 15.10}} {{::rhythmbox_ubuntu_15.10.png?direct&300|Rhythmbox sous Ubuntu 15.10}}

===== Logiciels de bureautique =====
La suite bureautique installée par défaut sous Ubuntu est [[LibreOffice]], qui existe aussi sous Windows et Mac, et qui est compatible avec les documents créés par Microsoft Office (Word, Excel, etc.).

Si vous avez installé [[:Xubuntu]] ou [[:Lubuntu]], et c'est sûrement le cas si vous êtes sur cette page, c'est alors [[AbiWord]] qui est installé par défaut.

{{libreoffice_ubuntu_15.10.png?300|LibreOffice sous Ubuntu 15.10}}

<note help>Si votre destinataire ne possède pas le logiciel libre et gratuit [[:LibreOffice]], il est recommandé pour plus de compatibilité d'[[:msttcorefonts|installer les polices de caractères Microsoft]] sur votre Ubuntu.</note>


===== Sécurité : antivirus, pare-feu =====
Pas besoin d'antivirus pour une utilisation standard d'Ubuntu -- la sécurité est l'un des gros avantages de GNU/Linux par rapport à Windows ! De même, un pare-feu est déjà intégré à Ubuntu et vous n'aurez sans doute jamais besoin d'y toucher !
Pour plus d'informations, consulter [[sécurité|cette page]].

===== Fichiers compressés =====

Le logiciel de compression/décompression intégré à Ubuntu s'appelle [[File-Roller]] et c'est un clone de WinZip pour ceux qui connaissent. Faites un clic-droit sur une archive (fichier ZIP par exemple) et "extraire ici" pour la décompresser. Par défaut il ne supporte que les formats libres **mais** lorsque vous installerez les formats propriétaires (unrar pour le format rar par exemple), il pourra aussi les gérer sans problème.

===== Téléphonie et webcam sur Internet =====

Ubuntu propose notamment [[Ekiga]] pour la téléphonie sur internet (avec vidéo) mais le logiciel propriétaire [[Skype]] fonctionne aussi sous Linux.

===== Messagerie instantanée =====

Pour écrire en temps réel avec vos contacts Gtalk, MSN, Yahoo, Jabber, ICQ, IRC, et autres, Ubuntu intègre par défaut un logiciel nommé [[Empathy]]. 

===== Gravure de CD =====

Le logiciel de gravure par défaut est [[Brasero]], il vous permettra aussi de graver des DVD et CD-R.
