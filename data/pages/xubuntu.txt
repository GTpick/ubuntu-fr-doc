====== Xubuntu ======

{{ ::xubuntu_logo.png?200 |}}

Xubuntu est une [[variantes#variantes_reconnues|variante reconnue d'Ubuntu]], elle reprend donc les mêmes concepts de base. La différence la plus visible est l'utilisation de l'[[environnements|environnement de bureau]] [[:Xfce]] à la place de [[:Unity]]. La base est la même ([[:kernel|noyau]], gcc, xorg ...), seule l'interface diffère.\\
L'un des buts de Xubuntu est d'offrir une distribution plus légère qu'[[:Ubuntu]], pouvant s'installer sur des ordinateurs plus anciens, et avec du matériel moins performant. L'environnement [[:Xfce]] a été choisi dans ce but, ainsi que les applications installées par défaut.

<note>
Surtout consultez la page dédiée à la [[:tutoriel:personnaliser_xfce|personnalisation d'XFCE]].
Et aussi ce tutoriel d'optimisation pour les versions égales ou postérieures à 12.04,  disponible sur ce [[http://forum.ubuntu-fr.org/viewtopic.php?id=1408621|fil]] (en fin du post #1, un script est aussi proposé)

</note>
===== Pourquoi Xubuntu ? =====

  * Xubuntu est léger et fonctionne donc mieux sur d'anciennes machines,
  * Xubuntu est aussi solide et stable qu'[[:Ubuntu]],
  * Xubuntu dispose de la totalité de la [[:software-center|logithèque d'Ubuntu]],
  * Xubuntu est fourni avec [[:xfce]] (pour ceux qui n'aiment pas [[:KDE]], [[:Gnome]] ou [[:Unity]]),
  * Xubuntu permet également aux machines plus récentes de faire tourner les logiciels plus rapidement, cela peut se révéler utile pour des machines spécialisées,
  * Xubuntu est conforme à [[http://www.freedesktop.org/wiki/|freedesktop.org]]
  * Xubuntu est modulaire : un programme pour un job

===== Configuration minimale =====
<note important>Il est fortement recommandé d'avoir au moins 512 Mo de mémoire ([[http://xubuntu.org/getxubuntu/requirements/|Minimum system requirements for Xubuntu 16.04|source]])</note>
Ceci est une traduction directe du paragraphe « //Minimum system requirements// » de la page « [[https://xubuntu.org/getxubuntu/requirements/]] » du site officiel.

  * Vous avez besoin de 256Mo de mémoire pour lancer le LiveCD ou pour l'installation. [[installation_alternate|L'alternate Install CD]] ne demande que 128Mo de mémoire pour l'installation ([[http://www.tutoriels-video.fr/installer-linux-sur-un-vieux-pc-alternate-cd/ | Tutoriel vidéo pour installer xubuntu en alternate CD]]).
  * Une fois installé, Xubuntu peut fonctionner avec 1 Go (ou juste 512) Mo de mémoire, mais il est vivement recommandé de disposer d'au moins 1 Go.
  * Pour installer Xubuntu avec l'installation standard, vous avez besoin de 6,1Go d'espace disque. L'installation avec "Alternate Install CD" nécessite 2Go d'espace disque disponible. 

=====Problèmes connus=====
====Perte des tableaux de bord ou des bords des fenêtres====
Ce problème est traité sur la page [[:xfce#Problèmes connus|XFCE]].

====Bogue du Live CD sur les vieux PC====
Sur de vieux PC, le [[live_cd|Live CD]] ne fonctionne pas : après avoir sélectionné //"Essayer Xubuntu"// l'écran de chargement n'est pas le bon et montre des points en dessous de //Xubuntu (version)//
<note important>Si cela vous arrive, essayez de redémarrer puis, si cela ne fonctionne toujours pas, attendez jusqu'au bout du chargement.</note>
Si après avoir essayé la méthode si dessus vous avez un écran noir, alors Xubuntu ne fonctionnera pas en [[live_cd|Live CD]] :-(
<note tip>Ne désespérez pas! Vous pouvez toujours essayer [[lubuntu|Lubuntu]] qui est encore plus léger!</note>

===== Conseils =====
  * **[[:tutoriel:personnaliser_xfce|Personnaliser Xfce]]**
  *  **[[:Applications par defaut sous Xubuntu|Logiciels installés par défaut sous Xubuntu]]**
  * **Compléments d'installation** :
    * [[:liste_applications_legeres|Autres applications légères disponibles]]
    * Le multimédia n'est pas intégré par défaut dans [[:Firefox]]. Vous pourrez installer au choix les plugins : [[apt>gecko-mediaplayer|gecko-mediaplayer]] ou [[apt>totem-mozilla|totem-mozilla]] ou [[apt>mozilla-plugin-vlc|mozilla-plugin-vlc]]
  * Dans le **cas d'une migration** (par exemple depuis [[:Gnome]]), vous souhaiterez probablement installer [[LibreOffice]] et/ou [[:Gedit]].

===== Si Xubuntu est trop lent =====

Consultez cette [[http://forum.ubuntu-fr.org/viewtopic.php?id=241092|page du forum]].

Et si votre machine possède peu de mémoire vive (moins de 512 Mo de RAM) et que vous trouvez Xubuntu trop lent :
  * vous pouvez essayer une **[[:installation_minimale|ré-installation minimale d'Ubuntu-Xfce]]**
  * pour d'autres solutions, consultez la page « [[mini_ram|Comment installer Ubuntu sur une machine ayant peu de mémoire vive (RAM) ?]] ».
