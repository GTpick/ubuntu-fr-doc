====== Counter-Strike ======

<note tip>Toutes les versions de Counter Strike sont désormais compatibles Linux via Steam (compatible aussi), il suffira simplement d'installer les jeux directement via Steam sans passer par Wine en quelques clics. De plus, le restant de la page est devenu inutile</note>

Counter-Strike est un des jeux de Valve le plus joués dans le monde, surtout depuis la sortie de la platforme [[:Steam]] qui a permis d'ailleurs le [[cssource|Counter-Strike: Source]], qui est le CS de la nouvelle génération.


===== 1. Introduction =====

Ce tutorial va vous expliquer la procédure pour installer une vieille version de Counter-Strike.
Cette version n'est plus jouable sur internet, mais c'est fun avec les bots: PODBot.

<note>Counter-Strike fonctionne très bien, le son inclus avec Wine version 1.1.31. La version 1.0.1 fonctionne, mais le son est souvent saccadé. C'est jouable sur internet maintenant.</note>

===== 2. Prérequis =====

Vous devez avoir [[wine]] installé et configuré.

<note>Sélectionné dans le logithèque Ubuntu la version 1.1.31 de wine pour une meilleur compatibilité surtout au niveau du son</note>

===== 3. Installation =====

La procédure d'installation suivante, est donné pour la version commerciale de Counter-Strike c'est à dire la version ne nécéssitant pas l'installation de Half-Life avant car le moteur du jeu y est inclus.

Rendez-vous dans le répertoire du CD-ROM.

Vous allez lancer l'installation:

<code>wine setup.exe</code>

Suivez la procédure normale.

Une fois le jeu installé, vous allez vous rendre dans le répertoire du patch 1.0.0.3 et y faire l'installation:

<code>cd Patch
wine fcs100.exe</code>

Suivez la procédure normale.


===== 4. Fin =====

Le jeu est maintenant installé, pour le lancer soit vous utilisez le raccourcis présent sur le bureau, soit vous allez dans le répertoire du jeu et vous faites:

<code>wine cstrike.exe</code>

Cela va lancer le jeu.



==== Problèmes ====

Il arrive que le jeu ait des problèmes soit au niveau du menu soit pendant le jeu.
Pour réparer cela, vous devez configurer votre wine pour qu'il fasse une émulation du bureau:

{{cs1003_wineconf.png|}}


=== Résolution graphique ===

Le mieux est de ne pas dépasser la résolution précédente à celle que vous utilisez actuellement, car cela cacherais votre barre de vie, bouclier et votre argent, sauf si bien entendu vous enlevez votre barre des taches, dans ce cas là il n'y a aucun soucis.


=== Menu du jeu ===

Le menu quand à lui, restera toujours assez lent, mais le jeu sera généralement soit aussi rapide que sous windows, soit encore plus rapide, tout dépend du rendu.


=== Rendu graphique ===

Si vous choisissez un rendu OpenGL, il sera aussi rapide que sous windows, si vous choisissez Software, il sera graphiquement affreux et lent comme sous windows. Maintenant si vous prenez DirectX(dit aussi DirectDraw), le jeu sera plus rapide.
Mais c'est une rapidité non pas quand on se déplace, mais au niveau de l'animation des tires principalement avec des armes comme le P90, la M249 ou encore le MP5 Navy.
