====== Brothers in Arms ROAD to HILL 30 ======

{{:jeux:jeu_brother_in_arm.png?600|Capture d'écran}}

Ce jeu se déroule en Normandie pendant la seconde guerre mondiale.

<note tip>Ce jeu semble fonctionne sans aucune manipulation avec [[wine]]. Il est également présent sous [[:Steam]]. Ce qui suit est à titre indicatif.</note>
===== Pré-requis =====

  * Disposer d'une connexion à Internet configurée et activée.
  * Disposer du DVD original " Brothers in Arms ROAD to HILL 30 " , ( Quelques € d'occasion, ou jouer avec la démonstration téléchargeable [[http://www.jeuxvideopc.com/demo/7785-brothers-in-arms-road-to-hill-30.php|ici]] Version en Anglais  ) 
  * Avoir [[:tutoriel:comment_installer_un_paquet|installé]] **[[wine]]**.

===== Installation =====

Pour installer ce jeu, il faut exécuter le fichiers setup.exe ( à la racine du CD ou après avoir dé-zipper la démonstration )

L'installation du jeux sous Wine commence...laisser les choix par défaut, et faire simplement suivant, accepter, suivant..., sauf tout à la fin, ne pas redémarrer , sélectionner ne pas s'enregistrer.

===== Configuration =====

Dans le menu applications, configurer  [[:Wine]], la** première fois dans Wine**, cliquer une fois sur l'onglet "**Audio**" pour activer le son ( valider OK puis retour sur l'onglet "Application" ), dans l'onglet application, paramètres  par défaut, onglet affichage, choisir les options : "permettre au gestionnaire de fenêtre de contrôler les fenêtres" et "émuler un bureau virtuel", ne pas cocher les 2 cases du haut, saisir la taille de votre bureau.( exemple 1280X1024, 1680x1050 ou autre selon votre matériel, pour vérifier la résolution idéale, et avoir les jeux parfaitement en plein écran, observer dans le menu Système, préférence, écran " configuration" ... ).

On peut maintenant, Ajouter une application, parcourir pour trouver le programme : program-files... ( dans le  dossiers Ubisoft..... en enfin **system**, choisir le fichier **BIA.EXE** ) on peut enfin faire ouvrir. Remettre les paramètres d'affichage du jeu BIA ( mettre la même chose que le par défaut ) ,  et le son ( clic sur l'onglet, ça suffit ), toujours sur le modèle de "paramètres par défaut". 

Si vous n'avez pas encore installer de jeux sous wine, il faut faire une seule modification qui permet d'éviter les problèmes de souris qui se bloque sur les cotés. ( sinon impossible de contrôler correctement votre soldat )

Pour cela,  lancer un terminal, saisir :

<code>wine regedit</code>

Dans l'arborescence de Regedit rechercher :  HKEY_CURRENT_USER/SOFTWARE/WINE ajouter une clé "DirectInput", en faisant clic droite, Nouvelle clé, dans la partie à droite, dans cette même clé ( à l'intérieur ), ajouter une valeur chaine "MouseWarpOverride" et enfin indiquer comme valeur "force"

===== Jouer à Brothers in Arms =====

Lancer le jeu, par le menu Applications, Wine, Programmes, Ubisoft...une fois trouver le " Jouer à Brothers in Arms" , on peut éventuellement créer un raccourci sur le bureau.

Si vous avez un message " Insérer le CD-ROM approprier ", aller dans Wine, onglet des lecteurs, il est probable que vous ne voyer pas le CD " BIA ", donc ajouter un lecteur, I ou K ou H ... chemin : parcourir dans le dossier media, choisir BIA.

Configurer ce jeux comme beaucoup d'autre, la résolution vidéo en premier ( sinon le jeu n'occupe qu'une petit partie de l'écran ), pour quelle soit la même que votre bureau, et aussi la même que dans la configuration du jeu adapté dans Wine ( on peut alors jouer en grand écran ), la qualité des graphismes, selon votre matériel, mais aussi les commandes avance, recule, droite et gauche et penser à afficher le viseur ( Cible oui ), plus tard on pourra également se passer du mode apprentissage.
