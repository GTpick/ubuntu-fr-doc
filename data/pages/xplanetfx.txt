====== xPlanetFX ======

Un fond d’écran de la Terre... en direct depuis le satellite.
===== Pré-requis =====

  * Disposer d'une connexion à Internet configurée et activée.









===== Installation =====

En suivant tranquillement les instructions d'installation sur le site très complet de l'éditeur  [[http://mein-neues-blog.de/xplanetfx]]  ( En Anglais ), vous devriez avoir une nouvelle application. ( Voir la section Installation )

<note tip>Astuce : Copier l'adresse dans Google traduction [[http://translate.google.fr/?hl=fr&tab=wT]] pour obtenir la traduction dans la langue de votre choix </note>

  * dans Ubuntu / interface Gnome Menu | Applications |  Accessoires
  * dans Ubuntu / interface Unity Tableau de bord : saisir Xpl.... dans la zone de recherche, pour trouver l’application à ouvrir.
  * dans Ubuntu / interface KDE Menu K , rechercher





===== Paramétrages rapides =====

Quelques réglages indispensables dans XplanetFX :

  * Dans l'onglet Outils : on peut activer le démarrage automatique et faire exécuter.

  * Dans l'onglet Paramètres :
           - Résolution ( clic sur le bouton Adapter, pour utiliser la résolution vidéo de votre écran)

//Avec Ubuntu 11.10 et l'interface Unity il semble qu'il n'y ait plus besoin d'indiquer un quelconque fichier de sortie, ça marche tout de suite, par contre avec KDE il faut indiquer un nom de fichier, cliquer sur KDE, pour pouvoir parcourir et donner un nom de fichier, exemple : terre.png , et le mettre dans le dossier Images, **enregistrer et actualiser** , ensuite, bureau KDE fond d'écran choisir ce fichiers, ouvrir dans le dossier Images, votre fichier terre.png (fichier PNG et penser à déverrouillez vos plasmoïdes pour cette opération)  //

  * Dans l'onglet vue, sélectionner une ville, **enregistrer et actualiser**
    ( si on ne peut pas enregistrer... attendre un peu )
    
===== Test  =====

Dans xplanetFX, dans l'onglet Affichage, modifier la taille de la Terre, Enregistrer et Actualiser.

Observer les changements sur votre bureau ( le fond d'écran doit changer )
Le changement, n'est pas immédiat, attendre quelques minutes.

C'est pendant cette phase de création de l'image que l'ordinateur est fortement sollicité.

//( ralentissement important, d’où l'importance de remettre le rafraîchissement 10, 20 minutes après vos test, et réglages  )//

===== Encore plus beau  =====

Dans l'onglet Outils, mises à jour des templates, téléchargement des thèmes...

Allez voir après, dans l'onglet Thèmes.

Choisissez une images pour agrémenter d'une décoration la planète Terre. ( **enregistrer et actualiser** )
