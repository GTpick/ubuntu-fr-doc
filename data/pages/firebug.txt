====== Firebug ======

Firebug est une extension pour Mozilla Firefox qui permet de déboguer, éditer et modifier le PHP, le HTML, le CSS et le Javascript d'une page web.



===== Installation=====

A partir de hardy 8.04, il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] [[apt://firebug|firebug]]

Sinon installer l'extension : https://addons.mozilla.org/en-US/firefox/addon/1843
