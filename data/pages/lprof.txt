====== LProf ======

**LProf** - //de son doux nom anglais : Little CMS Hardware Color Profiler// - est un gestionnaire de profils de couleurs ICC. Licencié sous GNU/GPL.

Il permet de gérer des profils de couleurs conforme ICC, pour des matériels tels que Caméra, scanners, imprimantes, écrans d'ordinateurs.
On peut utiliser ces profils de couleurs en coordination avec certains logiciels graphiques, tel que [[gimp|Gimp]], [[Scribus|Scribus]], etc...

===== Pré-requis =====

  * **Des connaissances, voire des compétences en infographie**
  * Disposer des [[:sudo|droits d'administration]] ;
  * Disposer d'une connexion à Internet configurée et activée ;

===== Installation =====

Disponible depuis le Hardy Heron (8.04), il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt://lprof|LProf]]**

===== Utilisation =====

Le logiciel, avec une interface graphique, fonctionne à partir du menu 'Infographie'.

===== Suppression =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]].
