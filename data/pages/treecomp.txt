====== TreeComp ======
 
**TreeComp** est un programme permettant de comparer ou de synchroniser deux hiérarchies de répertoires parallèles, en permettant de voir les différences des fichiers dossier par dossier.
 
<note tip>Il existe des applications semblables directement dans les [[dépôts]] officiels, [[Meld]], [[:freefilesync]] ou encore [[apt>Kdiff3]].</note>
===== Pré-requis =====
 
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  * Avoir [[:Wine]] d'installé.
 
===== Installation =====
 
Pour installer ce logiciel, il suffit de télécharger [[http://lploeger.home.xs4all.nl/TreeCompLatest_noinstall.zip|l'archive suivante]] et de l'[[:archivage#graphiquement|extraire]] dans le répertoire de votre choix.
 
===== Configuration =====
 
Il n'y a pas vraiment de configuration préalable à faire. Toutefois, on peut modifier les préférences d'utilisation du logiciel par le menu //Fichier -> Préférences//.
 
===== Utilisation=====
 
Lancer l'application depuis l'endroit où vous l'avez décompressée. Choisir les deux répertoires-racines de la comparaison par le menu //File -> Select directory// ou en cliquant sur le haut de la partie droite, juste en dessous des boutons…

Une fois les deux répertoires à comparer choisis, cliquer sur //OK//, puis choisir la méthode comparaison : //TimeStamp// et //Size// sont en général un bon choix.

Il ne reste plus qu'à attendre la fin de la comparaison et de naviguer dans le résultat…

  - Pour les fichiers :
    - Ce qui est noir est synchronisé entre les deux côtés.
    - Ce qui est en bleu n'existe que dans le répertoire de gauche.
    - Ce qui est en rouge n'existe que dans le répertoire de droite.
    - Ce qui est de deux couleurs existe des deux côtés : les flèches indiquent le côté le plus récent et les ronds indiquent (pour les  fichiers) que le programme ne sait pas lequel il convient de choisir entre les deux.
  - Pour les dossiers :
    - Ce qui est noir est synchronisé entre les deux côtés.
    - Ce qui est en bleu n'existe que dans le répertoire de gauche.
    - Ce qui est en rouge n'existe que dans le répertoire de droite.
    - Ce qui est de deux couleurs est en partie synchronisé seulement.

===== Désinstallation =====
 
Pour supprimer cette application, il suffit de supprimer les fichiers que l'on a décompressés (ou le répertoire racine les contenant).
