====== Yo Frankie! ======

{{ :jeux:yo_frankie.jpg}}

**Yo Frankie!** est un jeu Open Source de l'institut Blender qui a pour but de créer un jeu vidéo libre avec des outils libres, notamment le modeleur 3D [[Blender|Blender]] et le moteur Crystal Space. Le personnage principal est issu du court-métrage [[http://www.bigbuckbunny.org|Big Buck Bunny]].

Le jeu est disponible en deux version utilisant chacune un de ces moteur de rendu. On notera que pour la version Crystal Space il existe différents niveaux de qualité.


===== Installation =====

==== Depuis le dépôt officiel ubuntu (logithèque)====

Pour installer ce logiciel, il vous suffit d'[[:tutoriel:comment_installer_un_paquet#paquet_present_dans_les_depots|installer le paquet]] **[[apt>yofrankie]]**. Il s'agit de la version //BGE (Blender Game Engine)// : [[Blender]] sera installé automatiquement. 


==== Avec l'archive officielle ====

=== Version BGE (Blender Game Engine) ===

== Pré-requis ==

Vous aurez besoin d'[[:tutoriel:comment_installer_un_paquet#paquet_present_dans_les_depots|installer]] **[[apt>blender]]** (version ≥ 2.49a, disponible dans les dépôts Universe).

== Téléchargement et lancement ==

Vous pouvez télécharger directement la dernière version du jeu [[http://www.yofrankie.org/download|ici]] (archive zip - Version BGE). Décompressez l'archive .zip et cliquez sur le lanceur Linux pour jouer.

== Si le jeu ne se lance pas ==

Essayez depuis un [[:terminal]] :
  * sous Ubuntu 32 bits
<code>
cd yofrankie*
chmod +x yofrankie-linux-i386.bin
./yofrankie-linux-i386.bin
</code>
  * sous Ubuntu 64 bits
<code>
cd yofrankie*
chmod +x yofrankie-linux-x86_64.bin
./yofrankie-linux-x86_64.bin
</code>

===== Touches du clavier =====

La description des contrôles est accessible pendant le jeu en pressant la touche F1.

|    | Joueur 1: | Joueur 2: |
| Action | N | H |
| Sauter | M | G |
| Jeter | espace | J |
| Avancer | Flèche haut | S |
| Reculer | Flèche bas | W |
| Gauche | Flèche gauche | A |
| Droite | Flèche droite | D |


Maintenir la touche N pour courrir

Presser deux fois M pour faire un double saut

Maintenir M pour planer

Presser la touche "echap" pour quitter.
