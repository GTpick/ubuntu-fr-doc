====== Abanq ======

**Abanq** est un ERP (Entreprise Ressource Planning)  Libre orienté vers l'administration, la gestion commerciale, la finance et les processus administratifs. Son application s'étend depuis la gestion financière et commerciale dans des entreprises jusqu'à l'adaptation à des processus complexes de production.

===== Fonctionnalités =====

  * Progiciel Mono et multi-poste ( Jusqu'à 50 terminaux en réseau); 
  * Interface graphique multi-fenêtres; 
Tant les modules de l'application comme les données sont  gardées dans une base de données différente, de sorte qu'une même installation de Abanq supporte la gestion de plusieurs entreprises, ou d'une même entreprise avec différents modules. Vous pouvez choisir la base de données que vous voulez accéder au lancement de Abanq ;

** Modules:**

  * Finance: La comptabilité et rapports
  * Facturation: Stocks, rapports, facturation, trésorerie
  * RH: Gestion des Ressources Humaines
  * Produit: Gestion des produits, listes des prix
  * Auto: pour les société de ventes et locations de voitures

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]] ;
  * Disposer d'une connexion à Internet configurée et activée ;

===== À Savoir =====

  * Abanq supporte les bases de données PostgreSQL, MySQL et SQLite. Dans des environnements en production, l'éditeur recommande d'utiliser PostgreSQL ou MySQL, puisque SQLite  ne fonctionne pas correctement avec quelques caractéristiques avancées d'Abanq.
  * L'utilisation de PostgreSQL requiert l'installation préalable du serveur PostgreSQL. PostgreSQL est OpenSource disponible pour Linux (et Windows).
  * L'utilisation de MySQL requiert l'installation préalable du serveur  MySQL. MySQL est OpenSource. 
  * L'utilisation de SQLite ne requiert pas d'installations additionnelles, et pour les motifs précédemment exposés cette base de données doit être utiliser seulement en mode de teste.
