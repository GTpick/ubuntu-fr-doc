====== FuseAPI ======
FuseAPI est une série de petits utilitaires permettant de gérer vos points de montage Fuse.\\

===== Avant tout =====
Vous pouvez utiliser le dépôt de [[utilisateurs:spomky|Spomky ]] afin de faciliter l'installation du logiciel et ses dépendances.

Le code source  est disponible sous [[https://edge.launchpad.net/fusible|Launchpad]] et peut être téléchargé avec [[bzr|Bazaar]].

===== Installation =====
FuseAPI est disponible sous forme de paquets binaires x86 et x86_64 pour Lucid Lynx.

Installez simplement le paquet [[apt://fuseapi]].
Pour développer en utilisant FuseAPI, installez le paquet [[apt://fuseapi-dev]].

<note important>Vous devrez quitter votre session KDE puis revenir pour que FuseAPI soit pris en compte.</note>

===== Activation/Désactivation =====
FuseAPI contient un module KDED ; il s'exécutera en tâche de fond dès l'ouverture d'une session.

Si vous souhaitez arrêter FuseAPI ou désactiver le lancement automatique, rendez-vous dans le panneau "Configuration du système" de KDE, choisissez l'onglet "Anvancé" puis sélectionnez "Gestionnaire de services".

Recherchez ensuite "Service FuseAPI" dans la partie "Services au démarrage" et activez-le comme il vous plaira.

===== Configuration =====
FuseAPI peut être configuré via le panneau "Configuration du système" de KDE.
Rendez-vous à l'onglet "Avancé" puis choisissez "FuseAPI".\\
Vous changements sont pris en compte dès que vous les aurez validés.

FuseAPI utilise KNotify pour avertir les utilisateurs dès qu'un point de montage est ajouté ou supprimé. Dans l'onglet "Général", choisissez "Notifications" puis cherchez "Service FuseAPI".\\
Sélectionnez ensuite les méthodes de notifications selon votre convenance.

===== Désinstallation =====
Supprimez le paquet [[apt://fuseapi]] ou [[apt://fuseapi-dev]].\\
Vous devez quitter la session en cours et vous reconnecter pour que la désinstallation soit complète.
