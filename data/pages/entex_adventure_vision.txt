====== Émuler l’Entex Adventure Vision ======
{{ https://ia801400.us.archive.org/13/items/adventurevision_library/logo_header.jpg?250}}L'Adventure Vision est une console de jeux vidéo à cartouches intégrant un moniteur vidéo. Produite par Entex Industries et commercialisée à partir de 1982, l'Adventure Vision est la seconde console conçue par Entex.

===== AdViEmulator =====
[[https://sourceforge.net/projects/adviemulator/|AdViEmulator]] est un émulateur d'Adventure Vision dont il existe un portage pour Linux. L'émulateur est toutefois bugué, puisque l'activation du son le fait planter.

La version Windows fonctionne en revanche sans problème avec Wine 2.4.
>[[https://sourceforge.net/projects/adviemulator/|AdViEmulator v1.0]]
>[[https://mega.folderflex.com/sharefolder/MEGA/Adventure+Vision/id/80892|AdViEmulator v1.1]]

Il est possible de glisser/déposer les roms dans la fenêtre de l'émulateur. 

Les contrôles se composent d'une manette et de quatre boutons ; ils se configurent facilement dans le menu : ''System > Options > Controls''. Il semble toutefois obligatoire de jouer au clavier, les manettes n'étant apparemment pas prises en charge.

Par défaut les touches sont les flèches haut, bas, droite, gauche et les touches Q, S, D, W (en clavier azerty).

===== Utilisation de MAME =====
  * Installez [[mame|MAME]] : cliquez sur **[[apt>mame]]**, ou dans un terminal :
  <code># apt-get install mame</code>
  * Se procurer le BIOS de la console (fichier advision.zip).
  * Placez ce fichier dans le dossier roms de MAME (par défaut : /usr/local/share/games/mame/roms).
  * Procurez-vous des fichiers de jeux au format .bin (à placer n'importe où).
  * Utilisez la commande :
   <code>mame advision -cart "chemin/du/fichier/jeu.bin"</code>
