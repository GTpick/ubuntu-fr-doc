====== Comment lire des animations Shockwave avec Firefox ======

<note help>Il existe un autre tutoriel plus complet dans la documentation : [[:installer_shockwave]]</note>

En fait il n'existe aucun plugin Adobe Shockwave sous Linux. Mais nous avons de la chance, car Shockwave marche parfaitement bien sous la version Windows de [[:Firefox]]. Comme il est très facile d'émuler des programmes Windows sous Linux avec l'excellent logiciel [[:Wine]], il suffit simplement d'installer la version Windows sur votre disque.


===== Pré-requis =====

  * Avoir installé [[:Wine]]


===== Marche à suivre =====

  - Télécharger la version Windows de firefox : [[http://www.mozilla.com/en-US/firefox/all.html#fr]]
  - Faire un simple clic sur le fichier téléchargé. Si cela ne marche pas, ouvrir un [[:terminal]] et entrer Wine suivi du nom du .exe
  - Lors de l'installation, cliquer sur oui, suivant, etc… Puis cocher « Lancer Firefox maintenant »
  - Aller sur un site ayant besoin de Shockwave
  - Cliquer sur « Installer les plugins additionnels… »
  - Suivre l'installation de Shockwave
  - Et voila, ça marche.
  - Pour lancer la version Windows de Firefox ultérieurement, un raccourci a été créé dans le menu Applications (ou menu K sous [[:Kubuntu]]) sous-menu Wine => Programmes => Mozilla Firefox => Mozilla Firefox

<note warning>Les titres de menu, tels que « Lancer Firefox maintenant » ont été écrits de tête ; il se peut qu'en réalité le titre diffère.</note>
