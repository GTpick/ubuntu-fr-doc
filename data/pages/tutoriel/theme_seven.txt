====== Comment métamorphoser Ubuntu en Windows ? ======

Dans cette page vous apprendrez à changer l'interface d'Ubuntu et d'un ensemble d'autres logiciels pour les rendre comme windows XP, 7, 8 !

**Mais en gardant les fonctionnalités et la sécurité d'Ubuntu ! et du logiciel libre.**
Cette méthode a été testée sous [[:Unity]], [[:Xfce]], [[:Lxde]], [[:Mate]], [[:Cinnamon]].
<note warning>
Attention, des éléments graphiques de ces thèmes  proviennent de Windows et appartiennent à Microsoft, **vous n'êtes autorisé à les utiliser que si vous possédez une licence authentique de Windows correspondante aux éléments que vous voulez installer**.

Détails : 
  * les fonds d'écrans (usage personnel non commercial), (( http://www.microsoft.com/about/legal/permissions/default.mspx#E4 ))
  * une partie des icônes, (pose problème), ((http://www.microsoft.com/about/legal/permissions/default.mspx#EPE ,))
  * les polices. ((http://www.microsoft.com/about/legal/permissions/default.mspx#EMF ))
</note>
{{:personnalisation:Look_win7_cinnamon.png}}

=====Installation du thème et des icônes=====
====Via un ppa====
Pour avoir une interface identique à Windows il vous faudra installer deux paquets à partir d'un ppa, le paquet du thème et celui des icônes.
===Thèmes Windows===
Pour installer les différents thèmes Windows:
  * [[:ppa#ajout_d_un_ppa_a_vos_sources_logicielles|Ajoutez le PPA]] **ppa:noobslab/themes**((https://launchpad.net/~noobslab/themes/+archive/ppa)) dans vos sources de logiciels,
  * Puis [[:tutoriel:comment_modifier_sources_maj#recharger_la_liste_des_paquets|rechargez la liste des paquets]],
  * [[:tutoriel:comment_installer_un_paquet|Installez le paquet]] **[[apt>win-themes]]** pour windows 7 et 8, et **[[apt>ylmfos-theme]]** pour Windows XP.


===Les icônes Windows===
Pour installer les icônes Windows 7 et 8:
  * [[:ppa#ajout_d_un_ppa_a_vos_sources_logicielles|Ajoutez le PPA]] **ppa:noobslab/icons**((https://launchpad.net/~noobslab/icons/+archive/ppa)) dans vos sources de logiciels,
  * Puis [[:tutoriel:comment_modifier_sources_maj#recharger_la_liste_des_paquets|rechargez la liste des paquets]],
  * [[:tutoriel:comment_installer_un_paquet|Installez le paquet]] **[[apt>win-icons]]**.

{{:personnalisation:look_win7_unity.png}}


Pour installer les icônes Windows XP téléchargez et [[:tutoriel:comment_installer_un_paquet|installez le paquet]] **deb** des icônes [[http://drive.noobslab.com/data/icons/ylmfos-icons_1.0.deb|ici]].


===Utilisation===
Vous pourrez changer le thème en utilisant le gestionnaire d'apparence propre à votre [[:variantes|variante d'Ubuntu]]. Sous [[:Unity]] vous pourrez y accéder grâce à **[[apt>unity-tweak-tool]]**, ou [[:ubuntu_tweak|Ubuntu-Tweak]].
Le thème win7, win8 ou winXP vous est proposé au choix.


====Via un pack complet====

<note warning>
Les thèmes peuvent être difficiles à désinstaller proprement donc il pourrait être intelligent de faire un nouveau compte utilisateur sur votre machine pour n'appliquer ces modifications que pour ce compte. Exemple utilisateur : Windows. Ce pack peut ne pas fonctionner sur des versions récentes d'Ubuntu. Il est peut-être utile d'installer [[:gnome-shell]] au préalable. Il est également probable qu'il ne fonctionne que sur les environnements basés sur [[:Gnome]] (Cinnamon, Gnome-session-fallback et peut-être Unity).
Un pack pour les environnements sous Gnome2, (à télécharger [[http://gnome-look.org/content/show.php/Win2-7+Pack?content=113264|ici]]) ne fonctionne plus sur les versions récentes d'Ubuntu.
</note>

Un pack avec le thème Windows 7, **Win2-7** a été porté pour Gnome 3 : [[http://gnome-look.org/content/show.php/Win2-7+Remix?content=153233|pack de transformation pour ubuntu]].
Ce pack contient un script d'installation graphique, qui se chargera de tout, reconnaîtra votre langue, paramétrera votre installation selon vos choix, sinon vous pouvez juste laisser les paramètres par défaut.

[[http://gnome-look.org/content/show.php?content=113909&forumpage=0|{{http://gnome-look.org/CONTENT/content-pre2/113909-2.png}}]]

Le pack contient :

    * un pack d’icônes,
    * un thème GTK,
    * les fonds d’écrans,
    * les sons Windows 7,
    * le thème Gnomenu,
    * les fonds Windows pour les différentes fenêtres,
    * Les curseurs,
    * La barre des tâches.

=====Le démarrage=====
Vous pouvez également modifier [[:plymouth|l'écran de démarrage]] pour afficher le logo Windows grâce à [[:superbootmanager|super-boot-manager]].
