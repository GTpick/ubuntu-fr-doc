====== UtilIser le thème Ubuntu sous Windows ======

<note tip>Si vous êtes habitués à utiliser Ubuntu mais que vous avez besoin d'utiliser Windows, cette page vous permettra de transformer votre Windows pour lui donner l'apparence d'Ubuntu.
</note>

===== Thème Ambiance =====
{{ http://th03.deviantart.net/fs70/PRE/i/2010/285/d/e/ubuntu_light_for_windows_xp_by_freddi67-d2xoqxh.jpg?500 }}

==== Pré-requis ====

<note tip>Le thème Ambiance, introduit avec Ubuntu 10.04, est disponible sur DevianArt. Pour l'installer, quelques pré-requits sont nécessaires :</note>
  * L'outil [[http://www.numerama.com/telecharger/9580-uxtheme-multi-patcher.html|Uxtheme Multi-patcher]] pour débloquer les limites sur les thèmes créés par les partis tiers de Microsoft.
  * Le thème [[http://freddi67.deviantart.com/art/Ubuntu-Light-for-Windows-XP-177514325|Ubuntu Light pour Windows XP]]

==== Installation ====

  * Télécharger le thème [[http://freddi67.deviantart.com/art/Ubuntu-Light-for-Windows-XP-177514325|Ubuntu Light pour Windows XP]] et l'outil [[http://www.numerama.com/telecharger/9580-uxtheme-multi-patcher.html|Uxtheme Multi-patcher]]
  * Exécuter Uxtheme Multi-patcher puis redémarrer l'ordinateur
  * Décompresser l'archive du thème
  * Ouvrir UbuntuLight\ puis double-cliquer sur Light.msstyles pour modifier le menu-démarré, les bordures de fenêtres, etc.
  * Ouvrir UbuntuLight\wallpaper et appliquer le fond d'écran par un clic droit sur l'image
