====== Personnaliser le menu "Applications" de Mate ======
Il est possible de personnaliser à son goût, le menu **Applications** de l'environnement [[:Mate]], cette page décrit la procédure à suivre.

Le paquet **[[apt>mozo]]** est l'utilitaire utilisé par Mate pour faire cela.

{{ :personnalisation:menu-applications_xenial.png?450 }}

=====Utilisation=====
Pour accéder à **Mozo**, appelé aussi **Menu principal**, vous pouvez soit :
  * faire un clic droit sur l'icône **Applications** et **Éditer les menus**,
  *  ou par **Système -> Préférences -> Apparences -> Menu principal**.

{{ :personnalisation:mozo_xenial.png?450 }}
=====Configuration=====

==== Définir un sous menu ====
Pour définir un sous-menu, dans la fenêtre principale sélectionnez l'élément principal puis cliquez sur **Nouveau Menu**. Donnez-lui un nom, vous verrez alors votre sous-menu apparaître dans lequel vous pourrez ajouter les éléments de votre choix.

==== Ordonner un menu ====
Pour ordonner un menu, il suffit de sélectionner l'élément de votre choix et de le déplacer à l'endroit désiré. Vous pouvez également cochez ou décocher ce que vous voulez voir apparaître ou pas.

==== Supprimer/Ajouter un menu/élément ====
Vous pouvez également cochez ou décocher ce que vous voulez voir apparaître ou pas, que ce soit un menu ou un élément le principe est le même.
