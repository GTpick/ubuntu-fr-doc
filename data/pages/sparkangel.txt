====== SparkAngels ======

**SparkAngels** est un [[wpfr>logiciel propriétaire]] de télé-intervention qui permet de prendre le contrôle d'un PC à distance.\\
Bien qu'il existe d'excellents logiciels pour faire ce travail SparkAngels a l'avantage de ne pas avoir besoin d'une configuration pare-feu, est simple d'utilisation et est en français.

===== Pré-requis =====
 
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  * Avoir [[:tutoriel:comment_installer_un_paquet|installé le paquet]] **[[apt>icedtea-7-plugin]]**.


===== Installation =====
  - Téléchargez le script java [[http://www.spark-angels.com/sparkangels/sangels/sparkangels.jnlp|ici]]. 
  - Lancez-le par un double-clic ou avec IcedTea. Le script va télécharger l'application. 
  - Acceptez le [[wpfr>Licence_de_logiciel#Contrat_de_licence_utilisateur_final|contrat de licence]] ainsi que les différentes étapes par YES ou RUN.
Il faudra ensuite vous inscrire pour accéder à toutes les fonctionnalités de SparkAngels.

Il existe également une extension SparkAngels pour [[:LibreOffice]] téléchargeable [[http://www.spark-angels.com/static/plugins/ooo/toolbar/InstantCoaching.oxt|ici]] et une autre pour [[Firefox]] [[http://www.spark-angels.com/web/fr/trouvez-de-laide/telechargements-gratuits.html|ici]].


===== Utilisation =====
Normalement, l'installation a créé un raccourci sur votre bureau par lequel vous pouvez lancer l'application.\\
Il est possible après s'être inscrit à la fois de recevoir de l'aide mais aussi de devenir accompagnateur pour aider les autres. Pour cela allez dans **Options -> Mon statut accompagnateur -> Me déclarer disponible**.
