====== Purrr : Renommer en masse ======  

Purrr est une application GNOME pour le renommage de masse de fichiers. Le renommage de masse est un cas spécial de renommage, où de nombreux fichiers doivent être nommés de la même manière, les rendant utilisables pour des processus de traitement automatiques.
Purrr s'intègre parfaitement dans l'environnement GNOME. Si vous cherchez une application identique plus intuitive reporetez-vous à la page [[:Pyrenamer]].

===== Pré-requis =====
 
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation =====

Pour [[precise|Ubuntu 12.04]], [[:tutoriel:comment_installer_un_paquet|installez le paquet]] **[[apt>purrr]]**.\\
Pour [[trusty|Ubuntu 14.04]], téléchargez les paquets [[http://za.archive.ubuntu.com/ubuntu/pool/universe/p/python-central/python-central_0.6.17ubuntu1_all.deb|python-central]] et [[http://www.ubuntuupdates.org/purrr|purrr]], puis installez-les.

===== Utilisation =====
Pour lancer l'application faites une recherche dans [[unity#tableau_de_bord_dash|le tableau de bord Unity]] avec le mot clé **purrr**.


{{:purrr_trusty.png?300}}


Dans le champ de saisie en haut, rentrez le masque de renommage pour vos fichiers. Les options disponibles sont :


  * [C,a,b,c] : compteur qui possède trois options (a, b, c) qui peuvent être omises
    * a indique le numéro de départ (commencer la numérotation à partir de 2, 8, ou 64)
    * b indique le pas de numérotation (de 2 en 2, de 5 en 5)
    * c indique le nombre de chiffres sur lequel se fera la numérotation (1, 2, 3, avec c=1 ; 01, 02, 03 avec c=2 ; etc.)
  * [N] : est le nom du fichier courant. Il accepte l'option "_" qui permet de remplacer toutes les occurences de "_" par des espaces.
  * [B] : nom du fichier sans extension
  * [E] : extension du fichier. Il accepte l'option "." qui permet de rajouter un . avant l'extension, si le fichier comporte une extension.

Note : Les lettres N, B et E utilisées en minuscule permettent de changer la casse.
  * Ex.: ABcd.EFG -> [n] -> abcd.efg, alors que ABcd.EFG -> [N] -> ABcd.EFG

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. La configuration de l'application sera conservée ou supprimée selon la méthode de désinstallation que vous choisirez.
