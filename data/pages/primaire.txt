====== Logiciels éducatifs pour le primaire ======

Les logiciels éducatifs pour le primaire sont classés dans deux sous-catégories :
  * Les [[cycle_2|Logiciels éducatifs pour le cycle 2]] regroupe les logiciels pour les enfants de la grande section de maternelle jusqu'au CE1
  * Les [[cycle_3|Logiciels éducatifs pour le cycle 3]] regroupe les logiciels pour les enfants du CE2 jusqu'au CM2 (fin du primaire).
