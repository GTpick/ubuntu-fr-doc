====== NeverNoid ======

{{ nevernoid.jpg?200}}

**Nevernoid** est un freeware basé sur le jeu breakout sorti en 1976, et a été écrit en Pascal.

Vous dirigez une barre horizontale et devez renvoyez une balle afin de casser des briques.

===== Installation =====

  - [[:tutoriel:comment_installer_un_paquet|Installez le paquet]] [[apt://libsdl-mixer1.2|libsdl-mixer1.2]].
  -Télécharger l'archive du jeu en cliquant sur [[http://download.softpedia.ro/dl/07b399e5e5cf804fc45850fc19933d62/499e8e15/500008309/linux/games/NeverNoid_v12_linux_x86.tar.gz|ce lien]].
  -Extraire l'archive (sur Ubuntu: clic-droit sur le fichier puis "Extraire ici") ((ou avec le terminal: ''tar -xvf NeverNoid_v12_linux_x86.tar.gz''))
  -Se placer dans le dossier du jeu((avec Nautilus ou avec le terminal: ''cd nevernoid''))


===== Utilisation =====

Lancer((double-clic avec Nautilus, ou avec le terminal: ''./nevernoid'')) le fichier **nevernoid** qui se trouve dans le dossier du jeu.


===== Configuration =====


==== Cacher le dossier « nevernoid » ====


=== Gnome ===

Pour cacher le dossier « nevernoid »:
  * Lancer Nautilus.
  * Sur le dossier « nevernoid » faites //clic-droit → Renommer//.
  * Donnez-lui le nom de « .nevernoid ».

==== Créer un lanceur dans le menu Applications->Jeux ====
Voir le tutoriel **[[:raccourci-lanceur#creation_d_un_nouveau_lanceur_dans_le_menu_ubuntu_ou_sur_le_bureau|Création d'un nouveau lanceur dans le menu Ubuntu]]**.
La commande permettant de lancer nevernoid est ''dossier_ou_se_trouve_nevernoid/nevernoid''
