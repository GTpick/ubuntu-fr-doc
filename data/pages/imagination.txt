====== Diaporama Imagination ======

**Imagination** est une application de création de [[:diaporama|diaporama vidéo]] simple et intuitive. Elle comprend plus de 60 effets de transition. L'exportation du diaporama au format FLV (lisible par les lecteurs [[:flash]]) est également prise en charge.\\
Écrite en langage C et utilisant Gtk2, elle est disponible sous FreeBSD et %%GNU/Linux%% en licence libre ([[http://www.gnu.org/licenses/old-licenses/lgpl-2.0.html|LGPL v2.0]]). \\

{{ :application:imagination:imagination_quantal_01.png?500 |}} 

===== Pré-requis =====
  * Disposer des [[:sudo|droits d'administration]] ;
  * Disposer d'une connexion à Internet configurée et activée ;
  * Avoir installé l'encodeur [[:FFmpeg]] ([[Avconv]] pour [[trusty|Trusty 14.04]] et [[precise|Precise 12.04]]).

===== Installation =====

L'application **imagination** est présente dans les dépôts officiels. Pour l'installer, il suffit d'[[:tutoriel/comment_installer_un_paquet|installer le paquet]] **[[apt>imagination]]**. Ce paquet installera toutes les dépendances nécessaires à son fonctionnement. 

=====Configuration=====

====Activer l'ajout de formats audio non-libres (mp3, etc.)====
Si vous n'arrivez pas à importer un fichier audio (mp3 etc), [[:tutoriel:comment_installer_un_paquet|installez le paquet]] **[[apt>libsox-fmt-all]]** et redémarrez.
Si le problème persiste, consultez [[http://forum.ubuntu-fr.org/viewtopic.php?pid=3329533#p3329533|cette discussion]].

===== Lancement =====

Lancer l'application depuis depuis le [[:unity#tableau_de_bord_dash|tableau de bord]] d'Unity. (Pour les menus classiques, via le menu //Applications -> Son & Vidéo-> Imagination//). \\
Ou depuis un [[:terminal]] en exécutant la [[:commande_shell|commande]] suivante :
<code>imagination</code>
===== Utilisation =====
==== Astuce: diaporama facile et rapide====

Petite astuce pour créer un diaporama en quelques minutes:
  * Vous pouvez importer plusieurs images en même temps (Shift+clic pour sélectionner plusieurs fichiers dans votre gestionnaire de fichiers)
  * Vous pouvez sélectionner plusieurs images en même temps dans la barre d'images du bas dans Imagination, via la souris ou via la combinaison <Ctrl>+<A>. Il vous est ainsi possible de créer plusieurs transitions d'un coup, ou bien changer la durée d'apparition de toutes les diapos d'un coup !

==== Multi-zoom sur une image====

  * Sélectionner une image dans la barre d'images du bas ;
  * Puis dans le cadre "Déplacement de l'image", modifiez le zoom ;
  * Cliquer (clic-gauche) sur la prévisualisation de l'image, et sans relâcher le clic la faire glisser vers le point où on veut zoomer ;
  * Dans le cadre "Déplacement de l'image", cliquez sur "Ajouter". Vous pouvez répéter l'opération afin de zoomer sur un autre point.

====Ajouter une vidéo au diaporama====

Cette opération n'est pas (encore) possible dans ce logiciel. Par contre vous pouvez aisément créer votre diaporama d'images via Imagination, puis y ajouter des vidéos via un logiciel de [[montage vidéo]], comme Kino, Kdenlive, Pitivi, Openshot, Cinelerra par exemple.

=====Exporter le projet====
Votre diaporama est maintenant terminé. Il vous faut alors l'exporter pour pouvoir le visionner ou le diffuser. Mais avant de se rendre dans l'entrée du menu "Fichier → Exporter", vous devez configurer le format de sortie. Pour cela:
  - Se rendre dans l'entrée "Propriété" du menu "fichier". Puis:
  - Choisir le format vidéo de sortie et la taille de la vidéo et si vous le souhaitez, modifier les paramètres avancés.
  - Vous pouvez ensuite vous rendre dans l'entrée du menu "Fichier → Exporter".
  -Finaliser les derniers paramètres de format et lancer l'encodage.
Cela peut prendre du temps... beaucoup de temps, en fonction de la taille de votre projet, du format de sortie et de la vitesse de calcul de votre ordinateur.

===== Désinstallation =====
 
Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Si vous ne l'utilisez plus pour une autre application d'encodage, vous pouvez [[:tutoriel:comment_supprimer_un_paquet|supprimer le paquet]] **libav-tools**. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leur dossier personnel sont toujours conservés.
