====== Téléphone mobile ZTE F160 ======

Le ZTE F160 est un téléphone mobile 3G, ce n'est pas un "smartphone" bien qu'il propose quelques fonctionnalités avancées et puisse se connecter à l'Internet. Ce téléphone - de faible coût - a été commercialisé en France par l'opérateur de téléphonie mobile //Free//. Il n'est plus commercialisé. On peut encore en trouver d'occasion.

===== Transfert des données multimédia =====
Les sons, photos et vidéos sont stockés soit dans la mémoire de base du téléphone (20 Mo) soit sur une carte microSD qui peut y être insérée (8 Go max).
<note warning>Le téléphone n'est pas reconnu par Ubuntu quand on le branche avec son cordon USB.</note>
Le téléphone est fourni avec un logiciel de synchronisation //Join me// utilisable sous Windows uniquement (ne fonctionne pas sous Wine car ne peut se connecter au téléphone).

==== Solution 1 : liaison Bluetooth ====
Si votre ordinateur est équipé de la fonctionnalité [[bluetooth|Bluetooth]], vous pouvez connecter le téléphone.

Bluetooth doit être activé et visible à la fois coté ordinateur et coté téléphone. Définir un code d'accès sur l'ordinateur, rechercher les périphériques visibles, sélectionner le téléphone. Entrer le code sur le téléphone à l'invite.

Permet d'accéder aussi bien à la mémoire de base qu'à la carte microSD.

==== Solution 2 : lecteur de carte microSD ====
A défaut de Bluetooth, vous pouvez extraire la carte microSD du téléphone et la brancher sur l'ordinateur :

- dans un lecteur de carte SD, via un adaptateur microSD -> SD

- sur un port USB, via un adaptateur [micro]SD -> USB

Cette solution est lourde à mettre en œuvre (extinction et ouverture du téléphone) et ne permet pas l'accès à la mémoire de base du téléphone.
