====== SkyChart, cartes du ciel ======
{{skychart3.png?600|}}

**Skychart** est un excellent logiciel libre de **cartographie céleste**. Conçu à l'origine pour Windows, il vient rejoindre le monde libre avec sa version 3. Ce programme très complet se distingue par la richesse de ses catalogues, ses nombreuses fonctions paramétrables, le fait d'être utilisable aussi bien par le simple débutant que l'astronome chevronné.

Ainsi, il est possible de choisir spécifiquement ou automatiquement :
  * quels catalogues sont utilisés ;
  * le lieu et la date d'observation ;
  * la couleur et la dimension des étoiles et nébuleuses ;
  * le mode de représentation des planètes ;
  * l’affichage de labels et de grilles de coordonnées ;
  * la superposition de photographies ;
  * les conditions de visibilités.
Ajoutez à cela :
  * une fonction recherche basée sur les catalogues d'objets ;
  * l'accès à de multiples ressources en ligne ;
  * l'autoguidage d'un télescope ;
  * etc.
Il a de plus l'avantage de produire des cartes fiables et précises, ce qui ne gâche rien.

Il existe entre autres un logiciel équivalent (KDE) : [[:Kstars]]. À comparer…

===== Pré-requis =====
  * disposer des [[:sudo|droits d'administration]] ;
  * disposer d'une connexion à Internet configurée et activée ;
  * avoir modifié [[:tutoriel:comment_modifier_sources_maj|les sources de dépôts]] pour ajouter la ligne suivante :
<file>deb http://www.ap-i.net/apt stable main</file>
Si vous désirez des mises à jour plus fréquente avec les versions de développement ajoutez cette ligne:
<file>deb http://www.ap-i.net/apt unstable main</file>
  * avoir [[:tutoriel:comment_installer_un_paquet|installé les paquets]] **[[apt://libgtk2.0-0,libglib2.0-0,libpango1.0-0,libjpeg62,libpng12-0,libsqlite3-0|libgtk2.0-0 libglib2.0-0 libpango1.0-0 libjpeg62 libpng12-0 libsqlite3-0]]**.

<note tip> N'oubliez pas d'obtenir la clé authentification en exécutant la ligne suivante dans un terminal : 
<file>apt-key adv --keyserver keyserver.ubuntu.com --recv-keys D79BF92A</file></note>

<note tip>Après avoir modifié les sources de dépôts, pensez à mettre à jour la liste des paquets disponibles.</note>


===== Installation =====

  * Pour afficher les images des planètes, [[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt://xplanet|xplanet]]**.
  * Pour piloter un télescope avec le logiciel,  [[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt://indi|indi]]**. 
  * Puis [[:tutoriel:comment_installer_un_paquet|installer l'application]] **[[apt://skychart|skychart]]** 
  * Si vous avez beaucoup de place disque, [[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt://skychart-data-pictures|skychart-data-pictures]]**.

Pour information vous pouvez 
[[http://www.ap-i.net/skychart/fr/documentation/installation_pour_linux_debian
|consulter la procédure d'installation du site officiel pour distribution Debian]].

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer ses paquets]].
