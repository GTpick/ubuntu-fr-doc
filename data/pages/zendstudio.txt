====== Zend Studio ======

{{icon64_studio.png}}

**Zend Studio** est un IDE intégré complet, servant au développement, débogage, déploiement et gestion des applications [[PHP]]. 

Zend Studio est développé en Java, ce qui le rend disponible pour tous les systèmes d'exploitation ayant une machine virtuelle [[java|java]].

<note tip>Cette application nécessite une licence payante, une seule fois sauf si vous désirez avoir un support</note>


===== Installation =====
[[http://www.zend.com/fr/downloads/|Téléchargez zendstudio]] sur le site officiel et décompressez l'archive dans le dossier /home/user/.\\
Ensuite exécutez le fichier .bin du dossier décompressé et laissez-vous guider.\\ \\
{{installation_zend.png}} \\




===== Utilisation =====

Une fois installé, désactivez [[:compizconfig-settings-manager|compiz-fusion]] pour le 1er lancement afin de ne pas avoir de bug d'affichage désagréable. \\
Le lancement de celui-ci se fait en cliquant sur le fichier Zend_Development_Environment qui se trouve dans le dossier ZendStudio-5.5.0 ou le nom choisi pendant l'installation. \\

<note help>Attention, si vous réactivez [[:compizconfig-settings-manager|compiz/fusion]] et que cela bug, désactivez-le pendant votre travail sous zend</note>



==== Création d'un lanceur sur le bureau de façon graphique ====
{{zend_launch.png}}

Pour [[raccourci-lanceur|faire un raccourci]] sur le bureau faites: clic-droit sur le bureau ⇒ Créer un lanceur.

  *Choisissez dans Type "application" dans la fenêtre qui s'affiche.

  *Nom : "ZendStudio"

  *Commande : "la commande permettant de la lancer
<code>/home/utilisateur/ZendStudio-5.5.0/Zend_Development_Environment</code>

  *Commentaire:Commentaire/description du programme (apparaissant quand on laisse la souris un petit moment sur l'élément du menu)
<note tip>Les icônes sont disponible dans le dossier zend <code>/home/utilisateur/zend/bin</code></note>
