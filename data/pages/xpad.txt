====== Xpad : des Post-it™ sur le bureau ======

**Xpad** est un logiciel libre permettant la création de //notes// ou //papillons// (le terme français recommandé pour Post-it™) sur le bureau. 
Cette application est conçue pour le bureau [[:GNOME]] 2.x mais fonctionne également très bien avec [[:Unity]].

Si vous n'avez pas besoin de toutes les fonctionnalités de [[:tomboy|Notes Tomboy]], ou cherchez simplement un moyen d'afficher en permanence de simples notes sur le bureau, comme de vrais Post-it™ : alors Xpad est pour vous.

{{applications:xpad_trusty_1.png}}

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation =====

Pour installer ce logiciel, il suffit d'[[:tutoriel:comment_installer_un_paquet#paquet_present_dans_les_depots|installer le paquet]] **[[apt>xpad]]**.

Installation de xpad en ligne de commande dans un [[:terminal]] :

<code>sudo apt-get install xpad</code>




===== Utilisation =====
 
Lancez l'application via le [[:unity#tableau_de_bord_dash|tableau de bord Unity]] ou via le [[:terminal]] (toutes versions d'Ubuntu) avec la [[:commande_shell|commande]] suivante :
<code>xpad</code>

Une fois le logiciel lancé, il suffit de cliquer droit sur la note vierge pour ouvrir un menu avec les options principales du logiciel xpad :

  * **Note** : un menu comprenant les options basiques de création, suppression et mise en forme (contenues dans les //propriétés//) de la note visionnée ;
  * **Édition** : menu comprenant les options classiques de mise en forme du texte et les préférences de l'application ;
  * **Affichage** : menu permettant d'afficher les différents menus et barres d'outils du logiciel ;
  * **Notes** : menu permettant de gérer les notes enregistrées ;
  * **Aide** ;

Une barre d'outils apparaît quand le curseur survole une note. Un clic droit dessus permet de lui ajouter des outils supplémentaires.

Pour voir les notes sur le bureau de manière permanente, Il est possible de l'ajouter aux  [[:tutoriel/application_demarrage|application au démarrage]] d'Ubuntu. 

==== Ne pas afficher la bordure de fenêtre ====

Pour que les notes n'affichent pas la bordure de fenêtre, et qu'ils aient ainsi l'apparence de vrais Post-it™, faites un clic droit sur une note, et dans le menu contextuel //Affichage//, décochez //Décorations de fenêtre//.


{{applications:xpad_trusty_2.png}}

==== Personnaliser l'apparence d'une note ====

Faites un clic droit sur la note, puis sélectionnez le menu //Note -> Propriétés//. 
Vous pouvez changer la police de caractères, les couleurs d'arrière-plan et d'avant-plan. Ce changement n'affectera pas les préférences d'affichage générales (que vous pouvez modifier dans le menu //Édition -> Préférences//).

{{applications:xpad_trusty_3.png}}


===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, le configuration globale de l'application est conservée ou supprimée. 
Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
