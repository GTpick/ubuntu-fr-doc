====== Bomberclone ======

===== Introduction =====

**[[http://bomberclone.sourceforge.net/homepage/|Bomberclone]]** est un clone du jeu Atomic Bomberman.

===== Installation =====

[[:tutoriel:comment_installer_un_paquet|Installez le paquet]] **Bomberclone** [[apt://bomberclone|{{apt.png}}]]

Hélas il ne met pas à jour le menu Jeux, il faut donc le lancer par le terminal ou ALT+F2

  bomberclone

<note>Sous gnome, il y'a une solution simple à ce problème. Sélectionnez système->préférence->menu principal puis choisir jeu et faites nouvel élément. Mettez bomberclone dans nom et commande et le jeu apparaitra dans la liste des jeux. </note>


===== Comment ca marche ? =====

Bomberclone est avant tout un jeu réseau, il faut donc un joueur qui décide d'héberger la partie, les autres s'y connectent.

Le seul port a ouvrir/rediriger pour héberger une partie est le port 11000 (en udp)

Avant de rejoindre la partie passez par les options afin de mettre votre nom de joueur.

Les joueurs apparaissent dans la salle pré-partie et peuvent discuter des règles de la prochaine...

{{applications:jeux:capture-bomberclone_0.11.6.2.png?200}}

Pour jouer effectivement (ne pas être spectateur) appuyez sur la touche Ctrl afin de sélectionner un personnage.

Quant tout le monde est prêt l'hôte doit appuyer sur F4 pour lancer le jeu.

{{applications:jeux:bomberclone2.png?200}}


Une fois que tout le monde est arrivé, on peut encore préparer le coup dans le mini salon de discussion. Pour lancer le jeu proprement dit : F4 une nouvelle fois pour l'hébergeur.

Les touches :

  * de direction pour se diriger...
  * Ctrl pour poser une bombe
  * Shift pour utiliser les spéciaux


{{applications:jeux:bomberclone3.png?200}}

Il y a plusieurs mode de jeu :

  * Classique, chacun pour soi, une fois élimine, on attends le prochain round
  * Deathmatch, chacun pour soi, par contre on revient en jeu même après avoir été élimine, la partie dure ~ 5 min et on compte les points mort/tué
  * Team, la dernière équipe ayant au moins un joueur vivant remporte la partie

Le jeu propose plusieurs plateaux plus ou moins agréables à jouer, méfier vous des cartes aléatoires... l'ajout des tunnels est un gros plus.

===== Problème rencontré =====

Impossibilité de jouer en mode multijoueur une erreur UDP empêchant la connexion: version touché 0.11.7
