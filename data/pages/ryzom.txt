===== Ryzom, présentation =====
{{http://www.ryzom.fr/data/rz_ryzom_logo.png  |Ryzom}}

  * **[[http://www.ryzom.fr/fr/mmorpg-rpg-mmo-features.html|Ryzom]]** (//anciennement [[http://linuxfr.org/2008/09/19/24502.html|The Saga of Ryzom]]//) est un [[http://fr.wikipedia.org/wiki/Jeu_de_r%C3%B4le_en_ligne_massivement_multijoueur|jeu de rôle en ligne massivement multijoueurs]].
  * [[http://atys.ryzom.com/projects/pubfr/wiki/Lore#Histoire|L'histoire]] de [[http://www.ryzom.fr/fr/|Ryzom]] se déroule sur la planète [[http://atys.ryzom.com/projects/pubfr/wiki/Lore#Atys|Atys]] plus de 2'000 ans dans le futur. 
  * [[http://www.fsf.org/blogs/licensing/ryzom-free-software|Depuis le 6 mai 2010]], le jeu est publié sous licence libre, en plaçant le code source sous licence AGPL, et l'ensemble des données artistiques sous licence CC-BY-SA.   
  * [[http://ryzom.hg.sourceforge.net/hgweb/ryzom/ryzom|Le code]] de la partie client et serveur, les outils de développements ainsi que les [[http://media.ryzom.com/|ressources graphiques]] sont donc libres, néanmoins, une partie du gameplay et le level design restent privés. //(Voir [[http://ryzom.mondespersistants.com/?news=629|l'interview de l'équipe]] pour plus de précisions. [4"16])//
  * Depuis le 9 mars 2011, la [[http://forums.ryzom.com/showthread.php?t=35223|version d'essai gratuite de Ryzom]] est passée en illimitée. Il est donc possible de découvrir le jeu plus longuement. Les restrictions concernent l'impossibilité d'acquérir un quelconque stockage (mektoubs de bât ou coffre d'appartement) et les branches de compétences sont limitées au niveau 125. Ce qui laisse tout de même des heures et des heures de jeu.
  * Depuis le [[http://linuxfr.org/news/bilan-2015-du-mmorpg-ryzom|6 mai 2016]], le jeu est disponible sur la plateforme [[http://store.steampowered.com/app/373720/|Steam]]. De [[http://app.ryzom.com/app_forum/index.php?page=post/view/168802|nombreuses améliorations]] ont été apportées, notamment graphiques, pour la publication du jeu sur Steam.

===== Installation =====

Pré-requis :
  * // Disposer d'une connexion à Internet configurée et activée. //

Le jeu était disponible avec un ancien client dans la logithèque Ubuntu jusqu'à la version 14.04. L'installateur du nouveau client (V3) devrait être bientôt de retour dans les dépôts officiels.

L'installateur est téléchargeable directement depuis le site officiel de Ryzom.
  
Le jeu est aussi disponible sur la plateforme [[https://doc.ubuntu-fr.org/steam|Steam]]. Vous le trouverez donc [[http://store.steampowered.com/app/373720/|ici]] avec la version la plus à jour et bénéficiant des dernières améliorations graphiques.


==== Depuis la Logithèque Ubuntu ====

* Version du client obsolète pour le moment*

Cherchez Ryzom dans la logithèque Ubuntu (Ubuntu Software Center) et cliquez sur **Acheter** (même s'il est gratuit) pour l'installer. A chaque lancement du jeu à partir de son icône, les données du jeu seront mises à jour si nécessaire.

En cas de soucis lors du téléchargement, vous pourrez trouver les fichier du jeu dans le répertoire ~/.ryzom et les données téléchargées dans ~/.ryzom/data. Vous pourrez supprimer ces fichiers pour reprendre le téléchargement du début.


==== Avec le programme d'installation ====

Le client officiel (V3) peut être exécuté sur toutes les distributions GNU/Linux dont Ubuntu. Un fois installé, il se met à jour tout seul de façon automatique...

Depuis octobre 2016, il y a un nouvel installateur disponible depuis le [[http://ryzom.com/?lang=fr|site officiel]] :

  * Enregistrer le fichier [[http://devl1.ryzom.com/ryzom_installer_linux.tar.bz2|ryzom_installer_linux.tar.bz2]] sur votre ordinateur et extraire le contenu de l'archive.
  * Lancer le script ryzom_installer_linux.sh, ou ryzom_installer_linux32 pour un système 32 bits, ou ryzom_installer_linux64 pour un 64 bits.
  * Suivez les instructions. Les fichiers vont être téléchargés et placés par défaut dans votre home (.local/share/Ryzom) donc uniquement disponible pour votre utilisateur.

==== Via Steam ====

Depuis le 6 mai 2016, vous pouvez jouer à Ryzom via la plateforme de jeu Steam. Pour cela, suivre la procédure d'installation de [[https://doc.ubuntu-fr.org/steam|Steam]]. Le jeu est [[http://store.steampowered.com/app/373720/|disponible]] en version d'essai illimitée dans le temps et en abonnement comme sur le site officiel. La version Steam du client est aussi la plus à jour...

==== Depuis les archives officielles (toutes distributions) ====

Tous les fichiers permettant d'installer Ryzom manuellement se trouve sur [[https://sourceforge.net/projects/ryzom/files/installer/|ce dépôt Sourceforge]].

La méthode est détaillée sur [[http://forge.ryzom.com/wiki/Installation_Linux_G%C3%A9n%C3%A9rique|cette page]]

==== Via le PPA ====

Il existe un dépôt PPA contenant les dernières versions de Ryzom Installer. Ceux qui utilisent Ubuntu ou Debian (ou n'importe quelle autre distribution basée sur Debian) peuvent l'utiliser. A noter que ces paquets sont compilés sous Debian Squeeze (6.0) afin de garantir une compatibilité binaire avec toutes les versions récentes des distributions GNU/Linux.

Il peut être ajouté et le jeu installé par la commande suivante :
<code>
sudo add-apt-repository ppa:ryzom-isv/ppa
sudo apt-get update && sudo apt-get install ryzom
</code>
ou

[[:PPA|Ajoutez le PPA]] **[[https://launchpad.net/~ryzom-isv/+archive/ubuntu/ppa|ppa:ryzom-isv/ppa]]** dans vos sources de logiciels, puis [[:tutoriel:comment_modifier_sources_maj#recharger_la_liste_des_paquets|rechargez la liste des paquets]].
Ensuite vous pouvez [[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt://ryzom|ryzom]]**.

L'Installateur de Ryzom sera installé dans /opt/ryzom avec le nom ryzom_installer_qt. Il peut être lancé à partir du menu d'applications. Au premier lancement, il vous demandera où télécharger et installer Ryzom (ce sera fait avec votre compte utilisateur, il est donc conseillé de ne pas choisir un répertoire appartenant à un autre utilisateur). Le répertoire par défaut est ~/.local/share/Ryzom.

Il est conseillé d'utiliser l'Installateur pour lancer le jeu car il peut gérer un système de profils, mais si vous désirez lancer le jeu à la main, vous pouvez taper :
<code>
~/.local/share/Ryzom/ryzom_live/ryzom_client
</code>

Si vous souhaitez utiliser les profils, il faut ajouter le paramètre **-p** suivi du répertoire du profil. Le premier profil est créé dans le répertoire **~/.local/share/Ryzom/0** et porte donc l'ID 0, pour l'utiliser il suffit de taper :
<code>
~/.local/share/Ryzom/ryzom_live/ryzom_client -p 0
</code>

Vous pouvez créer autant de profils que vous le souhaitez (plutôt utile si vous avez plusieurs comptes), mais leur gestion est plus facile en utilisant l'Installateur (qui peut même mémoriser votre mot de passe).

===== Utilisation=====

Lancez le jeu depuis le menu //Applications -> Jeux -> Ryzom// ou depuis le lanceur //Ryzom Installer// et cliquer sur //jouer//.

On débute sur l’île de Silan, avant de rejoindre le continent.

Si vous faites toutes les quêtes de Silan, vous arrivez sur le continent riche et avec du bon matériel. Sur Silan, on découvre toutes les bases pour jouer, il est recommandé d' y faire toutes les missions. Chiang (PNJ à qui il faut parler à votre arrivée) vous dira s'il pense que vous êtes prêt à partir...

N'hésitez pas à demander de l' aide sur le canal français qui est très actif (uni fr) - et si vous ne trouvez pas le chat fr, faites SHIFT+W et cliquez sur le drapeau français. 

Pour rappel, le canal de chat UNIVERS est dédié à toutes les questions (pour toutes langues: anglais, russe, allemand...).

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
Il est par ailleurs recommandé de supprimer ou désactiver le dépôt PPA.

Tous les fichiers de configuration, marqueurs sur la carte ou macros sont dans le sous-dossier "/save" qui est à conserver et peut être replacé le cas échéant dans une nouvelle installation du jeu...
