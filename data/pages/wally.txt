====== Wally ======


**Wally** est un changeur de fonds d'écran Qt4 automatique et personnalisable, qui utilise de multiples sources comme les fichiers, les répertoires et les dossiers FTP  (Bing, Buzznet, Filckr, Ipernity, Panoramio, Photobucket, PikeSmugMug, Suzznet et Yahoo).

Il est compatible avec les gestionnaires de fenêtres : GNOME, KDE 3, KDE 4, Xfce 4.

===== Pré-requis =====
 
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation =====
Il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>wally]]**.

===== Utilisation =====
Lancez l'application via le [[:terminal]] (toutes versions d'Ubuntu) avec la [[:commande_shell|commande]] suivante : <code>wally</code>
Lors de la première utilisation, un message d'avertissement de l'auteur informe sur sa non responsabilité vis à vis des fichiers affichés par les sources externes.
{{ :applications:wally-avertissement_xenial.png?250 }}

Pour accepter, valider le "__Oui__".
