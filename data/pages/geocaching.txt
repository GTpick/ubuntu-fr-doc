======= Géocaching =======
Le géocaching (ou geocaching) est un loisir qui consiste à utiliser la technique du système de positionnement par satellites (GPS, GLONASS...) pour rechercher ou dissimuler un contenant (appelé « cache » ou « géocache ») dans divers endroits à travers le monde. Une géocache typique est constituée d’un petit contenant étanche et résistant comprenant un registre des visites et un ou plusieurs « trésors », généralement des bibelots sans valeur. Plus d'un million et demi de géocaches sont répertoriées dans 222 pays sur les différents sites web communautaires dédiés à ce loisir.\\
Source : wikipedia

===== Importation des cartes OSM =====
Les GPS sont en général fourni avec une carte restreinte d'une zone géographique. Si vous souhaitez avoir une carte "presque complète" de votre territoire, vous pouvez télécharger la cartographie libre d'Openstreetmap.\\
Pour télécharger la carte de votre zone, rendez-vous sur [[http://wiki.openstreetmap.org/wiki/FR:OSM_Map_On_Garmin/Download|cette page]], sauvegardez, décompressez la carte et copiez-la dans le répertoire de votre GPS.

=====Installation du GarminPlugin=====
<note important>Rappel : en installant les paquets fournis sur un dépôt PPA, vous faites confiance à la personne qui les a mis à disposition. En aucun cas ces paquets ne sont supportés par Canonical.</note> 
<note important>Il semblerait que le développement du plugin soit stoppé depuis 2014. Garmin développe uniquement le plugin pour IE sous Windows. De plus le plugin disponible dans le catalogue de Firefox ne fonctionne pas.</note> 
Si vous souhaitez importer les caches depuis le site geocaching.com, avec certains GPS Garmin, vous devrez installez le GarminPlugin pour que Firefox reconnaisse votre GPS.\\
Pour installer le plugin, ouvrez un terminal et tapez :
<code>sudo add-apt-repository ppa:andreas-diesner/garminplugin
sudo apt-get update
sudo apt-get install garminplugin </code>
Lien vers la documentation (anglais) du [[http://www.andreas-diesner.de/garminplugin|GarminPlugin]]
=====Générer des roadbook et des fichiers GPX grâce à GCTOUR=====
GCTour est un script conçu pour le géocaching fonctionnant uniquement sur Firefox.
Il permet de créer très rapidement des listes de caches et d'en générer la carte, le roadbook et le fichier GPX.
Pour l'installer, [[http://gctour.madd.in/index|Visitez le site de gctour]] \\
