====== Macinux ======

Macinux est un outil de préparation rapide pour l'installation d'Ubuntu sur la plateforme Macintosh. 


===== Prérequis =====

* Un compte Administrateur.
* Le module SIP désactivé. 


===== Installation =====

Vous pouvez télécharger Macinux depuis sourceforge : . 
Glissez l'application vers votre dossier Applications.


===== Désactivation du Module SIP =====

Le module SIP doit être désactivé avant toute utilisation du logiciel. 
Pour désactiver le module SIP, redémarrez votre ordinateur Macintosh en maintenant les touches CMD+R. 
Dirigez vous ensuite dans le menu Utilitaire, puis Terminal. 
Tapez dans le terminal "csrutil disable", puis redémarrez votre Macintosh. 

===== Utilisation =====

Démarrez le programme en double cliquant sur Macinux.app. Sélectionnez l'installation complète, puis entrez votre mot de passe administrateur. 
A la fin de l'installation le popup "All Done" doit s'afficher. 
Redémarrez ensuite votre mac, et vous disposez du nouveau menu Refind. 
Réentrez en mode recovery pour réactiver le module SIP.
