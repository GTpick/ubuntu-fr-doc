====== Belooted: jeu de belote ======
<note warning>Le jeu n'est hélas plus développé. Vous pouvez toujours essayer avec des [[http://www.belotegratuit.eu/linux|jeux en ligne]].</note>
Belooted est un jeu de belote écrit en GTK+, publié sous licence GPL.\\ 

{{beloote.png|}}

===== Installation =====
Le paquet n'est plus disponible directement sur le  [[http://boby.joe.free.fr| site du développeur]]



===== Configuration =====
Il existe un fichier de configuration permettant de régler quelques options : [[:tutoriel:comment_modifier_un_fichier|modifiez le fichier]] **~/.belooted/conf**


Les options disponibles (version 0.1.4):
  * animations => activer les animations lorsqu'une carte se déplace.
  * click_drop => activer le dépôt des cartes sur simple click
  * declarations => jouer avec les annonces (déconseillé de changer cela, car ça n'a pas été testé)
  * player_names => inutile pour l'instant.
  * background-image => chemin vers l'image à utiliser en fond
  * background-stretch => remplir la zone de jeu avec l'image de fond
  * card-stretch => augmenter la taille des cartes lorsque la fenêtre de jeu s'agrandit
  * card-keep-ratio => ne pas déformer les cartes lors d'un agrandissement
  * card-face => nom du dossier où se trouve les cartes du jeu
  * card-type => l'extension du nom des fichiers
