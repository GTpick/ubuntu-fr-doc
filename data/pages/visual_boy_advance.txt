====== Visual Boy Advance ======

//Visual Boy Advance// est l'émulateur le plus populaire pour les consoles **GameBoy**, **GameBoy Color** et **GameBoy Advance**. 

{{http://vbaexpress.tuxfamily.org/capture_fr_affichage.jpg}}

===== Installation =====

Il existe trois versions de Visualboy advance :

- l'ancienne version non mise à jour

- VBAM qui est la continuation du projet Vba par une nouvelle équipe. Cette version résout un très grand nombre de bugs, je vous conseille donc de l'utiliser.

-  VisualBoyAdvance Link, une version modifiée de VisualBoyAdvance pour Linux. Celle-ci supporte maintenant le Link (câble permettant de lier deux ou plusieurs consoles) jusqu'à 4 joueurs

==== GVBA ====

[[tutoriel:comment_installer_un_paquet|Installez]] les paquets **[[apt>visualboyadvance|visualboyadvance]]** et **[[apt>visualboyadvance-gtk|visualboyadvance-gtk]]**.

==== VBAM-GTK ====

Téléchargez la version 32 ou 64 bits suivant le choix de votre système d'exploitation et [[:tutoriel:comment_installer_un_paquet|installez le paquet.]]

[[http://sourceforge.net/projects/vbam/files/VBA-M%20GTK%2B%20svn%20r1001/vbam-gtk_1.8.0.1001-1_i386.deb/download|Version 32 bits]]

[[http://sourceforge.net/projects/vbam/files/VBA-M%20GTK%2B%20svn%20r1001/vbam-gtk_1.8.0.1001-1_amd64.deb/download|Version 64 bits]]

==== VisualBoyAdvance Link ====
Version modifiée de VisualBoyAdvance par Jacques-Olivier IMBERT, incluant le support multijoueurs (jusqu'à 4 joueurs) et accompagnée d'une interface graphique Java.

A noter que cet émulateur n'est disponible qu'en version 32 bits pour le moment. 

Le téléchargement est disponible sur le [[http://jg.imbert.free.fr/index.php?article3/projet-visualboyadvancelink|site du créateur]]
===== Utilisation =====

==== GVBA ====

Lancez l'application depuis le menu //Applications -> Jeux -> VisualBoy Advance// ou //via// la [[:commande_shell|commande]] suivante :
  gvba

Vous pouvez ouvrir une ROM depuis le menu Fichier/Ouvrir.

L'interface est assez intuitive, vous ne devriez pas avoir de problème pour l'utiliser.


===== Désinstallation =====
Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
