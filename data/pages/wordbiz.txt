====== Wordbiz ======
Avec Wordbiz, vous pourrez jouer au scrabble classique ou duplicate avec des partenaires de tous les niveaux et du monde entier.





===== Pré-requis =====
[[Java]] 6 installé et fonctionnel.

===== Installation =====
Téléchargez le fichier [[http://www.isc.ro/linux/WordBiz18linux.zip|WordBiz18linux.zip]] et dézippez le dans le dossier de votre choix.


==== Lancer le jeu ====

Vous avez deux possibilités  :
  * En faisant clique droit sur //wordbiz.jar// puis ouvrir avec Java.
  * Depuis votre console naviguez jusqu'au dossier contenant //wordbiz.jar// et tapez <code>java -jar wordbiz.jar </code>

<note important>Pour pouvoir se connecter au serveur ISC, vous devez créer un compte à cette adresse : [[

===== Jouer à WordBiz avec du son =====

Pré-requis : Avoir installé OpenJDK depuis la Logithèque Ubuntu ;-)

Afin d'éviter de louper votre tour, jouez avec du son depuis un simple lanceur sur le bureau, 
suivez ce chéma :

Téléchargez l'archive susmentionnée puis décompressez la sous votre dossier personnel (Celui qui porte le nom de votre identifiant Linux.) Vous avez donc dans dossier personnel un dossier nommé WordBiz. 

Sous Ubuntu 10.04 Lucid Lynx Lancez le **Gestionnaire de paquet Synaptic** et installez **alsa-oss** ensuite faite clic droite sur le bureau et "Créer un lanceur". Nommez WordBiz ou Scrabble puis entrez la commande suivante dans le champ "Commande":

<code>xterm -e 'cd /home/Dossier_personnel/WordBiz/ && aoss java -jar ./wordbiz.jar'</code>

Vous avez juste à remplacer /Dossier_personnel/ par votre identifiant Linux.

Source : [[http://forum.ubuntu-fr.org/viewtopic.php?pid=2360652]]

===== Pour changer l'icône =====
Pour avoir le carré officiel WordBiz, clic droit sur le lanceur, Propriété puis cliquez sur l'icône par défaut afin de la remplacer. Une fenêtre s'ouvre, accédez au dossier WordBiz puis sélectionnez l'icône présente dans ce dossier. Faites ouvrir et le tour est joué.
