====== Serialclient ======

Serial client est un client en ligne de commande pour port série. Serialclient, concu selon les principes KISS, se revendique d'être simple d'utilisation et de conception. Ainsi, contrairement à la plupart des clients pour port série qui sont en réalité des émulateurs de terminal, serialclient utilise votre propre shell pour saisir et afficher les échanges avec votre appareil connecté sur le port série.

Serialclient est tout particulièrement indiqué si vous cherchez simplement à communiquer sur un port série avec une carte comme arduino par exemple.

===== Installation =====

Commencez par installer ruby:

<code>sudo apt-get install ruby rubygems </code>

Puis, installez serialclient:

<code>sudo gem install serialclient</code>


Note : Sous Ubuntu 13.10, il faut installer ruby1.9.1 pour que l’installation puisse se faire :

<code>sudo apt-get install ruby1.9.1-dev rubygems
sudo gem install serialclient</code>
===== Utilisation =====

Si votre appareil est monté sur  /dev/ttyUSB0, serialclient s'utilise de cette maniere:
<code>serialclient -p /dev/ttyUSB0</code>


Pour plus d'informations, la commande help permet d'avoir plus de choix:
<code>serialclient --help</code>
