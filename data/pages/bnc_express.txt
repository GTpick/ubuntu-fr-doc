====== Presentation ======

BNC Express est un  logiciel de comptabilité spécialisé pour les professions libérales.
C'est un logiciel commercial sous licence propriétaire et développé pour Windows.


====== Installation ======

N'étant dévelopé que pour Windows, la solution est d'utiliser l'émulation Windows grace à Wine / PlayOnLinux.

Pour cela :
  - Installer [[playonlinux|PlayOnLinux]] (si non installé)
  - Installer le fichier .exe d'installation depuis PlayOnLinux
  - Installer la bibliothèque **oleaut32** (à partir de la page principal de PlayOnLinux : Configurer => onglet Wine => Configurer Wine => Bibliothèques)
  - Installer le composant Visual Basic  **vbrun6** (à partir de la page principal de PlayOnLinux : Configurer => onglet Installer les composants)

====== Notes d'utilisation à partir de Linux ======
  * **Importation** : pour que l'importation fonctionne, il faut que le format de fin de ligne de fichier à importer soit au format Windows (CR/LF) au lieu du format Unix (LF)
