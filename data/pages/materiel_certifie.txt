====== Matériel certifié pour Ubuntu ======
La société [[canonical|Canonical Ltd.]], qui développe [[ubuntu_distribution|la distribution Ubuntu]], travaille de concert avec les plus grandes entreprises informatiques afin de certifier du matériel compatible avec Ubuntu. Dans le site Web d'Ubuntu, une liste de matériels (ordinateurs de bureau, ordinateurs portables et ultra-portables, serveurs, périphériques pour ordinateurs, mobiles et tablettes) est exposée; chacun de ceux-ci s'est vu apposé un label concernant sa certification de compatibilité avec Ubuntu.

  * **[[https://certification.ubuntu.com/server/|Aller à la liste de matériels certifiés pour Ubuntu]]**
  *  **[[https://certification.ubuntu.com/phone/|Aller à la liste de mobiles et tablettes certifiés pour Ubuntu]]**

===== Certifications par Canonical =====

<box 49% left round green|**Ubuntu Certified**>

{{  http://webapps.ubuntu.com/certification/stickers/ubuntucertified.png?50|Certifié pour Ubuntu}}

Le label //Ubuntu Certified// est apposé aux ordinateurs complets testés et certifiés par les ingénieurs de Canonical. Les intégrateurs matériels (OEMs) soumettent leurs ordinateurs au laboratoire de tests de Canonical, où les ingénieurs de Canonical effectuent des tests de certification et de validation.
</box>

<box 49% left round blue|**Ubuntu Enabled**>

{{  http://webapps.ubuntu.com/certification/stickers/ubuntuenabled.png?50|Compatible avec Ubuntu}}

Le label //Ubuntu Enabled// est apposé aux ordinateurs complets lorsque Ubuntu est spécialement modifié pour fonctionner avec ceux-ci. Les ingénieurs de Canonical modifient Ubuntu pour que ces ordinateurs fonctionnent avec les mêmes performances que les ordinateurs certifiés pour Ubuntu.
</box>
~CL~
<box 49% left round|**Ubuntu Ready**>

{{  http://webapps.ubuntu.com/certification/stickers/ubuntuready.png?50|Prêt pour Ubuntu}}

Le label //Ubuntu Ready// est apposé aux ordinateurs complets testés par les intégrateurs matériels (OEMs). Ils effectuent la suite de tests de certification de Canonical. Ils envoient leurs données aux ingénieurs de Canonical qui valident les résultats.
</box>
~CL~

===== Testés par la communauté =====
Tous les ordinateurs et les périphériques ne peuvent être testés par les ingénieurs de Canonical. Généralement, Ubuntu fonctionne très bien même sur le matériel qui ne possède aucune certification. La communauté des utilisateurs d'Ubuntu utilise un très large éventail de matériels énormément diversifié; elle occupe une place importante dans les tests de compatibilité.

  * **[[https://wiki.ubuntu.com/HardwareSupport/|Aller à la liste d'ordinateurs et matériels testés par la communauté mondiale des utilisateurs d'Ubuntu]]**
  * [[:materiel|Liste de matériels testés par les utilisateurs de la communauté Ubuntu-FR]]

===== Devenir partenaire avec Canonical =====
Si vous êtes une entreprise qui travaille dans la conception, l'intégration ou le test de matériels informatiques, devenez un partenaire de Canonical afin de certifier votre matériel comme étant compatible avec Ubuntu. Pour ce faire, inscrivez-vous auprès de Canonical. Canonical prendra contact avec vous pour établir les détails de votre partenariat.
  * **[[https://forms.canonical.com/partners/|Remplir le formulaire de demande de partenariat]]**
