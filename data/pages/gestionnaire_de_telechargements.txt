===== Serveur de fichier =====

====Upload====
  * [[Plowshare]] ([[https://github.com/mcrapet/plowshare|site]]), en ligne de commande, aussi disponible par des [[:nautilus_scripts#internet|scripts Nautilus]].
====Créer un serveur====
===Type megaupload ( 1 fichier 1 URL )===
  * [[:OpenUpload]] ([[http://openupload.sourceforge.net/|site]]) Open Upload est une application PHP pour créer un serveur de téléchargement de fichier (privé et public) similaire à MegaUpload ou encore RapidShare.
  * [[https://github.com/FileZ/FileZ|FileZ]] :  est une application web développée par l’Université d’Avignon et des Pays de Vaucluse qui permet de déposer et gérer des fichiers destinés à être partagés grâce à une URL unique.
  * [[http://codingteam.net/project/koload|KOLoad]] gestionnaire de fichiers en ligne très léger, permettant de partager simplement des fichiers dans un répertoire public (ou privé selon votre configuration).
  * [[http://code.google.com/p/clients-oriented-ftp/|ProjectSend]] Anciennement nommé cFTP, ProjectSend est une plate-forme de partage de fichiers entre personnes authentifiées.
  * [[http://home.gna.org/jyraphe/|Jyraphe]] une application web de dépôt de fichier sous licence AGPL v3.
  * [[http://bigfilesharing.sourceforge.net/|BigFileSharing]] une application web de transfert de fichiers volumineux disponible en Français et Anglais sous licence GPL.
  * [[F*EX]] ([[http://fex.rus.uni-stuttgart.de:8080/index.html?locale=french|site]]) F*EX pour Frams’ Fast File EXchange est un service pour échanger des gros fichiers également sous Perl Artistic Licence.
