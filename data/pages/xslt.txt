====== Processeurs XSLT ======

Différents processeurs [[wpfr>XSLT]] sont disponibles sous Ubuntu. Ici seront présentés xsltproc et Xalan.

===== Pré-requis =====
 
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation =====
==== xsltproc ====
**xsltproc** est l'implémentation en C dans GNOME de XSLT 1.0.  Pour installer ce logiciel, il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>xsltproc]]**.
==== Xalan ====

[[http://xml.apache.org/xalan-j/index.html|Xalan-java]] est un processeur XSLT créé par l'Apache Software Fondation. Pour l'installer, il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>xalan]]**.

===== Utilisation =====
==== xsltproc ====
On peut l'utiliser dans un [[:terminal]] avec la commande "xsltproc" comme ceci : <code bash>xsltproc stylesheet.xsl fichier.xml > page.html</code>
ce qui générera dans le répertoire courant une page HTML à partir des fichiers XML + XSL.

==== Xalan ====
Pour l'utiliser de manière très simple, dans un [[:terminal]] : <code bash>xalan -in documentatraiter.xml -out documentdesortie -xsl feuilledestyle.xsl</code>

===== Désinstallation =====
 
Pour supprimer ces applications, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer leur paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
