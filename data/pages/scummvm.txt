====== ScummVM ======


Si vous êtes ou avez été un fan des jeux LucasArts (Monkey Island, Indiana Jones, Day of the Tentacle...) et des jeux d'aventure en général, le projet open source ScummVM est assurément un projet qu'il vous faut suivre.

ScummVM est une implémentation de l'interpréteur SCUMM de LucasArts, utilisé dans les jeux dits "point & click", tels que Monkey Island, Indiana Jones, Day Of The Tentacle et Sam and Max.

Il fonctionne sous Linux, Windows, Mac 0S X, Amiga, PSP, PS2, PocketPC, NintendoDS... Très simple d'utilisation, vous n'aurez quasiment rien à configurer pour faire fonctionner votre jeu. Il suffit de posséder les fichiers de données du jeu qui vous intéresse.
Les fichiers des jeux de n'importe quelle plateforme sont utilisables (voir les listes sur le site officiel de ScummVM).

On peut donc jouer aux versions PC ou Amiga ou Mac de Monkey Island, par exemple.

Pour le successeur de Scumm, GrimE, voyez ResidualVM.
===== Installation =====

==== Par les dépots officiels ====

Vous pouvez l'[[:tutoriel:comment_installer_un_paquet|installer ]] via les [[:depots#depots_officiels|dépots Universe]] : [[apt>scummvm]]

==== Par le site officiel ====

Directement en .deb depuis le site officiel :

www.scummvm.org/downloads.php#stable

Une fois installé, l'émulateur est dans le menu Applications => Jeux => ScummVm


===== Utilisation =====

Vous pouvez voir apparaître la fenêtre suivante :

{{ svm.png }}

Le jeu que nous allons utiliser ici se nomme : //Monkey 1 : The Secret of Monkey Island// 
 
Donc, cliquez sur " Add Game...". Sélectionnez le répertoire où vous avez copié les fichiers nécessaires au lancement du jeu, puis cliquez sur " Choose "

{{ svm2.png }}

Ensuite, appuyez sur OK, puis sélectionnez le jeu et cliquez sur " Start "

{{ svm3.png }}

Bienvenue au ScummBar ! 

{{ svm4.png }}

Tout fonctionne très bien ! Appuyez sur F5 pendant le jeu pour accéder au menu. On peut modifier quelques paramètres comme la vitesse d'affichage des sous-titres, sauvegarder, charger une partie.

===== Optimiser ScummVM ====
Exemple avec "//beneath a steel sky//".\\
**Graphisme** : vous pouvez activer différentes sortes de lissages et obtenir un rendu agréable compte tenu de la résolution (de l'époque) de ces jeux.\\
Voici les paramètres que je vous conseille, onglet " Graphique " :

{{:jeux:scummvm:scumm-opt.png?nolink&600|}}

**Son** : Mieux encore, vous pouvez métamorphoser complètement la qualité sonore de la bande son en utilisant des synthétiseurs MIDI et une sound font (SF2) de qualité !\\
__Requis__ : installer les paquets [[:tutoriel:comment_installer_un_paquet|fluidsynth]] et [[:tutoriel:comment_installer_un_paquet|fluid-soundfont-gm]].\\ Onglet audio :

{{:jeux:scummvm:scumm-opt2.png?nolink&600|}}

Onglet MIDI :

{{:jeux:scummvm:scumm-opt3.png?nolink&600|}}

Pour "Beneath a steel sky", cette optimisation est indispensable, la qualité de la bande son du jeu étant très médiocre (faites le test) !\\ 
Étant donné que l'on a changé la soundfont, il se peut que cela change radicalement l'ambiance de certains morceaux, c'est à vous de voir si vous êtes nostalgique ou non du vieux son MIDI !
