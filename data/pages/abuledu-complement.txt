====== Complément ======

**Complément fait partie de la suite de logiciels éducatif du [[le_terrier|Terrier]].**

<note>(Ci-dessous, extrait du site officiel http://www.abuledu.org/leterrier/complement )</note>

{{ http://www.abuledu.org/_media/leterrier/complement/complement_a_10-desordre.png?350}}

Le jeu consiste à effacer une liste de nombres, de l'ordre des unités, des dizaines, des centaines, des dixièmes ou des fractions simples.

Le joueur choisit un niveau, qui fixe la recherche du complémentaire d'un nombre proposé. Le joueur doit alors choisir ce nombre et/ou son complémentaire puis valider ce choix. 

//**Pour une description complète et des pistes pédagogiques, voir [[http://www.abuledu.org/leterrier/complement|la documentation sur le site officiel du Terrier d'Abulédu]].**//

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  * Avoir activé l'accès aux [[le_terrier#ajouts_des_depots|dépôts du Terrier]].
  

===== Installation =====

[[:tutoriel:comment_installer_un_paquet|Installez le paquet]] **[[apt>abuledu-complement|abuledu-complement]]**.


===== Utilisation =====

Comme tous les logiciels du Terrier, vous retrouverez un lanceur dans le menu **Applications --> Éducation**

===== Configuration =====

Comme les autres logiciels du Terrier, //Complément// permet à l'enseignant d'éditer les exercices. Pour cela, vous devez faire partie du groupe "profs". Si le groupe "profs" n'existe pas, créez-le. (Voir [[users-admin#gerer_les_groupes_d_utilisateurs|Gérer les groupes d'utilisateurs]])

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
