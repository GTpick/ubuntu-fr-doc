====== Osmose ======

**Osmose** est un émulateur pour la Sega Master System. 

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  * Avoir activé les [[:depots|dépôts Universe]].
  * Avoir [[:tutoriel:comment_installer_un_paquet|installé]] les bibliothèques **[[:QT]]** (sauf si vous êtes sous Kubuntu) FIXME quelles bibliothèques?

=====Installation=====
  * [[http://bcz.asterope.fr/osmose/Osmose-0-9-96-QT-64bits-bin.zip|Téléchargez]] la dernière version de **Osmose** puis [[:archivage|décompressez l'archive]] .zip dans votre **Dossier Personnel**.

===== Compilation =====

  * Code source : https://github.com/coringao/osmose-emulator
===== Utilisation =====
Dans le dossier décompressé, double-cliquez sur **Osmose-0-9-96-QT**.

Pour charger un fichier, allez dans //File > Open SMS/GG ROM//.

Par défaut Osmose utilise les touches Ctrl, Alt, ainsi que les flèches de direction. Il est possible de modifier leur configuration avec le menu //File > Configure// puis de naviguer dans les onglets.
