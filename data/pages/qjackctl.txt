====== QjackCtl ======

{{tag>audio jack multimedia}}

QjackCtl est une interface graphique pour le serveur de son [[jackd|JACK]].

Très pratique pour corriger visuellement les problèmes de connexions.

{{:audio:qjackctl_fenetre_principale.png?}}

===== Installation =====

QjackCtl est dans les dépôts pour les systèmes tournants sous Debian GNU/Linux, pour l'installer  **[[apt>qjackctll]]**

En console :
<code bash>sudo apt-get install qjackctl</code>
<note>Selon votre gestionnaire de paquet  :  [[apt-get|apt-get]] ou [[aptitude|aptitude]]</note>

===== Configuration =====

<note important>Lors du lancement, l'[[indicator-applications|indicateur d'application]] s'affiche en haut à droite, mais le menu associé n'est pas accessible. Pour contourner ce [[https://bugs.launchpad.net/sni-qt/+bug/970412|bug]], et accéder à la fenêtre de Qjackctl, il faut désactiver l'option **Démarrer minimisé dans la zone de notification système** de l'onglet **Divers**, accessible dans les **Réglages**.</note>
