====== Généalogie ======


===== Introduction =====
Un logiciel de généalogie peut être assimilé à une base de données de personnes (Nom, Prénom, date de naissance, date de décès, etc.) dans laquelle sont établis des liens (parents, cousins, enfants...).

De ces "bases", ces logiciels sont capables de générer des arbres généalogiques, des rapports textuels et graphiques.


===== Logiciels de Généalogie =====
[[:ancestris|Ancestris (anciennement GenjFr)]] : Logiciel open source en java dont le développement est assurée par une communauté française active ;

[[:gramps|Gramps]] : Genealogical Research and Analysis Management Programming System ;

[[:genj|genj]] : un logiciel open source en java.;

[[:geneweb|Geneweb]] : un logiciel de généalogie efficace avec une interface web ;

[[:phpgedview|PHPgedview]] : un logiciel de généalogie pour internet en PHP. Ce logiciel n'est plus développé. Il est remplacé par webtrees. Même principe, mêmes développeurs, forum Anglais/Français

**Heredis** : logiciel de généalogie non-libre et payant (voir [[http://forum.ubuntu-fr.org/viewtopic.php?id=144406|cette discussion]] concernant l'installation Heredis8 sous ubuntu via Wine)

[[http://www.framasoft.net/article4547.html|Lifelines]]

[[http://webtrees.net/fr|webtrees]]

[[http://www.geneamania.net/|Généamania]] : logiciel open source PHP-MySQL multiplate-forme ; forum de support et hébergement de sites.
