====== Wingware Python IDE ======

**WingIDE** est un //environnement de développement professionel// (IDE = integrated development environment) pour programmer en //python//.

**WingIDE** est disponible sous licence professionnelle, personnelle ou éducation (avec des fonctionnalités réduites voir [[http://www.wingware.com/wingide/features]]).

{{ application:wingide.png?800 }}

**Wing IDE** est multiplateforme et est disponible pour GNU/Linux, Windows, Mac OS X.

===== Installation =====

Il n'existe pas de paquet dans les dépôts pour installer **WingIDE**.

Vous trouverez néanmoins sur le site officiel un paquet "deb" (32 et 64 bits) pour installer facilement **WingIde** :

[[http://wingware.com/downloads|Zone de téléchargement officielle]]. Vous y trouverez 3 versions, une version Pro payante, une version personnelle gratuite, et une version débutant gratuite.

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. 

La configuration de l'application sera conservée ou supprimée selon la méthode de désinstallation que vous choisirez.
