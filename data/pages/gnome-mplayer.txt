====== gnome-mplayer ======


**Gnome-mplayer** est une interface simple pour [[:mplayer]] en GTK+. Il est le lecteur vidéo installé par défaut sous [[:Lubuntu]].

{{ :applications:gnome-mplayer_trusty.png?300 }}
===== Installation =====

Si vous n'utilisez pas [[:Lubuntu]] il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>gnome-mplayer]]**.

===== Utilisation =====

==== Ajouter des sous-titres ====

  * Ouvrez votre fichier vidéo.
  * Allez dans **Édition -> SetSubtitle** puis sélectionner un fichier de sous-titres.
  * Choisissez un fichier dont l'extension est .str ou .ssa.

==== Configurer les sous-titres ====

Pour décaler les sous-titres, il faut appuyer, pendant la lecture de la vidéo, sur **x** pour avancer les sous-titres de 100 millisecondes ou sur **z** pour les reculer de 100 millisecondes.

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
