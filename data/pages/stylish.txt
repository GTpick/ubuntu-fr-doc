====== Stylish : Styliser le Web ! ======

Stylish vous permet de restyliser le web. C'est un gestionnaire de style, avec une communauté importante. Également une Extension utile pour les personnes mal voyantes, ou professionnel toujours devant le pc - ou tout simplement pour donner du neuf au web ! 

Donc arrêtez de vous abîmer les yeux et geekez le web ! De plus les écrans foncés ou stylisés noir économisent de l'énergie (sur les écrans cathodiques et OLED).

[[http://userstyles.org|Site officiel pour les scripts (communauté userstyles.org). ]]

Voir aussi l'[[firefox#nosquint_-_zoom_de_la_page|extension firefox NoSquint - pour affecter un zoom sur les sites]].

<note>Il existe "deux" types de scripts :
  * Les scripts pour un site en particulier, exemple google, 
  * Les scripts qui s'appliquent et redessinent tous les sites du web (appelées parfois 'Global Style', la catégorie 'Global Style" est un peu confuse sur ce point)
</note>
