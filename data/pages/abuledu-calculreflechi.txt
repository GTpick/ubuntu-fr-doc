====== Calcul réfléchi ======

{{ :logo_calcul_reflechi.png}}

**Calcul réfléchi fait partie de la suite de logiciels éducatif du [[le_terrier|Terrier]].**

<note>(Ci-dessous, extrait du site officiel http://www.abuledu.org/leterrier/calcul_reflechi )</note>

{{ :capture_calcul_refelechi.png?350}}

    
Le logiciel propose différentes activités de calcul réfléchi, organisées par différents types d'opération et suivant une progression dans les difficultés.

Il comporte un éditeur,

il est paramétrable

et propose deux outils de suivi :

    *      une fiche “bilan synthétique sommatif”

    *      une fiche détaillée pour l’analyse des difficultés.


//**Pour une description complète et des pistes pédagogiques, voir [[http://www.abuledu.org/leterrier/calcul_reflechi|la documentation sur le site officiel du Terrier d'Abulédu]].**//

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  *  ** Avoir activé l'accès aux [[le_terrier#ajouts_des_depots|dépôts du Terrier]].**
  

===== Installation =====

[[:tutoriel:comment_installer_un_paquet|Installez le paquet]] **[[apt://abuledu-calcul-reflechi|abuledu-calcul-reflechi]]**.


===== Lancement =====

Comme tous les logiciels du Terrier, vous retrouverez un lanceur dans le menu **Applications --> Éducation
**

===== Configuration =====

Comme les autres logiciels du Terrier, Aller permet à l'enseignant d'éditer les exercices. Pour cela, vous devez faire partie du groupe "profs". Si le groupe "profs" n'existe pas, créez-le. (Voir [[users-admin#gerer_les_groupes_d_utilisateurs|Gérer les groupes d'utilisateurs]])


===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. La configuration de l'application sera conservée ou supprimée selon la méthode de désinstallation que vous choisirez.
