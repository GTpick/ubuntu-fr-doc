====== Wii Backup Fusion ======
 
Application de gestion et sauvegarde simple des backups Wii : [[http://sourceforge.net/p/wiibafu/home/Wii%20Backup%20Fusion|WiiBaFu - Sourceforge]]
<note importante>
WiiBaFu ne doit être utilisé qu'à des fins de copie privée. Il est, bien sûr, interdit de charger des ISO de jeux dont vous ne possédez pas l'original sur votre disque dur WBFS.
</note>
<note importante>
Tutoriel pour Ubuntu 64 bits (amd64), la version 32 bits nécessite simplement l'installation du paquet disponible sur  [[http://sourceforge.net/projects/wiibafu/files/Wii%20Backup%20Fusion%201.1/wiibafu_1.1-1_i386.deb/download/|sourceforge]]
</note>
===== Pré-requis =====
 
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  * WiiBaFu nécessite l'installation de wiimms-iso-tools [[http://wit.wiimm.de/|wiimms-iso-tools]]
  
  

 
===== Installation de wiimms-iso-tools =====
 
Pour installer ce logiciel, il suffit d'installer le packet "wit-v2.31a-r6005-x86_64.tar.gz"
pour cela télécharger le paquet : [[http://wit.wiimm.de/download/wit-v2.31a-r6005-x86_64.tar.gz|WIT]]
puis l'installer 
<code>wget http://wit.wiimm.de/download/wit-v2.31a-r6005-x86_64.tar.gz
tar -xvzf wit-v2.31a-r6005-x86_64.tar.gz
cd wit-v2.31a-r6005-x86_64
sudo ./install.sh</code>

===== Installation de WiiBaFu =====
 
Pour installer ce logiciel, il suffit d'installer le packet "wiibafu_1.1-1_i386.deb"
pour cela télécharger le paquet : [[http://freefr.dl.sourceforge.net/project/wiibafu/Wii%20Backup%20Fusion%201.1/wiibafu_1.1-1_i386.deb|WiiBaFu]]
puis l'installer 
<code>
wget http://freefr.dl.sourceforge.net/project/wiibafu/Wii%20Backup%20Fusion%201.1/wiibafu_1.1-1_i386.deb
sudo dpkg -i wiibafu_1.1-1_i386.deb</code>
 
===== Configuration =====
 
 * [[:users-admin#modifier_les_proprietes_d_un_groupe_et_ajouter_ou_retirer_des_comptes_d_utilisateurs_d_un_groupe_administrateurs_seulement|ajouter votre nom d'utilisateur]] au groupe << ''**disk**'' >> 
<code>sudo gpasswd -a $USER disk</code>
  * fermez et relancez votre session.
 
===== Utilisation =====
 
Lancez l'application via le [[:unity#tableau_de_bord_dash|dash]] ou via le [[:terminal]]  avec la [[:commande_shell|commande]] suivante :
<code>WiiBaFu</code>
 
===== Désinstallation =====
 
Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
