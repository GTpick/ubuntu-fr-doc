====== EtterCap ======

Ettercap est un logiciel libre d'analyse du réseau informatique. Il est capable d'intercepter le trafic sur un segment réseau, de capturer les mots de passe, et de réaliser des attaques dites de l'homme du milieu (Man In The Middle) contre un certain nombre de protocoles de communication usuels tels que HTTP, FTP et certains protocoles chiffrés.

<note interdite>
<del>Nous vous rappelons qu'il est interdit d'utiliser EtterCap à des fins illégales.</del>
Toute chose illégale est par définition interdite. Les utilisateurs d'EtterCap en sont avisés.
</note>



===== Installation de EtterCap =====

Pour installer EtterCap, il suffit de taper la commande suivante dans le Terminal ou bien de cliquer sur le lien apt :

**EtterCap en version ligne de commande:** [[apt://ettercap|{{apt.png}}]]

       sudo apt-get install ettercap

**EtterCap en version graphique (GTK):** [[apt://ettercap-graphical|{{apt.png}}]]

       sudo apt-get install ettercap-graphical
