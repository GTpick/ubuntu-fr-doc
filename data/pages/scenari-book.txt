====== Scenari-Book ======

Le modèle documentaire Book permet de structurer un contenu éditorial, et de le publier sous forme de livre (format [[OpenOffice.org]]), et de site web. Une publication expérimentale pour [[ebook|livre électronique (liseuse)]] est également disponible au format [[ePub]]. Ce modèle est basé sur la chaîne éditoriale [[Scenari]].

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
   * Avoir [[scenari#activation_des_depots_scenari|activé la liste de dépôts Scenari]]
  * Avoir parcouru la page [[Scenari]] de la documentation

===== Installation =====

==== La ScenariApp Book ====
<note>
La scenari-app présente dans la liste des dépôts est ancienne, choisir la méthode "Scenari + modèle Book" décrite plus bas. Si toutefois, vous souhaitez installer la version Scenari-Book 1.1 lisez la ligne qui suit.

Pour installer ce logiciel, il suffit d'[[:tutoriel:comment_installer_un_paquet#installer_un_paquet_present_dans_les_depots_recommande|installer le paquet]] **[[apt://book1.1|Book1.1]]**.
</note>

==== Scenari + modèle Book====

Il est tout à fait possible d'[[scenari#installation|installer ScenariChain]] puis le modèle Book que vous récupérerez ici [[http://scenari-platform.org/projects/book/fr/pres/co/teleBook.html]].

Vous aurez ainsi la dernière version de Scenari-Book et vous pourrez également utiliser d'[[http://scenari-platform.org/projects/scenari/fr/models/co/|autres modèles de Scenari]].

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
