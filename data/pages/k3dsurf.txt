====== K3DSurf ======

Logiciel de géométrie : Modeleur de surface (peut être utilisé de façon plus ou moins anecdotique)\\
Ce logiciel n'est plus disponible dans les dépôts depuis la version Ubuntu 12.10. 
  * Site Web : [[http://k3dsurf.sourceforge.net/index_fr.html]]
  * Se trouve dans les dépôts universe : [[apt://k3dsurf]] (<12.10)
<note>sous 12.10, on peut [[:tutoriel:comment_installer_un_paquet|installer des paquets]]  debian idoines : http://packages.debian.org/squeeze/amd64/libqt3-mt/download, http://packages.debian.org/squeeze/amd64/k3dsurf/download
  * sudo apt-get install libjpeg62
  * sudo dpkg -i libqt3-mt_3.3.8b-7+b1_amd64.deb
  * sudo dpkg -i k3dsurf_0.6.2.4-1_amd64.deb 

</note>
