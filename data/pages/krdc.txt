====== KRDC, est un client de bureau distant de KDE ======

Le client de bureau distant de KDE (KRDC) est une application cliente permettant d'afficher, voire de contrôler la session du bureau sur une autre machine qui exécute un serveur compatible. Les protocoles VNC et RDP sont pris en charge.

===== Installation =====
KRDC est installé par défaut dans Kubuntu.

Si vous n'utilisez pas [[Kubuntu]], il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt://krdc|krdc]]**

===== Utilisation =====

KRDC peut être lancé depuis le menu K --> Applications --> Internet --> Connexion à un bureau distant (KRDC)
