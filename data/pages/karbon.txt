====== Karbon ======

**Karbon**, anciennement Karbon14, est un logiciel libre de [[wpfr>image_vectorielle|dessin vectoriel]] faisant partie du projet [[:KDE]], en particulier de la suite [[Calligra]] (anciennement KOffice). Son interface est très conviviale, et est hautement configurable. Bien que conçu pour [[KDE]] (il est donc idéal pour [[:Kubuntu]]), il peut aussi être installé dans d'autres [[:environnements|environnements de bureau]].

Son format de fichier est le SVG. Les formats qu'il reconnaît sont les suivants :
  * **Importation :** ODG, SVG, WPG, WMF, EPS/PS ;
  * **Exportation :** ODG, SVG, PNG, PDF, WMF.

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.


===== Installation =====

Pour installer ce logiciel, il suffit d'[[:tutoriel:comment_installer_un_paquet#paquet_present_dans_les_depots|installer les paquets]] **[[apt>karbon,calligra-l10n-fr]]** (le second est pour installer la langue française).


==== Améliorer l'intégration au bureau GNOME ====

Il est possible de faire en sorte que **Karbon** s'intègre mieux à l'environnement d'Ubuntu (thème, polices, icônes). Pour plus de détails, consultez la page [[:integration_qt_gtk#applications_kde|Intégration Qt / GTK]].

===== Utilisation=====

=== Sous Kubuntu ===
Lancez l'application depuis le menu **Kickoff** -> //Applications -> Graphisme -> Karbon// ou //via// la [[:commande_shell|commande]] suivante :

  karbon

=== Sous Ubuntu ===
Lancez l'application en faisant une recherche dans [[unity#tableau_de_bord_dash|le tableau de bord Unity]] avec le mot clé **Karbon**.

 

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, le configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
