====== Gnome-voice-control ======

<note warning>Il y a de nombreux bugs dans ce programme, sa fiabilité laisse douter.</note>

**gnome-voice-control** est un logiciel de [[reconnaissance vocale]], permettant d'agir sur l'environnement [[GNOME]] par la voix (en anglais pour le moment).

===== Installation =====
Voir les instructions [[https://wiki.gnome.org/Projects/GnomeVoiceControl|ici]].
=====Configuration=====

  * **Ajoutez l'applet sur votre tableau de bord :**
    * Clic droit sur le tableau de bord
    * Ajouter au tableau de bord
    * Choisir **Voice-Control**
    * Ajouter
  * Une icône est rajoutée sur votre tableau de bord.
  * Vous pouvez commencer à commander votre PC à la voix !


===== Utilisation =====

Actions possibles :

  * FILE .. OPEN
  * FILE .. PRINT
  * FILE .. NEW
  * SELECT ALL
  * DELETE
  * UNDO
  * REDO
  * HELP .. ABOUT
  * RUN TERMINAL
  * RUN BROWSER
  * RUN MAIL
  * MAXIMIZE WINDOW
  * MINIMIZE WINDOW
  * CLOSE WINDOW
  * NEXT WINDOW
  * RUN TEXT EDITOR
