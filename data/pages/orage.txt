====== Orage ======
**Orage** est la nouvelle application calendrier moderne pour l'environnement de bureau [[:Xfce]].
Son interface est simple et intuitive. Il vous permet d'enregistrer des évènements et d'avoir une alarme visuelle et/ou audible à l'approche de ces derniers.

{{http://pix.toile-libre.org/upload/original/1352639737.png?recache}}

Orage inclut également un plugin pour s'intégrer totalement au panneau Xfce :

{{http://pix.toile-libre.org/upload/img/1352626166.png?recache}}
===== Installation =====

**Orage** est installé par défaut sur toutes les versions actuellement supportées de Xubuntu.
Vous pouvez l'[[:tutoriel:comment_installer_un_paquet|installer]] sur un autre environnement de bureau via le paquet [[apt://orage|orage]].

===== Utilisation =====

L'utilisation d'Orage est intuitive. Vous pouvez cependant consulter la [[http://www.xfce.org/projects/orage/?lang=fr|documentation Xfce officielle]] [fr/en] ainsi que le [[http://www.kolumbus.fi/~w408237/orage/index.html|site officiel d'Orage]] [en] pour plus d'informations sur son fonctionnement.

===== Désinstallation =====

Si vous utilisez Xubuntu, il est préférable de laisser Orage en place vu qu'il est installé par défaut.
Pour les autres variantes, il suffit de supprimer le paquet << orage >>.
