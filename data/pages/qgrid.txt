====== QGRID : Calcul de distances/orientation entre 2 localisations ======

===== Présentation =====
Permet de calculer l'orientation et la distance entre deux coordonnées.

===== Installation =====
  * Installer le nécessaire à la compilation : 

  sudo apt-get install qt4-qmake libqt4-dev build-essential

  * Télécharger et décompresser les sources :

  wget http://users.telenet.be/on4qz/qgrid/downloads/qgrid_3.0.0.tgz
  tar -xzvf qgrid_3.0.0.tgz
  rm qgrid_3.0.0.tgz

  * Configurer, compiler et installer :

  cd qgrid_3.0.0
  qmake
  make
  sudo make install
  cd /
  sudo rm -R qgrid_3.0.0
