====== Nuxeo EP ======

Nuxeo EP (Enterprise Platform) est une solution serveur d'Enterprise Content Management (ECM) déployée sur un serveur d'application Java EE 5.
Nuxeo EP est conçu pour être utilisé à partir d’un navigateur Web standard, avec une interface basée sur les technologies Ajax. Il est également possible de l’utiliser depuis une suite bureautique

Dans sa configuration par défaut, Nuxeo EP fournit à la fois des fonctions de travail collaboratif et de publication.
