======IBP : Prévision de l'émission des balises HF de l'International Beacon Project======

===== Présentation =====

L'//**International Beacon Project**// est un réseau de balises couvrant l'ensemble de la planète émettant successivement des signaux aux fréquences de 14,100   18,110   21,150   24,930  et  28,200  MHz.

Il vous est alors possible de mesurer l'intensité de ces signaux afin de savoir sur quelle fréquence il sera préférable de communiquer en fonction des conditions atmosphériques et solaires. 

Le logiciel **IBP** vous permettra de suivre en direct sur une carte les lieux, instants et fréquences d'émission de chaque balise.

===== Installation =====
La compilation du logiciel à partir des sources peut se faire de 2 manières, avec ou sans interface graphique.

  * Dans les 2 cas, commencer par télécharger et décompresser le code source :

  wget http://wwwhome.cs.utwente.nl/~ptdeboer/ham/ibp-0.21.tgz
  tar -zxvf ibp-0.21.tgz
  rm -ibp-0.21.tgz

====Version en mode texte dans le terminal====
{{ http://img15.hostingpics.net/pics/558762Capturedu20120521165229.png?400 |}}

  cd ibp-0.21
  make
  sudo cp ibp /usr/local/bin/
  cd ..
  sudo rm -R ibp-0.21

====Version avec interface graphique====
{{ http://img15.hostingpics.net/pics/430770Capturedu20120521165251.png?400 |}}
  sudo apt-get install xutils-dev
  cd ibp-0.21
  xmkmf
  make
  sudo cp ibp /usr/local/bin/
  cd ..
  sudo rm -R ibp-0.21
