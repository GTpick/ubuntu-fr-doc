====== S.C.O.U.R.G.E : Heroes of Lesser Renown ======

[[http://scourgeweb.org/|Scourge]] est un RPG en 3D, assez proche dans sa réalisation et son esprit du jeu Neverwinter Nights premier du nom. 

{{:application:scourge.jpg|}}

=====Pré-requis=====

[[:tutoriel:comment_installer_un_paquet|Installez les paquets]] **[[apt://libfreetype6-dev,libgl1-mesa-dev,libglu1-mesa-dev,libsdl1.2-dev,libsdl-image1.2-dev,libsdl-mixer1.2-dev,libxmu-dev,subversion,libwxgtk2.6-dev,libsdl-ttf2.0-dev,libsdl-net1.2-dev,scons,build-essential,autoconf,libtool|libfreetype6-dev libgl1-mesa-dev libglu1-mesa-dev libsdl1.2-dev libsdl-image1.2-dev libsdl-mixer1.2-dev libxmu-dev subversion libwxgtk2.6-dev libsdl-ttf2.0-dev libsdl-net1.2-dev scons build-essential autoconf libtool]]**.

===== Installation =====

Télécharger le jeu sur le [[http://scourgeweb.org/|Site officiel]]

Pour être sur d'avoir la dernière version, le plus simple reste d'utiliser SVN :

<code>
svn co https://svn.code.sf.net/p/scourge/code/trunk scourge
</code>

Les derniers fichiers sources vont ainsi être téléchargés dans votre répertoire home\scourge, on continue, toujours dans le même terminal :

<code>
cd scourge/scourge
</code>

<code>
autoreconf -i
./translate.sh
./configure
make
</code>


===== Lancement =====

Ouvrez Raccourcis->Dossier perso->scourge->scourge->scr-> et double clique sur scourge.
