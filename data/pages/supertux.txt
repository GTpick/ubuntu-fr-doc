====== Supertux ======
{{ :supertuxlogo.png?200 |}}

{{ supertux.png?300}}
**Supertux** est un clone de Super Mario.

Vous devez finir les niveaux et collecter les pièces. Dans chaque niveau, il peut y avoir un ou plusieurs passages secrets (contenant des pièces le plus souvent). Les pièces sont votre assurance vie ; à chaque fois que vous mourrez vous en perdez et dès lors que vous n'en avez plus le jeu est terminé.

===== Pré-requis =====
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation et utilisation =====
Deux versions du jeu sont disponibles dans les dépôts : la version 0.1.3 (aussi nommée //Milestone 1//), et la version 0.3.3 (aussi nommée //Milestone 2// ou encore //Supertux 2//).

Le jeu est aussi présent dans le dépôt de [[djl|DJL]].

Sous [[xenial]], il faut [[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt://supertux|supertux]]**. 

==== Version 0.1.3 (Milestone 1) pour ubuntu <= 16.04 ====
Pour avoir une version stable, il faut [[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt://supertux-stable|supertux-stable]]**. Cette version est très ancienne (arrêt de développement en 2004).

Lancez l'application via le dash (Ubuntu 11.04 et ultérieures) ou via le terminal (toutes versions d'Ubuntu) avec la commande suivante : ''supertux''
==== Version >= 0.3.3 (Milestone 2) ====
=== Jeu de base ===

Cette version contient plus de niveaux (environ 2 fois plus, 4 fois si on ajoute les addons), 2 fois plus d'ennemis, et des nouveaux mondes ((comme la forêt, qui est inexistante dans la version 0.1.3 (Milestone 1) )), dispose d'une histoire, et est traduite en français ((Lors du premier démarrage du jeu, aller dans **options** puis **language**)). En revanche, elle est plus gourmande en ressource (cela n'est handicapant que sur les très vieux ordinateurs).

Le monde __Antarctique__ est accessible via le menu jouer. Les autres mondes (y compris le monde __Forêt__) sont accessibles via le menu //Contributions//((Add-Ons, en version anglaise)).

Il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt://supertux|supertux]]**.
Lancez l'application via le dash (Ubuntu 11.04 et ultérieures) ou via le terminal (toutes versions d'Ubuntu) avec la commande suivante : ''supertux2''

=== Add-Ons ===
De nombreuses personnes utilisent l'éditeur de niveau, et leurs mondes et niveaux peuvent être ajoutés sous la forme d'une archive dans le dossier //.supertux2// (dossier visible dans votre Dossier Personnel après avoir [[:fichier_cache|afficher les fichiers cachés]]).

L'archive est disponible ici : [[http://www.mediafire.com/?k47v61nz6i230|ici, les Add-Ons parus avant juin 2013]]((http://www.mediafire.com/?k47v61nz6i230)) (si le lien change : [[http://supertux.lethargik.org/wiki/Levels#Add-on_Collection_for_0.3.3_and_0.3.3-SVN.2FGIT|ici]]((http://supertux.lethargik.org/wiki/Levels#Add-on_Collection_for_0.3.3_and_0.3.3-SVN.2FGIT)), et si même ce lien ne fonctionne pas, [[http://supertux.lethargik.org/wiki/Levels|ici]]((http://supertux.lethargik.org/wiki/Levels)) ).

Il n'est pas nécessaire de décompresser l'archive.
Descriptions des Add-Ons ici : [[http://supertux.lethargik.org/wiki/Add-ons]]

<note important>Si le jeu crash alors que vous utilisez un Add-Ons, ce n'est pas grave : relancez le jeu ( seul votre progression dans le niveau est perdu). C'est le cas lorsque il y a un bug dans le niveau qui n'a pas été détecté (ou corrigé). C'est pourquoi il vaut mieux remettre à jour souvent les Add-Ons.</note>

==== Installation de l'éditeur de niveau pour Milestone 1 (0.1.3) ====
L'éditeur est naturellement présent dans la version supertux 0.1.3 et est accessible via le menu principal du jeu.
Les niveaux ainsi créés son accessibles via le menu //Bonus levels//.

==== Installation de l'éditeur de niveau pour Milestone 2 (0.3.3) (par archive) ====

=== Pré-requis ===
Il faut tout d'abord [[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>mono-runtime]]**, et avoir [[:tutoriel:comment_installer_un_paquet|Installer le paquet]] **[[apt>supertux]]**

=== Installation ===
Télécharger le paquet "Editor_r5932_linux32" sur le site: [[http://elektromaniak.wz.cz/SuperTux/Editor_r5932_linux32.tar.gz|http://elektromaniak.wz.cz/SuperTux]], et [[:archivage#graphiquement|extraire l'archive]] dans votre Dossier Personnel.

=== Lancement ===
Lancer l'éditeur à partir du terminal:
<code>sudo mono --runtime=v4.0 ./Editor_r5932_linux32/supertux-editor.exe</code>
{{ :applications:alacarte_lucid_04.png?300|}}
Pour [[:raccourci-lanceur|créer un raccourci]] sur le bureau, il faut faire //alt// + //F2// et écrire <code>gnome-desktop-item-edit --create-new ~/Bureau</code> Et valider.

Dans la fenêtre qui s'ouvre, choisir
   * //Application dans un terminal// dans la rubrique **Type** ;
   * //''sudo mono --runtime=v4.0 ./Editor_r5932_linux32/supertux-editor.exe''//dans **commande**
   * Le **nom** : ce que vous voulez.
   * Le **commentaire** : inutile
   * L'**image** : Prenez celle disponible dans ~((Votre dossier personnel))/Editor_r5932_linux32/supertux-editor.png
