====== GWorldClock ======

GWorldClock est un logiciel libre qui vous permet de connaître l'heure actuelle dans différentes villes du globe. Vous avez ainsi en un coup d'oeil l'heure locale de plusieurs fuseaux horaires.

{{:applications:gworldclock_trusty.png}}

===== Installation =====

Pour installer ce logiciel, il suffit d'[[:tutoriel:comment_installer_un_paquet#paquet_present_dans_les_depots|installer le paquet]] **[[apt>gworldclock]]**.


===== Utilisation=====
Pour lancer l'application faites une recherche dans [[unity#tableau_de_bord_dash|le tableau de bord Unity]] avec le mot clé **GWorldClock**.
Quand vous lancez GWorldClock, l'heure courante s'affiche dans tous les fuseaux horaires sélectionnés.

Pour voir une autre heure que l'heure actuelle, cochez l'option //Options -> Rendezvous// et réglez la date et l'heure de votre choix.
<note important>Il peut arriver que la fenêtre soit réduite à un trait lors de son ouverture. Si tel est le cas il vous suffit avec le curseur de l'agrandir.</note> 

===== Configuration =====

Dans le menu //Options -> Add Timezone//, choisissez d'abord un continent, puis un pays, puis une ville. Cliquez sur //Add Zone//, et recommencez pour chaque ville qui vous intéresse.


===== Pour aller plus loin... =====

Votre liste de ville est enregistrée dans //~/.tzlist//.\\
L'application TZWatch fait la même chose en ligne de commande.


===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
