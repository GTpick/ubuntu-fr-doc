====== Tags des anciennes versions d'Ubuntu ======

<note tip>Cette page est une sous partie de la page [[:wiki/chantiers|des chantiers du wiki]].</note>

Cette page liste les pages de la documentation qui contiennent un ou des tags des anciennes [[versions|versions d'Ubuntu]]. Selon l'ancienneté de ces tags, il faut faire une vérification plus ou moins approfondie du contenu de la page avant de les remplacer par les tags des [[:versions#Versions d'Ubuntu activement soutenues|versions d'Ubuntu activement soutenues]]. **Il ne faut pas simplement remplacer les tags qui ne sont plus utilisés par ceux qui le sont, mais vérifier que le contenu de la page correspond au(x) tag(s) que l'on ajoute.**

Le principe de la démarche est le suivant :
  * Aller sur une page qui contient des **tags qui ne sont plus utilisés : [[:warty]] [[:hoary]] [[:breezy]] [[:dapper]] [[:edgy]] [[:feisty]] [[:gutsy]] [[:intrepid]] [[:jaunty]] [[:karmic]] [[:maverick]]**.
  * Vérifier que le contenu de la page est d'actualité (si ce n'est pas le cas, il faut le mettre à jour). Et si la page n'a plus lieu d'être, il faut la supprimer et retirer tous les liens vers cette page dans le reste du wiki.
  * Une fois que la page est d'actualité, ajouter les **tags qui sont utilisés : [[:xenial]] [[:trusty]] [[:precise]]**.
  * Retirer les tags qui ne sont plus utilisés.

Pour les pages qui ne sont pas à jour, leur ajouter le tag [[:vétuste]] et suivre la [[:vetuste|démarche pour les mettre à jour]].

Pour plus d'informations sur les tags en général, voir la page « [[:wiki:tags|Les règles et la syntaxe à respecter pour les tags]] ».

Vous trouverez ci-dessous une liste des pages qui contiennent des tags des [[:versions|versions d'Ubuntu]] qui ne sont plus utilisés sur cette documentation. Elles sont classées en différentes catégories, selon leur ancienneté.

===== Pages très anciennes qui n'ont plus de tags de versions supportées =====

<note important>**Liste des pages qui n'ont que des tags de versions qui ne sont plus supportées.**
  * Ces pages nécessitent une **vérification approfondie**.
  * Si la page n'a plus lieu d'être il faut :
    * la supprimer.
    * retirer tous les liens vers cette page dans le reste du wiki.
  * Si la page est encore nécessaire il faut :
    * retirer les tags des versions d'Ubuntu non supportées.
    * vérifier que les informations dessus sont encore valides pour les versions supportées.
    * mettre les tags des versions d'Ubuntu compatibles.
</note>
