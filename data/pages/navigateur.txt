==== Utilisant le moteur de rendu Gecko ====

  * [[Firefox]] : navigateur web libre de référence.
    * [[Iceweasel]] : version Debian de Firefox sans composant non-libre (logo, etc). Depuis peu, Debian utilise la version originale de Firefox et non Iceweasel.
  * [[tor|Tor browser]] : un navigateur basé sur Firefox qui protège la vie privée et et permet de surfer anonymement sous certaines conditions. Il permet d'accéder au réseau anonyme TOR. 
  * Suites logicielles :
    * [[Seamonkey]] : contient un client irc, un client mail, et un créateur de page web HTML.
    * [[http://packages.debian.org/sid/iceape|IceApe]] : version Debian de Seamonkey sans composant non-libre (logo, etc).

==== Utilisant le moteur de rendu KHTML ====
  * [[Konqueror]] : Navigateur web et gestionnaire de fichiers pour KDE.

==== Utilisant le moteur de rendu Webkit ====
  * Pour Unity :
    * [[Webbrowser-app]] navigateur développé par [[Canonical]] pour la convergence //desktop//, //smartphone// et tablette. Il est notamment le navigateur sur la tablette BQ Aquaris M10 avec Ubuntu.
  * Pour Gnome :
    * [[epiphany|Gnome Web]] : Navigateur web du projet Gnome.
  * Pour Xfce :
    * [[Midori]] : Navigateur ultra-léger récemment intégré au projet Xfce. C'est le navigateur par default dans Elementary OS.
  * Pour KDE :
    * [[Rekonq]] : Navigateur ultra-léger basé sur Qt et très intégré à [[KDE]]. 
    * [[arora]] : Navigateur ultra-léger basé sur Qt. **PROJET ABANDONNÉ EN 2010**, il est préférable d'utiliser Rekonq qui propose plus ou moins les mêmes fonctionnalités.
  * Pour Enlightnment
    * [[Eve]]: navigateur web en cours de développement pour e17, basé sur les EFL et Webkit.  
  * [[http://www.qupzilla.com|QupZilla]] : navigateur basé sur QtWebEngine (qui remplace QtWebKit). Ce serait le plus léger des navigateurs d'après [[http://www.makeuseof.com/tag/using-best-web-browser-linux|MakeUseOf.com]]

==== Utilisant le moteur de rendu Blink ====
  * [[chromium-browser|Chromium]] : Navigateur libre de Google servant de base à Google Chrome. 
  * [[:Google Chrome]]. Navigateur non libre de Google basé sur Chromium.
  * [[Iron]] : Navigateur basé sur Chromium mais avec (supposément, voir sur sa page) une meilleure confidentialité des données personnelles.
  * [[Opera]] : Un navigateur gratuit mais non-libre qui est basé sur Chromium.
  * [[Vivaldi]] : navigateur non libre et multi-plateforme basé sur Chromium qui a pour but d'implémenter les fonctionnalités avancées d'Opera 12. Il utilise cependant les extensions du Chrome Web Store. De fait, Vivaldi est un navigateur web qui fait la transition entre Opera et Chromium/Google Chrome. Le détail avantageux est dans le fait que ce navigateur à été développé "Pour nos amis", donc sans le pistage de navigateurs comme Google Chrome. Plus de détail sur 
    * [[https://vivaldi.com/|Site Officiel de Vivaldi]]
    * [[https://fr.wikipedia.org/wiki/Vivaldi_%28navigateur_web%29|Vivaldi sur Wikipedia]]
    * [[http://quebecos.com/?p=3470|Article Vivaldi sur Quebec OS]]
  * [[http://www.qtweb.net/|QtWeb]] un navigateur libre multiplate-forme. 
  * [[http://www.slimjet.com/fr/|SlimJet]] un autre navigateur basé sur Chromium  [[http://www.slimjet.com/fr/dlpage.php|download slimjet]]
  * [[https://iridiumbrowser.de/|iridium]] un autre navigateur basé sur Chromium, qui se veut protecteur de votre vie privé ("A Browser securing your privacy. That’s it.")

==== Utilisant d'autres moteurs de rendu (non-libre) ====
  * [[ie4linux|Internet Explorer]] : Navigateur par défaut de Microsoft pour Windows jusqu'à Windows 8.1. Installable via PlayOnLinux jusqu'a la version IE8.
   
===== Les navigateurs web légers =====
  * [[Dillo]] : Un navigateur extrêmement léger (utilise moins d'un Mo de RAM)
  * [[http://www.netsurf-browser.org/|netsurf-gtk]] : [[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>netsurf-gtk|netsurf-gtk]]**.
  * [[uzbl]] : graphique, alliant lignes de commande et rapidité.
  * [[http://www.qupzilla.com|QupZilla]] : navigateur basé sur QtWebEngine (qui remplace QtWebKit). Ce serait le plus léger des navigateurs d'après [[http://www.makeuseof.com/tag/using-best-web-browser-linux|MakeUseOf.com]]
===== Les navigateurs web en ligne de commandes =====
  * [[elinks]] :  simple et complet.
  * [[lynx]] : pas trop compliqué.
  * [[w3m]] : plutôt complexe.
  * [[links2]] : mode permettant d'afficher les images, les couleurs, etc.
  * [[jumanji]] : basé sur Webkit, il se contrôle grâce à des raccourcis clavier similaire à ceux de [[VIM]].
