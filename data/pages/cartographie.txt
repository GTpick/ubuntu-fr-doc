===== Fond cartographique sous licence de libre diffusion =====
==== Open Street Map ====
[[OpenStreetMap]] est un projet fondé en 2004 destiné à réaliser une carte du monde, sous licence « libre » . Cela signifie, que tout le monde est libre d'utiliser, d'améliorer, de distribuer ou d'effectuer des travaux dérivés à partir des cartes, donnant ainsi une liberté bien plus grande que les solutions telles que Google Maps.

==== SCAN1000® ====
Les fichiers SCAN1000® appartiennent à l'IGN.     * SCAN 1000® couvre la France métropolitaine en 5 dalles et de nombreuses informations topographiques pour des usages à petites échelles. Formats TIFF - ECW ; Projections disponibles : Lambert-93 ; Caractéristiques techniques ; Résolution au sol : 100 m

Licence : Les fichiers SCAN1000® peuvent librement être téléchargés depuis notre site web http://www.ign.fr. Tous les usages de ces fichiers sont autorisés, à l'exclusion de toute exploitation commerciale. Pour l'exploitation commerciale des fichiers, une autorisation préalable doit être demandée auprès du réseau commercial de l'IGN. Toute reproduction doit être accompagnée de la mention IGN-"nom du produit" et toute représentation sur un site Internet doit être accompagné de la même mention associée à un lien vers l'adresse www.ign.fr.

Télécharger SCAN1000 au format ECW.  : http://jrepetto.free.fr/ttmaps/files/IGN_SCAN1000.zip
Télécharger SCAN1000 au format TIFF : http://professionnels.ign.fr/documentDossiers.do?idDoc=5327966&siteId=5059300

===== Accéder à d'autres fonds cartographiques  =====
==== Gmapcatcher ====
GMapCatcher est une visionneuse de cartes en offline qui permet de télécharger en masse les dalles depuis certains fournisseurs tels que CloudMade, OpenStreetMap, Yahoo Maps, Bing Maps, Google Map.

http://code.google.com/p/gmapcatcher/

==== Géoportail ====
Géoportail est un logiciel permettant l'utilisation offline des cartes du site du même nom

http://code.google.com/p/geoportail/

<note important>Ce logiciel ne fonctionne plus avec la dernière version du géoportail. On peut utiliser à la place GMapCatcher avec le patch disponible ici : [[http://ttmaps.free.fr/forums/viewtopic.php?f=28&t=568#p3463]].</note>
===== Logiciels de cartographie sous GNU/Linux =====
==== Les logiciels de cartographie natifs ====

Vous trouverez :
  * [[http://abc-map.fr|Abc-Map - http://abc-map.fr]]: un logiciel de cartographie créé dans un objectif de simplicité. Il permet d'importer un fond de carte, de le compléter et de le mettre en page en quelques minutes. Des tutoriels vidéos expliquent son fonctionnement basique. 
  *  [[:Emerillon]] ;
  * [[https://launchpad.net/gottengeography|GottenGeography]] : un logiciel qui permet de géolocaliser manuellement des images (un tuto d'installation [[http://la-vache-libre.org/gottengeography-ajoutez-manuellement-des-geotags-a-vos-images-ppa/|ici]]).
  * Le projet [[http://www.gpsdrive.de/|Gpsdrive]] est un logiciel de navigation GPS ; les cartes sont librement téléchargeables via le logiciel. La synchronisation avec le GPS se fait avec GPSD ;
  * [[gpsman|Gpsman]], un logiciel qui gère les GPS, offre les principales fonctions mais ne fournit aucune carte, il faut donc les scanner auparavant ;
  * [[marble|Marble]], cartogrophie, itinéraires routiers.
  * [[:Navit]] ;
  * Le projet [[OpenStreetMap]] cartographie le monde librement et est compatible via un utilitaire avec les GPS de marque Garmin ; [[josm]] est un des éditeurs existants pour OpenStreetMap ;
  * [[http://wiki.openstreetmap.org/index.php/Pyroute|pyroute]] ;
  * [[qlandkarte|QLandkarte GT]] est un logiciel complet de gestion de carte et GPS. Il est en cours de remplacement par [[https://bitbucket.org/maproom/qmapshack/wiki/Home|QMapShack]]
  * [[http://www.tangogps.org/gps/cat/News|TangoGPS]] ;
  * [[http://jrepetto.free.fr/ttmaps/|Ttmaps]] : Affichage de cartes sur les GPS TomTom ;
  * [[viking|Viking]] : un logiciel qui permet entre autres d'importer des cartes depuis OpenStreetMap et Terraserver et de dialoguer avec un GPS ;

 

==== Les logiciels de SIG (Systèmes d'Information Géographique) ====

Si pendant longtemps, les logiciels libres en SIG (GIS en anglais) étaient peu nombreux, ils se sont multipliés au cours de ces dernières années. En voici une liste des principaux :

  * [[http://grass.osgeo.org|GRASS]]: le premier logiciel libre de SIG, il est puissant mais peu convivial ;
  * [[http://www.qgis.org|QGIS]], ou Quantum GIS de son nom complet : logiciel libre,  il permet d'utiliser notamment les librairies de [[GRASS]] avec une interface plus simple et conviviale ;
  * [[http://www.gvsig.org|gvSIG]], un logiciel de SIG en Java ; (depuis Ubuntu 8.10, l'installation pose problème, il faut lancer manuellement la commande<code> ./gvSIG-1_11-1305-final-lin-i586-withjre-j1_5.bin</code>  une fois installé, pour lancer le programme : <code>"chemin_vers_le_repertoire"/gvSIG_1.11.0_final/bin/gvSIG.sh</code>
  * [[http://www.openjump.org|OpenJUMP]] : également en Java, le précurseur de gvSIG  (voir sa [[http://www.framasoft.net/article4339.html|fiche Framasoft]] pour plus de détails) ;
  * [[http://www.saga-gis.org/en/index.html|SAGA GIS]] : orienté création et analyse de modèle numérique de terrain (voir sa [[http://www.framasoft.net/article4296.html|fiche Framasoft]] pour plus de détails).



==== Les logiciels pour connecter un GPS ====

  * Le projet libre **[[http://www.gpsbabel.org/|gpsbabel]]** permet de communiquer avec son GPS en ligne de commande.
  * Le pGMapCatcher est une visionneuse déconnecté cartes. Il télécharge automatiquement les tuiles de nombreux fournisseurs tels que:

CloudMade, OpenStreetMap, Yahoo Maps, Bing Maps, Google Map.

Il les affiche en utilisant une interface graphique personnalisée. L'utilisateur peut afficher les cartes en mode hors connexion. GMapCatcher ne dépend pas de JavaScript de la carte il doit donc travailler même si elles les modifier. Il fournit également un outil de téléchargement.rojet **[[http://gpsd.berlios.de/|gpsd]]** permet de faire communiquer son GPS avec son PC en usb ou sur port série.
  * Le logiciel libre **[[http://activityworkshop.net/software/prune/index.html|Prune]]**, développé en Java, permet la visualisation, l'édition et la conversion de données GPS.

===== Solutions reposant sur des logiciels Windows =====
==== Solutions avec Wine ====

En installant **[[:Wine]]** il est possible d'installer des logiciels propriétaires conçus pour Windows comme :
=== Carto-Exploreur ===

[[http://www.bayo.com/cartographie/index.php?prod=1|Site de l'éditeur]]

- La version **1** du logiciel est composée d'un CD-rom. L'installation se déroule comme sous Windows(tm). La configuration Wine utilisée était **Windows98 ou windows XP**.

- La version **3** (version 3.06 testée) est composée de 2 CD-roms, un pour l'installation du logiciel et un pour les données. L'installation complète du CD1 ne pose pas de soucis, mais le premier lancement avec le CD2 inséré provoque des erreurs, [[http://forum.ubuntu-fr.org/viewtopic.php?id=89600|plus de détails ici]]. La configuration Wine utilisée était **Windows XP**.

<note important>Carto-exploreur 1 fonctionne normalement mais n'imprime pas correctement les fonds de carte 1/25.000

Carto-exploreur 3 semble poser plus de soucis de fonctionnement, [[http://forum.ubuntu-fr.org/viewtopic.php?id=89600|plus de détails ici]]</note>

Voir [[http://forum.ubuntu-fr.org/viewtopic.php?id=89600|carto-exploreur 3 sur le forum ubuntu]].

=== Géorando ===
**Géorando** (anciennement IGNrando) est propriétaire et développé par l'IGN.

Il existe une version DVD et une version téléchargeable. Cette dernière fonctionne correctement sous WineHQ, le test est disponible [[https://appdb.winehq.org/objectManager.php?sClass=application&iId=11094|ici]].

[[http://www.georando.com/|Site de l'éditeur]]

=== CompeGPS ===
Ce logiciel s'installe et fonctionne correctement (version 6.51 testée). Il est décliné en plusieurs variantes spécialisées (Air, Land)
Les fonds de carte ne sont pas fournis, mais c'est un des logiciels acceptant le plus de formats.

[[http://www.compegps.com|Site de l'éditeur]]
