====== SIGIL ======

SIGIL est un éditeur WYSIWYG de fichiers au format [[:ePub]] (format de prédilection pour les liseuses de livres électroniques). Sigil est open-source et multi-plateformes. Il existe d'autres outils libres tel que [[:Scenari-Book]].

Cet éditeur est encore en plein développement (les nouvelles versions sortent rapidement et régulièrement) tout en étant parfaitement utilisable en production, mais en voici les principales spécifications :
    * Support intégral Unicode. Format UTF-16 utilisé dans Sigil
    * Support complet [[:ePub]] {{ http://imgur.com/jzWyI.png?400}}
    * Édition WYSIWYG ➙ visuel instantané du résultat voulu
    * Vues multiples : Livre, Code, Divisée
    * Éditeur de métadonnées avec support intégral de plus de 200 entrées
    * Éditeur de table des matières
    * Support TOC sur plusieurs niveaux
    * La vue “Livre” supporte entièrement l’affichage de n’importe quel document XHTML sous OPS.
    * Support SVG
    * Support XPGT basique
    * Conversion avancée, automatique de n’importe quel document importé, au format Unicode.
    * Formats d’import supportés actuellement : TXT, HTML, [[:ePub]]
    * Formats d’export supportés actuellement : [[:ePub]] et SGF (format natif de Sigil)
    * Nettoyeur HTML intégré, tous les documents importés sont minutieusement analysés et corrigés dans le code, peu importe le mode de vue
    * Une interface visuelle agréable à utiliser
    * Application C++ native

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  * Savoir utiliser le [[:terminal]] pour une installation sans paquet (il n'existe pas encore de paquet .deb pour l'installation et aucune liste de dépôt ne propose encore Sigil)

===== Installation =====
À partir d'[[xenial|Ubuntu 16.04]] il suffit d'[[:tutoriel:comment_installer_un_paquet|Installez le paquet]] **[[apt>sigil]]**.\\

Avant [[xenial|Ubuntu 16.04]] [[:PPA|ajoutez le PPA]] suivant dans vos sources de logiciel :
  * **ppa:sunab/sigil-release**((https://launchpad.net/~sunab/+archive/ubuntu/sigil-release)) pour la version stable actuelle, testé sous Ubuntu 14.04 
Puis :
  * [[:tutoriel:comment_modifier_sources_maj#recharger_la_liste_des_paquets|Rechargez la liste des paquets]] ; update
  * [[:tutoriel:comment_installer_un_paquet|Installez le paquet]] **[[apt>sigil|sigil]]**.

Il est possible de télécharger SIGIL directement sur GITHUB
Plus de détail sur la méthode d'installation : https://github.com/Sigil-Ebook/Sigil/blob/master/docs/Building_on_Linux.md
sudo git clone https://github.com/Sigil-Ebook/Sigil.git

<note help>Pour linuxmint 17.3, sudo add-apt-repository ppa:i2p.packages/i2p (le dépôt sunab n'étant pas compatible avec [[Trusty]])</note>
==== Avec Wine ====
En ayant [[:Wine]] installé, il est possible d'utiliser la dernière version de Sigil. Je viens de tester avec Saucy, ça fonctionne parfaitement. 
===== Contribution =====

Le logiciel est nativement en anglais et il existe désormais une traduction française traduction (version 0.9.5). 

===== Utilisation=====

Un icône devrait apparaître sur votre bureau (un gros "S") mais vous pouvez également lancez l'application depuis le menu //Applications -> Sigil -> Sigil// ou //via// la commande suivante :

  /opt/sigil/sigil.sh

Sur la dernière version, la commande est: /opt/sigil/sigil

(au lieu de sigil.sh)
===== Utilisation des fichiers ePub générés =====

Vous pouvez utiliser votre liseuse d'[[:ebook]] habituelle ou bien un logiciel de lecture de fichiers au format [[:ePub]] comme :
  * [[:Calibre]] qui possède d'autres fonctionnalités intéressantes (bibliothèque, conversions de fichiers dont la conversion vers le format [[:ePub]], connexion avec votre liseuse...)
  * [[:FBReader]] qui possède également d'autres fonctionnalités intéressantes (bibliothèque...)
  * Sigil lui-même qui lit et édite ces fichiers
