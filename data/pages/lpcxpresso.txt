====== LPCXPRESSO======


LPCXPRESSO est un environnement de développement basé sur Eclipse pour les microcontrôleurs LPC de NXP (Philips).

La licence gratuite permet la programmation d'au maximum 256 ko de mémoire flash.

Les sondes supportées dans la version 7.3 sont:
  * LPC-Link
  * LPC-Link 2
  * Segger JLink
  * CMSIS-DAP

Testé sous Ubuntu 14.04. 64bits

===== Pré-requis =====

Ubuntu 64 bits

* Pour les versions d'Ubuntu supérieures à 13.10
[[:tutoriel:comment_installer_un_paquet|Installer un paquet]] **[[apt>libgtk2.0-0:i386 libxtst6:i386 libpangox-1.0-0:i386 libpangoxft-1.0-0:i386 libidn11:i386 libglu1-mesa:i386 libncurses5:i386 libudev1:i386 libusb-1.0:i386 libusb-0.1:i386 gtk2-engines-murrine:i386 libnss3-1d:i386]]** 
 
* Pour les versions antérieures
[[:tutoriel:comment_installer_un_paquet|Installer un paquet]] **[[apt>linux32 ia32libs]]** 

*Posséder une débogueur/programmateur de microcontrôleurs LPC. La carte LPCXPRESSO LPC1769 d'Embedded Artists vendue 25E et un câble USB A mini B font bien l'affaire.

===== Installation =====

Télécharger le fichier d'installation sur [[http://www.lpcware.com/lpcxpresso/download]].

Décompresser et lancer l'installation.

Activer le produit en s'enregistrant sur le site lpcware.com .

Copier /usr/local/lpcxpresso_7.2.0_153/UdevRules/85-lpcxpresso.rules dans /etc/udev/rules.d et redémarrer la machine.
