====== Freemind ======

**Freemind** est un éditeur de [[:carte heuristique]]. Une description plus complète est disponible sur le site de [[http://www.framasoft.net/article2894.html|Framasoft]].

Il existe un fork nommé [[:Freeplane]].


===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation =====
 
<note important>
Freemind n'est plus disponible dans les dépôts d'Ubuntu ni dans getdeb dès Xenial (16.04). Privilégier [[freeplane|Freeplane]] à la place.
</note>
==== Installation depuis les dépôts officiels  ====

Il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>freemind]]**.


==== Installation de la dernière version Freemind   ====
La dernière version est disponible dans le dépôt [[:getdeb]]. Il vous faut donc ajouter le dépôt [[:getdeb]] à vos sources de logiciels, puis [[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>freemind]]**.

<note tip>En complément du paquet principal freemind quelques plugs-in intéressants peuvent être installés, notamment celui pour l'export des fichiers au format PDF, SVG.</note>

<note important>
Le dépôt getdeb peut entraîner la mise à jour de ses logiciels à leur dernière version. Donc, faites attention.
</note>
===== Utilisation =====

Vous pouvez lancer Freemind en faisant une recherche dans [[unity#tableau_de_bord_dash|le tableau de bord Unity]] avec le mot clé **Freemind** ou //via// la [[:commande_shell|commande]] suivante : <code>freemind</code>

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, le configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
