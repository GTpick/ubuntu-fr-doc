======Télécommande de la Carte TV : Terratec Cinergy T2 ======

Le syntoniseur Terratec Cinergy T2 est reconnu nativement sous Linux. la télécommande est compatible de base avec Ubuntu. Cependant, il est difficile de configurer celle-ci pour correspondre aux différents logiciels et aux configurations personnelles.**Dvb-remote-key-editor** est un outil qui permet de changer facilement la configuration des touches de la télécommande.\\
Cet outil propose aussi de modifier la configuration pour d'autre télécommande.

===== Installation =====

<note warning>
Attention il y a possibilité de conflit avec [[:LIRC]], s'il est installé veuillez au préalable le désinstaller.</note> 

Téléchargez le programme [[https://github.com/sclayer/dvb-remote-key-editor/releases/tag/v0.2.8|ici]]. Puis :
  * [[:archivage#graphiquement|Extraire l'archive]] dans votre **Dossier Personnel**,
  * Ouvrir un [[:terminal]] et aller dans le dossier décompréssé avec la commande : <code>cd dvb-remote-key-editor*</code> Il y a deux fichiers extrait de l'archive : ''dvb-remote-editor.v0.2.6.py'' et ''input.h''
  * Pour éxécuter l'application, il suffit de taper dans le [[:terminal]] :<code>gksudo python dvb-remote-editor.v0.2.8.py</code>

Puis cliquer sur "Cinergy T2" ou Load->Cinergy T2

Vous pouvez maintenant modifier toutes les touches que vous souhaitez :
  * Utilisez "Keymap->Load from file" pour charger un fichier (comme celui juste au dessus).
  * Utilisez "Keymap->Save to file" pour sauver votre configuration dans un fichier.

Une fois vos changement effectués, cliquez sur le bouton "Save changes to kernel module"

Fermez le programme et rentrez dans le terminal (n'oubliez pas de fermer Kaffeine) : 
<code>sudo rmmod dvb_usb_cinergyT2
sudo modprobe dvb_usb_cinergyT2</code>

Et voilà, vous pouvez maintenant utiliser la télécommande 8-)
