====== CUPS-PDF : imprimer en PDF ======

**CUPS-PDF** fournit une imprimante [[:PDF]] au système d'impression [[:CUPS]]. CUPS-PDF est utilisée comme une imprimante virtuelle ce qui est bien pratique pour imprimer plus tard si on n'a pas d'imprimante sous la main, ou pour des raisons écologique, ou simplement pour produire un document PDF afin de l'envoyer par e-mail et être sûr de la compatibilité de lecture du destinataire.

<note tip>Depuis [[:precise|Ubuntu 12.04 LTS]], la fonctionnalité d'impression au format PDF ou PostScript est intégrée par défaut. Il suffit de demander l'impression dans un fichier et de sélectionner le format souhaité.</note>

Depuis la version 12.04, cela peut encore s'avérer utile pour certains programmes ne permettant pas d'imprimer dans un fichier PDF tel que pour le lecteur de carte d'identité belge EID.

===== Installation =====

Il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>cups-pdf|cups-pdf]]**.

L'installation ajoutera une imprimante « virtuelle » dans la liste de vos imprimantes. Les documents PDF produits seront dans le dossier **PDF** de votre dossier personnel.

===== Utilisation et configuration =====

==== Changer le répertoire d'impression ====

Par défaut, CUPS-PDF imprime dans le dossier **~/PDF**. Pour changer ce répertoire, [[:tutoriel:comment_editer_un_fichier|éditez le fichier]] **/etc/cups/cups-pdf.conf** avec les droits [[:sudo|administrateur]] et modifiez la ligne suivante :
<file>
Out ${HOME}/PDF
</file>

Par exemple, pour mettre vos fichiers PDF sur votre bureau, remplacez-la par la ligne suivante (où //Desktop// est le nom du répertoire de votre Bureau) :
<file>
Out ${HOME}/Desktop
</file>

Après modification de ce fichier, il faut relancer cups :
<code>
sudo service cups restart
</code>
