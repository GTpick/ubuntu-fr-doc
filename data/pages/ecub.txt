====== eCub ======

ECub est un éditeur WYSIWYG de fichiers au format [[:ePub]] (format de prédilection pour les liseuses de livres électroniques). **ECub est un freeware et N'EST PAS UN LOGICIEL LIBRE.** Il existe d'autres éditeurs [[:epub]] comme [[:Scenari-Book]] ou [[sigil]] (logiciels libres), ou des éditeurs en ligne (voir la page [[epub]] pour plus d'informations à ce sujet). ECub est simple d'emploi.


<note important>
eCub n'étant pas un logiciel libre on ne sait pas exactement ce que contient son code qui pourrait être potentiellement dommageable pour votre système. Il est recommandé d'**utiliser [[Scenari-Book]] ou [[sigil]] (logiciels libres) à la place d'eCub**, ces deux premiers bénéficiant d'un développement rapide et très actif.
</note>

{{ :applications:epub_xenial.png?400 }}

===== Pré-requis =====
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation =====

[[:tutoriel:comment_installer_un_paquet|Installez le paquet]] eCub (prendre le paquet //eCub X.XX Debian package// sur le [[http://www.juliansmart.com/ecub|site officiel d'eCub]])

Il vous suffit, pour cela, de double-cliquer sur le fichier téléchargé et apt se lance pour l'installation d'eCub.

ECub n'a pas de traduction en français. 

===== Utilisation=====

Lancez l'application via le [[:unity#tableau_de_bord_dash|tableau de bord]] (Unity) ou via le [[:terminal]] (toutes versions d'Ubuntu) avec la [[:commande_shell|commande]] suivante : <code> ecub</code>


===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. La configuration de l'application sera conservée ou supprimée selon la méthode de désinstallation que vous choisirez.
