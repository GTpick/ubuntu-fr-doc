===== Installation du logiciel Sporttrack====

 =====

==== Pré-requis ====

  * [[:tutoriel:comment_installer_un_paquet|Installez les paquets]] 




  * Téléchargez les sources de [[http://www.zonefivesoftware.com/SportTracks/Downloads/Versions/Mono/sporttracks-2.1.3478.tar.gz]].

  * décompacter le fichier dans le répertoire utilisateur
  
 
Attention : Le fichier tar.gz ne semble pas disponible pour la version 3 de sportTracks

===== Utilisation et configuration =====

Depuis le répertoire  où vous avez décompacté le fichier taper "mono SportTracks.exe"

Et le logiciel se lance.
