====== Jedi Knight ======

<note important>
Wine n'est plus nécessaire, les sources ont été libérées, voir l'article  [[http://linuxfr.org/news/les-codes-sources-des-jeux-jedi-outcast-et-jedi-academy-passent-sous-licence-gnu-gplv2|suivant]].</note>

**Jedi Knight** est une série de jeux de tir subjectif se déroulant dans l'univers élargi de [[wpfr>Star Wars]]. Le joueur assume le rôle d'un apprenti Jedi et se voit confier des missions. Cette page explique comment installer [[wpfr>Jedi Knight 2 : Jedi outcast]].

===== Pré-requis =====
 
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  * Avoir [[:tutoriel:comment_installer_un_paquet|installé les paquets]] **[[apt>libopenal1,ia32-libs]]**.
  * Avoir une copie originale du jeu.
===== Installation =====
Pour installer Jedi Knight :
  * Téléchargez les fichiers binaires [[https://fast.sandboxes.io/~jkhubcdn/jko/code/client/JK2_SP_Linux.zip|ici]]. 
  * [[:archivage#graphiquement|Extraire l'archive]] dans votre Dossier Personnel,
  * À partir du cd ou bien de votre version [[Steam]] copiez simplement les fichiers **assets0.pk3, assets1.pk3, assets2.pk3 and assets5.pk3** dans le dossier précédemment décompressé.
=====Utilisation====
Dans le dossier décompréssé durant l'installation, **JK2_SP_Linux** lancez l'exécutable **jk2sp** en double cliquant dessus, ou depuis un [[:terminal]] ouvert dans le dossier concerné :<code>./jk2sp</code>
