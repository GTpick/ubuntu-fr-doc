====== Pencil Evolution ======

Pencil Evolution est un logiciel permettant de créer des prototypes d'interfaces graphiques (mockup). Il est libre, gratuit et multiplateforme (Windows, OSX, Linux et extension Firefox).

===== Installation =====

Se rendre sur le [[http://pencil.evolus.vn/Downloads.html|site web du logiciel]] et télécharger le paquet .deb. Ensuite, [[tutoriel/comment_installer_un_paquet|installer le paquet]].

<note>
Le lien direct de téléchargement est mort (04.09.2016). Pour télécharger le logiciel, cliquer sur "linux builds" et télécharger le fichier .deb le plus récent.
</note>

<note important>
Le paquet pencil présent dans les dépôts est celui d'une [[pencil|autre application du même nom]] permettant de réaliser des animations en 2 dimensions.
</note>


===== Utilisation =====

Lancez l'application via le tableau de bord (Unity) ou via le terminal (toutes versions ou variantes d'Ubuntu) avec la commande suivante :

<code>
pencil
</code>

L'interface est composée de 3 parties : l'affichage principal qui est le lieu de conception du projet, la barre d'outil (haut) permettant entre autres de formater les objets insérés sur la page et le volet latéral (gauche) d'où les objets (appelés stencils) peuvent être insérés sur l'affichage principal par cliquer-déposer. L'affichage principal est lui-même décomposé en onglet, chaque onglet représentant une page du projet d'interface. Le déplacement des onglets ne peut se faire que par clic droit et non pas par glisser-déposer, ce qui rend le travail sur de grands projets rapidement laborieux.

Pour éditer un stencil, il suffit de double-cliquer dessus. Pour activer visuellement une partie d'un stencil (tel un onglet dans une tab box), il faut faire précéder le texte de cette partie du symbole "#".

Il est possible d'étendre le nombre de stencils utilisables. Des modèles sont disponibles [[http://pencil.evolus.vn/Stencils-Templates.html|ici]]. Pour installer une série de modèles, procéder de la manière suivante :

  - Télécharger un fichier zip contenant des modèles (ne pas décompresser l'archive)
  - Dans la barre d'outil de pencil, cliquer sur "tool/install new collection"

===== Problèmes connus =====

==== Onglet de l'objet tab box ====


En cas de copier-coller de l'objet tab box, il est impossible de changer l'onglet actif sur les éléments copiés, le premier onglet étant à nouveau le seul affiché comme "activé" après quelques minutes malgré l'utilisation du symbole "#". Le bug est signalé sur Github.

===== Désinstallation =====


Pour supprimer cette application, il suffit de supprimer son paquet. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
