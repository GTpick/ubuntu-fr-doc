====== Suggestions pour améliorer Ubuntu ======


Cette page est la synthèse des suggestions émises par les utilisateurs francophones d'Ubuntu sur le [[http://forum.ubuntu-fr.org/viewtopic.php?pid=3072652|forum du projet "Suggestions pour Ubuntu"]].


===== Principe du projet =====

Ce projet a pour but de permettre à tous, même les non-anglophones((Jusqu'à peu il était nécessaire d'être anglophone pour proposer ses idées d'amélioration aux développeurs d'Ubuntu.)), de proposer des idées pour améliorer Ubuntu.


====Comment proposer une idée pour améliorer Ubuntu ?====

  * **Si vous ne maitrisez pas l'anglais:** faites-nous part de votre suggestion sur le [[http://forum.ubuntu-fr.org/viewtopic.php?pid=3072652|forum du projet]]: [[http://forum.ubuntu-fr.org/viewtopic.php?pid=3072652|ICI]] !
  * **Si vous êtes anglophone:** rédigez votre suggestion en anglais directement sur Brainstorm, puis laissez un petit message sur le [[http://forum.ubuntu-fr.org/viewtopic.php?pid=3072652|forum du projet]] si vous voulez en discuter avec la communauté francophone.

<note tip>N'ajoutez pas vos suggestions sur cette page, mais sur le [[http://forum.ubuntu-fr.org/viewtopic.php?pid=3072652|forum du projet]] ! </note>



====Comment ça fonctionne ?====

  - vous faites part sur [[http://forum.ubuntu-fr.org/viewtopic.php?pid=3072652|le forum du projet]] d'idées pour améliorer Ubuntu
  - on en discute tous ensemble sur le [[http://forum.ubuntu-fr.org/viewtopic.php?pid=3072652|forum]]
  -  si une idée est retenue, un membre de l'Equipe de ce projet l'ajoute cette page de synthèse du projet (celle-même que vous lisez)
  - l'Equipe du projet traduit la suggestion en anglais pour la proposer aux développeurs, et indique le "lien Brainstorm" sur cette page de synthèse
  - Tous les gens intéressés peuvent promouvoir l'idée en votant celle-ci sur Brainstorm (pour cela il faut vous [[http://brainstorm.ubuntu.com/user/register|créer un compte Brainstorm]]).


====Comment rejoindre l'Equipe du projet ?====

  * laissez un petit message sur [[http://forum.ubuntu-fr.org/viewtopic.php?pid=3072652|le forum du projet]] en indiquant si vous êtes anglophone ou pas.


<note tip>Lorsque la suggestion a été postée sur Brainstorm, le lien Brainstorm est mis juste à coté.

Lorsque la suggestion a été implémentée (ou éventuellement refusée) elle est supprimée de la page.</note>

=====Suggestions retenues sur le forum =====

==== Améliorations de logiciels ====

  * alacarte (l'éditeur de menu) : améliorer l'ergonomie en simplifiant à une seule arborescence. 
    * [[http://brainstorm.ubuntu.com/idea/23412/|Voter sur Brainstorm]] ([[https://bugzilla.gnome.org/show_bug.cgi?id=602749|Wishlist sur Bugzilla]])
  *  evolution: améliorations générales
    * [[http://brainstorm.ubuntu.com/idea/21935/|Voter l'idée sur Brainstorm]]
  * gnome-default-applications-properties ("Applications préférées") : Étendre les catégories de l'interface à d'autres types de fichiers. 
    * [[https://bugzilla.gnome.org/show_bug.cgi?id=171871|Wishlist sur Bugzilla]]
  * gnome-mouse-properties : pouvoir activer le simple-clic dans tout Gnome.
  * gnome-network: pouvoir créer des profils réseau ("Maison", "Travail", "Café", etc) - avec adresse réseau, divers pseudos, partages, imprimante, TOR, paramètres de pare-feu et proxy etc. 
    * [[http://brainstorm.ubuntu.com/idea/2813/|Voter l'idée sur Brainstorm]]
  * gnome-panel (tableau de bord): empêcher de supprimer un tableau de bord ou un applet par erreur en demandant confirmation 
    * [[https://bugzilla.gnome.org/show_bug.cgi?id=602751|Wishlist sur Bugzilla]]
  * gnome-panel (tableau de bord) : depuis MenuBar, pouvoir supprimer les applis par simple glissé/déposé dans la corbeille avec confirmation par boite de dialogue "supprimer l'appli, ou supprimer le lanceur, ou annuler ?". 
    * [[https://bugzilla.gnome.org/show_bug.cgi?id=602754|Wishlist sur Bugzilla]]
  * gnome-panel (tableau de bord) : dans MenuBar, ajouter les choix "Supprimer l'appli" et "Supprimer le lanceur" dans le menu quand on clic-droit sur un lanceur.
    * [[https://bugzilla.gnome.org/show_bug.cgi?id=602754|Wishlist sur Bugzilla]]
  * gnome-panel (tableau de bord) : dans MenuBar, pouvoir retrouver facilement dans les menus les applications nouvellement installées (par surlignage ou avec un petit label ou autre…). 
    * [[https://bugzilla.gnome.org/show_bug.cgi?id=602755|Wishlist sur Bugzilla]]
  * gnome-screenshot : améliorer l'outil capture d'écran de tel sorte qu'il gère la capture vidéo de l'écran (screencast) 
    * [[https://bugzilla.gnome.org/show_bug.cgi?id=304933|Wishlist sur Bugzilla]]
  * gnome-search-tool (outil de recherche gnome) : pouvoir choisir plusieurs dossiers dans lesquels on désire effectuer ses recherches. 
    * [[https://bugzilla.gnome.org/show_bug.cgi?id=603062|Wishlist sur Bugzilla]]
  * nautilus : pouvoir ajouter facilement des actions préconfigurées dans le menu clic-droit de Nautilus.
    * [[http://brainstorm.ubuntu.com/idea/23411/|Voter l'idée sur Brainstorm]]
  * nautilus : pouvoir sélectionner plusieurs dossiers/fichiers avec la souris en mode "vue en liste" par une simple "clique-glisser de la souris". 
    * [[https://bugzilla.gnome.org/show_bug.cgi?id=138931|Wishlist sur Bugzilla]]
  * nautilus : créer un système de "Dossiers Intelligents" qui contiennent des liens symboliques vers les fichiers correspondant au résultat d'une recherche (exemple : tous les fichiers photos, tous les fichiers d'un certain nom,...). 
    * [[https://bugzilla.gnome.org/show_bug.cgi?id=603065|Wishlist sur Bugzilla]]
  * nautilus : pouvoir mettre en Pause ou Annuler la copie d'un gros fichier. 
    * [[http://brainstorm.ubuntu.com/idea/6635/|Voter l'idée sur Brainstorm]]
  * nautilus : lorsque que l'on copie un texte et qu'on le colle dans un dossier, il serait bien que ça créer un nouveau document texte avec le texte copié dedans.
  * nautilus : lors d'une copie de fichier, si un fichier du même nom existe déjà dans le répertoire destination, il serait bien que Nautilus, en plus de nous demander si on veut le remplacer, nous précise la date de dernière modification du fichier source et de celui que l'on risque éventuellement écraser. 
    * [[https://bugs.launchpad.net/nautilus/+bug/136702|Voter l'idée sur Launchpad]]
  * nautilus : inverser le sens de la flèche de classement alphabétique des colonnes afin d'uniformiser le comportement avec les autres applications Gnome (Totem, Rhythmbox), cf [[http://forum.ubuntu-fr.org/viewtopic.php?pid=3209867#p3209867|captures d'écran]]. 
    * [[https://bugs.launchpad.net/hundredpapercuts/+bug/509656|Voter l'idée sur Launchpad]]
  * nautilus : intégrer File-roller dans Nautilus de la même manière que le créateur de CD/DVD est intégré dans Nautilus
    * [[http://brainstorm.ubuntu.com/idea/25434/|Voter l'idée sur Brainstorm]]
  * notifications (notify-osd ou python-notify ?) : menu pour activer/désactiver les notifications.
  * notifications (notify-osd ou python-notify ?) : afficher une notification lorsque l'on connecte un périphérique.[[http://brainstorm.ubuntu.com/idea/18023/|Voter l'idée sur Brainstorm]]
  * ubiquity (installateur d'Ubuntu): sauvegarde automatique du MBR (et des EBR ?) sur les derniers secteurs du disque, ceux qui ne sont pas utilisés car ne tombant pas sur des frontières de cylindres; cela permettrait de récupérer plus aisément les partitions en cas de crash/dual-boot/pc-tatoué. 
    * [[http://brainstorm.ubuntu.com/idea/10410/|Voter l'idée sur Brainstorm]]
  * ubiquity (installateur d'Ubuntu): faciliter la réinstallation et la migration vers un autre PC en proposant de réinstaller automatiquement les logiciels. 
    * [[http://brainstorm.ubuntu.com/idea/22688/|Voter l'idée sur Brainstorm]]
  * ubiquity (installateur d'Ubuntu): permettre de désinstaller proprement Ubuntu
    * [[https://bugs.launchpad.net/ubuntu/+bug/653474|Voter l'idée sur Launchpad]]
