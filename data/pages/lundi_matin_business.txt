====== Lundi Matin Business ======

**LUNDI MATIN BUSINESS** est une application Open Source de gestion d'entreprise, disponible sur Internet.

Parfaitement adaptée à vos besoins, cette solution vous apporte l'ensemble des outils nécessaires à la gestion de votre entreprise :
  * Gestion commerciale (devis, commandes, livraisons, factures)
  * Gestion de l'annuaire (clients, fournisseurs, collaborateurs, Partenaires, etc.)
  * Gestion du catalogue d'articles
  * Gestion du stock
  * Gestion financière (suivi des règlements, relances, encaissements) 

<note important>**Depuis Janvier 2013, il n'existe plus de versions Open Source/Communautaire.**

La société a confirmé (fin Février 2013) que suite à des orientations stratégiques différentes (plus de fonctions "poussées"), il n'y a plus de versions communautaires et seulement des versions payantes.
</note>

→ [[http://www.lundimatin.fr/|Plus d'informations sur le site officiel de Lundi Matin Business]]

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]] ;
  * Disposer d'une connexion à Internet configurée et activée ;

===== Installation =====

//Lundi Matin Business// peut être installé sur un serveur Internet (version zip) ou sur un poste de travail traditionnel (version .deb).

==== Téléchargement ====

En fonction de vos besoins, sélectionnez la version du logiciel :
  * Version ZIP : sur le site officiel et téléchargez la:   [[http://www.lundimatin.fr/dll/linux/install_lmb.rar|Téléchargement sur le site officiel.]]
  * Version .deb : sur [[http://www.informalys.com|le site du packager]].

==== Installation ====

[[:tutoriel:comment_installer_un_paquet|Installez le paquet]] lmb. Le mot de passe root pour MySQL vous sera demandé s'il n'est pas installé.

==== Configuration de la base de données ====

Il faut ensuite créer la base de données :

  mysql -u root -p

MySQL vous demandera le mot de passe root (créé à l'installation).

  create database lmb;
  exit

Votre base de données est maintenant créée et se nomme « lmb ».

===== Utilisation =====

==== Premier lancement ====

//Applications → Internet → LMB//

Vous vous trouverez confronté au configurateur. Vous devrez entrer les informations de connexion pour la base de données et créer votre entreprise.

==== Commencer à utiliser LMB ====

Une fois la configuration terminée, vous pourrez vous connecter et commencer à utiliser Lundi Matin Business.

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. La configuration de l'application sera conservée ou supprimée selon la méthode de désinstallation que vous choisirez.
