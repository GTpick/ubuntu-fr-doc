====== World of Padman ======


**World of Padman** est un [[wpfr>jeu de tir subjectif]] en ligne et libre, avec des graphismes rigolos, de style bande dessinée. À l'origine un [[wpfr>Mod_(jeu_vidéo)|mod]] pour [[wpfr>Quake_III|Quake III Arena]], il est devenu un jeu à part entière ([[wpfr>standalone]]) avec la libération du moteur de Quake III.

La plupart des cartes illustrent des lieux de la vie quotidienne (cuisine, chambre, garage, ...) où l'on y incarne un être minuscule. Il n'y a pas de mode de jeu solo; un mode permet de jouer contre des robots, sans être connecté à internet.

===== Installation =====

**World of Padman** peut être installé depuis [[http://www.getdeb.net|GetDeb]] (avec un paquet *.deb), ou depuis le [[http://www.worldofpadman.com/|site officiel]] (avec un fichier exécutable *.run). Nous vous recommandons la première méthode.

==== Installation depuis le site PlayDeb ====

Le site [[:Getdeb]] fournit un paquet pour Ubuntu, qui se trouve dans la section [[http://www.playdeb.net/updates/Ubuntu/all|PlayDeb]] (réservée aux jeux).

  * Activer le dépôt PlayDeb en suivant [[:getdeb#comment_ajouter_le_depot_de_playdeb|ce tutoriel]].
  * Rendez-vous sur la page de [[http://www.playdeb.net/software/World%20of%20Padman|World of Padman sur PlayDeb]]
  * Cliquez sur le bouton « Install this now ».

Une fois le paquet téléchargé et installé, désactivez le dépot PlayDeb, **avant toutes installations nouvelles ou toutes mises à jour** : //Système => Administration => Sources de logiciels//, puis dans l'onglet « Autres logiciels » décochez la ligne « http://archive.getdeb.net/ubuntu ».


==== Installation depuis le site officiel====

Téléchargez **Linux Full Installer v1.1** et **Linux Patch Installer v1.2** depuis [[http://padworld.myexp.de/index.php?files|la page de téléchargement]]. Attention à bien choisir « Enregistrer le lien sous ». 

Exécutez les commandes suivantes dans un [[:terminal]] en prenant soin de vous placer dans le dossier où se trouvent les fichiers téléchargés (par exemple, tapez « cd Bureau » si les fichiers se trouvent sur le bureau) :

  chmod +x worldofpadman.run
  sudo ./worldofpadman.run

ou
 
Exécutez le fichier *.run (voir [[:/installation_non_standard_de_logiciel#executable_run_et_les_binaire_bin_et_les_scripts_sh|ici]]).

Dans les deux cas, une fois l'installation terminée, lancer l'installation du patch :

  chmod +x wop_patch_1_2.run 
  sudo ./wop_patch_1_2.run

<note important>Si vous avez installé le jeu dans un dossier dont le nom contient des espaces, il faut faire une petite correction dans le script « wop » du répertoire du jeu : éditez le script wop, et à la ligne 26, ajoutez des guillemets (''"'') autour de ''$script''.</note>

<note>Vous devez disposer de 850Mo de mémoire au minimum.</note>

<note>D'après [[installation_non_standard_de_logiciel#executable_.run_et_les_binaire_.bin_et_les_scripts_.sh|cette page]] de la documentation, on peut avoir une interface plus conviviale pour l'installation via le *.run en installant le paquet [[apt://gtk-engines-pixmap|gtk-engines-pixmap]]</note>

==== Map pack supplémentaire ====

Le **Map pack** ajoute des cartes supplémentaires.

  * Télécharger le fichier **wop_padpack.zip** depuis un des miroirs liés [[http://padworld.myexp.de/index.php?news&update_id=63|sur cette page]] (//« PadPack released - A first Map-Pack »//).
  * Extraire **wop_padpack.pk3** dans votre dossier "World of Padman/wop/", qui se trouve sous /usr/local/games/WoP/wop par défaut.

===== Utilisation =====
Lancez l'application via le [[:unity#tableau_de_bord_dash|tableau de bord]] (Unity).

==== Modes de jeu ====


=== Spread your tag ===

Le but est d'éliminer les autres joueurs pour récupérer des pots de peinture, puis d'aller dans le téléporteur (en général au centre de la carte), et de peindre le mur (et le bon mur si l'on joue en équipe !!!)

=== Deathmatch ===

Le classique tue-tout-le-monde.

=== Big Balloon ===

Rapprochez-vous d'une des 3 boîtes avec un ballon, plus vous arrivez à la garder longtemps de votre couleur, plus vous marquez des points. Si votre équipe tient les 3 boites, vos points augmentent encore plus vite.
