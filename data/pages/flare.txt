====== Flare ======

Anciennement OSARE, Flare est un jeu de rôle d'action. Il est écrit en C++ avec SDL et fonctionne sur Windows, OSX et Linux. Il est distribué sous la licence GPL (version 2.0 ou ultérieure) et CC-BY-SA (3.0 ou version ultérieure).


{{:osare2.png}}{{:osare3.png}}{{:osare4.png}}{{:osare5.png}}{{http://clintbellanger.net/rpg/images/screenshots/0015.jpg?195}}{{http://clintbellanger.net/rpg/images/screenshots/0025.jpg?195}}{{:minotaur_run.gif}}


Flare est avant tout:

  * Un classique jeu de rôle fantastique d'action en 2D isométrique
  * Un ensemble simple de système de règles
  * Un jeu dont les graphismes, l'histoire et les maps sont personnalisables dans des fichiers humainement lisibles

Flare n'est pas:

  * Suffisamment flexible pour permettre de construire tout type de jeu isométrique
  * Un générateur de donjon aléatoire
  * Multijoueur

Flare utilise les attributs de base suivants pour les personnages: //Physique, Magiques, Offensifs, Défensifs//. Ils s'échelonnent de 1 à 5. Ces attributs se combinent pour former un archétype classique de jeux de rôle:

{{ :osare6.png }}

Le projet est en plein développement, et le travail de grande qualité, que ce soit les animations, les layouts, ou la musique. Si le projet ne se revendique pas comme étant un Diablo-like, l'auteur, incontestablement en est fan, et le jeu se trouve à cheval, qualitativement parlant, entre les deux épisodes mythiques - I et le II. L'ambiance est parfaitement retranscrite.


===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

=====Installation =====

Pour installer le jeu, il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>flare]]**.


===== Utilisation =====

Pour lancer le jeu il faut lancez l'exécutable **flare** en ligne de commande dans un [[:terminal]].

(note: si il n'y a pas de mods, lancez avec l'option ** - -game=flare-game**)
===== Configuration =====
Le jeu peut être paramétrés via le fichier **settings.txt** dans [[:fichier_cache|le dossier caché]] **.config/flare/** de votre **Dossier Personnel** :
<file>
fullscreen=0
resolution_w=1440
resolution_h=960
music_volume=96
sound_volume=128
mouse_move=0
hwsurface=1
doublebuf=1
frames_per_sec=30
</file>

 
===== Désinstallation =====
 
Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
