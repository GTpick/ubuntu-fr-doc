======Interface Neuronale : Contrôler un système par la pensée ======

Une [[wpfr>interface neuronale directe]] - aussi appelée IND, BCI ([[http://en.wikipedia.org/wiki/Brain%E2%80%93computer_interface|brain-computer interface]]), interface cerveau-machine ou encore interface cerveau-ordinateur - est une interface de communication directe entre un cerveau et un **dispositif externe** (ordinateur, outil électronique, etc.).


{{ http://img256.imageshack.us/img256/7759/epocneuroheadset01.gif?300}}

Les interfaces neuronales actuelle disponible sur le [[http://fr.wikipedia.org/wiki/Droit_des_march%C3%A9s_publics|marché public]] sont sous forme de casque, [[http://www.infohightech.com/spip.php?article30406|chaussure]], [[http://fr.wikipedia.org/wiki/Exosquelette_motoris%C3%A9|exosquelette motorisé]], plusieurs capteurs ( [[epoc]] , ...) identifie les zones activées dans le cerveau : ce qui permet par exemple de contrôler le cube compiz, déplacer des fenêtres comme si vous le faisiez avec une interface tactile, jouer à des jeux, contrôler un [[robotique|robot]], etc.
  * [[http://www.youtube.com/watch?v=tc82Z_yfEwc&feature=player_embedded#at=89|Vidéo : Contrôle d'un smartphone]]
  * [[http://www.youtube.com/watch?v=NNkkuMO5l7A|Vidéo : contrôle robot spybot]]

Dans la prochaine version d'interface neuronale ( les projets de recherche ), ces interfaces seront capable d'identifier les mots et les images auxquels vous pensez.
  * [[http://nextbigfuture.com/2010/08/intel-human-brain-imaging-and-machine.html|Intel Human Brain Imaging and Machine Learning Research]]


Une interface neuronale sans danger : est un dispositif passif : un capteur eeg.

Il y a également les interfaces neuronales actives : qui envoient un signal au cerveau, pour par exemple permettre à des aveugles de voir ([[https://encrypted.google.com/search?q=blind+implant&num=50&hl=fr&safe=off&prmd=ivns&source=lnms&tbs=vid:1&ei=5V8STaexJYm48gPmlqyCBw&sa=X&oi=mode_link&ct=mode&ved=0CBEQ_AU|vidéo]]).
