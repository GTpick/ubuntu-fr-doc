====== Applications par défaut sous Ubuntu ======
**
Méthode 1** :Pour changer les applications par défaut sous Ubuntu, lancez [[:gnome-control-center|gnome-control-center]]  (Paramètres système), allez dans la catégorie « Détails », puis « Applications par défaut ».

**Méthode 2** : La méthode précédente ne permet de modifier l'application par défaut que pour quelques types de fichiers. Impossible, par exemple, de définir quelle application sera utilisée par défaut pour lire un pdf.
Pour définir l'application par défaut quel que soit le type de fichier, ouvrez votre explorateur de fichier, faites un clic-droit sur le fichier concerné puis cliquez sur "Propriétés".
Allez à l'onglet "Ouvrir avec" puis choisissez l'application voulue et cliquez sur "Définir par défaut".
