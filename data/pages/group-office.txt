====== Group-Office ======

Group-Office est un Groupware publié par la socité Intermesh qui le distribue sous une version commerciale et une version GPL. Le logiciel fonctionne sur une base PHP MySQL et la difference notable entre la version commerciale et la version Open-Source concerne la synchronisation avec des assistants personnels.

__**Fonctionnalités :**__
  * Agendas
  * CRM (Gestionnaire de Relation Client)
  * Web Courriel
  * Gestionnaire/partage de fichier
  * Gestion de projet
  * Synchronisation

<note tip>Vous pouvez tester Group-Office sur le [[http://go3.group-office.com|site Web officiel]] (login:demo, mdp:demo).</note>


===== Installation de Group-Office 3.0 =====

Téléchargez l'archive sur [[http://sourceforge.net/projects/group-office|SourceForce - Group-Office]].

Décompressez cette archive sur votre serveur, rendez-vous sur votre dossier group-office via votre navigateur Web et suivez tous simplement les instructions d'installation.

<note important>Le paquet **libwbxml2** a été remplacé par les paquets [[apt://libwbxml2-0|libwbxml2-0]] et [[apt://libwbxml2-utils|libwbxml2-utils]], sans ceux-ci la synchronisation sera impossible</note>
