====== HjSplit ======

**HjSplit** est un logiciel multiplateforme permettant de découper et recoller un fichier en plusieurs morceaux.

===== Pré-requis =====
 
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation =====
==== Via le site officiel ====
Sur [[http://www.hjsplit.org/linux/|la page du site officiel consacrée à la version Linux]], téléchargez le fichier **hjsplitlx.tar.gz** d'un des trois emplacements suggérés, puis [[archivage|décompressez-le]].

==== via un fichier .deb ====
Télécharger le fichier http://dl.dropbox.com/u/964512/HJ-Split_1.0_all.deb puis l'installer comme n'importe quel paquet.
Attention, il s'agit d'une ancienne version ! (1.0 contre 3.x actuellement, janvier 2013).

===== Utilisation=====

Dans le dossier extrait, double-cliquez sur le fichier **hjsplit** pour lancer l'application.

<note tip>On peut aussi réassembler les fichiers avec la commande cat.
Ouvrez un [[terminal]], placez-vous dans votre dossier qui contient tous les fichiers 001 002 … et UNIQUEMENT ceux-ci, puis :

  cat * >fichier.avi
</note>
