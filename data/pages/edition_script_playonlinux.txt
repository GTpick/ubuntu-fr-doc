====== Edition de scripts PlayOnLinux ======






===== Introduction =====


Pour commencer, sachez que la contribution est importante pour faire avancer le projet. Le forum du site propose une [[http://www.playonlinux.com/fr/documentation.html|documentation officielle]] permettant de créer son propre script.
----
Cette documentation n'est que le condensé de la documentation officielle afin de savoir comment éditer les scripts donnés en texte (où les mettre, comment les publier, etc.) car les scripts disponibles sur le site ne sont pas téléchargeables ; il faut copier/coller les lignes de codes dans l'éditeur. 

Ainsi le but de cette doc est:

  * Installer playonlinux script creator
  * Vous montrer comment cet outil fonctionne.



===== Installation du Script Creator (playonlinux) =====





==== Prérequis ====

Pour l'installation de playonlinux script creator vous devez:
  * Avoir installé [[playonlinux]]
  * Avoir téléchargé [[http://repository.playonlinux.com/plugins/ScriptCreator0.5.pol|le plugin script creator]].







==== Ajout du plugin script creator ====

Après avoir téléchargé le plugin script creator, ouvrez playonlinux, puis Plugin/Gestionnaire de plugins. Faites "Ajouter", ouvrez le plugin que vous venez de télécharger et activez le, si ce n'est déjà fait.

Relancez PlayOnLinux.












===== Utilisation de PlayOnLinux Script Creator =====



Pour exécuter un plugin donné en texte, il suffit d'ouvrir PlayOnLinux -> Plugins -> Script Creator, puis une fois l'éditeur ouvert, de copier/coller le texte avant d'enregistrer le script dans un endroit accessible pour enfin l'exécuter : 
  * soit en double-cliquant dessus, 
  * soit en l'ouvrant dans PlayOnLinux -> Outils -> Exécuter un script local.
Une fois le script ouvert, il suffit de suivre l'installation de votre logiciel.
