====== Associations ======

**Associations fait partie de la suite de logiciels éducatif du [[le_terrier|Terrier]].**

<note>(Ci-dessous, extrait du site officiel http://www.abuledu.org/leterrier/associations )</note>

{{ :capture_associations.png?350}}

"Associations est un logiciel ouvert qui inclut un éditeur permettant de créer des associations de mot ↔ image ↔ son, et de générer différents exercices répartis par type : entendre, voir, reconnaître, combiner, écrire."


//**Pour une description complète et des pistes pédagogiques, voir [[http://www.abuledu.org/leterrier/associations|la documentation sur le site officiel du Terrier d'Abulédu]].**//

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  *  ** Avoir activé l'accès aux [[le_terrier#ajouts_des_depots|dépôts du Terrier]].**
  

===== Installation =====

[[:tutoriel:comment_installer_un_paquet|Installez le paquet]] **[[apt://abuledu-associations|abuledu-associations]]**.


===== Lancement =====

Comme tous les logiciels du Terrier, vous retrouverez un lanceur dans le menu **Applications --> Éducation
**

===== Configuration =====

Comme les autres logiciels du Terrier, Associations permet à l'enseignant d'éditer les exercices. Pour cela, vous devez faire partie du groupe "profs". Si le groupe "profs" n'existe pas, créez-le. (Voir [[users-admin#gerer_les_groupes_d_utilisateurs|Gérer les groupes d'utilisateurs]])


===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. La configuration de l'application sera conservée ou supprimée selon la méthode de désinstallation que vous choisirez.
