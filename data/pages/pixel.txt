====== Pixel Image Editor ======

{{  http://pix.toile-libre.org/upload/original/1270248091.png?350|Pixel, en version de démonstration}}

**Pixel** est un logiciel d'édition graphique propriétaire multiplate-forme développé par Pavel Kanzelsberger. Son interface semblable à celle de Adobe(r) Photoshop(r) séduira les habitués de ce logiciel de traitement d'images n'appréciant pas la présentation et les fonctionnalités de [[gimp|The GIMP]].

Il ne semble plus possible d'acheter Pixel à l'heure actuelle et son développement semble au point mort. La version complète de Pixel est toujours disponible en téléchargement aux utilisateurs ayant acquis une licence depuis leur espace membre. La version de démonstration proposée gratuitement est aussi encore téléchargeable ; cette version place un filigrane sur les images enregistrées.

===== Téléchargement et installation =====
  - Téléchargez l'archive d'installation de Pixel depuis son site officiel :
    * **Version de démonstration gratuite :** rendez-vous sur [[http://www.kanzelsberger.com/pixel/?page_id=4|la page des téléchargements]] et sélectionnez l'archive la plus récente pour Linux ;
    * **Version complète :** rendez-vous [[http://www.kanzelsberger.com/pixel/?page_id=21|dans l'espace membre]], connectez-vous à l'aide de votre adresse courriel et votre mot de passe, et téléchargez l'archive la plus récente pour Linux ;
  - Ouvrez l'archive de Pixel à l'aide de votre gestionnaire d'archives et extrayez-en le dossier **Pixel** :
    * Pour une installation pour votre compte d'utilisateur seulement : copiez le dossier vers votre dossier personnel (''/home/$USER/'') ;
    * Pour une installation pour tous les compte d'utilisateur : copiez le dossier vers le répertoire ''/opt/''. Les droits d'administration sont nécessaires pour réaliser cette opération ;
  - Pixel est désormais installé.



===== Utilisation =====
Pour exécuter Pixel, rendez-vous dans le dossier de Pixel (''/home/$USER/Pixel/'' ou ''/opt/Pixel/'') et double-cliquez sur le programme ''**pixel**''.
