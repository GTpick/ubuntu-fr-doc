====== Bluefish ======

{{  http://upload.wikimedia.org/wikipedia/commons/thumb/b/b9/Bluefish-icon.svg/200px-Bluefish-icon.svg.png?120}}
**Bluefish** est un [[éditeur de texte]] orienté vers le développement web, léger, rapide et libre supportant de nombreux langages au travers de sa coloration syntaxique (HTML, PHP, Javascript, CSS, SQL, XML, C, ColdFusion, Java, JavaServer Pages (JSP), Octave/Matlab, Pascal, Perl, Python, R, Ruby).


===== Installation =====

[[:tutoriel:comment_installer_un_paquet|installez le paquet]] **[[apt>bluefish]]**.

===== Configuration =====

Pour travailler directement sur un serveur FTP, il faut que[[:tutoriel:comment_installer_un_paquet| les paquets]] **[[apt>libgnomevfs2-0]]** et **[[apt>libgnomevfs2-extra]]** (support FTP, HTTP, SMB) soient  installés.

Ensuite définissez le répertoire de travail de votre projet comme suit : ''ftp://user:pass@server/someshare''

Exemple pour l'opérateur 1&1 : on rentre  ''ftp://uxyxyxyxy@machin.com'' où ''uxyxyxyxy'' est le nom d'utilisateur FTP et ''machin.com'' est le nom du serveur. Ensuite le mot de passe est demandé... en fait c'est [[:Nautilus]] qui sait gérer ce type de connexion.

===== Utilisation=====

Pour lancer l'application faites une recherche dans [[unity#tableau_de_bord_dash|le tableau de bord Unity]] avec le mot clé **Bluefish** ou via la [[:commande_shell|commande]] suivante :<code>bluefish</code>
