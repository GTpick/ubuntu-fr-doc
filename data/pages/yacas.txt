===== Présentation =====
Yacas est un logiciel libre de calcul formel.

===== Installation =====
==== Téléchargement des sources ====
Le code source de Yacas est disponible ici : [[http://yacas.sourceforge.net/homepage.html]]
==== Compilation ====
<code>tar zxvf yacas-latest.tgz
cd yacas-votreversion
./configure
make
make test (facultatif)
make install</code>

__Explications__ :
  - Extraire les fichiers de l'archive dans un dossier.
  - Se placer en console dans ce dossier
  - Préparer la compilation
  - Compiler le programme

Pour lancer le programme utilisez la commande :
<code>yacas</code>
