====== Gérer une TODO List ======

**Tasque** est un outil qui permet de tenir à jour une TODO list.

{{application:tasque.png|Tasque}}



===== Installation =====

==== Depuis les dépôts officiels ====

[[tutoriel:comment_installer_un_paquet|Installez le paquet]] **[[apt://tasque|tasque]].** 

===== Utilisation =====

**Tasque** se trouve dans **Applications** -> **Bureautique** -> **Tasque** du menu classique.

==== Lancer Tasque au démarrage ====

Dans le menu **Système** -> **Préférences**, sélectionnez **Sessions**. Puis allez dans l'onglet **Programmes au démarrage**, cliquez sur **Ajouter** et entrez : //tasque//

==== Fonctionnement ====

Tasque est très simple d'utilisation au démarrage, une icône se loge dans la zone de notification, un clic sur cet icône vous permet de faire apparaitre ou disparaitre la fenêtre principale de Tasque.

La barre d'outils permet l'ajout rapide d'une nouvelle tâche, et de la classer dans une catégorie.
Une fois la tâche créée, vous pouvez lui définir une date de fin (deadline) ainsi qu'une priorité entre 0 et 3.

==== Plugin Tomboy ====

Un plugin **Tomboy** permet de créer une nouvelle tâche à partir d'une note.
