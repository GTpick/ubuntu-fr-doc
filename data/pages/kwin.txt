**KWin** est le [[gestionnaire de fenêtres]] de [[KDE]].

===== Installation =====

Aucune installation n'est nécessaire sous Kubuntu, il est présent par défaut. Cependant, pour d'autres variantes d'Ubuntu, ou s'il a été désinstallé, il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt://kwin|kwin]]**

Utilisez les commandes suivantes :

''sudo apt-get install kde-window-manager''

''sudo apt-get install kde-workspace-bin''
