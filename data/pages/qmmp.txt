====== Qmmp ======

Qmmp est un [[lecteur audio]] dont l'interface est similaire à Winamp ou [[Xmms]].

{{audio:qmmp.png|}}

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation =====

==== via les dépôts ubuntu  ====

[[:tutoriel:comment_installer_un_paquet|Installez le paquet]] **[[apt://qmmp|qmmp]].**

====via les dépôts launchpad officiel ( dernière version )====
  * [[:PPA|Ajoutez le PPA]] **ppa:forkotov02/ppa**((https://launchpad.net/~forkotov02/+archive/ubuntu/ppa)) dans vos sources de logiciels ;
  * [[:tutoriel:comment_modifier_sources_maj#recharger_la_liste_des_paquets|Rechargez la liste des paquets]] ;
  * [[:tutoriel:comment_installer_un_paquet|Installez le paquet]] **[[apt://qmmp|qmmp]]**.



===== Utilisation=====

Lancez l'application depuis le menu //Applications -> son & vidéo -> Qmmp // ou //via// la [[:commande_shell|commande]] suivante :

  qmmp

===== Configuration =====

Vous pouvez configurer le logiciel en faisant un clic droit sur l'interface et en sélectionnant : **configuration**, ou en utilisant le raccourci clavier **Ctrl-p**. 

====Thèmes====
Vous pouvez aussi modifier l'apparence en téléchargeant et installant un thème disponible sur **[[http://browse.deviantart.com/customization/skins/media/winamp/classic/?order=9&alltime=yes|deviantart.com]].** 

====Modules visualisation - projectM====

Aller dans modules, visualisation et activer [[projectM]] - fare clic droit sur l'interface, visualisation et cocher [[projectM]]

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
