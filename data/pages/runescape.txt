====== RuneScape ======
[[wpfr>RuneScape]] est un jeu de rôle en ligne massivement multijoueur basé sur le langage Java et géré par la compagnie britannique Jagex. Reconnu comme étant le MMORPG gratuit le plus populaire au monde par le Livre Guinness des records, RuneScape compte environ quinze millions de comptes Free to play (jouables gratuitement) et plus de deux millions de comptes Pay to play (par abonnement payant).

Le jeu lance le joueur dans un univers fantastique, où il pourra combattre des monstres, découvrir et exploiter une panoplie d'habiletés ou tout simplement visiter l'immensité du jeu.

===== Installation =====
RuneScape devrait bien fonctionner si vous possédez les paquets OpenJDk Java 6 Runtime, OpenJDK Java Web Start et le greffon Java IcedTea pour le navigateur web. Vous devriez pouvoir aisément les trouver sur la Logithèque en tapant OpenJDK. 

Le jeu démarre dans tous bons navigateurs; bien configuré, il donnera pour certain un gain significatif en terme de performances.

Il existe un paquet .deb construit à partir des sources du lanceur pour Windows. Il est a noter que ce paquet contient du code propriétaire http://code.google.com/p/rs-client/.

=====Optimisations=====
Il est possible d'optimiser les performances graphiques du jeu à l'aide de quelques astuces.

Si votre ordinateur ne possède pas de carte graphique distincte, une puce intégrée Intel par exemple, il est préférable de mettre l'API du jeu en mode Software. L'idée est d'axer une plus grande partie des calculs sur le processeur et de réduire la charge de travail qu'une simple puce intégrée aurait à gérer.

Si vous possédez une carte graphique relativement correcte, il est préférable d'activer l'API [[opengl:OpenGL]]. Les cartes graphiques distincte tel que les carte [[nvidia:nVidia]] ou [[amd-ati:AMD-ATI]] possèdent généralement beaucoup plus de ressources pour traiter les modèles 3D, textures et autres, que les puces intégrées. Tout ce que l'on recherche est de réduire la tâche du processeur et de profiter plus amplement des capacités de la carte graphique.

Si votre processeur est capable de fonctionner à des fréquences variable, ce qui est le cas pour la majorité des CPUs récents, il sera probablement judicieux, si vous fonctionner avec l'API Software, de maximiser son utilisation lorsque vous jouez. Vous n'avez qu'à ajouter l'applet « Moniteur de changements de fréquence du processeur » sur votre barre GNOME.

<note important>
Faire rouler le processeur à fréquence maximale peut causer, à long terme, des problèmes de bruit de ventilateur, de surchauffe, et de bris de matériel. Il est donc fortement conseillé de se servir du mode Performance uniquement lorsque vous jouez et le mode Ondemand (sur demande), pour toute autres utilisations.
</note>

Pour de plus amples informations sur les optimisations possibles relatives aux cartes graphiques, vous pouvez consulter la page d'[[intel_graphics:Intel]], [[nvidia:nVidia]] ou [[amd-ati:AMD-ATI]].
