=====Éditeurs graphiques=====

  * [[:Gedit]] : éditeur de texte par défaut d'[[Ubuntu]].
  * [[:Gvim]] : vim avec une interface graphique.
  * [[:Kate]] : éditeur de texte par défaut de [[Kubuntu]].
  * [[Mousepad]] : éditeur de texte par défaut de [[Xubuntu]].
  * [[Leafpad]] : éditeur de texte par défaut de [[Lubuntu]].
  * [[wpfr>KWrite]] et [[wpfr>Kedit]] : éditeurs de texte conçus pour l'environnement [[:KDE]].
  * XEdit : éditeur basé sur emacs.
  * [[http://www.tuteurs.ens.fr/unix/editeurs/nedit.html|Nedit]]: éditeur aussi disponible sous Mac et Windows.
  * [[http://packages.ubuntu.com/precise/x2|x2]] éditeur simple et léger pour le bureau GNOME.
  * [[http://tea-editor.sourceforge.net/#news|TEA]] un éditeur multiplate-forme qui propose de nombreuses fonctionnalités.
  * [[Emacs]] : Emacs peut aussi être utilisé avec une interface graphique si on installe les paquets spécifiques

===== Éditeurs orientés développement =====

  * [[https://atom.io/|Atom]] : éditeur de texte orienté développement web, extrêmement modulaire.
  * [[Bluefish]] : éditeur de texte orienté développement web.
  * [[http://brackets.io/|Brackets]] : éditeur de texte libre, orienté Web-design, créé par Adobe
  * [[gPHPEdit]] : éditeur de texte orienté PHP/HTML/CSS.
  * [[Cssed]] : éditeur et validateur de feuilles de style CSS.
  * [[http://scribes.sourceforge.net/|Scribes]] : éditeur de texte équivalent à Textmate sous Mac OSX.
  * [[https://sourceforge.net/projects/quanta/|Quanta+]] : développement Web (XHTML, PHP, Javascript, CSS)
  * [[SciTE]] : pour développer dans plusieurs langages ((x)HTML, PHP, CSS, C, C++, Ruby…).
  * [[jEdit]] : IDE en Java.
  * [[KDevelop]] : IDE dévelloppée en Qt.
  * [[Geany]] : IDE rapide et simple en GTK.
  * [[NetBeans]] : IDE pour Java, Python, C, C++, Ruby, XML, PHP et HTML.
  * [[Gobby]] : Un éditeur de texte collaboratif qui permet à plusieurs personnes de travailler sur un même fichier de manière simultanée.
  * [[:Etherpad]] : Un autre éditeur de texte collaboratif.
  * [[apt>Medit]] : Un éditeur de texte qui permet de lancer un terminal dans la fenêtre d'édition ([[http://mooedit.sourceforge.net/|Site]]).
  * [[http://foicica.com/textadept/|Textadept]] éditeur multiplate-forme, supporte de nombreux langages, permet de "spliter" la fenêtre à convenance, [[:ppa]] disponible ([[http://www.webupd8.org/2013/02/try-textadept-fast-cross-platform-text.html|voir ici]]).
  * [[http://www.sublimetext.com/|SublimeText]] (propriétaire) : un éditeur de texte pour la programmation

===== Éditeurs de texte discrets ====

Les éditeurs de texte "discrets" ("distraction-free writing" en anglais) ont des interfaces épurées au maximum, le but étant de ne pas distraire visuellement l'utilisateur. Ils sont utilisés notamment par les littéraires qui passent beaucoup de temps à écrire.

  * [[:FocusWriter]]
  * [[http://uberwriter.wolfvollprecht.de/|UberWriter]] : un éditeur qui permet le formatage du texte (gras, italique, puces, titres)
  * [[http://www.codealchemists.com/jdarkroom/|JDarkroom]] 
  * [[http://code.google.com/p/marave/|Marave]] 
  * [[http://pyroom.org|Pyroom]] 
  * [[http://rubyroom.rubyforge.org/|Rubyroom]] 
  * [[http://textroom.sourceforge.net/|Textroom]]

===== Éditeurs en ligne de commande =====

  * [[Emacs]] : éditeur de texte très puissant, extensible et personnalisable. Emacs peut servir d'environnement de développement pour beaucoup de langages (LaTeX avec l’extension auctex, html…), de navigateur internet, de courrielleur, de client irc, de psychanalyste… 
  * [[http://www.jedsoft.org/jed/|JED]] Éditeur de texte multiplate-forme. 
  * [[http://joe-editor.sourceforge.net/|Joe]] (Joe's Own Editor) :: éditeur texte fondé sur WordStar, un ancien programme de traitement de texte très utilisé par le passé sous MS-DOS.
  * [[https://www.mirbsd.org/jupp.htm|Jupp]] disponible dans les dépôts depuis Ubuntu 12.04.
  * [[Nano]] : éditeur de texte en ligne de [[commande_shell|commande]] très simple, installé par défaut sur Ubuntu.
  * [[Vim]] : est très apprécié des développeurs pour toutes ses fonctions qui en font un très bon IDE (coloration syntaxique de 200 langages, complétion automatique, comparaison de fichiers, recherche évoluée, …) et est extensible par des scripts. 
  * [[midnight_commander|mcedit]] : L'éditeur de Midnight Commamder. 
  * [[apt>fte]] : L'éditeur de Borland. Ne supporte pas l'utf8. 
 
===== Éditeurs de texte spécifiques =====

  * [[http://www.trelby.org/|Trebly]] Un éditeur de texte conçu spécifiquement pour écrire des scénarios.
  * [[http://www.serandibsoft.com/cutepad1.html|CutePad]] Un éditeur orienté rédaction (texte en couleur, taille de police, insertion d'images etc.).
