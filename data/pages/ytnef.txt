====== TNEF : décoder les fichiers « winmail.dat » ======

Si un correspondant vous envoie un email depuis un logiciel comme MS Outlook, il est possible que vous receviez une pièce jointe **winmail.dat**.

**tnef** va vous permettre de décoder ce fichier et donc de récupérer les pièces jointes reçues...

===== Installation =====

Il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt://tnef|tnef]]**.

Une solution alternative consiste à installer le module LookOut de Thunderbird. https://addons.mozilla.org/fr/thunderbird/addon/lookout/
===== Utilisation =====

**Si c'est le lecteur TNEF qui a été installé**

  - Depuis votre lecteur de courriels, enregistrez la pièce jointe //winmail.dat// (dans votre dossier personnel par exemple) ;
  - ouvrez un terminal et depuis l'emplacement où vous avez enregistré le fichier, lancez la commande suivante :<code>tnef -f . winmail.dat</code> Les pièces jointes sont extraites dans le dossier courant.


**Si c'est l'extension LookOut qui a été installée dans Thunderbird**

Il suffit d'ouvrir le message dans Thunderbird, les pièces jointes apparaissent alors en clair.
