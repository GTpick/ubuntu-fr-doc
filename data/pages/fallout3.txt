====== Fallout 3 ======

**Fallout 3 New Vegas** est un jeu vidéo de type RPG. Il n'est pas disponible en version Linux. Cette page détaille l'installation de la version Windows sous Ubuntu.

===== Pré-requis =====
 
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  * Avoir installé et configuré [[Wine]].

===== Installation =====

==== Installation des dépendances =====

Avant de procéder à l'installation du jeu, il faut au préalable installer des bibliothèques d'exécution Windows.

À l'aide de [[Winetricks]], installez les bibliothèques suivantes : **directx9 vcrun2005 d3dx9 cc580**

Ensuite, téléchargez le fichier [[http://download.microsoft.com/download/5/5/8/55846E20-4A46-4EF8-B272-7F988BC9090A/gfwlivesetupmin.exe|msasn1.dll]]  et copiez-le dans le dossier ''**/home/mon_pseudo/.wine/drive_c/windows/system32**'', en écransant le fichier existant si besoin est.

Puis lancez l'[[wine#configuration_de_wine|outil de configuration de Wine]]. Dans l'onglet //Bibliothèques//, ajoutez la bibliothèque  **msasn1.dll**.

==== Installation du jeu ====
Pour installer Fallout 3 New Vegas, montez d'abord le CD-rom. Vérifiez que [[Wine]] reconnait bien le CD : dans l'[[wine#configuration_de_wine|outil de configuration de Wine]], onglet //Lecteurs//, vous devriez avoir une entrée **E : /media/cdrom**.

Parcourez le cd-rom, puis double-cliquez sur l'exécutable pour démarrer l'installation avec [[Wine]], et suivez les instructions.
