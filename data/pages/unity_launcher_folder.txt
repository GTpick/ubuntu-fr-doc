====== Ubuntu Unity Launcher Folders ======

Cette application permet d'aller beaucoup plus loin dans la personnalisation d'Unity, celle ci permet en effet d'avoir des "conteneurs" ou "tiroirs" dans lequel on place des applications par exemple par type (ex : Utilitaires, Outils, etc...)
Cette application à un équivalent sur Windows : [[https://www.stardock.com/products/fences/|« Fences de Stardock Corporation »]]


===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  * Avoir [[:tutoriel:comment_installer_un_paquet|installé le paquet]] **[[http://unity-folders.exceptionfound.com/unity-launcher-folders_1.0.3_all.deb|unity-launcher-folders_1.0.3_all.deb]]**.

<note warning>ne fonctionne plus depuis ubuntu 16.04</note>
<note tip>pour ubuntu 16.04 un paquet adapté est proposé par lynn dans [[https://forum.ubuntu-fr.org/viewtopic.php?id=2007155|cette discussion]] (post #3)</note>


===== Installation =====

Pour installer ce logiciel, il suffit de télécharger et installer le package Debian.
[[:tutoriel:comment_installer_un_paquet|installer les paquets]] **[[http://unity-folders.exceptionfound.com/unity-launcher-folders_1.0.3_all.deb|unity-launcher-folders_1.0.3_all.deb]]**.

Double cliquer sur le package pour lancer l'installation via la **Logithèque Ubuntu**

{{:unity:unity_launcher_folder.png?150|}}
===== Utilisation=====

pour exécuter ce programme il suffit d'ouvrir le [[unity|tableau de bord d'Unity]] (Dash en Anglais) (l'icône Ubuntu en haut de la barre Untiy)

et rechercher **Unity Launcher Folder** : {{:unity:unity_launcher_folder_icon.png?150|}}

On peut placer Unity Launcher Folders dans la barre Unity.

__L'interface **LauncherFolder Editor** est très dépouillée, vous disposez de 5 icônes :__
  - Nouveau tiroir
  - Ouvrir tiroir
  - Sauvegarde tiroir et ajout au lanceur
  - Paramètres tiroir
  - Prévisualisé tiroir

{{:unity:unity_launcherfolder_editor.png?400|}}

==== Création des "conteneurs" ou "tiroirs" : ====
(ex: //Utilitaires, Outils, etc...//)

Cliquer sur la **1ère icône** (//Nouveau tiroir//) attribuer un nom au conteneur.
Pour placer vos applications, ouvrir l'explorateur de fichier puis dans **Système de fichiers** ouvrir le répertoire **usr/share/applications.**
Faites glisser (drag and drop) les lanceurs de vos applications dans le "conteneur" qui vous convient.

<note>Dans le conteneur, on ne peut malheureusement pas déplacer les icônes celons notre convenance.</note>

==== Sauvegarde : ====

Pour sauvegarder vos conteneurs avec vos applications cliquer sur la **3ème icône** représentant un disque dur (//Sauvegarde tiroir et ajout au lanceur//)
Le conteneur va directement ce placer en bas de la barre Unity. (vous pouvez déplacer le conteneur ou vous voulez dans la barre Unity)
En survolant le conteneur avec la souris, le nom attribué apparaît.

{{:unity:unity_launcherfolder.png?400|}}
===== Configuration =====

Très peu de réglages à effectuer (à votre convenance)

En cliquant sur la **4ème icône** (//Paramètres tiroir//) vous pouvez par exemple choisir le type d'affichage de vos "tiroirs" (//Vertical, Horizontal, Box//)
Je n'ai pas modifier les autres réglages, à vous de voir et tester leurs actions.

{{:unity:unity_launcherfolder_editor_-_drawer.png?300|}}

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
