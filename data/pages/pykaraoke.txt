====== PyKaraoke ======



**PyKaraoke** est un logiciel simpliste de karaoké, il gère les fichiers *.kar, *.mid et cdg.
Qui ne gerre pas l'UTF8 correctement, utilisateurs modernes passez votre chemin.
===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation =====

Pour installer ce logiciel, il suffit d'[[:tutoriel:comment_installer_un_paquet|installer les paquets]] **[[apt://pykaraoke,timidity|pykaraoke et timidity]]**.

===== Configuration =====

Ouvrez Pykaraoke (via le menu //Applications -> Son & vidéo//), puis paramétrez :

  * **Dans le menu "Config" :**
    * Si vous voulez que la fenêtre de lecture se lance en plein-écran ou non.
    * L'encodage des caractères (défaut : iso-8859-1)

  * **Dans le menu "Addsong" :**
    * Dans quel dossier(s) se trouve vos fichiers .kar

    * Les autres options (types de fichier, regarder dans les *.zip ou non).
    * Rescannez les dossier ("Scan now") et sauvez ("Save all")


===== Utilisation =====

**Fermez toutes vos applications audio** (Y compris Firefox si vous utilisez Flash).

\\

Le logiciel s'utilise  2 manières :

==== Avec Pykaraoké ====

  * Utiliser la recherche (chercher "kar" pour que tout soit trouvé)
  * Utiliser le folder-view (peu pratique...)

Vous pouvez ainsi créer votre propre playlist !!


==== Avec pykaraoke_mini ====

Vous aurez directement accès à la liste de vos fichiers (si vous avez tout paramétré correctement) dans la fenêtre noire.



==== Raccourcis clavier ====


^ Touche ^ Raccourci ^
| Haut et bas | Se déplacer dans la playlist |
| Entrée | Jouer le morceau |
| 6 et 8 | Dézoomer |
| = et + | Zoomer |
| Echap et q | Quitter |
