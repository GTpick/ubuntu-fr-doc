====== shaiya évasion ======

{{banier10.jpg?600x100|shaiya evasion}}

shaiya évasion est un serveur privé, axé pvp [[wpfr>PvP|
Joueur contre joueur]], 100% gratuit, 100% francophone.

===== pré-requis =====

  *   disposer d'une connexion internet configurée et activée
  *  disposer de wine installé
  * créer un compte [[http://shaiya-evasion.com/index.php?file=User&op=reg_screen|ici]]

===== installation =====

  * télécharger le fichier : [[http://shaiya-evasion.com/index.php?file=Page&name=Telechargement|ici]]
  * décompresser l'archive

===== configuration =====

 lancer le fichier "updater.exe" avec wine, il va installer les mises à jours éventuelles. \\
Pour modifier les paramètres audio et vidéo, ouvrez, avec wine, le fichier config.exe (le jeu ne doit pas être lancé). 

===== utilisation =====

Pour jouer, il suffit de relancer le fichier "updater.exe", avec wine toujours, et le jeu va se lancer.

=====  désinstallation =====

pour désinstaller le fichier, il suffit de supprimer le dossier du jeu.

=====  Problèmes connus =====

*  La touche raccourci ' ne fonctionne pas avec un clavier français, seul l'utilisation d'un clavier us peut régler le problème. \\
* Les items placés sur le raccourci "w" s'active automatiquement lorsque vous avancez. \\
* On ne peut pas changer la résolution lorsqu'on est en jeux, il faut le faire via config.exe \\
