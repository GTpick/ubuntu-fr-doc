====== Emilia Pinball ======

{{ :jeux:Pinball.png?300}}
**Emilia Pinball** est un jeu libre de flipper. Il se joue exclusivement au clavier.

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]]
  * Disposer d'une connexion à Internet fonctionnelle

=====Installation=====

Il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt://pinball|pinball]]**.



=====Utilisation=====

Pour lancer le jeu, allez dans le menu //Applications -> Jeux -> Emilia Pinball//.

Avec les flèches du clavier et la touche Entrée, rendez-vous dans le menu "Load table" afin de charger l'un des 2 plateaux de jeu : "Professor" ou "Tux".
Puis sélectionnez le menu "Play" pour démarrer une partie.

Le jeu se joue par défaut avec les touches Shift (gauche et droite), Ctrl et Espace. Echap pour quitter la partie.

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, le configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
