====== Driconf ======

**Driconf**, pour //Direct Rendering Configuration//, est un utilitaire permettant de paramétrer le [[wpfr>Direct_Rendering_Infrastructure|Direct Rendering Infrastructure]] (DRI) de votre carte graphique.

Il permet notamment d'activer le support de la compression [[wp>S3TC]] (nécessaire à certains jeux) qui ne peut être activée par défaut dans les pilotes libres pour cause de licence, ou encore l'[[wp>HyperZ]] pour améliorer les performances des vieilles cartes graphiques.

===== Pré-requis =====
 
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation =====

Il vous suffit, pour cela, d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>driconf]]**.

===== Utilisation =====
 
Lancez l'application via le [[:unity#tableau_de_bord_dash|tableau de bord]] (Unity) ou via le [[:terminal]] (toutes versions d'Ubuntu) avec la [[:commande_shell|commande]] suivante : <code>driconf</code>

Lors du premier lancement il créera un fichier « .drirc » dans votre dossier personnel, dans lequel il enregistrera votre configuration.

==== Mode basique ====

Par défaut, vous pouvez configurer le DRI pour tout votre système :

{{:systeme:driconf_xenial.png?400|driconf en mode basique}}

==== Mode expert ====

Il existe un mode expert qui semble permettre de créer une configuration pour une application donnée :

{{:systeme:driconf-expert_xenial.png?400|driconf en mode expert}}

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer le paquet]] **driconf**. La configuration de l'application sera conservée ou supprimée selon la méthode de désinstallation que vous choisirez.
