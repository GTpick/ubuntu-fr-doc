====== Souris et Claviers Logitech :  Le logiciel HID Point======

**HIDPoint** est un pilote propriétaire accompagné d'une interface de configuration pour les claviers et les souris de marque Logitech. Ce pilote permet de bénéficier des mêmes avantages que les utilisateurs de Windows.

{{hidpoint..jpg|}}



===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]] ;
  * Disposer d'une connexion à Internet configurée et activée,
  * Pour les [[:architecture_materielle|architectures]] 64 bits entrez les commandes suivantes dans un [[:terminal]] : <code>sudo ln /lib/i386-linux-gnu/libpng12.so.0 /lib/i386-linux-gnu/libpng.so.3
sudo ln /lib/x86_64-linux-gnu/libpng12.so.0 /lib/x86_64-linux-gnu//libpng.so.3</code>
FIXME pour la dernière

===== Installation =====

Téléchargez le fichier binaire sur le site officiel de HID Point : [[http://www.hidpoint.com/hidpoint/download/hidpoint.html|ici]]. Il vous faudra entrer votre type de matériel aussi que votre version d'Ubuntu, le clavier et la souris utilisés et téléchargez le binaire : **hidpoint1-0.bin**. Puis :
  * Rendez le fichier [[:tutoriel:script_shell#executer_un_script|exécutable]].
  * Ouvrez un [[:terminal]] dans le dossier où se trouve le **.bin** (probablement le dossier Téléchargements) et lancez l'installation avec la commande : <code>./hidpoint1-0.bin</code>

Passer tranquillement les écrans jusqu'à l'écran informant que le noyau n'est pas compatible.

Cocher **Do Force Install** et continuer jusqu'à l'écran de mise à jour. Fermez la fenêtre, il n'y a pas de mise à jour.

===== Utilisation =====
 
Lancez l'application via le [[:unity#tableau_de_bord_dash|tableau de bord Unity]] avec le mot clé **hidpoint**.

===== Désinstallation =====
 
Lancez l'application via le [[:unity#tableau_de_bord_dash|tableau de bord Unity]] avec le mot clé **hidpoint**, et choisissez **Remove hidpoint**. Ou bien via le [[:terminal]] (toutes versions d'Ubuntu) avec la [[:commande_shell|commande]] suivante :
<code>sudo uninstall_hidpoint</code>
