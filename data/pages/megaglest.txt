====== Megaglest ======


**Megaglest** est RTS en 3D, qui vous plonge dans un monde fantastique où s'affrontent deux factions, l'une dominée par la magie, l'autre utilisant la technologie.
C'est un [[wpfr>Fork_(développement_logiciel)|embranchement (fork)]] de **Glest** au développement très actif.

{{applications:jeux:glest.jpg?500}}

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  * Disposer d'une carte vidéo supportant l'[[:accélération graphique]].

===== Installation =====

Il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt://megaglest|megaglest]]**.

Vous pouvez aussi ajouter le [[getdeb|dépôt PlayDeb]], afin d'obtenir une version plus récente.

===== Utilisation =====

==== Lancement ====
Lancez l'application via le [[:unity#tableau_de_bord_dash|dash]] (Unity) ou via le [[:terminal]] (toutes versions d'Ubuntu) avec la [[:commande_shell|commande]] suivante :
<code>megaglest</code>

==== Raccourcis clavier ====
**Quelques raccourcis clavier (extraits de /home/lolo/megaglest/docs/readme.txt)**

shift + clic gauche => ajouter une unité à la sélection

control + clic gauche => enlever une unité à la sélection

double clic => sélectionner toutes les unités du même type dans les environs

clic droit => ordre automatique (attaquer, couper du bois...)

control + chiffre => mémoriser un groupe d'unités

chiffre => sélectionner un groupe mémorisé

p => pause

i => sélectionner une unité inactive

b => sélectionner le bâtiment suivant

d => sélectionner l'unité blessée suivante

- + => modifier la vitesse du jeu
