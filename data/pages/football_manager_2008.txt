====== Football Manager 2008======


{{ http://www.lunivers-de-lentraineur.com/images/stories/FM2008/Logo_UDE_FM2008_200.png}}

**[[wpfr>Football Manager]]** est le jeu de management de football par excellence. Il est développé par [[http://www.sigames.com|Sports Interactive Games]] et édité par [[http://www.footballmanager.net|SEGA]].

Ce jeu n'est pas disponible sous GNU/Linux. Cette page explique comment installer la version d'essai (démo) ou la version payante à l'aide de logiciels créant un environnement Windows sous Ubuntu.


===== Pré-requis =====

  * Pour installer et jouer avec la démo ou le jeu il est primordial d'installer **[[:Wine]]** si ce n'est déjà fait.
  * Il vous faudra aussi installer **[[:winetricks]]**
  * Ensuite lancez les commandes suivantes :
<code>winecfg</code>
Ou dans le menu des applications allez sur //Wine// >> //configure Wine//. Une fenêtre s'ouvre, il faut ensuite aller dans l'onglet lecteur : 
{{http://www.manager-online.fr/images/tuto/linux/linux1.gif|}}

Une fois dans l'onglet lecteur : 
  - Cliquez sur Autodétection
  - Choisir l'unité /media/cdromX (X est un chiffre représentant votre numéro de lecteur) correspondant au lecteur où est inséré le CD.
  - Vérifier que le type est bien noté CDROM


===== Installation =====

Maintenant nous pouvons lancer l'installation du jeu. Pour cela il faut aller dans le contenu du CDROM. Il faut utiliser le terminal comme ceci :

<code>cd /media/cdrom0</code>

Et ensuite il suffit de taper une deuxième commande :

<code>wine setup.exe</code>

Vous pouvez sinon vous rendre dans //Raccourcis// >> //CDFM2008//. Trouvez install.exe, faites clic-droit dessus, puis cliquez sur //Ouvrir avec une autre application// et choisissez //Wine//.

Dès lors une fenêtre s'ouvrira, il vous suffit de suivre l'étape d'installation comme sur Windows :

{{http://www.manager-online.fr/images/tuto/linux/linux2.gif|}}


===== Jouer =====

Pour lancer le jeu un lien est créée dans le répertoire utilisateur.
Vous pouvez aussi le lancer à partir de ce chemin Wine > Programmes >Sport Interactive > Football Manager 2008

Et le jeu se lance normalement !

{{http://www.manager-online.fr/images/tuto/linux/linux3.gif|}}


===== Installer un patch =====

Pour cela téléchargez le patch (version PC). Ensuite placez-vous dans le répertoire à l'aide du terminal, si vous avez mis l'application sur le bureau, tapez :

<code>cd Bureau/</code>

puis

<code>wine NomExecutable.exe</code>


Si vous l'avez installé ailleurs que sur le bureau adaptez simplement la première commande.

Vous pouvez aussi cliquez-droit sur le fichier en question puis cliquez sur Ouvrir avec une autre application et choisissez Wine.
