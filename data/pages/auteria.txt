====== Auteria ======

Auteria est un rpg 3D sous licence commerciale.

Vous êtes sur la terre de Tergratia, vous obtiendrez une profession et ferez évoluer votre personnage au fil des quêtes qui vous seront proposées.

{{ http://www.auteria.com/shots/rel1_9.jpg }}

=====Installation=====

    * [[http://www.auteria.com/|Télécharger le fichier pour linux]]
    * Décompressez le fichier dans le dossier de votre choix.
    * Si vous avez installer une version 64bits d'Ubuntu((ou dérivé)), [[:tutoriel:comment_installer_un_paquet|installez également le paquet]] **[[apt>ia32-libs]]**.
    * Suivez les instructions incluses dans le fichier **README.LINUX**



<note>**Lors du premier lancement l'application téléchargera les autres éléments requis**

{{ :jeux:auteria:update_auteria.png?400 |}}
 </note>
=====Desinstallation=====

Pour supprimer cette application, il suffit de supprimer le dossier contenant l'application.
