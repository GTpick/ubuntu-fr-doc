====== SecureW2 ======

SecureW2 est utilisé pour sécuriser des réseaux et propose une méthode d'authentification. Ce logiciel n'étant pas disponible pour linux, vous trouverez ici une méthode pour vous connecter à un réseau utilisant SecureW2, tirée d'une doc de [[http://docs.google.com/Doc?docid=dfmq2m4v_128g3m3bjf5&hl=en|Copenhagen Business School]].

===== Méthode =====


  * Téléchargez [[http://cbs.dk/content/download/64739/902957/file/SecureW2_TTLS_320_2KXP.zip|SecureW2]]
  * Décompressez l'archive et placez le fichier //radiator.cer// sur votre disque dur
  * Dans Gnome Network Manager choisissez le réseau auquel vous souhaitez vous connecter
  * Network Manager demande les détails, remplissez comme suit :
    * //Sécurité : WPA et WPA2 entreprise//
    * //Authentification : Tunneled TLS//
    * //Identité anonyme : - laisser vide-//
    * //Certificat du CA : aller chercher le certificat là où vous l'avez enregistré//
    * //Authentification interne : PAP//
    * //Nom d'utilisateur : votre nom d'utilisateur du réseau//
    * //Mot de passe : votre mot de passe//
  * Validez et vous devriez être connecté
