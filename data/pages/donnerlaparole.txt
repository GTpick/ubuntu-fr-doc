====== DonnerLaParole ======

{{ :applications:dlp01.png?150}}

**Donner La Parole ** est un logiciel multiplateforme sous licence Commons Public((Avec limitation à un usage civil uniquement.Tout usage militaire est prohibé)) qui permet de saisir des mots et des phrases facilement et de les restituer en sons.\\

Il y a 6 fonctionnalités principales :
  * un ensemble de claviers virtuels :
    *  un clavier d'ordinateur ;
    *  un clavier type téléphone ;
    *  un clavier pour la langue des signes ;
    *  des claviers pictogrammes personnalisables.
  * un ensemble de dictionnaires pour pouvoir sélectionner rapidement des mots sans avoir besoin de saisir toutes les lettres (appelé aussi saisie prédictive ou saisie intuitive) ;
  * un gestionnaire de phrases courantes, pour éviter de devoir saisir continuellement les mêmes mots et les mêmes phrases ;
  * un synthétiseur de voix, qui prononcera les phrases ou les mots saisis ;
  * un composeur de phrases iconiques ;
  * un gestionnaire de sélection par bloc (ou défilement par bloc, ou scan) pour une saisie sans déplacement de la main.

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]] ;
  * Disposer d'une connexion à Internet configurée et activée ;
  * Disposer d'un moteur [[:java|JAVA]] actif.

===== Installation =====
L' installation de l'application peut s'effectuer de deux façons:
  * un fichier exécutable lancé par [[:Java]] qui fonctionnera sur la plupart des plateformes;
  * un [[:paquet]] debian à installer sur la machine.

 ==== Installation via JAVA ====
  - Télécharger la dernière version du fichier .jar depuis le [[http://sourceforge.net/projects/donnerlaparole/files/donnerlaparole/|site du projet]];
  - Rendre le fichier exécutable en modifiant les [[:permissions]];
  - Exécuter le fichier ((dans [[:nautilus]] Cliquer droit sur le fichier / Ouvrir avec/ choisir //sun java 6 runtime// ou //openJDK JAVA6 runtime//)).

==== Installation via paquet Debian ====
  - Télécharger la dernier version du paquet .deb depuis [[http://sourceforge.net/projects/donnerlaparole/files/donnerlaparole/|site du projet]];
  - [[:tutoriel:comment_installer_un_paquet|Installez le paquet]] depuis le dossier où vous avez téléchargé le paquet. 



===== Utilisation =====
L'utilisation du logiciel est très intuitive :
  * **Utilisation du clavier Alphanumérique ou langage des signes**
    - Tapez votre texte à l'aide de l'interface choisie,
    - Cliquez sur le logo "// Parler //" et le logiciel restituera votre texte vocalement.

  * **Utilisation des claviers additionnels** \\ À chaque pictogramme est associé une phrase, une idée... Cliquez sur le pictogramme et le texte associé sera prononcé voire affiché quand celà est prévu.\\ Il vous est loisible de créer vos propres "claviers" en fonction de vos besoins personnels. Pour plus d'informations à ce sujet, reportez-vous à la documentation du projet : [[http://sourceforge.net/apps/trac/donnerlaparole/wiki/GuideDePersonnalisation|Guide de Personnalisation]]

===== Configuration =====
Dans le coin supérieur droit un bouton "// Réglages //" vous donnera accès aux panneaux de configuration.\\
Vous pourrez y régler notamment la "// Voix //", le fonctionnement de la souris et quelques autres options. 

===== Clavier additionnel =====

Vous trouverez des claviers additionnels, dont la bibliothèque de pictogrammes AXELIA((Très bien fournie et a été adaptée aux spécificités du logiciel)), sur le site du projet à [[http://sourceforge.net/projects/donnerlaparole/files/claviers|cette adresse]].\\
Choisissez les fichiers ayant pour extension //**.jar**//

Pour l'installation voir la procédure d' installation [[#Installation_via_JAVA|via JAVA]]\\
Que l'installation du logiciel de base soit faite par JAVA ou .deb, l'installateur mettra les composants au bon emplacement.

=====Désinstallation=====
  * Si vous avez choisi la [[#Installation_via_java|méthode d'installation via JAVA]], un désinstalleur est accessible dans la rubrique de lancement du logiciel.
  * Si vous avez choisi la [[#Installation_via_paquet_Debian|méthode d'installation via paquet Debian]], pour supprimer cette application, il vous faudra [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]].\\ La configuration de l'application sera conservée ou supprimée selon la méthode de désinstallation que vous choisirez.
