====== igotu2gpx ======
{{ https://launchpadlibrarian.net/24373701/igotu2.jpg|Traceur I-GotU }}
**igotu2gpx** est une application pour l'exploitation des traces du traceur GPS I-GotU de [[http://www.i-gotu.com/|Mobile Action]], une interface graphique (GUI) (en Qt) est disponible.


{{ :applications:igotu2gpx_xenial.png?400 }}

Actuellement, il permet dores et déjà de :
  * afficher la configuration du traceur,
  * récupérer les traces enregistrées dans la mémoire du traceur,
  * exporter les traces (//tracks// et //waypoints//) au format gpx.
  * La possibilité d'effacer les traces mémorisées dans le traceur. 

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation =====
Pour installer igotu2gpx :
  * Ajoutez les [[:ppa]] suivants :
    * Pour Ubuntu 12.04 et 14.04 [[:ppa#ajout_d_un_ppa_a_vos_sources_logicielles|Ajoutez le PPA]] **ppa:igotu2gpx/daily**((https://launchpad.net/~igotu2gpx/daily/+archive/ppa)) dans vos sources de logiciels.
    * Pour Ubuntu 16.04 et suivantes  [[:ppa#ajout_d_un_ppa_a_vos_sources_logicielles|Ajoutez le PPA]] **ppa:hackie/igotu2gpx-ppa**((https://launchpad.net/~hackie/igotu2gpx-ppa/+archive/ppa)) dans vos sources de logiciels.
  * Puis [[:tutoriel:comment_modifier_sources_maj#recharger_la_liste_des_paquets|rechargez la liste des paquets]],
  * Enfin installez [[:tutoriel:comment_installer_un_paquet|installez le paquet]] **[[apt>igotu2gpx]]**.

===== Utilisation=====

(Il semblerait pour le moment il faille être root pour accéder aux traceur).

(edit au 26/2/2017 par Markorki)
Cette restriction n'est plus vraie depuis (selon ma mémoire, qui peut se tromper) au plus tard la fin 2015 (je m'en sers sous mon compte normal depuis cette date ou plus tôt).

__1. En ligne de commande :__

pour afficher la configuration du traceur : <code>igotu2gpx info</code>

pour exporter le contenu du traceur au format gpx : <code>igotu2gpx dump --gpx > track.gpx</code>


__2. Avec le GUI :__
Lancez l'application via le [[:unity#tableau_de_bord_dash|tableau de bord]] (Unity).


===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. La configuration de l'application sera conservée ou supprimée selon la méthode de désinstallation que vous choisirez.
