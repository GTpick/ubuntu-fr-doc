====== Project IGI ======

{{:applications:jeux:projectigi_pochette.jpg}}

<note important> Jeu non compatible Linux, créé pour Windows. Préférez les jeux natifs Linux</note>

Project I.G.I. (titre anglais : Project I.G.I.: I'm Going In) est un jeu vidéo d'infiltration en vue subjective sorti en décembre 2000 sur PC. Le jeu a été développé par Innerloop Studios et édité par Eidos Interactive.

Le jeu est un précurseur en matière de jeu vidéo ultra réaliste notamment sur les armes et les situations de combat, il s'est fait remarquer par sa difficulté de jeu et par ses qualité graphiques utilisant parfaitement la profondeur de champ de vision permise par l'utilisation d'un moteur 3D de simulateur d'avion développé précédemment par Innerloop pour Joint Strike Fighter.

Pour jouer à Project IGI, il vous faut au préalable installer [[:Wine]] [[:tutoriel:comment_installer_un_paquet|installez le paquet]][[apt://wine]]



===== Installation =====

Pour installer Project IGI, insérer votre CD dans votre lecteur et allez à la racine du CD :

    cd /media/cdrom0

Et lancer le fichier d'installation du jeu de cette manière :

    wine autorun.exe

Poursuivez l'installation avec Project IGI. 

Une fois le jeu installé,  il ne vous reste plus qu'à lancer le jeu de cette manière :

    cd ~/.wine/drive_c/Program\ Files/Eidos\ Interactive/Project\ IGI/pc/
    wine IGI.exe


voila, vous pouvez désormais jouer. :-)
