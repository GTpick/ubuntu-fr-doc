====== Qucs ======

Logiciels de [[cao|Conception Assistée par Ordinateur]] pour les circuits électriques et électroniques 2D.

===== Installation =====
Ce logiciel est présent dans les dépôts: il suffit [[:tutoriel:comment_installer_un_paquet|d'installer le paquet]] **qucs** [[apt://qucs|{{apt.png}}]]

<note important>Qucs n'est pas présents dans les dépôts officiels de la 12.10 et de la 13.10 sûrement a cause d'une dépendance non satisfaite : libqt3-mt.

Une solution est donc d’installer manuellement la version de développement (par essence instable) de ce paquet trouvable ici [[https://launchpad.net/ubuntu/quantal/amd64/libqt3-mt/3:3.3.8-b-8ubuntu3]], puis de faire de même pour Qucs [[https://launchpad.net/~archetipo/+archive/ppa/+build/3594642]] (indisponible le 17 janvier 2014) ou http://packages.debian.org/squeeze/qucs</note>


<note important>Une autre solution pour avoir une version stable de Qucs, en ajoutant les dépots PPA de Qucs :
      sudo apt-add-repository ppa:qucs/qucs
      sudo apt-get update
      sudo apt-get install qucs
     
</note>
===== Utilisation =====
{{:qucs:qucs.png|}}
<note important>A compléter</note>
Tout d'abord lancez le logiciel, comme vous pouvez le remarquer l'interface est assez intuitive.
Il suffit de glisser-déposer les composants nécessaires sur la surface du schéma.
Il existe plusieurs type de composants:
  * composant discret qui contient tout les composant dits "passifs" (en effet un AO n'est pas si passif que ça) vous y retrouverez donc tout ce qui est résistances, bobines, etc.
  * sources nous trouvons ici tout ce qui peut être une source de tension ou d'intensité
 


===== Désinstallation =====
Il suffit de désinstaller le paquet **qucs**.
