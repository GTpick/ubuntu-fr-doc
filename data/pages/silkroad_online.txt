====== Silkroad Online ======

{{silkroad.jpg|}}

Qui n'as pas envie de tester ce fameux MMORPG ? Nous allons voir comment l'installer avec l'aide de PlayOnLinux.

===== Téléchargement du client =====

Pour commencer, sachez que vous ne pourrez pas aller sur le site officiel avec Firefox ou Opéra. Seul Internet Explorer peut y accéder.

Pour pallier ce problème, soit vous téléchargez le client officiel depuis Windows, ou bien en émulant Internet Explorer 6 (disponible dans le dépot WorkOnLinux de PlayOnLinux).

<note tip>ON peut aussi utiliser l'extension Firefox User Agent Switcher[[https://addons.mozilla.org/fr/firefox/addon/59]] en simulant Internet Explorer (Outils > Default User Agent > choisir IE8) pour naviguer sur le site officiel</note>

[[http://www.silkroadonline.net/|Site officiel]]

Créez un compte, puis téléchargez le client en ddl si possible (~950 mo).

<note tip>Sachez que les liens proposés par le site officiel sont très lents, n'hésitez pas à chercher le client sur un site plus rapide. Ma version actuelle (13/01/08) est la 1.140.</note>

===== Installation du client =====

Démarrez PlayOnLinux.

Allez dans Installer > Jeux > SilkroadOnline

Le script va vous demander si vous possédez le client, répondez "oui" et grâce au bouton parcourir sélectionnez le fichier en question.

Normalement pour l'installation, il n'y a aucun problème, quelle que soit la version de Wine.

<note important>Il ne faut pas encore démarrer le jeu!</note>


===== Assignation de la bonne version de Wine =====

Démarrez PlayOnLinux.

Cliquer sur SilkroadOnline dans la liste du menu principal, et allez dans Outils > Gérer les versions de Wine > Installer > 0.9.50

POL va installer la version de Wine 0.9.50

Une fois fini, allez dans : Outils > Gérer les versions de Wine > Assigner > SilkroadOnline > 0.9.50

Après, vous pouvez démarrer le jeu, grâce au menu principal.

<note warning>Pour ma part, j'ai dû redémarrer deux fois le jeu pour qu'il lance les mises à jours (la première fois je n'avais pas le bouton "start"). La deuxième fois j'ai eu une minute de mise à jour, et j'ai eu accès au bouton "Start".</note>

Le jeu fonctionne très bien chez moi, quelques rares lag dûes au serveur. J'ai l'impression que les files d'attentes sont moins longues avec Linux.

<note warning>Depuis quelques années, la connexion est devenue plus difficile, les serveurs étant surchargés ("Crowded") malgré leur grand nombre. Pour les motivés, l'ultime recours semble donc l'achat d'un compte premium, avec une priorité de connexion aux serveurs (Temps d'attente pratiquement nul).</note>

En cas de problème, le forum de PlayOnLinux est à votre disposition:

[[http://www.playonlinux.com/fr|PlayOnLinux]]

Pour commencer à jouer à Silkroad, voilà un bon site en français :

[[http://silkroad-france.servhome.org/|Silkroad France]]

Bon jeu à tous!
