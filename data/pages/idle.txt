====== IDLE ======
**IDLE** est un éditeur de programmes [[python|python]] multiplate-forme (Windows et Unix).

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  * Avoir [[:tutoriel:comment_installer_un_paquet|installé le paquet]] **[[apt://python|python]]** et/ou **[[apt://python3|python3]]**.

===== Installation =====

La version d'IDLE à installer dépend de votre version de Python. 

==== Connaitre sa version de python ====

<note>Python 2 et 3 peuvent être installés côte à côte</note>

Pour python 2.7 et antérieur :
<code>python --version</code>

Pour python 3 (3.0 et supérieur) :
<code>python3 --version</code>

==== Installer IDLE ====

Maintenant que vous connaissez votre version de Python, installez la version correspondante d'IDLE. 

<note>
On peut [[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt://idle|idle]]** et/ou **[[apt://idle3|idle3]]** (respectivement pour python et/ou python3) qui détecte automatiquement les versions installées de python
</note>


^ Version de Python ^ Paquet à [[:tutoriel:comment_installer_un_paquet|installer]] ^
| Python 2.4.x | [[apt://idle-python2.4|idle-python2.4]] |
| Python 2.5.x | [[apt://idle-python2.5|idle-python2.5]] |
| Python 2.6.x et 2.7.x | [[apt://idle-python2.6|idle-python2.6]] ou [[apt://idle-python2.7|idle-python2.7]] |
| Python 3.0.x | [[apt://idle-python3.0|idle-python3.0]] |
| Python 3.1.x | [[apt://idle-python3.1|idle-python3.1]] |

Par exemple, pour python 2.6.2, il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt://idle-python2.6|idle-python2.6]]**.


===== Utilisation=====
Lancer IDLE par le **menu** ou en **console** (X.X est à remplacer par le numéro de version à lancer) :
<code>idle-pythonX.X</code>
<note tip>ou si vous avez installé les paquets correspondants :
<code>idle</code><code>idle3</code> </note>
