====== Childsplay ====== 

Childsplay est une suite de jeux éducatifs de type [[Gcompris]].

Il contient des activités variées de type mémory, manipulation du clavier, reconnaissance image / son... Exercices disponibles suivants le système de plug-ins.

{{childsplay_hardy_01.png|mode fenêtre}}

Childsplay est disponible sous Linux, FreeBsd, Mac OS et Windows en différentes langues sous licence GNU/GPL.

===== Configuration =====

<note warning>ATTENTION Childsplay est prévu pour fonctionner en mode plein écran avec une résolution de 800 x 600 ce qui peut poser quelques problèmes sur des écrans LCD.</note>

Il vous faudra [[:tutoriel:comment_editer_un_fichier|éditer le fichier]] **childsplayrc**  qui se trouve dans .childsplay/ConfigData du répertoire utilisateur (/home). Si vous utilisez nautilus, pensez à afficher les fichiers cachés !

==== Les configurations possibles ====

  * **Pour le mode fenêtré** : Section [default]

trouver :
<code>fullscr  = 1</code>

remplacer par
<code>fullscr  = 0</code>


  * **Pour exclure des jeux du programmes** :

rechercher :
<code>### Exclusion of games ############ </code>

Ajouter à la liste :
<code>exclude_games = "le_nom_du_jeux_que_vous_voulez_exclure"</code>

comme indiqué dans le fichier.
  * **Pour le mode Kiosque** :

rechercher :
<code>kiosk_mode = no</code>

remplacer par :
<code>kiosk_mode = yes</code>

Attention le mode kiosque désactive le clavier, je n'ai pas testé.

<note>La voix française (légèrement du sud) ne prononce apparemment pas les lettres accentuées.</note>

===== Utilisation =====

Au lancement de childsplay, une page d'accueil vous proposera de sélectionner un jeu parmi les 14 que comporte la version installée par le dépôt. D'un simple clic, l'enfant se retrouvera confronté à un jeu de mémoire visuel ou sonore, un puzzle, des situations de reconnaissance de caractères (différentiation des majuscules et des minuscules) reconnaissance des chiffres...

Cerise sur le gâteau, un pacman glouton de lettres à manger dans l'ordre ainsi qu'un ping-pong digne des premiers jeux vidéo... :-)

Chaque jeu réussi donne des points qui sont additionnés à l'écran sous forme d'un score. L'enfant peut arrêter le jeu en cours grâce au panneau stop.

Une aide est disponible pour chaque jeu en cliquant sur le point d'interrogation (l'éléphant).

<note tip>Tous les boutons de la souris donnent le même résultat, l'enfant peut donc utiliser celui qui lui convient le mieux.</note>

L'arrêt du jeu est provoqué par un clic sur le panneau stop, une confirmation est demandée afin de quitter définitivement childsplay.
