====== Annuaire des utilisateurs de Jabber de la communauté ubuntu-fr.org ======
===== Introduction =====
Cette page fait suite à [[http://forum.ubuntu-fr.org/viewtopic.php?id=271839|cette discussion]] sur le forum.

Les adresses affichées ici ont été récoltées sur [[http://forum.ubuntu-fr.org/viewtopic.php?id=5479|une autre discussion]] du forum.

N'hésitez pas à modifier cette page vous même pour ajouter/modifier/retirer votre adresse Jabber de cette liste.

Il ne s'agit pas d'adresses e-mail, mais d'adresses pour [[jabber|la messagerie instantanée Jabber]]. Il n'y a donc pas de risque de recevoir du [[http://fr.wikipedia.org/wiki/Pourriel|pourriel]] **sauf si votre adresse Jabber correspond à une adresse e-mail**. C'est notamment le cas pour les adresses Gmail. Dans ce cas, utilisez un format du type //nom POINT prenom AT gmail POINT com// par exemple.

===== Spam Jabber =====

**En ajoutant votre adresse sur cette page vous consentez à ce que ce vous soyez contacté par les visiteurs quelconques de cette page. Mais aussi à des robots et dans ce cas il vous suffira de supprimer l'adresse Jabber pour voir le spam cesser. **
 

Notamment grâce à un script qui permet de récupérer toutes les adresses Jabber ( correctement écrit et pour une bonne cause ) présent sur cette page pour les ajouter à la liste de contacts d'un utilisateur.
Le script est disponible [[http://forum.ubuntu-fr.org/viewtopic.php?id=283308&p=1|ici]].
Si vous ne voulez pas que ce script capture votre adresse Jabber vous pouvez simplement modifier votre adresse sous une forme littérale :

<pseudo> at <domaine> point <com>

Ainsi le script mis à la disposition ignorera votre adresse.

==== Utilisation répétée du script = **Inutile** ====

Une fois que vous avez utilisé le script, il est inutile de le relancer pour remettre à jour les contacts, en effet, tous ceux qui s'inscrivent sur cette page, supposant connaitre l'existence de ce script, ajouteront **ou non** via ce petit script vos adresses à la leur. Vous recevrez donc une demande d'autorisation et votre liste sera tenue à jour sans rien à faire !

Les nouveaux contacts ( moment à partir duquel vous aurez lancé et ajouté les contacts de cette page et supposant que vous ne relancerez pas ce script ) qui n'auront pas lancé le script tout en s'inscrivant ( et dont l'adresse sera informatiquement ( mot qui n'existe pas mais qui est adapté ) correct ) auront fait le choix de ne pas encombrer leur messagerie mais ils auront aussi à gérer l'ajout de nouveaux contacts ( moment à partir duquel ils auront lancé et ajouté les contacts de cette page et supposant qu'ils ne relanceront pas ce script ).

Il arrive aussi que certains utilisateurs laissent leur adresse Jabber informatiquement correct mais ils vous suppriment de leur liste, si vous relancer le script vous relancerez une demande d'ajout à ce contact qui, avec <del>agacement</del> tact,  refusera la demande, il en résultera des contacts qui seront hors ligne indéfiniment et qui ne vous auront pas sur leur liste.
