====== GPS Sport Trace ======

**GPS Sport Trace** est une application open source sous licence [[http://www.gnu.org/licenses/old-licenses/lgpl-2.1.html|GNU LGPL]] pour l'exploitation des traces GPS multisport pour ForeRunner et autre GPS sportif reconnu par Gpsbabel.
Utilisable sur de nombreux sports, de la simple randonnée jusqu'au sport mécanique.
Permet de visualiser les traces sur une carte IGN et d'obtenir des Statistiques et des graphiques de vos performances. 

Elle communique avec votre GPS ( avec l'aide de gpsbabel ) afin de récupérer les traces et permet ensuite de générer des cartes, des graphes et des statistiques sur les traces.

Voici une liste non exhaustive des fonctionnalités de GPS Sport Trace :
  * Récupération des traces depuis votre GPS (tout GPS reconnu par gpsbabel ) ;
  * La possibilité de rejouer les traces ;
  * Représentation graphique des traces entièrement personnalisables ;
  * Affichage des traces sur un fond cartographique IGN ;
  * Affichage des traces sous Google Map ;
  * Édition des paramètres des traces ;
  * Édition des points des traces ;
  * Gestion de Waypoint ;
  * Possibilité de visualiser des photographies sur une trace ;
  * Affichage de graphiques représentant vitesse, altitude, rythme cardiaque, accélération, pente, etc. ;
  * Statistiques filtrables par catégorie et date ;
  * etc

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  * Avoir [[:tutoriel:comment_installer_un_paquet|installé le paquet]] **[[apt>gpsbabel]]**.

===== Installation via dépôt PPA (Ubuntu 10.04 LTS) =====

Le logiciel ne semble disponible que pour [[lucid|Ubuntu 10.04 LTS]], //via// un dépôt PPA.
//Il fonctionnait aussi sous Ubuntu 10.10 en installant le paquet pour lucid. Mais depuis... ? -> Dépendance non satisfaite sous la 13.10 (gmabas2-runtime(>=1.9.48))...//

<note important>Ceci n'est pas un dépôt officiel du projet UBUNTU.</note>
[[:PPA|Ajoutez le PPA]] **ppa:stephane-brunner/lucid**((https://launchpad.net/~stephane-brunner/+archive/lucid)) dans vos sources de logiciels, puis [[:tutoriel:comment_modifier_sources_maj#recharger_la_liste_des_paquets|rechargez la liste des paquets]].

Enfin, [[:tutoriel:comment_installer_un_paquet|installez le paquet]] **[[apt>gps-sport-trace]]**.
  

===== Configuration =====

Tout se règle dans les options du logiciel.

//En cas de problème, les options sont stockés dans le fichier// **~/.config/gambas/GPS-Sport-Trace.conf**.

===== Utilisation=====

Lancez l'application depuis le menu //Applications -> Graphisme -> GPS Sport Trace// ou //via// la commande suivante :

  GPS-Sport-Trace.gambas

Les traces sont stockées au format texte dans le répertoire **~/Documents/Forerunner**.

Pour activer la visualisation GoogleMap dans l'application, il faut cocher la case GoogleMap dans le menu Cartographie.

<note> Si vous obtenez un message d'erreur [45] File or directory does not exist.  fmain.?.0   . il faut simplement créer un dossier GPS_Sport_Trace  dans le dossier Documents </note>

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. La configuration de l'application sera conservée ou supprimée selon la méthode de désinstallation que vous choisirez.
