===== Description =====

Pessulus est un éditeur de verrous. Il permet de limiter certaines fonctions de votre ordinateur et de fournir une configuration par défaut pour les bureaux des utilisateurs.

{{:pessulus.png|}}

Fonctions principales:
  * Général: Désactiver certaines fonctions du système.
  * Tableau de bord: Limiter des fonctionnalités, verrouillage pour éviter les modifications.
  * Navigateur Web Epiphany: Brider le navigateur et ses possibilités.
  * Économiseur d'écran: Fonction de session et de verrouillage d'écran.

===== Utilisation  =====

Vous pouvez retrouver l'éditeur de verrous dans le menu\\
Système -> administration -> Éditeur de verrou

**Conseil pratique:**

Avant de verrouiller votre système je vous recommande quelques actions pour une sécurité optimale.

  * Créer un compte utilisateur spécifique
  * Préparer votre tableau de bord pour l'utilisation finale.
  * Éditer le menu gnome (clic droit sur le menu -> Éditer les menu). Supprimez toutes les applications susceptible de modifier le système. n'oublier pas d'enlever le lien vers pessulus et le terminal.
  * modifier le compte utilisateur en desktop-user. (Système -> Administration -> Utilisateurs et groupes).
