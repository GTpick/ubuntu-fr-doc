====== Volubil ======

**Volubil fait partie de la suite de logiciels éducatif du [[le_terrier|Terrier]].**

<note>(Ci-dessous, extrait du site officiel http://www.abuledu.org/leterrier/volubil )</note>

{{ http://www.abuledu.org/_media/leterrier/volubil/production_et_lecture_par_phrase.png?350}}

Volubil est un logiciel éducatif fondé sur une synthèse vocale, c'est à dire un logiciel permettant de lire un texte écrit à l'écran.

//Attention : problèmes de droits liés à la licence du moteur de synthèse vocale.//

Volubil se compose de 2 modules : lecture et écriture.
    * En mode lecture, le texte peut être lu partiellement ou en totalité.
    * En mode écriture le logiciel peut oraliser toutes associations de lettres quel qu'elles soient.


//**Pour une description complète et des pistes pédagogiques, voir [[http://www.abuledu.org/leterrier/volubil|la documentation sur le site officiel du Terrier d'Abulédu]].**//

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  * Avoir activé l'accès aux [[le_terrier#ajouts_des_depots|dépôts du Terrier]].
  
===== Installation =====

[[:tutoriel:comment_installer_un_paquet|Installez le paquet]] **[[apt>abuledu-volubil|abuledu-volubil]]**.

===== Utilisation =====
Comme tous les logiciels du Terrier, vous retrouverez un lanceur dans le menu **Applications --> Éducation**

===== Configuration =====
Comme les autres logiciels du Terrier, //volubil// permet à l'enseignant d'éditer les exercices. Pour cela, vous devez faire partie du groupe "profs". Si le groupe "profs" n'existe pas, créez-le. (Voir [[users-admin#gerer_les_groupes_d_utilisateurs|Gérer les groupes d'utilisateurs]])

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
