====== Questions fréquemment posées sur #ubuntu-fr à propos de l'aide en ligne ======

<note warning>Cette FAQ concerne logiquement toutes les versions courantes et suivantes. Les commandes données ne sont pas documentées volontairement pour ne pas surcharger le document et peuvent être potentiellement **dangereuse pour votre système**. Elles sont à utiliser avec l'aide donnée sur le canal IRC [[irc://irc.freenode.net/ubuntu-fr|#ubuntu-fr]]</note>

===== Connaitre =====
[[Tutoriel:console_commandes_de_base|Les commandes de base en console]]

----

===== Aide commande =====
La plupart des commandes ont une aide synthétique, pour l'obtenir ajouter ––help à la commande.
<code>NomCommande --help
mv --help</code>

===== Liste des pages de manuel =====
Cherche le nom et la description des pages de manuel
<code>apropos NomCommande
apropos mv
whatis mv</code>

===== man =====
man permet d'obtenir la page de manuel d'une commande ou autre sujet. Pour sortir de la page de manuel, appuyez sur la touche q. Certaines commandes ont plusieurs sections, vous pouvez si vous le souhaitez spécifier cette section.
<code>man [section] NomCommande
man man
man passwd
man 5 passwd
man --all passwd
</code>

===== info =====
info permet d'obtenir une aide détaillée sur certaines commandes ou autres sujets.
<code>info NomCommande
info cp</code>

===== Pages de manuel en français =====
Installer les pages de manuel en français. Toutes les commandes ne sont pas traduites.
<code>sudo apt-get install manpages-fr
sudo apt-get install manpages-fr-extra</code>

===== Centre d'aide =====
Vous pouvez consulter toutes les pages de manuels, info, howto, etc, en installant dwww.
<code>sudo apt-get install dwww</code>
Dans votre navigateur tapez l'url http://localhost/dwww/
