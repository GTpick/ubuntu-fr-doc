====== Furius ISO Mount Tool ======

Cette application permet de monter facilement des images CD et DVD dans des formats divers (comme ISO, IMG, BIN, MDF ou NRG). Avec cette application, il n'est pas nécessaire d'être administrateur pour monter une image, car elle utilise [[:fuseiso]].

{{applications:furiusisomount_fr.png?400}}

Elle offre les fonctionnalités suivantes :
  * Montage automatique des fichiers images au format ISO, IMG, BIN, MDF et NRG
  * Création automatique d'un point de montage dans votre dossier personnel
  * Démontage automatique des fichiers image
  * Suppression automatique du répertoire de montage pour retrouver l'état précédent de votre dossier personnel
  * Enregistrement automatique de l'historique des dix dernières images montées
  * Prise en charge du montage de plusieurs images
  * Gravure de fichiers ISO et IMG sur disque optique
  * Génération des sommes de contrôle MD5 et SHA1
  * Récupération automatique de toutes les images préalablement démontées
  * Génération automatique d'un fichier journal de toutes les commandes nécessaires pour monter et démonter des images manuellement.
Vous pouvez aussi vérifier vos images en calculant leur empreinte [[:md5sum|Md5]] ou SHA1.

===== Installation =====

Pour installer cette application, il vous suffit d'[[:tutoriel:comment_installer_un_paquet#paquet_present_dans_les_depots|installer les paquets]] **[[apt>furiusisomount|furiusisomount]]**.

===== Utilisation=====

L'utilisation de cette application est très simple, pour le démarrer, lancez l'application via le [[:unity#tableau_de_bord_dash|tableau de bord Unity]] ou pour les menus, cliquez sur **Applications -> Accessoires -> Furius ISO Mount**.
 ou encore via le [[:terminal]] (toutes [[:variantes|variantes d'Ubuntu]]) avec la [[:commande_shell|commande]] suivante :
<code>furiusisomount</code>

Une fois lancé, sélectionnez l'image à monter à l'aide du bouton **Parcourir**, puis cliquez sur le bouton **Monter**. \\
Vous pouvez aussi glisser les images dans la liste des **images montées**.

<note>Si le mot de passe administrateur vous est demandé, c'est sans doute que vous avez coché la case //Loop// à la place de //Fuse//. Cochez donc la case Fuse et on ne vous demandera plus de mot de passe.</note>

<note>Si toutefois le format d'image que vous voulez monter n'est pas géré par **Furius ISO Mount Tool**, allez voir la page [[tutoriel:convertir_des_images_cd|convertir des images cd]]</note>

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, le configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
