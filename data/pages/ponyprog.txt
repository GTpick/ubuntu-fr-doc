====== PonyProg ======

===== Introduction =====

{{:2229_big.png|PonyProg}}

PonyProg est un programme multi-OS pour utiliser vos carte stk 200, 500, Jtag... pour programmer les microcontrôleur ATMEL

===== Installation =====

[[http://www.lancos.com/ppwin95.html|Télécharger les sources de PonyProg]]

décompresser les sources puis aller dans PonyProgxxxx-x.xxx/v et editer le fichier Config.mk

HOMEV - doit pointer vers votre chemin absolue v

LINUX_HEADERS - doit pointer vers vos en-têtes du noyau en cours

INSTALL_PREFIX - où installer l'exécutable

ARCH - doit etre configurer sur linux

puis

<code>make
sudo make install </code>
