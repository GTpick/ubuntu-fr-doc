====== Mousepad ======

**Mousepad** est l'[[éditeur de texte]] officiel de l'environnement de bureau [[XFCE]].

===== Installation =====

[[tutoriel:comment_installer_un_paquet|Installez le paquet]] **[[apt>mousepad|mousepad]]**. ((Mousepad est l'éditeur de texte installé par défaut dans [[Xubuntu]].))

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.


===== Principales caractéristiques =====
Ce qu'il fait :
  * Consommation de mémoire faible et lancement rapide.
  * Coloration syntaxique facilitant la lecture de code (HTML, Perl, Python, C, Shell...) (notamment grâce à la librairie libgtksourceview2 ou supérieure ! ;-)). Si vous cette librairie n'est pas installée, installez-la en premier lieu via le Gestionnaire de paquet [[synaptic|Synaptic]] par exemple.
  * Numérotation des lignes.
  * Gère les onglets.
  * Coloration syntaxique.

<note tip> Pour activer la colorisation syntaxique :

1) Ouvrez Mousepad -> Document -> Type de fichier (//-> "Faites votre choix selon ce que vous désirez"//)
{{ :001_document_menu_mousepad.png?direct&400 |}}

2) Allez dans Vue -> Color Scheme (//-> "Faites votre choix selon ce que vous désirez"//) 
{{ :002_vue_menu_mousepad.png?direct&400 |}}

3) Tapez le texte que vous voulez. Voilà ! La couleur est au rendez-vous. :-)

{{ :003_test_mousepad_colored.png?direct |}}
</note>
