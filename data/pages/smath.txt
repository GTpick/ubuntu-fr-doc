====== SMath:  ======
SMath est un logiciel freewar (pas open-source!) de calcul mathématique WYSIWYG, équivalent à MathCAD.


===== Pré-requis =====

Il est nécessaire d'avoir les libraires pour Mono installées:
<code>sudo apt-get install mono-complete</code>

===== Installation =====

Pour installer ce logiciel, il suffit de télécharger le fichier d'installation pour la version Mono sur le site officiel du logiciel: [[http://en.smath.info]]

Le logiciel ne nécessite pas d'être installé comme un paquet traditionnel, il faut simplement executer le fichier exe avec Mono:
<code>sudo mono ~/Downloads/SMathStudioDesktop.0_88.Mono/SmathStudio_Desktop.exe</code>

Plus d'information sur l'installation: [[http://en.smath.info/forum/yaf_postsm2314_Installation-in-Ubuntu.aspx#post2314| Forum de SMath sur l'installation]].

Pour ajouter le programme dans le menu d'Unity:
[[http://askubuntu.com/questions/311220/launching-smath]]
