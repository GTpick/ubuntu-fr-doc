====== GPRename ======

**GPRename** est un logiciel écrit en GTK2-Perl qui permet un renommage de masse de fichiers et/ou de répertoires. Vous trouverez ici la procédure d'installation, ainsi qu'une explication des fonctions de GPRename.

<note>**[[PyRenamer]]** possède à peu près les mêmes fonctionnalités que GPRename. Il est simple et convivial et dispose d'une meilleure intégration dans GNOME.</note>

===== Installation =====

Il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>gprename|gprename]]**.

Vous pouvez lancer l'application avec la commande ''gprename'' ou depuis le menu //Applications -> Accessoires//.

{{gprename2.2.png}}


===== Utilisation =====

Dans le menu « Language », choisissez votre langue (Français UTF-8 pour nous).

La fenêtre de travail se décompose en trois parties.

**1. Le navigateur de dossiers.** (à gauche)\\
Sélectionnez ici le dossier dans lequel vous voulez renommer des fichiers.

**2. Fenêtre de sélection.** (partie principale)\\
Choisissez ici les fichiers (dans l'onglet « Fichiers ») ou les répertoires (dans l'onglet « Répertoires ») à renommer.
Si vous ne souhaitez renommer que certains d'entre-eux. Les touches « Maj » et « Ctrl » permettent de sélectionner plusieurs fichiers.
Si vous souhaitez travailler sur tous les fichiers ou répertoires, il est inutile de les sélectionner.

**3. Fenêtre d'action.** (partie inférieure)\\
C'est ici que vous sélectionnerez les actions à faire pour renommer vos fichiers.

Après chaque action, appuyez sur le bouton « Aperçu » pour voir le résultat. Si cela vous convient, appuyez sur « Renommer ».

<note tip>L'aperçu automatique peut être activé dans le menu « Options ».</note>

Si vous vous êtes trompé vous pouvez appuyer sur « Annuler » pour retrouver le nom original de vos fichiers.

Le bouton « Rafraîchir » met à jour votre répertoire de travail.




===== Désinstallation =====
