====== Remobo ======

Remobo est une alternative à [[:hamachi]], il est disponible sur Windows, Mac et Linux.

Remobo n'est pas libre, il est gratuit (bien qu'une version payante soit envisagé lorsque le logiciel ne sera plus en bêta-test).

Remobo permet :
  * De contrôler un ordinateur a distance
  * De partager des fichiers
  * De créer un VPN facilement

===== Pré-requis =====

  * Avoir une connexion internet

===== Installation =====

  *Télécharger le .deb correspondant à votre architecture (32 ou 64 bits) sur cette page [[http://www.remobo.com/download.php]]
  * [[:tutoriel:comment_installer_un_paquet|installer le paquet.]]

===== Configuration =====

Afin de pouvoir utiliser remobo, vous devez vous créer un compte, soit via l'interface du logiciel, soit à cette adresse :
[[http://www.remobo.com/register.php]]

Les options peuvent être activées/désactivées en choisissant "Properties" dans le menu contextuel du bouton "Menu", puis dans l'onglet "Settings".

===== Utilisation=====

Le logiciel installe un raccourci dans le menu Applications => Internet => Remobo Instant Private Network

Dans la fenêtre, saisissez votre nom d'utilisateur et votre mot de passe.
Vous pouvez aussi cochez "Save & login automatically.

Saisissez un nom explicite pour le champ "Location".

Puis cliquez sur le bouton "Connect".
L'icône de Remobo s'ajoute alors à la zone de notification.

Lorsque la connexion a réussi, une fenêtre s'affiche affichant la liste de vos contacts (aussi appelé Buddy).
Pour l'instant cette liste ne comporte que votre nom d'utilisateur.

==== Ajouter un contact (Buddy) ====

Pour ajouter un ami, cliquez sur le bouton "Menu" situé en bas de la fenêtre, puis dans le menu contextuel, choisissez "Add a Buddy".
Une nouvelle fenêtre s'affiche vous invitant a taper le nom d'utilisateur de votre ami.

==== Supprimer un contact ====

Dans la fenêtre principal, faites un clic droit sur le nom du contact a supprimer puis choisissez "Remove Buddy".

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]].
