====== The widget Factory ====== 

{{ twk.png?300}}
**The Widget Factory** est un logiciel développé en GTK+ dans le but de vous aider à créer vos thèmes GTK. 

Vous verrez souvent des captures d'écran (screenshots) du logiciel pour présenter certains thèmes, par exemple sur [[http://gnome-look.org]]

===== Installation =====

==== Installation par les dépôts ====

En un clic :[[apt://thewidgetfactory|The Widget factory]] ( v0.2.1 )

En ligne de commande :
''sudo apt-get install thewidgetfactory''

==== Installation à partir des sources ====

Les sources sont disponibles sur le site à l'adresse suivante ( jusqu'à la v 0.7.7 ) : [[http://gwf.sourceforge.net/|Lien externe]]

===== Utilisation =====
Pour le lancer, entrez cette commande dans un terminal ou via le raccourci Alt+F2:
  twf

===== Voir aussi =====

2 autres outils existent pour visualiser vos thèmes GTK:

==== A Widget Factory ====
'A Widget Factory' est une application similaire à 'The Widget Factory' qui permet de visualiser à la fois les thèmes sous GTK2 et GTK3.
Des paquets existent pour Oneiric Ocelot, Precise Pangolin et Quantal Quetzal via les PPA mentionnés dans le README de l'application à l'adresse suivante: https://github.com/valr/awf

==== gtk3-widget-factory ====
Une autre application existe également depuis Precise Pangolin pour visualiser les thèmes GTK+3. C'est un outils, fort bien caché, qui se nomme **gtk3-widget-factory**. Il s'installe avec le paquet **[[apt>gtk-3-examples]]** disponible dans les dépots et se lance via le terminal (afin de repérer les erreurs éventuelles du thème) :
