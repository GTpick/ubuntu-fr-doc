====== EXTRAIRE LES PISTES AUDIO D'UN DVD AVEC DEVIDIFY ======

Devidify l'excellent logiciel du Californien Matthew Newton a pour seul objet l'extraction des pistes audio d'un DVD. 
Idéal lorsqu'on souhaite "ripper" les pistes audio d'un concert pour les écouter en ballade !!!


<note warning>Pré-requis:

Devidify fonctionne sur les systèmes Linux. Sur Ubuntu, assurez vous que les paquets suivants sont installés: [[apt://python]], [[apt://python-gtk2]], [[apt://mplayer]], [[apt://lsdvd]], [[apt://lame]], [[apt://vorbis-tools]].
</note>

Une fois ces paquets installés rendez-vous sur le site de Google Code pour télécharger l'archive: [[http://code.google.com/p/devidify/downloads/list]]
Une fois téléchargée, extrayez l'archive à l'endroit de votre choix, ouvrez un terminal et rendez-vous, à l'aide de la commande "cd" dans le répertoire d'extraction.
Tapez la commande:

<note>sudo python setup.py install</note>

Après l'installation Devidify se trouveras dans le menu "**Applications/son et vidéo**"

===== UTILISATION: =====

Insérez un dvd, lancez Devidify, cliquez sur "scan dvd" et sélectionnez les pistes que vous souhaitez ripper (un clique droit sur une piste, vous permet de visionner la piste concernée dans mplayer), enfin cliquez sur le bouton "Rip audio tracks" pour lancer l'extraction.

Le menu "Edit/Préférences" permet de paramétrer le format de sortie (mp3, ogg...), ainsi que le repertoire de votre choix.

Voilà, c'est aussi simple que ça !!!
