====== FAQ sur e17 ======
Ci-dessous une FAQ concernant le gestionnaire de fenêtres [[enlightenment|E17]].

===== Certaines applications s'ouvrent ou se ferment avec un message d'erreur =====
Pour régler ce problème, faites un clic droit sur l'icône de votre application qui pose problème, puis éditez les propriétés de l'icône en rajoutant un "&" après le nom de l'application puis dans la partie exécutable :\\
{{enlightenment:application-erreur.png?250}}\\

===== Exalt fonctionne, mais ne détecte pas correctement les réseaux sans-fil  ====
Essayez de lancer exalt-daemon avec le terminal :
<code>sudo exalt-daemon</code>

Ou bien essayez de lancer exalt-client :
<code>sudo exalt-client</code>
Si le module n'affiche toujours pas les réseaux wifi, redémarrez enlightenment.

Si un problème persiste, posez une question sur le [[http://forum.ubuntu-fr.org/viewtopic.php?id=108951|forum]].

===== Je voudrais avoir le thème gtk en harmonie avec le thème e17 par défaut  =====
Pour le thème gtk correspondant au [[http://gnome-look.org/content/show.php/E17+B_and_W?content=95680|thème black and white]].
Pour le thème gtk correspondant au [[http://gnome-look.org/content/show.php/E17+-GTK2+and+Metacity+%28not+official%29?content=60092|thème bling bling]].

Pour les icônes correspondant au [[http://www.gnome-look.org/content/show.php/black-white+2+Style?content=72619|thème black and white 2 style]] (voir également le thème [[http://gnome-look.org/content/show.php/B%26W+New?content=106551|thème black and white new]]).

Copiez ensuite les thèmes gtk dans **~/.themes** et les icônes dans **~/.icons** (si ces répertoires n'existent pas, créez-les). Appuyez simultanément sur "Alt" et "Échap", écrivez lxappearance et pressez **[entrée]** :\\
{{enlightenment:lxappearance.png?250}}\\
Sélectionnez alors le thème gtk et les icônes dans la fenêtre qui s'est ouverte.

===== Dans EFM, le gestionnaire de fichiers de e17, je ne réussis pas à avoir la barre d'outils alors que j'ai chargé les modules  =====
Assurez-vous tout d'abord d'avoir bien chargé les modules correspondant au gestionnaire de fichier :\\
{{enlightenment:EFM_modules.png?250}}\\
Lorsque les modules sont bien chargés, cliquez sur une icône présente sur le bureau pour afficher EFM, et faites un clic droit sur la barre d'outils. Vous aurez alors l'option de la configurer, et de rajouter les modules que vous souhaitez :\\
{{enlightenment:EFM_configuration.png?250}}\\

===== J'aime beaucoup e17, c'est très joli, est-ce qu'il existe des distributions qui l'utilisent comme environnement principal ?  =====
Référez-vous à la partie sur les différentes [[:enlightenment#variantes_de_gnulinux_avec_enlightenment|variantes de GNU/linux avec enlightenment par défaut]]

D'autre part, méfiez-vous des cd des magazines, comme ceux de Linux Pratique, ou encore des forums critiquant les distributions. Chacune a un point fort. Elive est très complet, et facile d'utilisation, Soad Linux possède une configuration par défaut originale et soignée, Opengeu vise l'esthétisme, etc.

Si vous souhaitez essayer alors téléchargez le livecd correspondant. Le plus simple reste alors d'utiliser [[:unetbootin|unetbootin]].

===== Quand je mets une clé USB ou un disque dur externe sur mon ordinateur, rien ne se passe, que dois-je faire ? =====
Il existe plusieurs manières de configurer le montage automatique des périphériques. Installez par exemple [[apt://ivman|ivman]], puis sélectionnez-le comme service au démarrage à l'aide de BootUp Manager :\\
{{enlightenment:ivman_bum.png?250}}\\
Si vous n'avez pas BootUp Manager dans le menu système, installez-le : [[apt://bum|BootUp Manager]]

===== Menu Applications vide===== 
Si vous avez installé e17 après une installation classique d'Ubuntu, Xubuntu, Lubuntu, il se peut que lorsque vous ouvriez votre menu Applications soit vide. Pour y faire apparaître les applications:
  * Ouvrez omni (clic gauche sur le bureau > Omni), taper "menu",
  * Allez dans l'onglet "menu",
  * Choissisez applications lxde ou xfce...
