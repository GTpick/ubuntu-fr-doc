====== Super Mario War ======

{{ http://www.supersanctuary.net/supermariowar/18Beta1/smw_1_8_pic4.png}}
**Super Mario War** (SMW) est un jeu de bataille se jouant seul ou en multi-joueurs (jusqu'à 4 joueurs ou bots en simultané), dans l'univers du célèbre Mario Bros de Nintendo.
Le but est d'éliminer les autres joueurs en leur sautant dessus ou en utilisant l'une des innombrables options (carapace, fleur, cape, boulet, etc).

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.


===== Installation =====

Pour installer ce logiciel, il suffit de :

  * [[:tutoriel:comment_installer_un_paquet#paquet_present_dans_les_depots|installer le paquet]] **[[apt://libpng3|libpng3]]**
<code bash>sudo apt-get install libpng3</code>
  * Télécharger la version "linux" du jeu ([[http://smw.supersanctuary.net/18Beta2/smw-1.8-beta2-linux.tar.gz|lien direct pour la beta2]]), et la décompresser dans votre Dossier personnel
<code bash>wget http://smw.supersanctuary.net/18Beta2/smw-1.8-beta2-linux.tar.gz
tar -zxvf smw-1.8-beta2-linux.tar.gz</code>

  * Avec les [[sudo|droits administrateur]], déplacer le contenu de chacun des sous-dossiers du jeu dans les sous-dossiers correspondants de votre système. Pour cela, vous pouvez par exemple saisir dans un [[terminal]] les commandes suivantes :
<code bash>sudo mv usr/games/* /usr/games/
sudo mv usr/share/games/* /usr/share/games/</code>



===== Utilisation=====

Pour lancer le jeu, il suffit de saisir la [[:commande_shell|commande]] suivante :

  /usr/games/smw

Vous pouvez aussi créer un [[raccourci-lanceur]].

===== Désinstallation =====

Pour supprimer cette application, il suffit de supprimer ses fichiers.
