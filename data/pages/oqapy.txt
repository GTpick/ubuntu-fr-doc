====== Oqapy ======

**Oqapy** est un outil de production pour photographe.

===== Pré-requis =====
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  * Avoir [[:tutoriel:comment_installer_un_paquet|installé]] [[:pyqt|PyQt]],  [[:gphoto|gPhoto2]],  [[:openCV]], **[[apt>dcraw]]**, et [[http://tilloy.net/dev/pyexiv2|pyexiv2]];


===== Installation =====

Le plus simple sera :
  * [[:PPA|D'ajouter le PPA]] **ppa:vincent-vandevyvre/vvv**((https://launchpad.net/~vincent-vandevyvre/+archive/ubuntu/vvv)) dans vos sources de logiciels; 
  * [[:tutoriel:comment_modifier_sources_maj#recharger_la_liste_des_paquets|De recharger la liste des paquets]];
  * [[:tutoriel:comment_installer_un_paquet|D'installer le paquet]] **[[apt>oqapy]]**. 

Il est aussi possible de trouver des paquets Debian sur le [[http://www.oqapy.eu|site]] de l'application.

===== Utilisation =====
===== Utilisation =====
 
Lancez l'application via le [[:unity#tableau_de_bord_dash|tableau de bord Unity]] ou via le [[:terminal]] (toutes versions d'Ubuntu) avec la [[:commande_shell|commande]] suivante :
<code>oqapy</code>

Au lancement du programme, cliquez sur le menu //Fichier > Source// pour ouvrir un dossier d'image.

L'ouverture d'un nouveau dossier demande que le programme construise les vignettes qui s'afficheront au bas de la fenêtre, les ouvertures suivantes de ce dossier seront plus rapides.

À la droite de la fenêtre principale se trouve un panneau de boîte d'outils. Les boutons situés dans la barre d'entête de ce panneau permettent de changer de page d'outils.

Dans la première page, on peut trouver les informations générales de l'image courante. Là, vous pouvez gérer les méta-données (aussi appelées //tags//) du fichier. Il est aussi possible de géolocaliser ses images avec l'outil situé au bas de cette page.

La deuxième page fournit des outils de composition comme le dimensionnement, le cadrage ou la rotation de l'image.

La troisième page permet de modifier le contraste et la luminosité de l'image.

La quatrième page fournit des outils de modification artistique tels que des virages sépia ou bleu.

La cinquième page permet de gérer ses dossiers d'images et aussi d'enregistrer ses modifications sur l'image courante en choisissant parmis une dizaine de formats différents.

La dernière page présente les outils de conversion des fichiers RAW.

Avec le menu //Fichier > Import// vous accédez à la boîte de dialogue d'import d'image depuis un appareil photo numérique.

Dans le menu //Outils// on peut trouver aussi un outil de recherche d'images, un outil de géolocatisation avec fichier .gpx (fichier créé par un GPS), une loupe, etc.

La documentation accessible par le menu //Aide// permet une prise en main rapide de l'application.

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.

===== Bug connu =====

Il se peut que la documentation n'apparaisse pas en français, c'est le cas si la langue de votre ordinateur n'est pas //fr_FR// mais //fr_BE// ou //fr_CH// par exemple. Dans ce cas, ouvrez le [[:fichier_cache|dossier caché]] //.oqapy/docs// et renommez le dossier //fr_FR// avec le nom de votre langue.
