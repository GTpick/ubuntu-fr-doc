====== Arronax ======
 {{ :applications:arronax.png}}
Arronax est une application qui permet de créer et de modifier [[Raccourci-lanceur|les lanceurs d'applications]] ou les raccourcis pour les liens et les dossiers. 
Il peut être utilisé comme application autonome ou bien comme plugin pour [[Nautilus]] accessible dans le menu contextuel.
 


===== Pré-requis =====
 
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  
===== Installation =====
 Pour installer **Arronax**:
  * [[:ppa#ajout_d_un_ppa_a_vos_sources_logicielles|Ajoutez le PPA]] **ppa:diesch/testing**((https://launchpad.net/~diesch/+archive/testing)) dans vos sources de logiciels.
  * [[:tutoriel:comment_modifier_sources_maj#recharger_la_liste_des_paquets|Rechargez la liste des paquets]],
  * Puis [[:tutoriel:comment_installer_un_paquet|installez le paquet]] **[[apt>arronax]]** ou **[[apt>arronax-nautilus]]** si vous souhaitez ajouter un raccourci vers Arronax dans le menu contextuel de [[:Nautilus]].

 
===== Utilisation =====
 
Lancez l'application via le [[:unity#tableau_de_bord_dash|dash]] (Unity) ou via le [[:terminal]] (toutes versions d'Ubuntu) avec la [[:commande_shell|commande]] suivante :
<code>arronax</code>

Pour accéder au raccourci Nautilus, faites un clic droit sur le fichier, l'application ou le dossier pour lequel vous souhaitez créer un lanceur et choisissez //Create a starter for this file//.

Il ne vous reste plus qu'à remplir les champs en fonction de vos besoins:

{{ :applications:arronax_trusty.png?400 }}
 
===== Désinstallation =====
 
Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
