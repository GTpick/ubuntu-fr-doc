====== Omnitux : Suite ludo-éducatifs ======

Le but d'Omnitux est de proposer des activités ludo-éducatives autour d'éléments multimédia (images, sons, textes):

  * // Associations// :
  * // Images à placer sur une carte ou un schéma// :
  * // Activités de numération// :
  * // Puzzles// :
  * // Cartes à mémoriser// :
  * // Trouver les différences// :
{{:omnitux1.png?200|}}  {{:omnitux2.png?200|}}

* Disponible en Français, Anglais, Allemand, Espagnol, Italien, Polonais et Portugais.
  
=====Installation=====

Omnitux n'est pas dans les dépôts Ubuntu.
Il suffit de télécharger et d'[[:tutoriel:comment_installer_un_paquet|installer]] le paquet .DEB du site officiel.


   * [[http://sourceforge.net/projects/omnitux/files/omnitux/v1.2.1/light/omnitux-light_1.2.1_all.deb/download|Télécharger Omnitux-light ]] 58 Mo
   * [[http://sourceforge.net/projects/omnitux/files/omnitux/v1.2.1/omnitux_1.2.1_all.deb/download|Télécharger Omnitux ]] 115 Mo


    * [[:tutoriel:comment_installer_un_paquet|Installez les paquets]]  en saisissant dans un [[:terminal]] la commande suivante :
<code>sudo dpkg -i omnitux </code>


===== Jouer =====

Un lanceur est créé automatiquement par l'installation dans le menu **Applications => Education** ou encore **Applications => Jeux**
On peut aussi lancer le programme en tapant "omnitux" dans le [[:unity#tableau_de_bord_dash|dash]] d'Unity ou en mode [[:console]].
