====== Gloobus, Amélioration de votre interface  ======
<note warning> Projet mort depuis longtemps.</note>
Gloobus pour une interface plus agréable, et très attractive.

Gloobus est une suite de solutions visant à améliorer l'interface homme machine (ihm) du bureau gnome : reprenant des idées de mac os.

Site officiel : 
    * http://gloobus.net/
    * https://launchpad.net/gloobus (meta-project)
    * http://gloobus.wordpress.com/

=====Gloobus-preview =====

{{ http://lazyubuntu.com/images/2009/09/gloobus.png?650 }}

Pour tout ouvrir avec une interface jolie et simple.

[[http://gloobus.net/wiki/index.php/Install#Installing_Gloobus_Preview_from_the_PPA|Wiki officiel]].
((Note: Les paquets de Gloobus en préversion sont également disponibles :\\ 
PPA pour les éléments du bureau : ppa:elementaryart/elementarydesktop \\ 
PPA pour les éléments de Nautilus : ppa:am-monkeyd/nautilus-elementary-ppa))
<code> 
sudo add-apt-repository ppa:gloobus-dev/gloobus-preview
sudo apt-get update && sudo apt-get install gloobus-preview
</code>

Allez dans Système -> Préférences -> CompizConfig Settings Manager ( Pour l'installer : [[apt://compizconfig-settings-manager]] ), activer "Commandes", puis cliquez dessus. Sur le premier onglet appelé "Commande", tapez "gloobus-preview" pour la première commande (appelée «ligne de commande 0"). Puis sur le deuxième onglet appelé "Key Bindings", sous la rubrique «commande Exécuter 0" entrez votre combinaison de touches souhaitée comme Ctrl + Espace ou ce que vous voulez.

Pour les utilisateurs de Nautilus Elementary : appuyer sur ESPACE après avoir sélectionné un fichier

=====Covergloobus=====

Affichage des images (cover : couverture d'album ou dvd) à la place de simples dossiers dans nautilus.

<note>Voir aussi [[https://launchpad.net/cover-thumbnailer|cover thumbnailer]]</note>

[[http://gloobus.net/wiki/index.php/Install#Installing_CoverGloobus_from_the_PPA|Wiki officiel]].


====Installation par dépôt====
<code>sudo add-apt-repository ppa:gloobus-dev/covergloobus
sudo apt-get update</code>

Puis installer [[apt://covergloobus]]
