====== Petits jeux sous Ubuntu ======

Voici une liste de petits jeux que l'on peut facilement installer sous Ubuntu.


===== gbrainy =====

Voir **[[:gbrainy]]**.


===== Iagno (Reversi) =====

**[[apt>iagno]]** (également connu comme Reversi ou [[wpfr>Othello]]) est un jeu de stratégie qui se joue sur un plateau de huit par huit avec des jetons ayant une face noire et une face blanche. L'objectif du jeu est de retourner à votre couleur autant de jetons de votre adversaire que possible tout en évitant qu'il n'en fasse autant avec les vôtres. Pour retourner les jetons de votre adversaire, piégez-les entre deux des vôtres.


===== Gnome-klotski =====

L'objectif de **[[apt>gnome-klotski]]** est de faire coulisser la pièce recouverte d'un motif vers les marqueurs de position (en bas à droite) en un minimum de mouvements. Utilisez la souris pour déplacer les blocs.

===== Mahjongg =====

Voir **[[:gnome-mahjongg|Mahjongg]]**.

===== Mines (Démineur) =====

Voir **[[:gnomine|Mines]]**.

=====Gnome-nibbles =====

**[[apt>gnome-nibbles]]** est un jeu de serpent. L'objectif est de guider le serpent autour de l'espace en mangeant les diamants. Pour chaque diamant mangé le serpent s'allongera. L'objectif du jeu est de manger tous les diamants sans se heurter à un mur ou sur le corps du serpent. Quand tous les diamants d'un niveau sont mangés, le jeu passe au niveau suivant. Faites le maximum de niveaux possibles sans heurter quoi que ce soit. Dirigez le serpent en utilisant les touches de direction Gauche, Droite, Haut, and Bas.

===== Quadrapassel =====

Voir **[[:Quadrapassel]]**.

===== Gnome-robots =====

L'objectif de **[[apt>gnome-robots]]** est d'éviter d'être attrapé par une bande de robots en maraude. Pour éviter les robots, vous pouvez vous déplacer verticalement, horizontalement et en diagonale le long de l'espace en utilisant le clavier numérique. Si les choses deviennent trop chaudes vous pouvez sauter à une nouvelle position par téléportation. Sélectionnez //Paramètres → Préférences// et cliquez sur l'onglet //Clavier// pour voir les options et les choix par défaut des touches. Si un robot vous attrape, c'est terminé. Éliminez les robots avant qu'ils ne vous éliminent en les faisant s'écraser les uns contre les autres ou dans une pile de poubelles.


===== Swell Foop =====

L'objectif de **[[apt>swell-foop]]** est de supprimer autant de carré (au lieu des balles) que possible avec le minimum de mouvement. Les carrés sont de trois couleurs (vous pouvez augmenter le nombre). Quand ils sont regroupés, cliquer sur un des carrés enlève le groupe. À chaque fois qu'un groupe est retiré les carrés restants s'ajusteront d'eux-mêmes pour former un nouveau groupe, ou peut-être pas.

Le score pour un groupe supprimable de n balles de même couleur est (n-2)².


===== Solitaire AisleRiot  =====

Voir **[[AisleRiot]]**.

===== Sudoku =====

Voir **[[:gnome-sudoku|Sudoku]]**.

===== Tali (Yams) =====

**[[apt>Tali]]** est un jeu de poker avec cinq dés, appelés aussi Yams. Jetez trois fois les dés et essayez d'avoir la main heureuse. Le deuxième et le troisième jet de dés peuvent n'inclure qu'une partie des dés, après que vous ayez indiqué ceux qu'il vous intéresse de conserver tels quels.

===== Gnome-tetravex =====

**[[apt>gnome-tetravex]]** est un jeu de puzzle. Il est plus simple qu'il n'y parait. L'objectif est de positionner les pièces afin que les nombres correspondent au nombre adjacent sur les pièces voisines comme dans un jeu de domino. Utilisez la souris pour modifier la position des pièces.

Une astuce comme point de départ de la résolution : repérer le point faible, comme par exemple un chiffre n'apparaissant qu'une fois.
