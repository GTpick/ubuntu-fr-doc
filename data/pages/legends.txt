====== Legends ======



Legends est un FPS (First Person Shooter → Jeu de tir à la première personne) de type freeware ([[http://legendsthegame.net/index.php?m=content&pageid=28|Licence du jeu]])

Site officiel: [[http://legendsthegame.net/|http://legendsthegame.net/]]

{{image:legends_loadscreen.jpg|}}

===== Installation (méthode 1)=====

Vous trouverez sur ce [[http://legendsthegame.net/index.php?m=fileswap|lien]] un paquet debian (Full installer) à télécharger d'une centaine de méga-octets.

Double cliquer sur le fichier pour lancer l'installation.


===== Installation (méthode 2)=====
Télécharger le "Linux installer (GUI)" sur la page de [[http://legendsthegame.net/index.php?m=fileswap|téléchargement de Legends]]

Dans le répertoire du fichier téléchargé, lancer la commande :
<code>$ sudo sh legends_linux_gui-0.4.1.43.run</code>

L'interface graphique d'installation se lance, il faudra accepter la licence et spécifier les options si nécessaire (emplacement des fichiers du jeu etc...).


=====Jouer =====
lancer cette commande:

<code>/usr/games/legends/runlegends</code>


Vous pouvez ajouter un raccourcis à votre en menu en passant par 
**Système->Préférence->Agencement du menu**

=====Problèmes connus =====


=== Le jeu ne se lance pas ===
Creer le fichier **prefs.cs** s'il n'existe pas


<code>unzip -p /opt/legends/legends/scripts.unf scripts/client/default.pref.cs > ~/.legends/legends/prefs/prefs.cs</code>

Puis éditer le fichier **prefs.cs**

<code>gedit ~/.legends/legends/prefs/prefs.cs</code>

et ajouter/ ou remplacer les lignes suivantes:

''$pref::Video::resolution = "1280 1024 32";''

''$pref::Video::fullscreen = "0";''


==== Les cartes ne se chargent pas ====

Couper les applications susceptibles de se servir du serveur son (VLC, Listen etc...)
