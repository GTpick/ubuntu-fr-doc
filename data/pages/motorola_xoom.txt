====== Motorola Xoom ======

===== Sous Honeycomb la fonction de Mass Storage est remplacée par le protocole MTP =====
//**[[wp>Media Transfer Protocol]]**// (**MTP**) est un protocole de transfert de fichiers destiné au monde du multimédia. Créé récemment (depuis 2004 environ), il s'agit d'un ensemble d'ajouts au //[[wp>Picture Transfer Protocol]]// déjà existant utilisé par les appareils photos numériques. Les ajouts apportés à ce protocole permettent de gérer les spécificités des divers appareils multimédia : baladeurs, appareils de photographie numérique, caméras vidéo, téléphones portables, etc. Il est de plus en plus utilisé par les périphériques mobiles.

Originalement créé par Microsoft(r), le protocole MTP est désormais développé par le USB Implementers Forum et est en cours de normalisation. Actuellement, le protocole MTP est bien supporté par Windows XP SP2 (Windows Media Player 11 requis, car c'est celui-ci qui apporte le support de MTP sous cet OS) et Windows Vista ; pour les autres systèmes d'exploitation, il est nécessaire d'avoir recours la la **''libmtp''**. 

===== Installation sous Ubuntu 11.10 =====  

  *  Ne pas connecter la tablette au PC
  * Lancer la commande "sudo apt-get install mtpfs" sous Ubuntu
  * attendre la fin de l'installation
  * Connecter la tablette qui devrait être reconnue par nautilus sous le nom de code "MZ604"
