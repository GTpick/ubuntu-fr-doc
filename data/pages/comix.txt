====== Comix/Mcomix ======
<note important>Comix est maintenant repris sous le nom de Mcomix, les deux sont présents dans les [[:dépôts]].</note>
**Comix** est une visionneuse d'image avancée et gratuite sous licence GPL spécialement conçu pour la lecture de bandes dessinées. 

Écrit en Python, il utilise les bibliothèques graphiques GTK+ pour l'environnement de bureau [[Gnome]], bien qu'il puisse fonctionner toutefois sous d'autres environnements tels que KDE. 

Il est relativement simple à utiliser avec une interface sobre. Il dispose néanmoins d'un nombre impressionnant d'options comme la possibilité de consulter et d'extraire des fichiers images contenus dans des archives (Tar, Zip, Rar), de créer sa propre bibliothèque ainsi que de créer des diaporamas, cet utilitaire s'avère rapidement essentiel pour les passionnés de bande dessinée.

===== Pré-requis =====
 
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
 
===== Installation =====
 
Pour installer ce logiciel, il suffit d'[[:tutoriel:comment_installer_un_paquet|installer les paquets]] **[[apt>comix]]** ou **[[apt>mcomix]]**.

===== Utilisation =====

Lancez l'application via le [[:unity#tableau_de_bord_dash|tableau de bord]] (Unity) ou via le [[:terminal]] (toutes versions d'Ubuntu) avec la [[:commande_shell|commande]] suivante :
<code>comix</code>

Trois modes d'affichages sont disponibles :
  * Le mode "manga" utile lorsque l'on tente de lire une BD manga, 
  * le mode "plein écran" qui permet de profiter au maximum de la taille des photos,
  * le mode "pages double" dédié au feuilletage de votre bande dessinée.

En outre, vous avez une loupe pour faciliter la lecture sur de petits écrans. Un œil sur les préférences du logiciel est conseillé notamment pour étirer les images à la taille de l'écran ou pour le mode d'affichage par défaut.

===== Désinstallation =====
 
Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
