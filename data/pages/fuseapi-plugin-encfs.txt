====== Extension EncFS pour FuseAPI ======
[[EncFS]] permet à n'importe quel utilisateur de créer un système de fichiers chiffrés grâce à FUSE.

Cette extension ajoute de nouvelles fonctionnalités à [[FuseAPI]] :
  * Reconnaissance des points de montage EncFS
  * Interfaces permettant d'ouvrir ou créer des points de montage chiffrés
  * Obtenir des informations sur un répertoire chiffré avec EncFS
  * Changer le mot de passe d'un répertoire chiffré avec EncFS

===== Pré-requis =====
Cette extension est disponible sous forme de paquets binaires x86 et x86_64 pour Lucid Lynx.

Rendez-vous [[utilisateurs:spomky|sur sa page]] pour pouvoir utiliser son dépôt.

===== Installation =====
Installez simplement le paquet [[apt://fuseapi-plugin-encfs]]

Si vous avez installé FuseAPI via le code source, cette extension a été installée en même temps.

===== Désinstallation =====
Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]].
