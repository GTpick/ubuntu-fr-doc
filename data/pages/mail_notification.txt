====== Mail-Notification ======

**Mail-Notification** vous permettra de savoir si de nouveaux courriels sont disponibles, indépendamment de votre courrielleur et même si celui-ci n'est pas lancé. 

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]] ;
  * Disposer d'une connexion à Internet configurée et activée.


===== Installation =====

Il vous suffit, pour cela, d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>mail-notification]]**.

===== Utilisation=====

Lancez l'application via le [[:unity#tableau_de_bord_dash|tableau de bord Unity]] ou via le [[:terminal]] (toutes versions d'Ubuntu) avec la [[:commande_shell|commande]] suivante :
<code>mail-notification</code>

Vous disposez de 2 entrées :
  * Avertisseur de courrier qui vous permet de lancer **Mail Notification**,
  * Avertisseur de courrier qui vous permet de paramétrer **Mail Notification**.


===== Configuration =====

{{ :applications:mail-notification_xenial.png?350 }}

**Onglet Général** : Dans cet onglet, vous pouvez paramétrer les boites de courriel à surveiller, prévoir un lancement automatique de Mail Notification au démarrage, ou lancer des commandes spécifiques lorsque du nouveau courrier est disponible.

**Icône de statut** : Tout ce qu'il vous faut pour paramétrer le comportement de l'icône de tableau de bord.

**Onglet Popups** : Visualisez une ou des popups dès que de nouveaux messages se présentent.

===== Problèmes connus =====


==== Mail-Notification ne propose pas l'option "Open Message" pour Hotmail et Yahoo! ====

Un lien vers la solution de plusieurs problèmes crédit [[http://dontsurfinthenude.blogspot.com/2009/11/email-notification-in-ubuntu.html| ici à FreewheelinFrank ]]

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. La configuration de l'application sera conservée ou supprimée selon la méthode de désinstallation que vous choisirez.
