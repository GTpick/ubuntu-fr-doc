====== Gestionnaires de fichiers ======

Un **[[http://fr.wikipedia.org/wiki/Gestionnaire_de_fichiers|gestionnaire de fichiers]]** est un logiciel qui fournit une interface utilisateur pour travailler avec les fichiers informatiques. Les plus communes utilisations sont : la création, l'ouverture, la visualisation, l'impression, la lecture, le renommage, le déplacement, la copie, la suppression, les propriétés et la recherche de fichiers.

===== Gestionnaires de fichiers "complets" =====

  * [[Dolphin]] : le gestionnaire de fichiers par défaut de Kubuntu (KDE4)
  * [[Konqueror]] : gestionnaire de fichiers KDE très complet
  * [[Nautilus]] : le gestionnaire de fichiers par défaut d'Ubuntu (Gnome)
  * [[Nemo]] : fork de Nautilus et gestionnaire de fichiers par défaut du bureau Cinnamon (depuis la version 1.6)
  * [[pcmanfm|PCMan File Manager]] : gestionnaire de fichiers léger 
  * [[SpaceFM]] : fork de PCMan File Manager
  * [[Thunar]] : le gestionnaire de fichiers par défaut de Xubuntu
  * [[Xfe|Xfe]] : gestionnaire de fichiers très léger et complet 
  * Caja : le gestionnaire de fichiers de [[mate|Ubuntu MATE]]

===== Gestionnaires multi-panels =====

Logiciels similaires à **Total commander** sous Windows :

  * [[double commander|Double Commander]] ([[http://doublecmd.sourceforge.net/|Site officiel]] **[en]**)
  * [[gnome-commander|Gnome Commander]] : pour Gnome
  * [[krusader|Krusader]] : pour KDE
  * [[http://sunflower-fm.org/|Sunflower]] : gestionnaire de fichiers léger et paramétrable

===== Gestionnaires de fichiers spécialisés dans le renommage de fichiers =====

  * [[PyRenamer]] : logiciel de renommage en masse de fichiers/répertoires bien intégré à Gnome
  * [[KRename]] : logiciel de renommage en masse de fichiers bien intégré à KDE
  * [[GpRename]] : logiciel de renommage en masse de fichiers/répertoires
  * [[Metamorphose2]] : logiciel de renommage en masse de fichiers/répertoires très complet, mais complexe
  * [[Purrr]] : logiciel de renommage en masse
