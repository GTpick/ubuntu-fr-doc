====== Go Ollie ======

**Go Ollie !** est un jeu de plates-formes se jouant à la souris. Il raconte l'histoire d'un petit vers de terre qui part trouver "une belle señorita". Arrivé dans la jungle, les obstacles se multiplient.

{{:goollie1small.jpg|}}

===== Installation =====

[[http://www.playdeb.net/software/Go%20Ollie!|Go ollie est disponible sur playdeb]] (FIXME ne fonctionne pas sous Ubuntu 10.04 LTS).

On peut aussi le trouver sur le site officiel du jeu : http://tweeler.com/index.php?PAGE=goollie_linux&PLATFORM=linux 


=== Installation sur ubuntu 64 bits ===
Il semble que playdeb ne supporte pas la version 64 bits, voici comment l'installer dans ce cas :

  * Télécharger l'installateur disponible sur [[http://tweeler.com/index.php?PAGE=goollie_linux&PLATFORM=linux|cette page]].
  * Ensuite il faut rendre exécutable le fichier téléchargé (clic gauche / permissions ou droits d'accès).
  * Un double clic sur le fichier pour lancer l'installateur, après l'installation une icône de lancement est disponible dans le menu jeu.
