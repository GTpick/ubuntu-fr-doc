====== Dim Dim - Web conference ======

Solution de web conference open source, existe également en version propriétaire avec des services associés.

Site officiel : http://www.dimdim.com/

Note : Peut être plus mature qu'[[openmeeting]].

<note warning>DimDim n'est pas véritablement opensource.</note> 
  * Seule la version 4.5 est opensource
  * Elle est packagée pour CentOS et sa mise en place n'est pas simple : un tuto remarquable pour Debian [[http://www.generation-linux.fr/index.php?post/2009/05/04/Presentation-et-installation-de-Dimdim-sur-Debian-Lenny|ici ]] (Bravo à Benjamin, du blog [[http://www.generation-linux.fr|Génération Linux]])
  * La version actuelle est la 5.2, pas du tout opensource et exploitée en mode SAAS
  * L'éditeur quand on lui pose la question dit clairement que la version opensource reste en ligne pour servir de "commercial trick", en bref un miroir aux alouettes...

===== Fonctionnalités =====

Voici une liste non-exhaustive de ses fonctionnalités :
  * Partage de documents
  * Audio et Vidéo conférence
  * partager votre écran
  * partager un tableau blanc
  * Chat public ou privé
  * Enregistrer les conférences
  * API permettant d’intégrer dimdim à différentes applications comme Claroline, Moodle, mais aussi SugarCRM, outlook, zimbra…
Dimdim est basé sur ces technologies opensource :
  * Mysql, Serveur de base de données SQL
  * Terracota, Solution de clustering pour les applications Java
  * LightHTTPD, Serveur Web
  * Tomcat, Serveur d’application Java
  * CherryPy, Framework pour le développement web, orienté objet et pythonic.
  * OpenOffice, Suite bureautique opensource
  * [[Red5]], serveur flash opensource
