====== Opérations ======

**Opérations fait partie de la suite de logiciels éducatif du [[le_terrier|Terrier]].**

<note>(Ci-dessous, extrait du site officiel http://www.abuledu.org/leterrier/operations )</note>

{{ http://www.abuledu.org/_media/leterrier/operations/operations-multiplications.png?350}}

Opérations n'est pas qu'un simple exerciseur demandant une résolution d'opération. C'est un logiciel pour **apprendre à poser** correctement une **opération** et à **gérer la retenue** de manière explicite. 

//L'algorithme de la division euclidienne est couvert par le logiciel ”[[:abuledu-calculs|Calculs]]”.//

//**Pour une description complète et des pistes pédagogiques, voir [[http://www.abuledu.org/leterrier/operations|la documentation sur le site officiel du Terrier d'Abulédu]].**//

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  * Avoir activé l'accès aux [[le_terrier#ajouts_des_depots|dépôts du Terrier]].
  
===== Installation =====

[[:tutoriel:comment_installer_un_paquet|Installez le paquet]] **[[apt>abuledu-operations|abuledu-operations]]**.


===== Uitilisation =====

Comme tous les logiciels du Terrier, vous retrouverez un lanceur dans le menu **Applications --> Éducation**

===== Configuration =====

Comme les autres logiciels du Terrier, //Opérations// permet à l'enseignant d'éditer les exercices. Pour cela, vous devez faire partie du groupe "profs". Si le groupe "profs" n'existe pas, créez-le. (Voir [[users-admin#gerer_les_groupes_d_utilisateurs|Gérer les groupes d'utilisateurs]])

===== Désinstallation =====
Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
