====== CaRMetal ======

CaRMetal est un logiciel de géométrie dynamique qui permet de travailler sur des figures géométrique, comme avec une règle et un compas et ainsi découvrir visuellement et ludiquement les règles de géométrie apprises au collège.

{{carmetal.gif|}}

CaRMetal est multiplateforme et développé pour Linux, Mac OS, Windows et autres OS sous licence GNU/GPL. CaRMetal est également disponible en version réseau.

**AUTEUR** : Eric Hakenholz

===== Pré-requis =====

  * [[Java]] doit être installé sur le système,
  * Disposer des [[:sudo|droits d'administration]].

===== Installation =====

[[:tutoriel:comment_installer_un_paquet|Installez les paquets]] **[[apt>carmetal]]**.


===== Installation Minimale =====

==== Objectifs : ====

Si les postes de la salle informatique de votre établissement sont pourvus d'un logiciel propriétaire, ou si le logiciel installé actuellement ne vous convient pas; vous n'êtes pas obligé de quémander une installation sur tous les postes.


==== Deux possibilités : ====

  - Vous récupérez le fichier CaRMetal.jar présent sur votre système (une fois que vous l'avez installé) à l'emplacement usr/share/carmetal/CaRMetal.jar

  - vous décompressez l'archive "autres systèmes" présent sur la page téléchargement du site : http://db-maths.nuxit.net/CaRMetal/


==== Méthode : ====

  - Vous placez le fichier sur internet ou sur le réseau, les élèves le copient sur le bureau, ils font un clic droit, ouvrir avec Sun Java, et le tour est joué 8-)

  - Vous décompressez l'archive sur le réseau, les élèves copient le dossier sur leur bureau, et c'est exactement comme s'ils avaient une version de CaRMetal installé sur leur poste!



===== Configuration =====

CaRMetal est internationalisé, le choix de la langue est automatique mais peut se configurer par le menu **Spécial -> Langues**.

La configuration du logiciel est assez sommaire et accessible depuis **Spécial -> Préférences**.

La première utilisation de CaRMetal va générer deux fichiers de configuration dans le dossier **carmetal_config**, ceux-ci se trouvent dans votre répertoire utilisateur (/home).

===== Utilisation =====

L'interface est soignée et présente dès la première utilisation des palettes d'outils détachées de la fenêtre de travail.

Une aide contextuelle est proposée <del>mais en anglais</del> en ligne (en français).

Un clic droit fait apparaître un menu contextuel bien pratique qui reprend les macros fournies avec CarMetal.

FIXME

Les figures peuvent être groupées dans un dosier de figures, accessibles en cliquant sur des onglets en bas de page (comme les feuilles d'un tableur).

Comparé à d'autres logiciels de géométrie dynamique, CaRMetal possède des fonctionnalités inédites: Courbe de niveau, magnétisme des objets, un peu de 3D, et possibilité d'écrire des scripts en JavaScript, à l'intérieur de la figure. Pour plus de détails , voir [[http://fr.wikipedia.org/wiki/CaRMetal]] et [[http://fr.wikipedia.org/wiki/CarScript]]

<note help>Ici une description plus complète de l'utilisation de Carmetal serait à faire par une personne plus compétente que moi Merci.</note>

Vous pouvez exporter votre travail en PNG, PostScript ou en format SVG ou encore créer une page HTML afin de publier votre travail sur le Net.

**Les avantages de ce logiciel par rapport à d'autres propriétaires :**

  * très intuitifs ;

  * clic droit donne accès à l'ensemble des propriétés de l'objet ;

  * exportation au format html avec un applet java, très pratique pour pouvoir exporter vos figures sur un cahier de texte électronique (plusieurs exemples, à cette adresse : http://biotrans.free.fr/?/math/p-voir/fichier-77/probleme-geometrique-plusieurs-solutions-alignement-points.htm ) ;

  * une installation minimale, c'est-à-dire, qu'il suffit de faire télécharger le carmatal.jar aux élèves et ils peuvent immédiatement l'utiliser sous n'importe quel OS, et sans installation. Par exemple, sous ubuntu, clic droit sur le fichier carmetal.jar, et ouvrir avec sun java ;

  * il est évolutif grâce à son système de macros. De nombreuses macros sont déjà présentes, et permettent le codage des figures.

  * Ce logiciel est totalement libre, et en constante évolution, vous pouvez même proposer des améliorations sur le forum de l'auteur : http://db-maths.nuxit.net/CaRMetal/forums/
