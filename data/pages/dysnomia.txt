====== Dysnomia ======

Dysnomia est un programme montrant la possibilité de télécharger des musiques depuis des sites de musique à la demande (dans la version 0.3, uniquement le site deezer.com). Il constitue une preuve de concept (Proofof concept) de cette possibilité, et son intérét principal réside dans son code et dans la manière dont il fonctionne, pas dans son utilisation pour du téléchargement de musique des sites avec lesquels il est capable d'intéragir. 

===== Pré-requis =====

Il vous suffit, pour cela, d'[[:tutoriel:comment_installer_un_paquet#paquet_present_dans_les_depots|installer les paquets]] **[[apt://build-essential,libwxgtk2.8-dev,libwxgtk2.8-0,libcurl4-gnutls-dev|build-essential libwxgtk2.8-dev libwxgtk2.8-0 libcurl4-gnutls-dev
]]**.

===== Installation =====

  wget http://dysnomia-project.org/dysnomia-fr-0.3.4.tgz // Lien mort
  tar -xzvf dysnomia-fr-0.3.4.tgz
  cd dysnomia-fr-0.3.4
  make
  sudo make install
