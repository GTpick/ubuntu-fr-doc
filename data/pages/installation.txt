====== Cas particuliers de l'installation d'Ubuntu ======
===== Pré-requis =====
==== Choix de l'architecture ====
Ubuntu est accessible à de nombreuses [[:architecture materielle|architectures]] (déterminées par le matériel) : officiellement sont supportés les ordinateurs à base de processeurs x86 (32 bits) et x64 (64 bits), mais non officiellement, Ubuntu supporte aussi PowerPC, HPPA, ia64 et SPARC.
(Canonical envisage fin support x86 (32 bits) à sa version 18.08. Souce: [[http://www.ginjfo.com/actualites/logiciels/linux/ubuntu/ubuntu-18-10-canonical-envisage-fin-support-32-bit-20160708]])

La très grande majorité des ordinateurs individuels sont compatibles avec la version **32 bits**, les ordinateurs neufs du commerce étant le plus souvent compatibles avec la version **64 bits**.
https://doc.ubuntu-fr.org/lib/images/wrap.gif
==== Choix du support ====
  * Installation depuis un CD voir **[[tutoriel:amorcer_sur_cd_ubuntu|Installation standard avec le CD Ubuntu]]**
  * [[:live_usb|Live-USB]] : **Installer Ubuntu sur (ou via) une clé USB** avec persistance des données, ou pas.
  * Autres supports : voir [[#Cas particuliers]]

==== Pré-requis système ====
voir l'[[:exigences_minimales|article détaillé selon les variantes]].
  -** Version Bureau:** ((Détails en Anglais: https://help.ubuntu.com/community/Installation/SystemRequirements))
    * Processeur de 1,2 GHz ou plus
    * 1,5 Go de RAM, conseillé 2 Go
    * 8 Go d'espace sur le disque dur
    * Une carte graphique affichant une résolution de 1024x768
  -**Version serveur:** ((Détails en Anglais: https://help.ubuntu.com/community/Installation/SystemRequirements))
    * Processeur de 300 MHz ou plus
    * 128 Mo de RAM
    * 1 Go d'espace sur le disque dur.
    * Une carte graphique affichant une résolution de 640x480

===== Mise à niveau vers une version ultérieure=====
Pour passer à une version supérieure d'Ubuntu, il n'est pas nécessaire de procéder à une réinstallation complète de votre système. Ubuntu peut être mis à niveau, directement depuis Internet ou d'un média d'installation d'Ubuntu, à l'aide de l'utilitaire de mise à niveau.
  *  [[:tutoriel:migrer_sans_connexion_internet|Se mettre à niveau sans connexion à Internet]] : si vous voulez mettre à niveau votre version //via// un disque amovible préparé.
  *  Pour mettre votre version à niveau depuis internet, une clé USB ou un CD/DVD, consulter la page [[:migration|Migrer vers une version ultérieure.]]

===== Cas particuliers =====
  * Via l'image ISO sans utiliser de lecteur CD :
    * [[:live_usb|Live-USB]] : **Installer Ubuntu sur (ou via) une clé USB** avec persistance des données, ou pas.
    * [[:unetbootin|Installer Ubuntu avec Unetbootin]]. Installer Ubuntu directement depuis une image ISO stockée sur un disque dur ou une clé USB (si vous avez internet et aucun lecteur CD par exemple) 
    * [[:tutoriel:Installer Ubuntu sans cd]] : installer Ubuntu depuis une image ISO présente sur un disque dur ou une clef USB (ou depuis les dépôts Internet), sans graver de CD. 
  * Via le réseau :
    * [[:netboot|Installation depuis un autre ordinateur (ordinateur du réseau)]]. L'ordinateur doit être capable de démarrer par le réseau (ex : PXE), et //un autre ordinateur// doit faire office de serveur de l'image sur le réseau. Cette méthode est assez complexe à mettre en place la première fois, mais beaucoup plus simple pour les installation suivantes, s'il y a //plusieurs ordinateurs à installer// et pour la //personnalisation des installations//.
    * [[tutoriel:comment_installer_ubuntu_en_pxe_depuis_windows|Installation quand on n'arrive pas à démarrer sur le CD-ROM et que l'ordinateur démarre sur réseau (ex : PXE)]]
    * [[tutoriel:installer_ubuntu_par_internet|Installer Ubuntu par internet]] : une ISO de ~13Mo qui télécharge et installe Ubuntu via internet
  * Installation sur un media secondaire :
    * [[:tutoriel:installation_sur_disque_USB|Sur un disque dur USB]] : installation classique, mais sur un disque USB ;
    * Voir aussi [[:live_usb|Live-USB]] pour utiliser une clé USB avec démarrage persistant, donc utilisable à partir de n'importe quel PC ;
  * Méthodes d'installation alternative :
    * [[installation_alternate|Installation en mode semi-texte]] à l'aide de l'//Alternate CD// : pour les ordinateurs moins puissants ou pour une installation plus personnalisée.
    * [[:rescue|Installation quand la phase d'installation se bloque ou pose un problème]] ;
    * [[:smart_boot_manager|Installation quand on n'arrive pas à démarrer sur le CD-ROM mais qu'on possède un lecteur de disquette]] ;
    * [[:tutoriel:installation_sur_nonpae|Processeur sans instruction PAE]] ;
  * Autres méthodes pour ordinateur peu puissant :
    * [[:mini_ram|Installation d'Ubuntu sur une machine ancienne]], avec peu de ressources matérielles ;
    * [[:installation_minimale|Installation du minimum vital]], pour avoir un bureau propre, sans tous les logiciels inutiles installés par défaut ;
====Si votre écran est cassé====
Si l'écran de votre ordinateur portable est cassé il suffit de brancher un autre écran avec un câble VGA ou HDMI, et de lancer l'installation de façon habituelle.
