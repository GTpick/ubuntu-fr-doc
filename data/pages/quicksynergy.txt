====== QuickSynergy ======

**QuickSynergy** est un utilitaire pour configurer [[Synergy]] graphiquement.

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]] ;
  * Disposer d'une connexion à Internet configurée et activée ;
  * Avoir activé l'accès aux [[:depots#universe_et_multiverse|dépôts Universe et Multiverse]] ;
  * Avoir installé **[[Synergy]]**.


===== Installation =====

Pour installer cette application, il suffit d'[[:tutoriel:comment_installer_un_paquet|installez le paquet]] **[[apt>quicksynergy]]**.

===== Utilisation =====

L'application est disponible dans //Applications → Accessoires → QuickSynergy//.
Vous pouvez aussi utiliser la combinaison de touches « Alt+F2 » puis saisir « quicksynergy », qui est le programme QuickSynergy.

Configurez selon les indications de la section [[#Configuration]].

Positionnez-vous sur l'onglet désiré et appuyez sur le bouton « Execute » afin de lancer le partage (mode serveur) ou l'utilisation (mode client).

===== Configuration =====

Sur chaque machine, un seul onglet doit être configuré à la fois.

Sur le serveur (ordinateur maitre, partageant son clavier et sa souris), cela se passe dans l'onglet « Share » : saisissez l'adresse IP ou le nom de l'ordinateur client (esclave) à contrôler selon la position de l'exclave par rapport au maitre. Pour d'autres choix, voir la page [[:Synergy]].

{{:applications:quicksynergy:onglet_share.png?nolink}}

Sur le client, cela se passe dans l'onglet « Use » : dans « Server », saisissez l'IP ou le nom d'hôte du maitre ; dans « Screen name », saisissez le nom ou l'adresse IP du client.

{{:applications:quicksynergy:onglet_use.png?nolink}}

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. La configuration de l'application sera conservée ou supprimée selon la méthode de désinstallation que vous choisirez.
