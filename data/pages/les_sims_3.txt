====== Les Sims 3 ======

Le jeu **[[http://fr.wikipedia.org/wiki/Les_Sims_3|Les Sims 3]]** est sorti en Europe le 4 juin 2009. Le gameplay des Sims 3 est le même que dans les précédents opus de la série (Les Sims, Les Sims 2), il s'agit de simuler la vie d'une famille ou d'un sim seul ou en couple, dans son quartier de Amou. Le jeu ne comporte ni violence physique, ni score à battre, il suffit uniquement de faire vivre son avatar dans sa vie de tous les jours. Si une grande partie consiste à faire interagir son Sim avec son entourage (objets, autres Sims), la construction d'une habitation et son ameublement tiennent également une certaine place comme toujours dans la série. Des codes existent, mais c'est bien plus marrant de faire sans ! Si vous tenez absolument à utiliser des codes triches vous pouvez aller sur ce site : http://simsminitroll.com/Sims3/Base/Codes/codes_sims3.php



===== Avertissement =====

Une fois le jeu installé via Play-On-Linux, un no-cd est nécessaire  pour lancer les Sims 3, sinon un message d'erreur vous indiquera qu'aucun CD n'est présent dans le lecteur et vous ne pourrez pas jouer.

Il suffit alors de trouver ce patch "no-cd" se nommant "TS3.exe" (Par exemple ici: http://www.t411.ch/torrents/les-sims-3-1-50-56-0210xx-crack-no-cd-dvd-only-fonctionnel-avec-add-on) et de le mettre dans *\Les Sims 3\Game\Bin. Il existe déjà un fichier "TS3.exe", alors remplacez-le ou renommer-le (comme "TS3bis.exe").

===== Installation =====

Etape 1:

Il vous faut disposer du jeu "Les Sims 3", soit en DVD original que vous aurez acheté, soit en téléchargeant un .iso depuis n'importe quel site de partage (piratebay, t411.ch).

Exemple (pas forcément fonctionnel ni le meilleur): http://www.t411.ch/torrents/Les-Sims-3-RELOADED-MULTI

Il vous faut également une clé CD, par exemple: 5HDV-PSQ6-EEUN-T763-9MEQ

Etape 2:

Installez Play-On-Linux.

Etape 3:

Installez les Sims 3 via Play-on-Linux en sélectionnant le fichier téléchargé à l'étape 1.

Etape 4:

Telecharger le patch no cd et le copier-coller dans le répertoire des sims correspondant.

Lancer le jeu en double cliquant sur TS3.exe, si vous avez correctement fait le copier-coller vous ne devriez pas voir apparaitre de message d'erreur relatif à l'absence du CD des sims 3.


===== Mise à jour =====

<note important>Ne faites pas de mise à jour via le launcheur des Sims 3 !</note>
Pour mettre à jour les Sims 3 :

  * Executez PlayOnLinux dans Applications->Jeux->PlayOnLinux
  * Cliquez sur "installer"
  * Allez dans l'onglet "Jeux" à gauche et tapez "sims" dans le moteur de recherche à droite
  * Double cliquez sur "The sims 3" qui apparait dans la liste en dessous et suivez les instructions qui apparaissent à l'écran

===== Problèmes possibles =====
Il est fort possible que vous ayez un petit problème d'affichage pour la police d’écriture dans le jeu, apparemment c'est la police d’écriture qui est pas supportée par mono. Une personne a réussi a contrer ce problème mais n'a pas eu l'intelligence de donner la démarche pour supprimer le fichier ttf pour que le jeu en utilise une par défaut.

Néanmoins une astuce consiste à lancer le jeu avec le terminal .

Aller dans le répertoire du jeu :

 <code>$ cd /répertoire/du/jeu/Les Sims 3/Game/Bin</code>


et lancer le jeu à l'aide de cette commande

<code>$ force_s3tc_enable=true ./TS3.exe</code>
