====== Chemins ======

**Chemins fait partie de la suite de logiciels éducatif du [[le_terrier|Terrier]].**

<note>(Ci-dessous, extrait du site officiel http://www.abuledu.org/leterrier/chemins )</note>

{{ http://www.abuledu.org/_media/leterrier/logo-chemins.png?350}}

“Chemins” est un logiciel permettant d'acquérir des compétences de repérage dans l'espace : lire et coder des déplacements absolus sur les cases ou les noeuds d'un quadrillage.

Une série de trois ou quatre chemins de difficulté croissante ayant été regroupés par activité :

    * Décodage,
    * Codage immédiat,
    * Codage différé,
    * Parcours caché

Il s'agit sur plusieurs séquences d'enchaîner ces activités pour aboutir au codage écrit du déplacement dans une situation de communication, comme dans [[:abuledu-anouslesnombres|A nous les nombres : activité-calapa]] et/ou [[:abuledu-labyrinthe|Labyrinthe]].

Une autre série d'activités concerne le déplacement sur des cases en fonction du tirage d'un ou deux dés, ou d'un chiffre écrit. Ces modules correspondent au déplacement sur piste en fonction d'un cardinal du type jeu de l'oie.

Chemins intègre un éditeur permettant de tracer les chemins (en cours de développement) et un bilan permettant le suivi de l'élève.


//**Pour une description complète et des pistes pédagogiques, voir [[http://www.abuledu.org/leterrier/chemins|la documentation sur le site officiel du Terrier d'Abulédu]].**//

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  * Avoir activé l'accès aux [[le_terrier#ajouts_des_depots|dépôts du Terrier]].
  

===== Installation =====

[[:tutoriel:comment_installer_un_paquet|Installez le paquet]] **[[apt>abuledu-chemins|abuledu-chemins]]**.


===== Utilisation =====

Comme tous les logiciels du Terrier, vous retrouverez un lanceur dans le menu **Applications --> Éducation
**

===== Configuration =====

Comme les autres logiciels du Terrier, //chemins// permet à l'enseignant d'éditer les exercices. Pour cela, vous devez faire partie du groupe "profs". Si le groupe "profs" n'existe pas, créez-le. (Voir [[users-admin#gerer_les_groupes_d_utilisateurs|Gérer les groupes d'utilisateurs]])


===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés..
