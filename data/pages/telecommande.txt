====== Les télécommandes ======

Cette page liste les applications, les tutoriels ou les procédures spécifiques à la gestion des télécommande sous Ubuntu.

=====Logiciels de configuration=====
  * [[:irkeytable|IR-Keytable]] : Premier outil à utiliser pour paramétrer une télécommande ;
  * [[:lirc|Installer une télécommande infrarouge avec lirc]] : si la première méthode n'a pas fonctionnée essayez celle-ci ;
  * [[:tutoriel:lircrc|Création de fichiers lircrc]],
  * [[:lirc_-_exemples_hardware.conf|exemple de fichier hardware.conf pour lirc]] ;
  * [[:tutoriel:creer_fichier_lircd.conf|Création d'un fichier lircd.conf pour une télécommande n'en possédant pas]] ;
  * [[:remuco|Remuco : contrôlez vos logiciels multi-médias depuis votre téléphone en bluetooth ou internet]] ;
  * [[anyremote|Anyremote  : contrôlez vos logiciels multi-médias depuis votre téléphone en bluetooth ou internet]] ;
  * [[:xbindkeys|XBindKeys est utile pour configurer une télécommande mal supportée]].
