====== Exigences matérielles pour utiliser Ubuntu ======

Comme pour n'importe quel logiciel et système d'exploitation, Ubuntu et ses variantes nécessitent des systèmes performants, répondant aux exigences minimales ci-dessous, afin de proposer une expérience d'utilisation agréable.

Puisque chaque variante officielle d'Ubuntu requiert des capacités matérielles différentes, les configurations minimales sont listées par variante. Ce document ne contient les exigences que pour les variantes officielles d'Ubuntu.

Si votre ordinateur ne rencontre pas les exigences pour la variante de votre choix, comparez-les avec celles d'autres variantes -- [[#xubuntu|Xubuntu]] et [[#lubuntu|Lubuntu]], par exemple, sont particulièrement adaptées aux configurations plus anciennes.

<note tip>Certaines des spécifications matérielles mentionnées ci-dessous sont des estimations. Vous pourriez être capable de faire fonctionner avec succès Ubuntu dans un ordinateur ayant des performances en-dessous de ce que nous recommandons. Néanmoins, les performances d'Ubuntu pourraient alors être médiocres.</note>

L'installation pour serveur fournit une base commune pour l'hébergement de toutes sortes de programmes et services. Les exigences inscrites ici reflètent celles requises par Ubuntu Server. Des exigences additionnelles pourraient être requises par les divers services que vous hébergerez. Ubuntu Server propose une expérience en ligne de commande seulement ; pour une expérience graphique, installez plutôt Ubuntu pour postes de travail.

Des médias d'installation différents sont proposés pour chaque architecture matérielle. Veillez à télécharger la bonne image d'installation pour votre situation. ([[http://www.ubuntu.com/download/server|Télécharger Ubuntu Server]])



===== Informations avancées =====
Les informations suivantes donnent des indications supplémentaires pour qui souhaiterait installer Ubuntu et ses variantes dans un ordinateur destiné à une autre utilisation que la bureautique.


==== Ordinateur agissant comme client léger LTSP ====
Si vous prévoyez d'utiliser un ordinateur comme un client léger (tel un client pour **LTSP Terminal Server** pour l'[[edubuntu|Édition Éducation d'Ubuntu]]), seul un ordinateur répondant aux exigences minimales énoncées ci-dessus est nécessaire. Cet ordinateur doit disposer d'une carte graphique et d'un écran, ainsi que d'une carte réseau.

==== Minimum absolu ====
La configuration minimale absolue s'adresse particulièrement aux administrateurs de serveurs, mais aussi à ceux qui souhaitent personnaliser à fond leur système. Ce mode d'installation n'installe que les paquets vitaux du système. Vous devrez ajouter après, les paquets que vous trouverez utiles.

Il est entendu qu'avec les exigences strictes citées ci-dessous, Ubuntu ne sera utilisé qu'en mode console (sans interface graphique). Vous pourrez installer et utiliser une interface graphique avec un ordinateur plus puissant, comme les configurations recommandées ci-dessus. De plus, il est possible que vous réussissiez à installer Ubuntu dans un ordinateur moins puissant que ce que décrivent les prochaines informations ; néanmoins, l'installation pourrait être très longue et Ubuntu pourrait ne pas fonctionner au mieux de ses capacités.

Pour profiter de l'installation minimale absolue, vous devez utiliser l'//Alternate CD// d'Ubuntu ou une variante.

== Absolu minimum (avec interface graphique) ==
  * Processeur Intel Pentium 66 MHz ou compatible
  * 48 Mio de mémoire vive (RAM)
  * 468 Mio d'espace disque
  * Carte graphique VGA

== Absolu minimum (sans interface graphique) ==
  * Processeur Intel 486 et compatible
  * 32 Mio de mémoire vive (RAM)
  * 300 Mio d'espace disque

La configuration recommandée est celle préconisée pour une utilisation bureautique ou personnelle typique (avec un environnement graphique).
