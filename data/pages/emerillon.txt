====== Emerillon - Visionneur de Carte ======

Emerillon est un logiciel utilisant les données du projet [[OpenStreetMap]] pour afficher des cartes.

===== Installation =====

====Depuis les dépôts officiels====
Emerillon est disponible dans le dépôts officiels depuis [[maverick|Ubuntu 10.10]].

[[:tutoriel:comment_installer_un_paquet|Installez les paquets]] **[[apt://emerillon|emerillon]]**.
 

===== Utilisation =====

Le logiciel se lance à partir du menu Applications, //Accessoires -> Visionneur de Carte//.

Pour faire des recherches, activez le greffon correspondant dans les préférences.
