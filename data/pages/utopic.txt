====== Ubuntu 14.10  (« The Utopic Unicorn ») ======

<note critique>**[[:old-releases|Fin de vie]] depuis le 23 juillet 2015** \\ Depuis le 23 juillet 2015, Ubuntu 14.10 est en fin de vie (//end of life//, //EOL//). Elle ne profite plus de nouvelle mise à jour de sécurité et les paquets logiciels pour cette version sont déplacés dans des serveurs d'archive. Pour continuer à recevoir des mises à jour de sécurité, veuillez mettre à niveau votre version d'Ubuntu vers [[versions#versions_d_ubuntu_activement_soutenues|une version activement soutenue]].</note>

{{http://upload.wikimedia.org/wikipedia/commons/1/1d/Ubuntu_14.10_Utopic_Desktop.png?300  |Ubuntu 14.10 "The Utopic Unicorn" est sortie en version stable le 23 octobre 2014}}

**Ubuntu 14.10** (nom de code : //The **Utopic Unicorn**//) est la vingt et unième version d'Ubuntu. Son développement a débuté en avril 2014 et s'est échelonné sur une période de six mois. La version stable, finale d'Ubuntu 14.10 est sortie le 23 octobre 2014. La durée du cycle de maintenance était de neuf mois ; les mises à jour étaient donc assurées jusqu'au 23 juillet 2015.

===À propos du nom de code de développement ===
Mark Shuttleworth a annoncé le nom de code dans [[http://www.markshuttleworth.com/archives/1363|un billet de son blog personnel]], le 23 Avril 2014. 
'Utopic‘ est dérivé du mot ‘Utopia‘ qui décrit une société ou un projet qui tend a atteindre un grand idéal. Unicorn (licorne) est un animal de la mythologie généralement représenté comme un équidé avec une unique longue corne en spirale sur sa tête. Cela évoque un cycle de développement placé sous le signe de l'étonnement, les développeurs vont laisser parler leur créativité pour obtenir un système unifié et droit, "quelque chose dont nous pourrons être fiers". Il y aura donc une accélération du développement pour la convergence des plate-formes desktop et mobile avec Unity 8.

===== Changements apportés depuis la version précédente =====

Les rencontres ayant eu lieu durant l'Ubuntu Online Summit (UOS), du 10 au 12 juin 2014, sont des événements majeurs du processus de développement d'une nouvelle version d'Ubuntu. C'est durant ces sessions, auxquelles assistent des ingénieurs de Canonical, des membres de la communauté, des partenaires, des vendeurs de logiciels indépendants, des membres d'équipes de développement en amont et d'autres participants, que sont discutés les grands axes de développement d'Ubuntu, dans un environnement tourné autour de discussions et de planification d'échéanciers.

Peu de changements sont visuellement remarquables par les utilisateurs finaux, la majorité des changements dans cette version d'Ubuntu se trouvant « sous le capot ». Canonical continue de préparer la version mobile d'Ubuntu, ainsi que la future génération d'OS convergents. Les développements majeurs, trop instables, sont testés dans une pré-édition parallèle appelée « [[http://cdimage.ubuntu.com/ubuntu-desktop-next/daily-live/current/|Ubuntu Next]] » ; le présent document ne traite que de l'édition stable d'Ubuntu 14.10.

==== Ubuntu ====
  * L'interface [[Unity]] 7.3.1 est mise à jour pour mieux gérer les affichages en haute définition (//highDPI//)
  * Le navigateur Web [[firefox|Mozilla Firefox]] passe en version 33
  * La suite bureautique [[LibreOffice]] 4.3 apporte une prise en charge améliorée du format PDF, ainsi que des nouvelles fonctionnalités dans Writer, Calc et Impress (//[[https://wiki.documentfoundation.org/ReleaseNotes/4.3|Plus d'infos sur le site de LibreOffice]]//)
  * La gestion de [[http://www.pwg.org/ipp/everywhere.html|IPP Everywhere]] permet un partage facilité des imprimantes configurées dans Ubuntu avec des appareils mobiles
  * Le développement d'un nouvel outil à destination des développeurs, [[Ubuntu Developer Tools Center]], facilite l'installation d'un environnement de développement complet pour Android

==== Kubuntu ====
  * Cette version inclut KDE 4.14. Une pré-version (instable) avec l'environnement KDE Plasma 5 est aussi disponible. Les deux versions ne peuvent pas cohabiter.

==== Lubuntu ====
  * Nouvelle version de maintenance corrective, en attendant la version LXQt.
  * Beaucoup de composants de LXDE ont été mis à jour avec des versions correctives.
  * Mise à jour du graphisme (+ d'icones, mise à jour du thème, + de compatibilités). 

==== Ubuntu GNOME ====
  * Ubuntu GNOME se base sur l'environnement de bureau [[GNOME]] 3.12. L'équipe met à disposition un [[ppa|dépôt personnel (PPA)]] proposant GNOME 3.14.
  * La session GNOME classique est disponible à partir du sélecteur d'environnement, au moment de l'ouverture de session.
  * gnome-maps et gnome-weather sont maintenant installés par défaut. Les programmes gnome-photos, gnome-music et polari sont disponibles pour installation dans les archives d'Ubuntu.
    

==== UbuntuKylin ====
  * Cette version inclut des mises à jour de logiciels: Ubuntu Kylin Software Center 1.1.3, Youker Assistant 1.3.1, Youker Weather 2.1.2, Youker Calendar 1.0.0, Youker Fcitx 1.0.0, Sogou IM 1.1.0, Kuaipan 2.0.0 et Wiznote 2.1.12


==== Ubuntu MATE ====
  * Première version de cette variante qui est officiellement reconnue par le projet Ubuntu!
  * La saisie du mot de passe pour déchiffrer un disque dur dans Plymouth est corrigée

==== Ubuntu Studio ====
  * Le nouveau noyau Linux apporte la prise en charge des périphériques FireWire par ALSA.
  * Plusieurs logiciels ont été mis à jour, dont qtractor, jackd2, blender et lmms.


==== Xubuntu ====
  * Pour célébrer la licorne, le thème par défaut a été ajusté pour que le surlignage des éléments d'interface soit de couleur rose. Pour changer ce comportement, utilisez le configurateur de thème.
  * À partir de Xubuntu 14.10, les usagers devraient utiliser ''pkexec'' plutôt que ''gksudo'' pour exécuter des applications graphiques en mode administrateur. L'équipe de Xubuntu a préparé et intégré de base les stratégies pkexec qui ont été jugées nécessaires pour les applications par défaut.
  * Un nouveau greffon de gestion d'énergie est inclus dans le tableau de bord. Note : les utilisateurs qui effectuent une mise à niveau de leur système continuent d'utiliser le précédent ''indicator-power'', et doivent retirer le greffon puis le rajouter à nouveau pour utiliser la nouvelle version.
  * Xfce Power Manager 1.4 dispose de plusieurs améliorations. Le contrôle de luminosité a été étendu afin de mieux prendre en charge les claviers rétro-éclairés et les nouveaux affichages d'ordinateurs portables.
  * Le réarrangement de l'agencement multi-écrans dans Xfce Display Settings est facilitée : il suffit simplement de déplacer l'écran dans l'espace disponible pour réagencer son espace de travail.


==== Ubuntu Server ====
  * Cette version inclut OpenStack 2014.2 pour neuf projets -- attention à bien tester les mises à jour
  * L'outil d'orchestration de services Juju est mis à jour à la version 1.20.10, pour une meilleure stabilité et performance

==== Sous le capot ====
  * Le noyau Linux 3.16 apporte plusieurs corrections de bogues et la prise en charge de nouveau matériel, dont une gestion initiale des processeurs graphiques nVidia Gk20A et GK110B. On note des performances graphiques de plusieurs périphériques nVidia, AMD et Intel, et une prise en charge améliorée de l'encodeur vidéo .264 Radeon. Des améliorations sont notables aussi au temps de mise en hibernation et de réveil. Au plan des systèmes de fichiers, les performances sont aussi accrues notamment du côté de XFS, qui stabilise son format v5 et gère les E/S directes étendues, et de Btrfs, qui gère désormais les modes de compression par répertoire.
  * AppArmor reçoit plusieurs corrections de bogues et s'adjoint des nouvelles mises à jour de stratégies.
  * Oxide, un visualisateur Web basé sur Chromium pour publier du contenu Web, est mis à jour à la dernière version de Chromium Content API.
  * Le serveur graphique X.Org 1.16 dispose d'une meilleure prise en charge de périphériques non-PCI. Xephyr gère maintenant DRI3. La mise à jour MESA 10.3 dispose d'une prise en charge préliminaire de Nouveau avec les périphériques Maxwell.

===== Obtenir et utiliser Ubuntu 14.10 =====
{{  http://shop.canonical.com/images/UBN00159.jpg?250|Téléchargez maintenant Ubuntu 14.10}}

Veuillez noter qu'Ubuntu 14.10 n'est plus une version activement soutenue depuis le 23 juillet 2015. Ceci signifie que plus aucune mise à jour de sécurité n'est publiée pour cette version d'Ubuntu. Il est vivement conseillé de mettre à niveau Ubuntu vers une version ultérieure, toujours activement soutenue.

Néanmoins, il n'y a pas de date d'arrêt de fonctionnement à votre système Ubuntu : les versions d'Ubuntu arrivées en fin de vie sont toujours techniquement fonctionnelles. Il n'y a pas de mécanisme bloquant artificiellement le fonctionnement de votre système Ubuntu, et vous pouvez continuer à l'utiliser en connaissance de cause, jusqu'à ce que vous décidiez à mettre à niveau votre système d'exploitation. Notez que les dépôts officiels ne proposent plus de paquets/logiciels pour Ubuntu 14.10. Ceux-ci ont été migrés vers [[old-releases|un dépôt d'archives, destinés à contenir les logiciels des versions obsolètes d'Ubuntu]]. Les paquets de ces dépôts ne sont évidemment plus mis à jour; ils vous permettent seulement de continuer à utiliser la Logithèque ou tout autre système d'installation de logiciels.

  * Pour récupérer un CD-ROM d'installation d'Ubuntu 14.10, accédez [[http://old-releases.ubuntu.com/releases/|au serveur de téléchargement des versions archivées]].
  * Pour installer des logiciels, modifiez votre fichier de sources de logiciels (///etc/apt/**sources.list**//) afin de le faire pointer vers les serveurs d'archives: <code>deb http://old-releases.ubuntu.com/ubuntu/ utopic main restricted universe multiverse
#deb-src http://old-releases.ubuntu.com/ubuntu/ utopic main restricted universe multiverse

deb http://old-releases.ubuntu.com/ubuntu/ utopic-updates main restricted universe multiverse
#deb-src http://old-releases.ubuntu.com/ubuntu/ utopic-updates main restricted universe multiverse

deb http://old-releases.ubuntu.com/ubuntu/ utopic-security main restricted universe multiverse
#deb-src http://old-releases.ubuntu.com/ubuntu/ utopic-security main restricted universe multiverse

deb http://old-releases.ubuntu.com/ubuntu/ utopic-backports main restricted universe multiverse
#deb-src http://old-releases.ubuntu.com/ubuntu/ utopic-backports main restricted universe multiverse

deb http://old-releases.ubuntu.com/ubuntu/ utopic-proposed main restricted universe multiverse
#deb-src http://old-releases.ubuntu.com/ubuntu/ utopic-proposed main restricted universe multiverse</code>
