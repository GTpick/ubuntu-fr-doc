====== Anomos - client BitTorent anonyme ======

**Anomos** est un client [[p2p|bittorent]] libre.

Anomos utilise un protocole chiffré anonyme multi-pair-à-pair. Il est basé sur le concept de peer tracker de BitTorrent, en combinaison avec un oignon routeur (TOR) pour l'anonymat, avec l'avantage supplémentaire d'un chiffrement de bout en bout ( pour plus d'anonymat). En combinant ces technologies, les auteurs ont créé une plate-forme où personne en dehors du tracker (tracker de confiance) aura d'information sur un pair ou sur ce qu'ils télécharge.

Anomos est conçu pour être facile à utiliser - l'utilisateur ne sera même pas au courant de la sécurité qu'elle offre, c'est transparent. Quiconque est déjà familier avec BitTorrent n'aura aucune difficulté à l'utiliser.




=====Installation=====

Anomos n'est pas disponible dans les dépôts officiels.

====Par dépôt GetDeb====
   - [[getdeb|Installez le dépôt getdeb]] ;
   - [[http://www.getdeb.net/software/Anomos|Allez sur la page d'anomos sur GetDeb]] ;
   - Cliquez sur le bouton « Install this now » pour l'installer.

===== Utilisation =====

Lancez l'application depuis le menu //Applications -> Internet -> Anomos// ou //via// la [[:commande_shell|commande]] suivante :

  anomos

Par défaut, le site a mis en place son tracker compatible avec le nouveau format atorrent.

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, le configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
