====== Gelemental: ======
 
Tableau périodique regroupant de nombreuses informations sur les éléments chimiques. C'est un réécriture du logiciel [[:gperiodic|GPeriodic]] en C++, il intègre également de nombreuses améliorations sur le plan esthétique et ergonomique.

{{:gelemental_quantal_01.png?600}}

===== Installation: =====
Pour installer ce logiciel, il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>gelemental|gelemental]]**.

===== Utilisation=====
Lancez l'application depuis le menu //Applications -> Accessoires -> Tableau périodique// ou //via// la [[:commande_shell|commande]] suivante :
<code>gelemental</code>

=====Fonctionnalités=====

Gelemental permet notamment de :
  * colorer les éléments suivant leurs propriétés (densité, température de fusion, ...)
  * afficher les informations sur le nom de l'élément (éthymologie, origine, lieu de découverte, ...)
  * afficher les principales propriétés des éléments (physiques et atomiques)
  * ...

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
