====== AstroMenace ======
{{ :jeux:astromenace.png}}
**AstroMenace** est un jeu libre de shoot'em up spatial. Il se joue à la souris, dans un environnement plutôt beau, en 3D, avec des décors somptueux.

===== Installation ======
Depuis [[trusty|Trusty 14.04]], **AstroMenace** est disponible dans les dépôts officiels. Il suffit donc d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>astromenace|astromenace]]**.

**[[:precise|Pour Précise 12.04]]**\\
Le projet ne propose plus de binaires, mais il est toujours possible de télécharger le code source et de le [[:tutoriel/compilation|compiler]] pour disposer de la dernière version du jeu.
Les instructions sont disponibles sur [[http://www.viewizard.com/download.php?forceos=linux&forcegame=astromenace|cette page]] **(en)**.

===== Lancement =====
Lancez l'application via le [[:unity#tableau_de_bord_dash|tableau de bord]] ou via le [[:terminal]] (toutes versions d'Ubuntu) avec la [[:commande_shell|commande]] suivante : <code>astromenace</code>

===== Désinstallation =====
 
Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
