======Löve : framework de jeux vidéo 2D ======

Löve est un framework de développement de jeux 2D cross-platform.

Utilise le langage de script LUA.

{{ http://www.ubuntugamer.com/wp-content/uploads/2010/12/Screenshot-500x391.png }}

=====Installation=====

====Par launchpad====

  * [[:PPA|Ajoutez le PPA]] **ppa:bartbes/love-stable**((https://launchpad.net/~bartbes/+archive/love-stable)) dans vos sources de logiciels ;
  * [[:tutoriel:comment_modifier_sources_maj#recharger_la_liste_des_paquets|Rechargez la liste des paquets]] ;
  * [[:tutoriel:comment_installer_un_paquet|Installez le paquet]] **[[apt://love|love]]**.


===== Utilisation =====

FIXME

Pour lancer un programme depuis son dossier :
  love /home/chemin/monjeu/
(cherchera le fichier "main.lua")

Pour lancer un .love (qui n'est rien d'autre qu'un zip) :
  love /home/chemin/monjeu/jeupackage.love
