====== Solitaire Aisleriot ======

{{ jeux:aisleriot_lucid_00.png?90}}

**Solitaire Aisleriot** est une collection de jeux de cartes simples, à jouer avec l'aide de la souris. Pour votre convenance, les règles des différents jeux ont été écrites dans le langage de script du projet GNOME (Scheme).



{{jeux:aisleriot_lucid_01.png?500}}

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.


===== Installation =====

Ce logiciel est installé par défaut dans Ubuntu. Pour l'installer sous d'autres [[:environnements|environnements de bureau]], il suffit d'[[:tutoriel:comment_installer_un_paquet#paquet_present_dans_les_depots|installer le paquet]] **[[apt>aisleriot|aisleriot]]**.

<note tip>Sous [[:Kubuntu]], préférez-lui **KPatience**, qui fait partie des jeux de l'environnement [[:KDE]].</note>

===== Utilisation =====

Lancez l'application depuis le menu //Applications -> Jeux -> Solitaire Aisleriot// ou //via// la [[:commande_shell|commande]] suivante :

  sol

Le jeu pré-sélectionné par défaut pour le solitaire est "Klondike".

<note tip>
Pour tous ceux qui n'auraient , suite à une migration , plus la possibilité de changer de style de carte , et pour qui le modèle de carte "par défaut" donne une impression de très mauvaise définition , il faut installer, via synaptic le paquet gnome-games-extra-data devenue gnome-cards-data sous ubuntu 13.04
</note>

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, le configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
