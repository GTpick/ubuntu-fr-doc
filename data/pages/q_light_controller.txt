====== Q Light Controller ======
 
**Q Light Controller** est un logiciel multi-plate-formes pour contrôler les systèmes d'éclairage DMX ou analogues comme des têtes mouvantes, dimmers, des scanners et d'autres effets encore. Notre but est de remplacer les coûteux logiciels propriétaires par un logiciel libre.

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.


===== Installation ======

Q Light Controller n'est pas disponible dans les dépôts officiels. Il faut l'installer depuis un PPA :
  * [[:PPA|Ajoutez le PPA]] **ppa:ninoaderri/studio**((https://launchpad.net/~ninoaderri/+archive/studio)) dans vos sources de logiciels,
  * [[:tutoriel:comment_modifier_sources_maj#recharger_la_liste_des_paquets|rechargez la liste des paquets]], puis 
  * [[:tutoriel:comment_installer_un_paquet|installez le paquet]] **[[apt>qlc]]**.

===== Utilisation =====
 
Lancez l'application via le [[:unity#tableau_de_bord_dash|tableau de bord Unity]] ou via le [[:terminal]] (toutes versions d'Ubuntu) avec la [[:commande_shell|commande]] suivante :
<code>qlc</code>

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
