====== MySMS ======

{{  http://www.topapps.net/wp-content/uploads/2012/12/Mysms-Application-App.jpg|}}

**MySMS** est une Application propriétaire qui permet d'utiliser votre ordinateur pour envoyer et recevoir des SMS ou MMS grâce à votre smartphone.

===== Pré-requis =====
 
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  * Avoir installé l'application sur votre iPhone ou moble Android : [[https://itunes.apple.com/fr/app/mysms-messenger/id533406258|iPhone]] [[https://play.google.com/store/apps/details?id=com.mysms.android.sms|Android]]
  * Avoir créé un compte MySMS : https://app.mysms.com/?locale=fr#

===== Installation =====
 
==== Par le dépôt ====

<code>sudo sh -c "echo 'deb http://download.opensuse.org/repositories/home:/mysms/xUbuntu_13.04/ /' >> /etc/apt/sources.list.d/mysms.list"
wget http://download.opensuse.org/repositories/home:mysms/xUbuntu_13.04/Release.key
sudo apt-key add - < Release.key
sudo apt-get update
sudo apt-get install mysms</code>


==== Par les paquets ====

Installer le paquet DEB correspondant à votre architecture :
[[http://download.opensuse.org/repositories/home:/mysms/xUbuntu_13.04/i386/mysms_2.0-1_i386.deb|32bit]] [[http://download.opensuse.org/repositories/home:/mysms/xUbuntu_13.04/amd64/mysms_2.0-1_amd64.deb|64bit]]
