====== RipIT ou CD Ripper ======
===== A propos =====
RipIT est un ripper de CD en ligne de commande développé à l'origine par Simon Quinn. Il permet entre autres d'enregistrer un CD en fichiers WAV, FLAC, MP3 tout en alimentant les tags ID3V2 en consultant soit la base de données de [[http://www.freedb.org/|FreeDB]] soit celle de [[http://musicbrainz.org/|MusicBrainz]].
===== Installation =====
<code>
sudo apt-get install ripit
</code>
===== Configuration =====
Le fichier de configuration se situe ici :

<code>
/etc/ripit/config
</code>
