  * le [[http://webtv.guidetv.orange.fr/|portail orange]] peut être facilement visualisé et enregistré avec [[webtvorange]].

  * [[http://vlc-media-player.org/m3u/orangetv.m3u|Lien externe vers le fichier .m3u]] : Liste de chaine de télévision pour VLC ou autres pour les abonnés d'Orange.
==== Free ====
Ce service est réservé aux abonnés Free dégroupés.

<note important> Pour les utilisateurs non dégroupés, une petite visite du site http://tv.freebox.fr/ vous expliquera que ce service est arrêté depuis le 1/02/2011. Plus d'informations [[http://www.infobidouille.com/actualites/2011/02/02/Free-coupe-la-TV-aux-non-degroupes|ici]] </note>


Ouvrez VLC, et dans le //Navigateur de médias// (ou //Liste de lecture//), cliquez sur //Internet//, puis //Freebox TV//. La liste des chaines et radios disponibles s'affichera alors. Double-cliquez sur un élément et le flux (chaine ou radio) sera lu ([[:vlc#streaming_freebox_multiposte|source]]).

Si vous n'utilisez pas VLC ou que vous voulez plus d'informations sur ce sujet, consultez [[:multiposte_free|cette page]].

==== SFR ====
<note tip>Ainsi que les abonnés Neuf Télécom, Club Internet</note>
=== Dans votre navigateur internet ===
Rendez vous sur le site de [[http://www.sfr.fr|sfr]] puis suivez les instructions ci dessous:
Cliquez sur **SFR & moi** puis sur l'onglet **Votre TV-VOD** ensuite sur la nouvelle page chargée il faut cliquer sur **TV sur PC** on vous demandera alors votre identifient et votre mot de passe.
Après vous être connecté il vous suffira de choisir quelle chaîne regarder.


=== Avec VLC ===
<note warning>Depuis fin 2011, SFR ne propose plus l'accès au [[http://televisionsurpc.sfr.fr/televisionsurpc.m3u|fichier m3u]] donnant la liste des chaines. Il faut alors utiliser la méthode décrite ci-dessous.</note>
La web TV de SFR via un plugin VLC est accessible aux abonnés après identification sur [[http://tv.sfr.fr/tv-pc/popup.php|le site de SFR]]. Il faut avoir leur propre version de VLC (VLC by SFR) développée par SFR eux-même. Il faut donc désinstaller toute trace de VLC déjà existante sur votre système, puis réinstaller leur version.

<note>La version de VLC by SFR correspond actuellement à la version 1.1.11 de VLC, donc attendez-vous à des fonctionnalités manquantes ou des bugs déjà corrigés dans les futures versions de VLC.</note>

Les sources et les paquets pour Linux sont disponibles sur [[http://maj.sfr.fr/web/vlc-hls/|ce site.]]

=== Méthode à employer pour installer avec le système de paquets. ===

<note warning>Il semblerait que ces paquets ne soient disponibles qu'en version 64 bits. Si vous êtes sous un système fonctionnant en 32 bits et que cette méthode ne fonctionne pas, suivez la méthode alternative (à partir des sources) décrite plus bas.</note>

Désinstallez toute version de VLC sur votre système : 

<code>sudo apt-get remove vlc*</code>

[[tutoriel:comment_modifier_sources_maj|Ajouter le dépôt externe]] : 
<file>deb http://loyik.org/9cegetel/hls/MA_DISTRIBUTION/ ./</file>
Remplacez évidemment MA_DISTRIBUTION par le nom de votre distribution, par exemple //precise// pour Ubuntu 12.04 LTS Precise Pangolin

Désactivez tous les dépôts où la version //officielle// de VLC risque de se trouver (dépôts officiels), pour être certain d'obtenir la version de SFR.

[[tutoriel:comment_modifier_sources_maj|Mettez à jour]] la liste des paquets.

Installez le paquet **[[apt>vlc]]**. Si vous utilisez firefox, installez aussi le paquet **[[apt>browser-plugin-vlc]]**.

Si la page [[http://tv.sfr.fr/tv-pc/popup.php]] se charge sans problème, c'est-à-dire que vous n'obtenez pas le message d'erreur "VLC à installer
la télévision sur ordinateur nécessite le plugin VLC by sfr en version 1.1.11, voulez-vous l'installer ?", et que la liste des chaines s'affiche et la vidéo se lance, c'est bon.

=== Installation à partir des sources ===
<note warning>Il semblerait que cette méthode ne fonctionne pas toujours. Si le paquet correspondant à votre version d'Ubuntu est disponible sur [[http://maj.sfr.fr/web/vlc-hls/|le site de SFR]], mieux vaut l'utiliser.</note>
Les sources sont sous deux formes différentes, Mercurial et Tarball. Une fois téléchargées, vous devez évidemment posséder en outre toutes les bibliothèques de développement des librairies que nécessite VLC, comme pour une compilation classique.

Une fois le nécessaire acquis, il faut utiliser la commande <code>./configure</code>.
Si vous utilisez Firefox, ajoutez l'option <code>./configure --enable-mozilla --with-mozillapkg=libxul</code> .
Ensuite, si cette étape s'est bien passée, entrez la commande <code>make</code> puis <code>make install </code>

Pour en savoir plus :
  * [[http://www.n9ws.com/forum/viewtopic.php?p=1011264#p1011264|Discussion sur le forum de n9ws avec les techniciens SFR.]] 
  * [[http://www.n9ws.com/forum/viewtopic.php?t=113826|L'annonce du nouveau système et le suivi de l'avancée de la solution Linux.]]
