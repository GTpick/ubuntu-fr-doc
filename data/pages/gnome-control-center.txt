====== Paramètres système ======

L'application **Paramètres système** regroupe les principaux éléments de configuration d'Ubuntu. Simple et intuitive, elle est facile à utiliser par les débutants.\\
D'autres applications de configuration et réglages plus complexes sont possibles (Utilitaires de disques par exemple). Pour ceux-ci, il suffit de les lancer directement depuis le tableau de bord d'[[ :Unity]] via la [[:unity-lens|loupe]] "**application**" puis "**filtrer les résultats**" et choisir "**système**".\\
\\
{{ :application:gnome-control-center:gnome-control-panel_13.10_01.png?500 |}}

===== Installation =====
L'application est installée par défaut dans Ubuntu.\\
Elle peut ne pas être installée sur certaines [[:variantes]] d'Ubuntu. Dans ce cas, il n'est pas conseillé de l'installer mais d'utiliser le gestionnaire de configuration propre à la variante. \\

Sous d'autres [[ :variantes]] comme Ubuntu [[:Gnome]], le gestionnaire **paramètres Système** est installé par défaut, mais n'est pas rendu accessible dans le [[menu Système]]. Pour l'activer :
   - Faites un clic droit dans votre **tableau de Bord** sur l’**applet** de votre **Menu principal**, puis sélectionnez **Éditer les menus**.
   - En bas de la liste, cliquez sur //Système// et cochez la case //Centre de contrôle//.

=====Lancement=====
Le lancement **Paramètres système** peut s'effectuer de différentes façons :
  * Par le lanceur **Paramètres systèmes** {{:application:gnome-control-center:gnome-control-panel_13.10_02.png?100|}}
  * Depuis le [[:unity#tableau_de_bord_dash|tableau de bord]] d'Unity ;
  * Depuis le menu de l'indicateur système {{:application:gnome-control-center:gnome-control-panel_13.10_03.png?30|}} dans la zone de notification de la barre supérieure du bureau ;
  * Ou bien dans un [[:terminal]], saisir la [[:commande_shell|commande]] suivante :
<code>unity-control-center –overview</code>
<note>Pour [[:precise|Precise 12.04]] la [[:commande_shell|commande]] est :
<code>gnome-control-center –overview</code></note>
=====Utilisation=====
Une fois lancé les **Paramètres système**, il vous suffit de choisir l'élément à modifier. Pas de difficultés particulières. Pour plus d'information sur les paramétrages, consultez dans la [[:|documentation]] la page dédiée à l'élément.
