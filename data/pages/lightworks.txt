====== Lightworks ======
**Lightworks** est une application de montage vidéo de niveau professionnel. Après de nombreuses annonces d'Editshare sur la possibilité de libérer le code de Lightworks, le logiciel reste finalement propriétaire. Une version gratuite a cependant vu le jour. La différence avec la version payante (135 euros par an) se joue sur le support des codecs.
===== Pré-requis =====
  * Avoir ouvert un compte sur le site [[http://www.lwks.com]]
  * Une fois l'inscription validée, télécharger le paquet (.deb) sur le site de l'éditeur
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
===== Installation =====
L'installation du paquet ne pose, a priori, pas de soucis (en tous cas, il n'en a pas posé chez moi).
===== Utilisation=====
Lancez l'application via le menu ou en utilisant la commande :
<code>lightworks</code>
