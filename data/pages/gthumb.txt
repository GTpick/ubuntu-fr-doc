====== gThumb =====
**gThumb** est un [[gestionnaire_de_photos|gestionnaire de photos]] pour [[GNOME]].\\
Outre les fonctions d'importation d'images, d'organisation, et des fonctions simples  communes à la plupart des gestionnaires de photos (rotation, recadrage, correction des yeux rouges, ajustements,...), il permet le traitement par lot de certaines de ses fonctions comme le redimensionnement ou la conversion de format. Il permet le diaporama (F5) affichant une série de photos (en anglais "slideshow"). Assez complet tout en restant accessible facilement, il offre aussi la possibilité de publier en ligne ses photos.

{{ :application:gthumb:gtumb_quantal_03.png?400 |}}

===== Pré-requis =====
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation =====

L'application **Gthumb** est présente dans les dépôts officiels. Pour l'installer, il suffit d'[[:tutoriel/comment_installer_un_paquet|installer le paquet]] **[[apt>gthumb|gthumb]]**
===== Lancement =====
Lancer l'application depuis le [[:unity#tableau_de_bord_dash|tableau de bord]] d'Unity. L'application se trouve dans la catégorie "Multimédia" une fois "filtrer les résultats" de la [[:unity-lens|loupe]] "application" .\\
(Pour les menus classiques :  //Applications -> Graphisme -> 
 Visionneur d'images gThumb//.) \\
Ou dans un [[terminal]] (toutes versions d'Ubuntu), saisir la [[:commande_shell|commande]] suivante :
<code> gthumb </code>

===== Paramétrages =====
  * **__Afficher le Nom des images sous les vignettes du navigateur de diapositives__** : 
<note tip>**Cliquez dans Gthumb** : //Édition -> Navigateur// Puis **cochez** "__Nom__" et retirez ou ajoutez ce que vous voulez voir sous vos vignettes (taille, date etc.)</note>
  * **__Affichage d'une image : Adapter l'image à la fenêtre__** : 
<note tip>**Cliquez dans Gthumb** : //Édition -> Préférences -> Visionneur -> Ajuster à la fenêtre// Puis **Sélectionnez** "__Ajuster à la fenêtre__" </note>
  * **__Paramétrer l'importation de vos images à partir d'un appareil photo ou d'une carte mémoire__** : 
<note tip>**Cliquez dans Ubuntu** : //Paramètres système -> Détails -> Applications par défaut -> Photos// Puis **Sélectionnez** "__GTHUMB__"</note>
Connectez votre appareil, lorsque l'outil d'importation de photos Gthumb apparaît : <note tip>**Cliquez** "__Destination__"</note>
Choisissez un nouveau **dossier de destination**, si le dossier Images ne convient pas.
Si vous le souhaitez, Gthumb peut stocker vos photos dans un dossier par jour en **créant une série de dossiers** en suivant la hiérarchie "//Année -> Mois -> Jour//" Exemple : //home / alex / mages / 2013 / 09 / 13// : toujours dans l'outil d'importation de photos Gthumb
<note tip>**Cochez** : "__Sous dossier automatique__"
----
**Sélectionnez** : "__Date du fichier__" **Puis** : "__Année-mois-jour__"
----
**Décochez** : "__comme sous dossier unique__"
</note>
Puis fermer.

=====Utilisation=====
====Modification de l'image====
Une fois sélectionnée l'image à modifier, cliquer sur le bouton {{:application:gthumb:gtumb_quantal_02.png?25|}} à droite dans la barre d'outils ou simplement avec un "doubleclic" sur l'image. De nombreuses fonctions sont à disposition.

==== Traitement par lots ====
Outil très pratique en l'utilisant par exemple avec le redimensionnement pour transférer des photos trop volumineuses par internet (Publication, expédition,...). \\
Sélectionner les images à traiter (en maintenant la touche "//ctrl"// enfoncée) puis, dans le menu //outils//, appliquer la fonction désirée.

===== Désinstallation =====
Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.

===== Problèmes connus =====

==== Fichiers .gif ====
Lors de la réalisation d'une planche contact, évitez de placer des fichiers .gif dans le dossier source, cette image ne sera pas mise à dimensions et crée des planches  incorrectes.
