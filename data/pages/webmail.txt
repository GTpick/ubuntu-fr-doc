====== Intégrer son webmail à Ubuntu ======

Véritable besoin ou nouvelle mode du cloud computing ? Peu importe ! Il n'empêche que les webmails sont bel et bien présents et que de plus en plus de monde les utilise.

Il peut être très intéressant de les intégrer dans notre environnement de travail préféré.

<note tip>Sachez qu'il est très souvent possible de récupérer les messages de son webmail via un courrielleur de type [[:Evolution]] ou [[:Thunderbird]] : **[[configurer_mail|configurer un client mail]]**. Voir également à ce sujet la page [[:courriel]] qui présente la plupart des clients courriel.</note>

===== Pré-requis =====

  * Disposer des droits d'administration.
  * Disposer d'une connexion à Internet configurée et activée.
=====Avec Firefox=====
==== Firefox comme logiciel par défaut pour accéder aux mails ====

Comme son nom l'indique, le webmail permet d'accéder à nos mails depuis une interface web. Il est donc nécessaire d'utiliser un navigateur internet pour accéder à nos mails du webmail. [[:Firefox]] est le navigateur par excellence qui va nous faciliter grandement l'intégration de notre webmail.

Pour cela, associez [[:firefox]] dans [[:tutoriel:changer_l_association_par_defaut_des_fichiers_avec_les_applications|les paramètres systèmes]] pour le logiciel de messagerie.

Firefox sait gérer nativement les adresses emails. Cette seule configuration est déjà très pratique.

==== Votre webmail comme client de courriel par défaut dans Firefox ====

Maintenant, il nous faut déclarer à Firefox qu'il doit utiliser notre webmail pour envoyer un email.

Pour cela, il suffit de sélectionner notre webmail dans le menu déroulant de **//Firefox -> Édition -> Préférences -> Applications -> mailto//**. 

Vous aurez le choix entre Gmail et Yahoo! Mail.

Si votre webmail n'est pas l'un des deux précédents, laissez le choix par défaut à savoir Thunderbird.
   

==== Utilisation====

Dorénavant, lorsque vous cliquerez sur une adresse email d'un document (dans un fichier odt, un pdf...) ou d'une page web, vous ouvrirez automatiquement une page de rédaction de votre webmail avec le champ de destinataire déjà rempli.

Bien entendu, ce fonctionnement automatique sous-entend que vous avez enregistré le mot de passe de votre webmail dans Firefox.


===== Recevoir des notifications automatiques =====

<note tip>Si vous avez configuré un client de [[:courriel]] pour récupérer automatiquement vos messages sur votre webmail (comme [[:Evolution]] ou [[:Thunderbird]] par exemple), le notifieur du logiciel fera très bien l'affaire.</note>

Il existe **plusieurs outils au choix** pour recevoir des notifications d'arrivées de messages :

==== Pour Gmail ====

  - [[:CheckGmail]] ;
  - [[:gnome-gmail-notifier]] ;
  - [[:gm-notify]] ;
  - [[:gmail-notify]] ;
  - **[[apt>kgmailnotifier]]** pour KDE ;
  - **[[apt>xfce4-mailwatch-plugin]]** pour XOffice 4 : Gmail, pop3, imap.


==== Autres Webmails ====

  - **[[apt>coolmail]]** : un notifier de mail avec des effets graphiques ;
  - **[[apt>bbmail]]** pour Blackbox / Fluxbox ;
  - **[[apt>xfce4-mailwatch-plugin]]** pour XOffice 4 : Gmail, pop3, imap
  - **[[Mail notification]]** pop3, imap...
