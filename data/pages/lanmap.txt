====== LANMAP, utilitaire pour générer le plan de son réseau ======


**Lanmap** est un utilitaire permettant de générer le plan d'une architecture de réseau local. Lanmap schématise dans un fichier PNG, SVG ou GIF la liste des stations du réseau avec leurs adresses MAC, IP, leurs nom d'hôte ou encore le système d'exploitation de la station détectée.

<note warning>Lanmap n'est plus disponible. Son développement est abandonné. Son développeur a ré-écrit une version lanmap2 disponible sur [[https://github.com/rflynn/lanmap2/tree/master]]</note>

===== Utilisation =====

Exemple :

  lanmap -i eth0 -r 30 -T png -o /tmp/

Pour plus d'informations consultez la documentation livrée avec le logiciel : 

  man lanmap

===== Interface graphique =====

Vous pouvez aussi utiliser une interface graphique pour //lanmap//. Elle se nomme [[http://code.google.com/p/wxlanmap/|wxlanmap]] et est développée par un membre du forum ubuntu-fr, à l'aide du langage de programmation Python et son API wxPython.
