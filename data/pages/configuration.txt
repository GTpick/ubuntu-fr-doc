====== Configuration Live du Thème. ======
===== Fonctionnalité =====

Cette fonctionnalité vous permet de modifie certain aspect du site avant leurs mise en dure dans la version finale du Thème. Elle ce sert d'un code java-script pour modifier les directives css et attrapé certains événement (click, pression de touche).
