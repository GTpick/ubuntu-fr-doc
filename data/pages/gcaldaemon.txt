====== GCalDaemon, l'outil de synchronisation des agendas ====

===== Introduction =====
[[http://gcaldaemon.sourceforge.net/|GCalDaemon]] est une application permettant de synchroniser dans les deux sens vos agendas [[http://google.com/calendar|Google Agenda]] et vos agendas installés avec [[Evolution]], [[Thunderbird]] ou toute autre application acceptant le format ICS.

GCalGaemon est distribué sous licence LGPL.

<note>GCALDaemon permet de servir d'interface entre votre logiciel (Sunbird, Evolution, Rainlendar, KOrganizer...) et Google Calendar.
**IL PERMET LA SYNCHRONISATION DANS LES 2 SENS**.
Pour Sunbird, il utilise les accès/synchronisation HTTP (donc très simple à paramétrer). En revanche, pour les autres logiciels, il faut modifier le fichier [[:tutoriel:comment_editer_un_fichier|gcal-daemon.cfg]] pour pouvoir activer la synchro.</note>

===== Installation =====
Téléchargez le deb [[https://blog.heckel.xyz/2008/09/30/gcaldaemon-deb-package-for-ubuntu-kubuntu/|ici]].


Vous aurez alors deux nouvelles entrées dans le menu "Accessoires" : l'une pour configurer vos agendas à synchroniser, l'autre pour lancer une synchronisation ponctuelle. Notez qu'une synchronisation automatique a lieu toutes les 15 minutes (option à configurer dans le fichier /etc/gcaldaemon/gcaldaemon.conf).
