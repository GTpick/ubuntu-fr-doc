====== Mirage ======

**Mirage** est une visionneuse écrite en [[wpfr>Python (langage)|Python]] avec GTK+. Elle est indépendante de l'environnement utilisé et se révèle légère et pratique.

{{:applications:mirage_xenial.png?450}}

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation =====
[[:tutoriel:comment_installer_un_paquet|Installez le paquet]] **[[apt>mirage]]**.

===== Utilisation =====
 
Lancez l'application via le [[:unity#tableau_de_bord_dash|tableau de bord]] (Unity) ou via le [[:terminal]] (toutes versions d'Ubuntu) avec la [[:commande_shell|commande]] suivante : <code>mirage</code>

===== Configuration =====

Il peut être intéressant d'activer l'option « Charger le dossier » dans les préférences, afin de pouvoir faire défiler les images avec les touches « ← » et « → ».
