====== Gsambad / Gadmin-Samba ======

**Gadmin-samba** (ancien nom GSambad) est une interface graphique qui permet de faciliter la configuration d'un serveur samba de façon avancée; si vous voulez utiliser samba de façon simple il est recommandé d'utiliser [[:system-config-samba]] (Voir la page [[:samba]]).

===== Installation =====

Le programme a changé de nom pour Gadmin-samba. 
 [[:tutoriel:comment_installer_un_paquet|Installez le paquet]] **[[apt>gadmin-samba]]** du [[:depots|dépôts universe]].

===== Utilisation =====

Pour ouvrir ce programme, faites une recherche dans  [[unity#tableau_de_bord_dash|le tableau de bord Unity]] avec le mot clé **SAMBA**.

À la première ouverture le programme vous demandera s'il peut modifier votre fichier **/etc/samba/smb.conf** pour y mettre sa configuration par défaut, faites oui, le programme sauvegardera l'ancien fichier et le remplacera par sa configuration.

<note important>Cette manipulation empêchera tout fonctionnement du raccourci **clic droit->Option de partage** disponible par défaut sous Ubuntu et désactivera tous les partages en cour ! Tout partage sera donc à paramétrer via gadmin-samba.

Bien sûr il est possible de faire machine arrière facilement, en ouvrant un [[terminal]] et en exécutant cette commande :<code>sudo mv /etc/samba/smb.conf.old.gadmin-samba* /etc/samba/smb.conf</code>
</note>
