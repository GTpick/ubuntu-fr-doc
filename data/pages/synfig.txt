====== Synfig: logiciel d'animation vectorielle ======

Synfig est un logiciel libre, multi plate-forme et gratuit d'animation vectorielle (2D) pour produire des films d'animation. 

Il est conçu pour créer des films d'animation de qualité et utilise le dessin vectoriel et bitmap. Il élimine la nécessité de créer les animations image par image, et vous permettra de produire des animations de plans (2D) d'une meilleure qualité avec moins de ressources.

===== Pré-requis =====
 
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  * Avoir activé l'accès au [[:depots|dépôt Universe]].

=====Installation=====

Sous [[:trusty|Ubuntu 14.04]] et ultérieures il suffit d'[[:tutoriel:comment_installer_un_paquet|installer les paquets]] **[[apt>synfig]]** et **[[apt>synfigstudio]]**, sans quoi vous n'aurez pas d'interface graphique.


Sous [[:Precise|Ubuntu 12.04]] ou pour installer la dernière version stable, téléchargez et [[:tutoriel:comment_installer_un_paquet|installez le paquet]] **.deb** adapté à votre système (i386 ou amd64) sur [[http://www.synfig.org/cms/en/download/stable|cette page]].

===== Utilisation =====

Pour lancer l'application faites une recherche dans [[unity#tableau_de_bord_dash|le tableau de bord Unity]] avec le mot clé **Synfig**.

Le logiciel fonctionne avec des objets et une ligne de temps. Il interpole automatiquement les objets entre deux animations avec des possibilités très avancés pour la gestion des façons de passer d'un point a l'autre de l'animation : de manière direct, exponentielles, tangete BLINe ... 

Ces objets se présentent sous formes de calques. Il peuvent être type primaires :
  * Calques de géométries (Damier, Cercle, Contour, Contour avancé, Polygone, Rectangle, Région, Couleur solide, Étoile)
  * Dégradés (Conique, Courbe, Linéaire, Bruité, Radial, Spiral)
  * ...
ou de type filtres :
  * Distorsion (Courbe, Sphérique, Étirer, Tourbillons, Bruitée, Inverse, Perspective)
  * Transformation (Rotation, Décaler, Échelle)
  * ...

Il permet également de manipuler des éléments de l'animation par la liaison de données et l'ajout de formules mathématiques entre les paramètres des calques pour créer de animations encore plus complexes et réalistes : trajectoires / particules / simili 3D / collisions ...

==== Import fichiers vectoriels ====
[[:Inkscape]] à partir de la version 0.49 peut enregistrer directement dans le format de fichier **sif** de Synfig.
Avant la version 0.49 il faut installer l'extension **svg2sif** (donc valable pour [[:precise|Ubuntu 12.04]] et [[:trusty|Ubuntu 14.04]]):
  * Téléchargez [[:archivage|l'archive]] [[https://github.com/nikitakit/svg2sif/|ici]] en cliquant sur "Download", vous pouvez aussi télécharger les sources en les clonant avec [[:git]] : <code>git clone git://github.com/nikitakit/svg2sif.git </code>
  * Pour l'installer pour tous les utilisateurs, avec les [[:sudo|droits d'administration]] [[:archivage|extraire]] les fichiers de type **.py** et **.inx** dans **/usr/share/inkscape/extensions**, pour l'installer uniquement pour l'utilisateur courant, dans [[:fichier_cache|le dossier caché]] **.config/inkscape/extensions** de votre **Dossier Personnel**,
  * Ouvrer votre fichier dans Inkscape. Préparez votre dessin pour l'export dans //Menu -> Extensions -> Synfig -> Prepare for export// (cela peut prendre du temps),
  * Puis "enregistrer sous" votre fichier dans un format de fichier directement lisible par Synfig //Menu -> Fichier -> Enregistrer sous : type de fichier Synfig Animation (*.sif)//,
  *Ouvrez votre fichier avec Synfig !

==== Lecteur html5 pour fichiers sif ====
Un lecteur de fichier **sif** (format de fichier animation Synfig Studio) pour html5 est en cours de développements. Il permet déjà de lire des fichiers d'animation simple. Voir [[https://github.com/haramanai/sifplayer|ici]].
