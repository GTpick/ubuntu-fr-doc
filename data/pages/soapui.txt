====== SoapUI ======

[[http://www.soapui.org|SoapUI]] est un outil de test pour [[wpfr>Architecture_orient%C3%A9e_services|SOA]] et [[wpfr>Service_Web|Web Service]].

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation pour Ubuntu 12.04 =====

  * [[:PPA|Ajoutez le PPA]] **ppa:upubuntu-com/web**((https://launchpad.net/~upubuntu-com/+archive/web)) dans vos sources de logiciels ;
  * [[:tutoriel:comment_modifier_sources_maj#recharger_la_liste_des_paquets|Rechargez la liste des paquets]] ;
  * [[:tutoriel:comment_installer_un_paquet|Installez le paquet]] **[[apt>soapui|soapui]]**.

Ou en une ligne :
<code>
sudo add-apt-repository ppa:upubuntu-com/web && sudo apt-get update && sudo apt-get install soapui
</code>

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]].
