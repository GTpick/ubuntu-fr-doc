====== Key-mon ======  
**[[http://code.google.com/p/key-mon/|Key-Mon]]** est un utilitaire permettant d'afficher en temps réel les commandes entrées au clavier et à la souris. Il est utile pour des démonstrations, des cours, ou lors de l'enregistrement d'une [[wpfr>vidéographie]].

{{ :accessibilite:keymon_xenial.png }}

===== Pré-requis =====
 
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
===== Installation  =====
[[:tutoriel:comment_installer_un_paquet|Installer le paquet]] **[[apt>keymon]]**. 

===== Utilisation  =====
Lancez l'application via le [[:unity#tableau_de_bord_dash|tableau de bord]] ou via le [[:terminal]] (toutes versions d'Ubuntu) avec la [[:commande_shell|commande]] suivante : <code>key-mon</code>. 

=====Configuration=====
Il est possible de déplacer la fenêtre où vous le souhaitez avec un clic gauche maintenu. 
Pour modifier les préférences, il faut faire apparaître le menu. Faites un clic droit sur la fenêtre. Le menu vous permettra d'accéder aux préférences du logiciel, ainsi que de quitter Key-Mon. 

Raccourcis clavier :  (fonctionnent une fois Key-Mon **sélectionné** avec un clic gauche sur la fenêtre)
  *Ctrl + S pour afficher le menu.
  *Ctrl + Q pour quitter Key-Mon.

===== Désinstallation =====
 
Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
