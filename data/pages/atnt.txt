====== ATNT ======

Logiciel équivalent à Regressi pour les TP de physique/chimie.

{{ :atnt_courbes.png?250}}

===== Pré-requis =====

  * Nécessite Java.


===== Installation =====

  * Site de téléchargement : http://atn.adullact.net/
  * Lien de téléchargement direct pour la version 0.16.0 (dernière version stable au jour du 17/02/2010) : [[https://adullact.net/frs/download.php/4288/install-ATNT_0.16.0.jar]]

Lancez l'installation //via// la [[:commande_shell|commande]] suivante (pour un fichier téléchargé situé dans //dossier personnel// et nom de fichier à modifier si besoin) :

  java -jar install-ATNT_0.16.0.jar
  

Pour le lancer, depuis le tableau de bord (Dash), taper : //ATNT//

===== Voir aussi =====

  * Plus d'informations : http://franck.fresnel.free.fr/ATNT/Presentation.php
