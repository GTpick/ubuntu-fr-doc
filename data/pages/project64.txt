====== Project 64  ======
{{:hdr_logo.png?nolink|}}
===== Presentation =====
Project64 (communément abgregé PJ64) est un emulateur pour la console nintendo 64. A l'heure actuelle, il s'agit du plus puissant emulateur pour cette console.
<note warning>Il a plusieurs defauts : Il n'est pas open source, n'est pas compatible sous ubuntu de manière native et l'installeur est truffé d'adwares !</note>
===== Installation =====
Installez Wine et vos drivers graphiques. Intallez ensuite la version que vous voulez. La version 1.6 tourne parfaitement.

Vous pouvez la télécharger ici : [[http://www.pj64-emu.com/downloads/func-startdown/34/]] (site signalé comme malveillant par Firefox).

Indiquez votre langue et un dossier pour vos roms.
===== Bugs =====
Bug du logiciel :
<note help>Le nom de ma rom est "bad rom! [...]"</note>
<note important> Checksum erroné. Retelechargez une autre rom. Verifiez que [!] (dump parfait) apparait a la fin du nom de fichier</note>

Bugs du a wine:
<note>Je n'ai jamais eu de problème de compatibilité, du moins avec la version 1.6</note>
