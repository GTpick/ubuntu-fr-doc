====== Liste des Imprimantes et Multifonctions BROTHER ======

Cette page à pour but de compléter le [[:imprimantes_brother|tutoriel d'installation]] en vous aidant à identifier et à installer les pilotes nécessaires au bon fonctionnement de votre imprimante ou multifonction de la marque Brother.

===== Si les pilotes sont installés par défaut =====

Certains modèles ont la chance d'avoir des pilotes pré-installés sous Ubuntu. Pour déterminer dans quel cas se trouve votre matériel, consultez les tableaux de cette page en vous basant sur la grille d'interprétation des icônes fournie ci-après. Si vos pilotes sont préinstallés, vous n'avez rien à faire et vous pouvez directement retourner au **[[:imprimantes_brother|tutoriel d'installation]]**. Dans les autres cas, installez les pilotes recommandés par les tableaux.

Gardez cependant à l'esprit que les pilotes préinstallés ne sont pas ceux fournis par Brother et qu'il est possible que ceux-ci ne permettent pas d'exploiter toutes les spécificités de votre imprimante.

Si, après test, vous n'êtes pas satisfait de ces pilotes, vous pourrez installer des pilotes d'impression alternatifs (si disponible pour votre modèle, référez vous à la colonne « remarque ») ou encore utiliser la « **[[#Méthode alternative d'obtention des pilotes]]**  » ci-dessous pour installer les pilotes fournis par Brother.

<note tip>
N'oubliez pas de compléter cette page si vous constatez que votre modèle a de meilleures performances avec un pilote autre que celui actuellement recommandé !((Les utilisateurs du même modèle que vous vous en remercieront grandement. 8-) ))
</note> 

<note help>
N'hésitez pas à compléter cette documentation en rajoutant des modèles qui n'y seraient pas référencés ou encore d'éventuelles remarques que vous jugez intéressantes !

Cependant, cette page étant déjà longue, essayez (autant que faire se peut) de rester concis.
</note>

===== Interprétation des icônes =====

Dans les listes ci-dessous les icônes ont la signification suivante:\\

| {{:apt.png?nolink|Installer les paquets…}} | Vous pouvez facilement installer les pilotes d'impression en cliquant sur cette icône. Notez que vous devrez préalablement avoir activé le dépôt [[:depots#universe_et_multiverse|multiverse]]. |
| [[:liste_imprimantes_brother#methode_alternative|{{:icons:important.png?nolink| Fonctionne avec la méthode alternative}}]] | Fonctionne avec la méthode alternative. |
| {{:icons:process-stop.png?nolink|Ne fonctionne pas}} | Le modèle ne semble pas être supporté par Ubuntu. |

En vous aidant de la table des matières en haut à droite de cette page, repérez votre modèle dans la liste ci-dessous.
S'il n'y figure pas, référez-vous à la **[[:liste_imprimantes_brother#Méthode alternative d'obtention des pilotes|méthode alternative d'obtention des pilotes]]** décrite en fin de page.

**Si vous n'arrivez pas à installer les pilotes en cliquant sur le lien "apt"** (icône), installez les avec les commandes suivantes :

<code>
sudo apt-get install pilote1
sudo apt-get install pilote2
</code>

où //''pilote1''// et //''pilote2''// sont les noms relevés dans l'info-bulle qui apparaît lorsque vous passez le pointeur sur l'icône.

**Pour les pilotes de numérisation**, le lien correspondant à votre modèle vous conduira sur la page de téléchargement du pilote en question sur le site de Brother.
Téléchargez et [[:tutoriel:comment_installer_un_paquet|installez les paquets]] au format //.deb// en 32 ou 64 bits (selon votre architecture).
Retourner ensuite sur le **[[:imprimantes_brother|tutoriel d'installation]]**.

===== Méthode alternative d'obtention des pilotes=====

<note important>Utilisez cette méthode uniquement si votre modèle n'apparait pas dans la liste ci-dessus.</note>

Vous pouvez télécharger les pilotes nécessaires depuis [[http://support.brother.com/g/s/id/linux/en/index.html?c=us_ot&lang=en&comple=on&redirect=on|le site officiel de Brother]] qui donne des liens pour les drivers imprimante, scanner et fax.

Depuis mars 2014 Brother à crée un script facilitant l'installation de votre imprimante, si vous désirez vous pouvez suivre ce tutoriel: [[:brother_driver_install_tool|brother_driver_install_tool]]

==== Pilotes d'impression ====

<note important>Si vous utilisez une **version [[:Ubuntu 64bits|64 bits]] d'Ubuntu**, sachez que Brother ne fournit pas de pilote d'impression pour cette architecture ((Leurs pilotes pour les scanners supportent pourtant le 64 bits, allez comprendre...)). Information à revoir : si vous utilisez "brother_driver_install_tool" cité ci-dessus, l'installation fonctionne sans autre ajout sur Ubuntu 64 bits depuis 13.10 pour une imprimante DCP-J

Vous devrez installer les pilotes en version 32 bits en utilisant la méthode suivante : [[:Ubuntu_64Bits#Utiliser des programmes 32 bits sur Ubuntu 64 bits]].
</note>


Vous devez tout d'abord créer manuellement les dossiers  **''/usr/share/cups/model''** et **''/var/spool/lpd''**.\\
Pour se faire, vous pouvez exécuter les commandes suivantes dans un [[:terminal]] :

  sudo mkdir /usr/share/cups/model
  sudo mkdir /var/spool/lpd


Repérez ensuite votre modèle puis téléchargez et [[:tutoriel:comment_installer_un_paquet|installez les paquets]] au format **//.deb//** correspondant sur la page suivante : **[[http://support.brother.com/g/s/id/linux/en/download_prn.html]]**


==== Pilote de numérisation ====

Repérez votre modèle dans la page suivante :

**[[http://support.brother.com/g/s/id/linux/en/download_scn.html]]**

Puis, téléchargez et [[:tutoriel:comment_installer_un_paquet|installez les paquets]] au format //.deb// correspondant en 32 ou 64 bits (selon votre architecture).\\ Retournez ensuite sur le **[[:imprimantes_brother|tutoriel d'installation]]**.

==== Pilotes pour 3 en 1 ====

Les pilotes de certains multifonctions faisant également office de fax sont disponibles sur la **[[http://support.brother.com/g/s/id/linux/en/download_pcf.html|page suivante]]**.

===== Que faire après l'installation des pilotes ? =====
Il ne vous reste plus qu'à retourner sur le **[[:imprimantes_brother|tutoriel d'installation]]** pour poursuivre et terminer la configuration de votre matériel. 

==== Si votre modèle ne figure dans aucun des liens précédents ====

Dans ce cas, cela signifie que Brother ne fournit pas de pilotes pour votre modèle.\\
Si votre modèle est récent, il est possible que Brother publie des pilotes lui correspondants sous peu. Cependant, cela reste à son bon vouloir.

Si votre imprimante n'est pas présente dans cette page, vous pouvez tester son installation avec le **[[:utilisateurs:demonipuch:script_brother|le script de Demonipuch]]** dont le suivi sur le forum est [[http://forum.ubuntu-fr.org/viewtopic.php?id=652931|ici]]
