====== Ubuntu 11.04 (« The Natty Narwhal ») ======

<note critique>**Fin de vie depuis le 28 octobre 2012** \\ Depuis le 28 octobre 2012, Ubuntu 11.04 est en fin de vie (//end of life//, //EOL//). Elle ne profite plus de nouvelle mise à jour de sécurité et les paquets logiciels pour cette version sont déplacés dans des serveurs d'archive. Pour continuer à recevoir des mises à jour de sécurité, veuillez mettre à niveau votre version d'Ubuntu vers [[versions#versions_d_ubuntu_activement_soutenues|une version activement soutenue]].</note>

{{:utilisateurs:alexandrep:unity_3.png?250  |Ubuntu 11.04, arborant la nouvelle interface par défaut, Unity.}}

**Ubuntu 11.04 //(nom de code : The Natty Narwhal)//** est la quatorzième version d'Ubuntu. Son développement s'est échelonné sur six mois, jusqu'à sa publication en tant que version stable le 28 avril 2011. En tant que version intermédiaire (située entre deux versions [[LTS]]), elle profite de mises à jour de sécurité pendant une durée de 18 mois, soit jusqu'en octobre 2012.

== À propos du nom de code de développement ==
Le nom de cette version, choisi par Mark Shuttleworth, //The Natty Narwhal// (//le narval chic//) a été divulgué sur [[http://www.markshuttleworth.com/archives/478|son blog, le 17 août 2010]]. Ce nom symbolise à la fois la volonté de soigner l'apparence d'Ubuntu, apparue depuis [[lucid#nouvelle_identite_visuelle|la précédente version LTS]] (Ubuntu se doit d'être agréable dès le premier coup d'œil) et la notion de diversité représentée par le narval, une espèce animale faisant penser à la licorne. Mark Shuttleworth y voit une image de respect, de partage et en même temps de rêve.

===== Changements apportés depuis la version précédente =====

==== Unity : interface par défaut... pour les postes de travail ! ====
{{  :utilisateurs:alexandrep:unity-natty.png?350|L'interface Unity remplace l'interface classique.}}

L'interface [[Unity]] avait fait ses débuts dans la précédente version d'Ubuntu en tant qu'interface privilégiée pour les //netbooks//, les ordinateurs à petits écrans. Avec Ubuntu 11.04, Unity est développé davantage pour devenir aussi l'interface principale pour les postes de travail. De gros travaux d'optimisation ont été apportés (passage du gestionnaire de fenêtres Mutter à Compiz, par exemple), et une attention particulière est apportée à la manière de gérer les fichiers et dossiers. Unity est activé par défaut dans les ordinateurs disposant de pilotes graphiques capables de générer des effets 3D. Lorsque les pilotes ne sont pas installés ou ne possèdent pas cette capacité, l'[[gnome-panel|interface habituelle de GNOME 2.x]] est utilisée.((Une version de Unity utilisant Qt fonctionnant uniquement avec des pilotes graphiques 2D est en cours de développement. Bien que Unity 2D soit installable par la [[software-center|Logithèque]], la version finale de Unity 2D [[oneiric|n'est attendue que pour Ubuntu 11.10]].))

Unity pour ordinateurs de bureau n'est pas un clone de la version pour netbook, mais plutôt une amélioration mixant plusieurs éléments : une barre de lanceurs des applications (Unity Launcher) remplaçant un tableau de bord unique listant les fenêtres ouvertes, un contrôle interactif (Dash) qui remplace les menus de raccourcis et App Menu qui déplace les barres de menus des applications dans le tableau de bord supérieur, pour une utilisation plus efficace de l'espace de travail.

La zone de notifications, où viennent se loger certains boutons d'applications réduites près de l'horloge, n'est pratiquement plus utilisée. L'accent est mis sur les indicateurs (//[[indicator-applications|app indicators]]//), ces menus contextuels basés selon l'utilité (comme le menu de son) qui ont commencé à apparaître dès Ubuntu 10.04 LTS. Les indicateurs actuels sont davantage peuplés et de nouveaux indicateurs font leur apparition, tels que l'indicateur de réseau.

Consultez [[unity|les documents traitant de Unity]] pour plus d'informations sur son utilisation et sa personnalisation.

==== Logithèque Ubuntu : avec des critiques et des évaluations ====
{{  http://www.chimerarevo.com/wp-content/uploads/2011/01/ubuntu-software-center-valutazione-recensione.png?350|La Logithèque permet désormais de laisser ses commentaires et ses évaluations à propos des applications.}}

La [[software-center|Logithèque Ubuntu]] permet désormais aux utilisateurs de lire et soumettre des critiques à propos des applications disponibles, ainsi que de leur donner une note. Cette évolution longuement attendue aidera les utilisateurs d'Ubuntu à découvrir des logiciels de qualité parmi la multitude des applications présentes dans la Logithèque.

Le mécanisme d'évaluation repose sur trois axes :
  * la Logithèque elle-même, pour afficher les évaluations des autres utilisateurs et enregistrer ses propres critiques ;
  * le service Ubuntu Single Sign-On (USSO) -- le même identifiant qu'utilisé avec [[ubuntuone|Ubuntu One]] --, pour identifier les utilisateurs soumettant des évaluations ;
  * un serveur de stockage dans Internet, pour contenir et publier l'ensemble des évaluations reçues des utilisateurs.

De manière optionnelle, les utilisateurs peuvent aussi partager leurs évaluations vers les réseaux sociaux auxquels ils sont inscrits grâce à l'intégration du client de réseaux sociaux [[Gwibber]]. Le système d'évaluation fonctionne sensiblement de la même manière que dans Android Market, avec la possibilité de rapporter des critiques offensantes ou inappropriées.

=== Test d'applications sans installation ===
Toujours de manière optionnelle, l'administrateur peut installer un greffon supplémentaire, [[weblive|WebLive]] ([[apt>qtnx|Installation ici]]), permettant de [[http://www.stgraber.org/2011/03/06/introducing-the-weblive-api/|tester une application sans installation]]. Ce greffon ajoute un bouton **Test drive** sous la capture d'écran des principales applications disponibles dans la Logithèque Ubuntu. Lorsque pressé, après quelques secondes de chargement, une fenêtre s'ouvre avec l'application concernée, qui s'exécute à distance dans un serveur distant, sans que l'application soit installée localement.

==== Logiciels par défaut : des maintiens et des nouveautés ====

=== Ubuntu pour postes de travail / pour netbooks ===
{{  http://www.omgubuntu.co.uk/wp-content/uploads/2010/10/147134841.png?250|Banshee sera le lecteur audio par défaut dans Ubuntu 11.04.}}

  * L'environnement de bureau [[gnome|GNOME 2.32]] et ses logiciels mis à jour. L'interface par défaut, constituée de [[Gnome-Panel]], est toutefois remplacée par [[Unity]]. L'interface classique reste disponible sous l'appellation //Ubuntu classique//. ([[unity#choisir_l_interface_de_session|Comment sélectionner l'interface de session?]]);
  * [[Banshee]] remplace le lecteur de musique [[Rhythmbox]] en tant que lecteur par défaut ;
  * [[LibreOffice]] remplace [[OpenOffice.org]] en tant que suite bureautique ;
  * [[Firefox]] 4.0 devient le navigateur Web par défaut ;
  * Le client d'affichage distant [[Remmina]] remplace [[tsclient|Terminal Server Client]].

=== Kubuntu ===
{{  http://www.bordel-de-nerd.net/wdp_photo/owncloud1.png?250|OwnCloud, la solution de stockage personnel nuagier développé par le projet KDE, est disponible en paquet pour Kubuntu 11.04.}}
  * L'environnement de bureau [[KDE|KDE 4.6]] et ses applications mises à jour ;
  * Le framework multimédia Phonon utilise désormais le système multimédia GStreamer, afin d'être cohérent avec les autres variantes de la famille Ubuntu et pour un développement facilité de greffons pour les navigateurs Web (en prévision de HTML5) et les lecteurs multimédias ;
  * Une compilation de jeux de cartes (solitaires), [[KPatience]], est inclus dans le CD-ROM de Kubuntu;
  * Un nouveau module de sélection de langue est proposé;
  * Le gestionnaire de fichiers [[Dolphin]] inclut désormais un onglet pour configurer facilement le partage de dossiers à travers un réseau local;
  * Le service de stockage à distance OwnCloud est mis en paquets et installable dans cette mouture de Kubuntu.

=== Xubuntu ===
  * L'environnement de bureau [[xfce|Xfce 4.8]] et ses applications mises à jour;
  * Un nouveau thème, GreyBird;
  * Les menus de Xfce 4.8 sont désormais modifiables avec n'importe quel éditeur de menu compatible avec la norme FreeDesktop.org. L'éditeur suggéré est [[Alacarte]];
  * [[Gmusicbrowser]] remplace [[Exaile]] en tant que lecteur audio par défaut.

=== Edubuntu ===
  * [[LibreCad]] remplace [[QCad]] en tant que logiciel de dessin assisté par ordinateur;
  * Des nouvelles applications sont proposées: [[Pencil]], [[Geogebra]], [[Calibre]], [[Freemind]] et [[Stellarium]].

==== Mise à niveau de la distribution : maintenant possible avec le CD-ROM pour postes de travail ====
Dans les versions antérieures d'Ubuntu, ceux qui souhaitaient effectuer une mises à niveau hors-ligne, à l'aide d'un CD-ROM, devaient obligatoirement récupérer un CD-ROM particulier (le //CD-ROM alternatif//). Ubuntu 11.04 inclut désormais les outils de mises à niveau directement dans le CD-ROM d'installation pour postes de travail. Ceci vous permet d'économiser de la bande passante, en ne récupérant qu'un seul CD-ROM pour à la fois tester la nouvelle version d'Ubuntu en session live, puis procéder à votre mise à niveau hors-ligne. [[http://www.guideubuntu.com/2011/04/mise-niveau-vers-ubuntu-1104-natty.html|Mise à niveau vers Ubuntu 11.04 Natty Narwhal à l'aide d'un Live CD]]

==== Ubuntu One ====
{{  :natty-u1controlpanel.png?250|Le nouveau centre de contrôle d'Ubuntu One}}
  * Le service de [[ubuntuone|synchronisation de fichiers d'Ubuntu One]] voit ses performances améliorées ;
  * Il est désormais possible de sélectionner les dossiers à synchroniser avec chaque ordinateur (par exemple, synchroniser son dossier //Musique// entre Ubuntu One et son poste de travail, mais pas vers son netbook qui dispose de trop peu de mémoire). Ceci est gérable à travers le centre de contrôle d'Ubuntu One, qui est remanié. Si un nouveau dossier apparaît dans la liste, une notification vous en informe;
  * Lors d'une synchronisation, l'icône dans la barre de lanceurs d'Unity contient une barre de progression de la synchronisation;
  * La gestion de Ubuntu One et ses divers services s'effectue dans un centre de contrôle entièrement retravaillé.

==== Sous le capot ====
  * Le noyau Linux 2.6.38, contenant le fameux patch de 200 lignes de code qui améliore le temps de réponse pour le traitement du multi-taches. D'autres avancées, telles les performances accrues de la gestion des systèmes de fichiers, des nouveaux pilotes de nouveaux périphériques et un plus grand nombre de cartes graphiques prises en charge par des pilotes 3D natifs, sont incluses dans ce nouveau noyau;
  * Le serveur X.org 1.10 avec Mesa 7.10 propose une meilleure prise en charge de cartes ATi Radeon et introduit la gestion des puces Sandy Bridge d'Intel;
  * [[Upstart]] a été mis à jour à la version 0.9.4-1, apportant de nombreuses fonctionnalités : détection du mode "chroot", visualisation basique de tâches/d’événements, nouvelles instructions pour ''initctl'' (''show-config'' et ''check-config'') etc.;
  * [[Ubuntu Netbook Edition]] n'est désormais disponible que pour les puces OMAP3 et OMAP4. Les autres fonctionnalités qui faisaient de l'édition pour netbooks à processeurs x86/x64 une variante bien spécifique (entre autre, l'interface Unity) étant intégrées dans l'édition pour postes de travail, il n'est plus nécessaire de conserver deux variantes spécifiques;
  * udisks et upower, utilisés pour la reconnaissance des périphériques de stockage et pour la gestion de l'énergie, remplacent HAL dans Kubuntu.

===== Obtenir et utiliser Ubuntu 11.04 =====
{{  http://shop.canonical.com/images/UBN00159.jpg?250|Téléchargez maintenant Ubuntu 11.04}}

Veuillez noter qu'Ubuntu 11.04 n'est plus une version activement soutenue depuis le 28 octobre 2012. Ceci signifie que plus aucune mise à jour de sécurité n'est publiée pour cette version d'Ubuntu. Il est vivement conseillé de mettre à niveau Ubuntu vers une version ultérieure, toujours activement soutenue.

Néanmoins, il n'y a pas de date d'arrêt de fonctionnement à votre système Ubuntu : les versions d'Ubuntu arrivées en fin de vie sont toujours techniquement fonctionnelles. Il n'y a pas de mécanisme bloquant artificiellement le fonctionnement de votre système Ubuntu, et vous pouvez continuer à l'utiliser en connaissance de cause, jusqu'à ce que vous décidiez à mettre à niveau votre système d'exploitation. Notez que les dépôts officiels ne proposeront bientôt plus de paquets/logiciels pour Ubuntu 11.04. Ceux-ci seront migrés vers [[old-releases|un dépôt d'archives, destinés à contenir les logiciels des versions obsolètes d'Ubuntu]]. Les paquets de ces dépôts ne sont évidemment plus mis à jour; ils vous permettent seulement de continuer à utiliser la Logithèque ou tout autre système d'installation de logiciels.

  * Pour récupérer un CD-ROM d'installation d'Ubuntu 11.04, accédez [[http://old-releases.ubuntu.com/releases/|au serveur de téléchargement des versions archivées]].
  * Pour installer des logiciels, modifiez votre fichier de sources de logiciels (///etc/apt/**sources.list**//) afin de le faire pointer vers les serveurs d'archives: <code>deb http://old-releases.ubuntu.com/ubuntu/ natty main restricted universe multiverse

#deb-src http://old-releases.ubuntu.com/ubuntu/ natty-updates main restricted universe multiverse

deb http://old-releases.ubuntu.com/ubuntu/ natty-security main restricted universe multiverse
#deb-src http://old-releases.ubuntu.com/ubuntu/ natty-security main restricted universe multiverse

deb http://old-releases.ubuntu.com/ubuntu/ natty-backports main restricted universe multiverse
#deb-src http://old-releases.ubuntu.com/ubuntu/ natty-backports main restricted universe multiverse

deb http://old-releases.ubuntu.com/ubuntu/ natty-proposed main restricted universe multiverse
#deb-src http://old-releases.ubuntu.com/ubuntu/ natty-proposed main restricted universe multiverse</code>
