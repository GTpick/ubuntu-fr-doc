====== Freezer : widget Deezer pour Ubuntu ======
Freezer est un widget (un élément graphique de bureau) qui permet d'écouter vos listes de lecture du site de musique en ligne [[http://www.deezer.com|Deezer.com]].

{{  :freezer.png  }}

===== Installation =====
==== Pré-requis ====
Il faut tout d'abord vérifier que Java est bien installé sur votre système. Si ce n'est pas le cas, rendez-vous sur [[java|cette page.]]

[[:tutoriel:comment_installer_un_paquet|installez le paquet]] [[apt://xulrunner|xulrunner]]
ou
  sudo apt-get install xulrunner

==== Téléchargement ====
Récupérez l'archive de Freezer : [[http://diuf.unifr.ch/pai/people/broccoa/freezer/freezer-0.01.zip|freezer-0.01.zip]] //(22,1 Mo)//

Décompressez la (clic-droit > Extraire ici) dans le dossier où elle se trouve (par exemple **/home/votre_login/Bureau** dossier de téléchargement de fichier par défaut dans Firefox).
==== Rendre exécutable le lanceur ====
Rentrez dans le dossier freezer-0.01 précédemment extrait et faites //clic-droit > Propriétés// sur **Freezer.linux.x86.sh** ou **Freezer.linux.x86_64.sh** suivant votre architecture 32 ou 64 bits.

Dans l'onglet //Permissions//, cochez //Autoriser l'e__x__écution du fichier comme un programme// si ce n'est pas fait par défaut.
===== Utilisation =====
==== Premier lancement ====
Placez-vous dans le répertoire où se trouve l'exécutable de Freezer
  cd /home/$USER/Bureau/freezer-0.01 #exemple
Dans un terminal, lancez la commande
  sh Freezer.linux.x86.sh
Une icône de notification devrait apparaître dans la zone comme montrée sur l'image en début de page.

==== Ajouter une liste de lecture ====
Sur Deezer les listes de lecture sont identifiées par une série de numéro, par exemple **4525618**

Faites un clic-droit sur l'icône de notification de Freezer, //Manage playlists// > //New playlist// > //Playlist name//. Entrez un nom, cliquez sur //Add//. Collez alors dans //Deezer path// l'identifiant de votre liste de lecture que vous pouvez trouver dans la barre d'adresse de cette dernière après le nom porté par la playlist. Par exemple:

  http://www.deezer.com/fr/#music/playlist/nom-de-la-playlist-4525618
==== Lancer la liste de lecture ====
Pour lancer la musique, clic-droit sur l'icône de Freezer, //Playlists...// > //votre_playlist//.

Ensuite, clic-gauche sur l'icône et vous devriez voir apparaître une fenêtre comme celle présentée sur l'image avec une photo de l'album, le nom de la chanson et des boutons de lecture, pause et lien vers le site Deezer.

Si cette fenêtre est plantée, tuez le programme si vous êtes dans un terminal avec Ctrl+D et relancez-le comme expliqué dans la partie premier lancement.
