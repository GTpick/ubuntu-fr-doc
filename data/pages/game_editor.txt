======GameEditor: IDE pour la création de jeux video 2D ======

Game Editor est un logiciel open source de conception de jeu qui vous donne le pouvoir de créer les jeux de vos rêves, et, contrairement à d'autres outils de création de jeu, vous donne la chance d'obtenir et de modifier le code source de l'éditeur de jeu pour concevoir et de développer les jeux pour n'importe quel ordinateur ainsi que les appareils mobiles.
{{ http://game-editor.com/wiki/images//4/48/Physicalresponse.jpg?300 }}

===== Pré-requis =====
Disposer d'une connexion à Internet configurée et activée.

=====Installation=====
   * [[http://sourceforge.net/projects/game-editor/files/gameEditor.zip/download|Téléchargez GameEditor]]
   * Décompressez l'archive
   * Lancez gameEditorLinux

==== Si le programme ne se lance pas  ====
Ouvrez un [[terminal|terminal]] et rendez vous dans le répertoire ou vous avez extrait l'archive (avec la commande [[tutoriel/console_ligne_de_commande#cd|cd]]):\\
Puis entrez le code ci-dessous 

<code>./ gameEditorLinux</code>
