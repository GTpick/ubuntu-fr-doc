Yabause est un émulateur libre de la Sega Saturn. Il est porté sous Windows, Linux, Mac OS X et FreeBSD. Il utilise OpenGL et SDL, un port moins avancé existe pour la Dreamcast.

===== Installation =====

==== Dépôt d'Ubuntu ====
{{ http://wiki.yabause.org/yabause.png?200}}
Yabause est disponible dans les dépôt :
 apt://yabause

==== Compilation ====

Pour la version de développement, le code se trouve sur GitHub : https://github.com/Yabause/yabause

  * Télécharger le code : https://codeload.github.com/Yabause/yabause/zip/master
  * Installez les dépendances :
   <note tip>sudo apt install git cmake qtbase5-dev qtmultimedia5-dev libqt5opengl5-dev libopenal-dev libsdl2-dev</note>
  * Décompressez l'archive du code et ouvrez le dossier « yabause ».
  * Ouvrez le fichier CMakeLists.txt avec CMake.
  * Indiquez dans les champs dédiés (en haut, en-dessous de la barre de menu) où se trouve le code et où vous voulez que le fichier binaire soit compilé.
  * Cliquez sur le bouton « Configure », puis « finish » et vérifiez que tout est installé.
  * Cliquez sur « Generate » pour obtenir un fichier makefile.
  * Ouvrez un terminal dans le dossier « Yabause » et tapez « make » puis //Entrée// pour démarrer la compilation.
  * Dans ce terminal, tapez « sudo make install » si vous voulez installer complètement Yabause.
  
==== Compatibilité ====
Yabause n'émule pas tout, mais environ 2 jeux sur trois. Une liste peut être consultée sur le wiki de l'émulateur : http://wiki.yabause.org/index.php5?title=Compatibility_list
