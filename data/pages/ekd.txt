====== EKD, post-production image et vidéo facile ======


EKD est un logiciel libre((sous licence GPL)) qui se propose de faire des opérations de post-production aussi bien pour des vidéos, que pour des images. En ce qui concerne les images, EKD fait du traitement par lots.

{{:application:ekd.png?400|EKD 2.0}}

===== Installation =====

====Version stable====

<note>Depuis la dernière version d'EnKoDeur-Mixeur, [[http://gmic.sourceforge.net | G'MIC]] et [[:ffmpeg |FFMPEG]] sont maintenant intégrées dans EKD (les versions portables de ces applications).
</note>

  * [[:tutoriel:comment_modifier_sources_maj|Ajoutez les dépôts]] correspondants à votre version d'Ubuntu sachant que le premier correspond à celui pour obtenir G'MIC et le second EKD.

^ Version d'Ubuntu ^ Logiciel ^ Dépôts à ajouter ^
| [[:Trusty|Ubuntu 16.04 LTS]] | //EKD// | <file>N'existe pas au 24 septembre 2016</file> |
| [[:Trusty|Ubuntu 14.04 LTS]] | //EKD// | <file>deb http://download.tuxfamily.org/ekdforum/ekd trusty contrib</file> |
| [[:Precise|Ubuntu 12.04 LTS]] | //EKD// | <file>deb http://download.tuxfamily.org/ekdforum/ekd precise contrib</file> |

  * Puis ajoutez la [[:tutoriel:comment_modifier_sources_maj#gerer_les_cles_d_authentification|clé d'authentification]] du dépôt, en saisissant dans un [[terminal]] la commande suivante :
<code>wget -q http://download.tuxfamily.org/ekdforum/ekd/cle-ekd.key.asc -O - | sudo apt-key add -</code>
  * [[:tutoriel:comment_modifier_sources_maj#recharger_la_liste_des_paquets|Rechargez la liste des paquets]]

dans un terminal
<code>sudo apt-get update</code>
puis
<code>sudo apt-get install ekd</code> 
puis pour lancer EKD
<code>ekd</code>

Un [[raccourci-lanceur|lanceur]] est automatiquement créé dans le menu //Applications -> Son&Vidéo//

====Version de développement====

Consulter [[http://ekd.tuxfamily.org/index.php/Developpement/SVN|cette page]].

=====Configuration=====


Pour pouvoir encoder en mp3 avec EKD, il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>libsox-fmt-all|libsox-fmt-all]]**.\\

===== Utilisation =====

Une documentation générale est disponible [[http://ekd.tuxfamily.org/index.php/Documents/Linux|ici]] ainsi que [[http://ekd.tuxfamily.org/index.php/Acceuil/Didacticiels|des didacticiels]].

=====Désinstallation=====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. La configuration de l'application sera conservée ou supprimée selon la méthode de désinstallation que vous choisirez.
