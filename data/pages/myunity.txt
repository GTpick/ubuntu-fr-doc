====== Myunity ======
{{ application:myunity_logo.png}}
**Myunity** est un utilitaire qui permet de configurer l'apparence d'[[:Unity]].
<note warning>Myunity n'est plus dans les dépôts d'Ubuntu 14.04 et suivantes. Vous pouvez vous tourner vers [[apt>unity-tweak-tool]] ou [[:Ubuntu tweak]], si vous cherchez un logiciel équivalent</note>
 ===== Pré-requis =====
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

 ===== Installation =====
     
 Pour [[:precise|Ubuntu 12.04]], pour installer ce logiciel, il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>myunity|myunity]]**.

<note important>FIXME <del>Pour le moment</del>  MyUnity n'est pas disponible pour Ubuntu 12.10 et supérieur, l'interpréteur gambas manque dans les dépôts (gambas2-runtime). **Projet abandonné**</note>
     
     
 ===== Utilisation =====
     
Il faut lancer le programme **myunity**.

L'interface se présente sous la forme de plusieurs onglets dont chacun donne accès à une série d'options.  Il est permis de configurer les principaux paramètres de l'agencement des menus et de la barre de lancement rapide ; de choisir d'y afficher ou non les programmes disponibles ou seulement ceux installés (NDLR:ouf!)  ; de changer le thème du gestionnaire de fenêtre,... 
     
 ===== Désinstallation =====
     
 Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
