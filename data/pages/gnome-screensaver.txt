====== Économiseur d'écran Gnome-Screensaver======

**Gnome-Screensaver** est un économiseur d'écran intégré au bureau [[:GNOME]]. L'avantage par rapport à [[:Xscreensaver]] est de ne pas nécessiter un nouveau daemon et de fonctionner de concert avec gnome-power-manager, ce qui se révèle très pratique sur un portable. 

Un autre point fort de Gnome-Screensaver est d'être compatible avec les économiseurs d'Xscreensaver car il en est juste l'interface utilisateur.

Il est conçu pour gérer, entre autres :

  * Le verrouillage de configuration,
  * L'intégration avec le bureau (thèmes, polices, boîte à outils),
  * Le changement d'utilisateur directement à partir de la boîte de dialogue déverrouiller,
  * La modification instantanée des paramètres de l'application.

<note tip>Sur les versions récentes d'Ubuntu, l'économiseur d'écran voit ses fonctionnalités réduites. Il ne permet plus que d'éteindre et/ou de verrouiller l'écran au bout d'une période d'inactivité déterminée. Il pourrait disparaître complètement sous GNOME 3.2(([[http://mail.gnome.org/archives/gnome-shell-list/2011-March/msg00340.html|What is the status of the screensaver in GNOME3?]])).\\
Pour un économiseur d'écran avec fonctionnalités complètes, tournez-vous vers [[:XScreensaver]].</note>

===== Installation =====

Il est pré-installé sur [[:Ubuntu]] et [[:Xubuntu]].

Pour installer ce logiciel, il suffit d'[[:tutoriel:comment_installer_un_paquet#paquet_present_dans_les_depots|installer le paquet]] **[[apt>gnome-screensaver]]**.

===== Utilisation =====

==== Sous Ubuntu 12.04 LTS ====
Lancez l'application depuis //Paramètres système -> Écran//.

Dans gnome 3, il n'y a plus de visualisations.

{{:applications:gnome-screensaver_oneiric_01.png?500|L'Économiseur d'écran sous Ubuntu 11.10.}}

==== Sous Ubuntu 14.04 LTS et supérieures ====
Avec le tableau de bord de [[unity#tableau_de_bord_dash|unity]], saisissez le début de "Luminosité & verrouillage" pour accéder à ces paramètres qui s'affiche alors comme la capture ci-dessus.

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
Ceci n'est valable que dans le cas sur une variantes d'Ubuntu, sur Ubuntu classique il est fortement déconseillé de supprimer le paquet Gnome-screensaver.
