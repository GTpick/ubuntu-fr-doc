====== Bureautique =======

Par défaut, Ubuntu est livrée avec [[:LibreOffice]] une suite bureautique [[wpfr>ramification_(développement_logiciel)|dérivé]] d'[[:OpenOffice.org]]. Elle est développée par [[wpfr>The_Document_Foundation|The Document Foundation]].\\
Cette suite bureautique est pleinement fonctionnelle, y compris pour des besoins professionnels courants. Elle utilise nativement le format ouvert [[wpfr>OpenDocument]].
LibreOffice permet aussi l'import et l'export des documents aux formats Microsoft Office(r) (.doc(x), .xls(x), .ppt(x)...) afin de permettre l'échange de documents avec les utilisateurs ne disposant pas d'une suite bureautique compatible [[wpfr>OpenDocument]].
===== Texte =====

Suite office :
  * **[[:LibreOffice]]** : suite bureautique à code en source libre basée sur OpenOffice.org. Libreoffice est installé par défaut dans Ubuntu.
  * ** [[:Calligra]]** : La suite bureautique du projet KDE desktop.
  * [[https://www.wps.com/|WPS office]] : suite bureautique clonant l'interface de Microsoft Office
  * [[http://www.onlyoffice.com/fr/apps.aspx|OnlyOffice]] : suite bureautique open-source
  * [[:OpenOffice.org|Apache OpenOffice]] : suite bureautique à code en source libre.
  * [[:Gnome office]] : La suite bureautique du projet Gnome desktop.

Suite office léger :
  * **[[:Abiword]]** ([[apt://abiword|installer]]) : Traitement de texte léger qui évite les lourdeurs d'une suite bureautique telle queLibreOffice, surtout si on ne cherche qu'un traitement de texte. 
  * **[[:Gnumeric]]** ([[apt://gnumeric-gtk|installer]]): tableur léger
  * [[:kate]] ([[apt://kate|installer]]) : éditeur de texte pour KDE :
  * [[:OOo4Kids]] ([[http://download.ooo4kids.org/fr/|téléchargement]]) : Suite bureautique libre légère pour les enfants... mais pas seulement.       
Traitement de texte ultra léger en terminal-console :
  * **[[apt>wordgrinder]]** : Traitement de texte ultra léger. Formats OpenDocument, HTML import and export; LaTeX, Troff et MarkDown export. Style gras, italique, souligné et titres hiérarchisés. 
Suite bureautique libre en ligne (alternative à Google doc) à installer sur son serveur personnel :
  * [[:OwnCloud]]
  * [[opengoo|Feng office]] : [[http://www.epn-ressources.be/opengoo-google-documents-en-logiciel-libre | article ]], [[>OpenGoo|wikipédia]]
  * [[http://www.freecloudalliance.org/ung-Home.Page|UNG]] : [[http://korben.info/ung-un-clone-open-source-de-google-docs.html |article]]
Pour compatibilité windows :
  * [[:MSttCoreFonts]] : installation des polices de caractères de Microsoft Office pour LibreOffice.

Pour les éditeurs :
  * [[:Scenari]] : Tous les logiciels de la chaîne éditoriale Scenari pour l'élaboration de contenus multimédia et papier.
  * [[http://www.calligra.org/news/calligra-announces-author|Calligra Author]] : logiciel pour les auteurs.
  * [[:Opale]] : Opale est une chaîne éditoriale libre permettant de réaliser des contenus de formation.
  * [[http://opcion.sourceforge.net/|Opcion]] : visualiseur de polices de caractères en JAVA.
  * [[:verbiste]] : logiciel de conjugaison. (voir aussi [[dictionnaires_encyclopedies|dictionnaires et encyclopédies]])
  * Voir aussi [[:scribus]]

Latex :
  * [[:LaTeX]] : éditeur utilisé couramment pour les publications scientifiques (utile pour créer des documents formatés de manière logique, avec un excellent rendu des équations et formules).
  * [[:XeLaTeX]] : variante de LaTeX gérant nativement l’Unicode.
  * [[:OOoLatex]] : équations latex dans les documents LibreOffice Writer ou Impress.

Groff :
  * [[:Groff]] : logiciel de typographie [[GNU]] utilisant des balises, très léger.

Collaboratif : 
  * [[:Gobby]] : éditeur de texte collaboratif
  * [[:Abiword]]
  * [[http://framapad.org/|Framapad]] : le traitement de texte du [[http://framalab.org/|Framalab]] est en ligne (dans le //cloud//) et Open Source (c'est un [[http://etherpad.org/|etherpad]])
  * [[http://framacalc.org/_start|Framacalc]] : le tableur du [[http://framalab.org/|Framalab]] est en ligne (dans le //cloud//) et Open Source (c'est un [[http://www.ethercalc.org/|ethercalc]])

===== Cartographie heuristique (mind mapping) =====
Conception de diagrammes qui représentent les connexions sémantiques entre différentes idées, les liens hiérarchiques entre différents concepts intellectuels. À la base, il s'agit d'une représentation principalement arborescente des données.
  *[[:carte heuristique]]
  *[[Freemind]] 
  *[[Freeplane]] (fork de **Freemind**)
  *[[Vym]] 
  *[[Semantik]]
  *[[xmind]]
  *Autres logiciels de Mind Mapping : [[http://www.mind-mapping.org/]]
  * [[http://framindmap.org/|Framindmap]] : le créateur de cartes mentales  de la suite [[http://framalab.org/|Framalab]] (c'est un [[http://github.com/drichard/mindmaps|mindmaps]])
