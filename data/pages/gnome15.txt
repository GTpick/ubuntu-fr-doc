======Gnome15 ======
 
Gnome15 fournit un indicateur de panneau (ou applet), un outil de configuration, un système de macro et des plug-in pour les claviers Logitech la série G, 
y compris le G15, G19, G13, G110 (d'autres peuvent être pris en charge). L'intention est de fournir la meilleure intégration avec le bureau Linux possible.

<note warning>Le site de Gnome15 est actuellement indisponible depuis 2014, aucune nouvelle récente de Nuno Araujo (alias Russo79) donc le projet pourrait être abandonné. Cependant, les fichiers sont disponibles dans un autre sitev: [[http://trynet.ch/?p=215|sur Trynet.ch]]</note>


===== Pré-requis =====
 
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
 
===== Installation =====
Il est nécessaire de compiler à partir des sources :
  * Commencez par télécharger l'archive [[http://gnome15.org/downloads/Gnome15/Required/gnome15-0.9.2.tar.gz|ici]],
  * [[:archivage|Décompressez-la]],
  * Déplacez-vous dans le dossier décompressé,
  *  [[:compilation|Compilez les sources]] de façons habituelles (configure, make, sudo make install).

Vous pouvez également utiliser [[git]], sachant que les sources se trouvent [[https://github.com/Huskynarr/gnome15.git/bash]|ici]]
 
===== Configuration =====
 
Accèdez a **Gnome 15** en cliquant sur l'applet G puis **Keyboard** -> **Driver**.
  * Dans **use** sélectionnez **Kernel driver**
  * Dans **Option** --> **Device** sélectionnez **/dev/fb0** 

===== Utilisation =====
L'exemple qui suit est tiré de [[Kubuntu]].\\
Vous trouverez une nouvelle icône dans la zone de notification. Comme ici sous KDE :

{{:materiel:claviers:logitech:g510:g510_000.png?nolink&|}}

Si cette icône clignote cela indique que le [[:services]] (daemon) n'est pas démarré.

Faites un clic droit sur cette icône --> Préférences, vous obtenez alors l'outil de paramétrage du clavier.


=== Keyboard ===

{{:materiel:claviers:logitech:g510:g510_001.png?300|}}

**Keyboard Backlight Colour** : Couleur du rétro-éclairage du clavier

**Invert LCD** : Inverser l'affichage

**Cycle screens** : Alterner les affichages d'écran (voir plugins)



=== Macros ===

{{:materiel:claviers:logitech:g510:g510_002.png?300|}}

Cette section permet d'ajouter des macros pour les touches G1 à G18, ainsi que les touches M1, M2, M3 et MR.

Il est possible de créer différents profils contenant leurs propres macros (un profil par jeu par exemple).

=== Plugins ===

{{:materiel:claviers:logitech:g510:g510_003.png?300|}}

Chacun des plugins apporte une fonctionnalité complémentaire au clavier. Certaines propres aux macros, mais la plupart pour afficher des informations sur l'écran de votre clavier.

Certains plugins peuvent avoir leurs propres options, utilisez pour cela le bouton préférences.

=== Configure ===

Cet écran s'obtient en cliquant sur le bouton "Configure" situé en bas à gauche.

{{:matériel:claviers:logitech:g510:g510_005.png?300|}}

Les paramètres par défaut sont ceux ci-contre. Il n'y a pas lieu de modifier si vous ne rencontrez pas de problème particulier.

===== Désinstallation =====
 Lancez la commande à partir du dossier des sources :
<code>sudo make uninstall</code>
