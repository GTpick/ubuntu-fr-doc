===== DmgExtractor =====

Dmgx est un programme écrit en java. Il n'est donc pas spécialement conçu pour linux et est multiplateforme, et fonctionnera sur tous les systèmes du moment qu'un client java est installé sur la machine.
Il permet de convertir les fichiers DMG natifs Mac OS en image ISO, IMG, etc. compatible pc.

==== Installation ====

Il n'y a aucune installation préalable. C'est un fichier jar à télécharger sur le site de l'auteur :
[[http://hem.bredband.net/catacombae/dmgx.html]]

==== Configuration ====

Une fois le .jar téléchargé, il faudra l'associer à votre application java. Si vous n'avez pas de client java installé, allez [[java|ici]].

==== Utilisation ====

Il suffit soit de faire un click-droit sur le fichier jar et choisir ouvrir avec l'application java, soit d'ouvrir le terminal :
<code>java -jar ./dmgextractor*.jar</code>
<note tip>Notez que sur le site de l'auteur, vous pouvez lancer directement le logiciel sans le télécharger. Choisissez pour cela "Run program with Java Web Start"</note>
