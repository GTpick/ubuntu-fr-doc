====== Galternatives ======

**GAlternatives**((il existe aussi le paquet kalternatives)) est une interface graphique au programme update-alternative. Il permet, dans le cas où vous auriez installé plusieurs logiciels susceptibles d'accomplir une même tache, de choisir celui qui sera utilisé par défaut.

Cette application permet la gestion des fichiers contenus dans /etc/alternatives/.

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation =====

Pour installer ce logiciel, il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>galternatives]]**.

===== Utilisation=====
 
Lancez l'application via le [[:unity#tableau_de_bord_dash|tableau de bord]] (Unity) ou via le [[:terminal]] (toutes versions d'Ubuntu) avec la [[:commande_shell|commande]] suivante : <code>galternatives</code>
Celle-ci nécessitant les droits d'administration, une fenêtre vous demandant votre mot de passe apparaitra.

  * Dans la partie gauche, nous pouvons voir toutes les commandes pour lesquelles des alternatives sont disponibles. Dans l'exemple ci-dessous, nous voyons tous les binaires disponibles pour la commande ''java''.
  * Dans la partie droite sont listée toutes les alternatives disponibles et installées sur le système. Dans l'exemple, OpenJDK, GCJ et Sun Java sont installés
  * Un clic sur le bouton radio de l'alternative choisie dans la colonne « Choix » permettra de choisir l'alternative à utiliser ; la modification est immédiate. Dans l'exemple, c'est OpenJDK qui sera utilisé derrière l'invocation de ''java''.

{{:applications:outils_systeme:galternatives.png|Changer de machine virtuelle Java (sous Ubuntu 10.04 LTS)}}

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
