====== Eggdrop : un robot IRC ======

**eggdrop** est un [[wpfr>Robot_IRC|robot IRC]] qui permet d'utiliser les fonctions du protocole IRC de manière automatisée.


===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.


===== Installation =====

Pour installer ce logiciel, il suffit d'[[:tutoriel:comment_installer_un_paquet#installer_un_paquet_present_dans_les_depots_recommande|installer le paquet]] **[[apt://eggdrop|eggdrop]]**.

===== Configuration ===== 

Un fichier de configuration nommé **eggdrop.conf.gz** se trouve dans le dossier **/usr/share/doc/eggdrop-data/examples**.

Il suffit de le copier dans le Dossier personnel, puis de le [[:archivage|décompresser]].

Ensuite, [[:tutoriel:comment_modifier_un_fichier|modifiez le fichier]] **eggdrop.conf** extrait.

<note important>
Le fichier n'est pas correctement configuré par défaut !!!! Il faut donc l'éditer avant de lancer  eggdrop.</note>

<note help>Pour l'édition du fichier **eggdrop.conf** je vous dirige vers ce lien [[http://wiki.eggdrop.fr/Configuration_d%27un_eggdrop|configuration d'un eggdrop]] ou vers cet outil de [[http://tools.eggdrop.fr/autoconf.php|configuration automatique]]</note>

===== Utilisation=====

Pour lancer le eggdrop, saisir dans un [[terminal]] : 
<code>
eggdrop -m eggdrop.conf 
</code>
L'option **-m** n'est à utiliser que pour la première utilisation. 



===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application sera conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs situés dans leurs dossiers personnels seront conservés.
