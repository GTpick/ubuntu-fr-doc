====== Serveur d'impression sans fil (Linksys WPS54G) ======

  Version Dapper

Rédigé par [[utilisateurs:calimarno]].

===== Introduction =====

Cette page présente la façon dont j'ai configuré l'impression sur un serveur sans fil (''Linksys WPS54G''). L'imprimante qui y est reliée est une ''Brother DCP-8045D'' (via USB).

===== Prérequis =====

  - Branchez votre imprimante au serveur sans fil, assurez vous que votre connexion sans fil fonctionne.
  - Donnez une IP fixe à votre serveur sans fil (192.168.1.144, par exemple).

===== Configuration =====

  - Dans ''Système -> Administration -> Impression'', double-cliquez sur ''Nouvelle imprimante''.
  - Sélectionnez dans la nouvelle boîte de dialogue ''Imprimante réseau'' puis ''Imprimante UNIX (LPD)''.
  - Dans le champ ''Hôte'', saisissez l'adresse IP de votre serveur d'impression, puis dans le champ ''File d'impression'' entrez ''L2'' pour une imprimante branchée sur le port USB (ou ''L1'' pour le port paralèlle).
