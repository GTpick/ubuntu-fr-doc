====== Ultracopier ======

**Ultracopier** est un [[:gestionnaire de copie]] multi-langues et multi-plateformes.

Ultracopier est l'équivalent du fameux [[http://ultracopier-fr.first-world.info/supercopier.html|Supercopier(windows)]] sous Linux, qui as justement été repris par l'équipe d'ultracopier.

{{:ultracopier.png|}}

Vouci quelques fonctions d'**Ultracopier**:
  * Écrites en C++, pour que tous les développeurs puissent utiliser,
  * Utilisation du framework Qt pour être utilisable sur toutes les plateformes,
  * Facilité de création et d'incorporation de thèmes, plugins et ressources,
  * Traduction facile et fourni avec des traductions,
  * Gestion par liste de copie,
  * Gestion des erreurs,
  * Gestion avancée des fichiers déjà existants,
  * Options avancées (vitesse, droits, ...)
  * Copie par ligne de commande pour les scripts automatiques,
  * Plugins.

===== Pré-requis =====
 
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation =====

Il vous suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>ultracopier]]**  disponible depuis les [[:depots#depots_officiels|dépôts]].


===== Utilisation =====

  * Votre explorateur est supporté (est compatible avec le protocole catchcopy protocole 0002), alors faites vos copies normalement
  * Votre explorateur n'est pas supporté :
    - Lancer UltraCopier (si il n'est pas dans la barre de notification / systray),
    - Faire un clic droit sur l'icône du systray (près de l'heure),
    - Ajoutez une nouvelle copie/déplacement par ce menu,
    - Rentrez dans l'interface qui va ouvrir la source et la destination.

Vous l'aurez compris, c'est bien plus simple quand les développeurs ont fait la compatibilité avec le protocole catchcopy.
Vous trouverez [[http://ultracopier-wiki.first-world.info/wiki/Guide|ici]] le guide complet officiel d'utilisation.




===== Compatibilité =====

Pour l'intégration natif dans le navigateur avec le copier / coller, il faut que votre gestionnaire de fichiers soit compatible avec [[wpfr>Ultracopier#Projet associé|le protocole catchcopy protocole 0002]].

Pour l'instant il n'est supporté par [[:Nemo]] qu'en appliquant cette modification : [[https://github.com/lestcape/Nemo-UltraCopier|https://github.com/lestcape/Nemo-UltraCopier]]
