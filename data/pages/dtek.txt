====== DteK, gérer sa collection de films ======
<note important>Ce logiciel n'est plus développé depuis 2008.</note>
**DteK** est une application programmée en Java pour gérer sa collection de films. DteK est donc multiplate-forme et peut ainsi être exécuté sur différents systèmes d'exploitation (Linux et Windows)...

DteK vous propose de gérer votre collection de films qu'elle soit sur votre ordinateur ou sur d'autres supports (VHS, CD, DVD, HD-DVD...). Il est à même de rechercher les films sur votre disque dur dans les dossiers que vous lui avez indiqués. Vous pouvez ainsi lancer votre film avec votre lecteur de vidéo par défaut depuis le logiciel sans aller le rechercher sur votre ordinateur.

DteK gère aussi la recherche des informations de vos films depuis internet sur des sites tels que imdb, allocine, cinemotions.

{{:applications:dtek_lucid_01.png?500}}

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  * Avoir installé de [[java|Java (OpenJDK ou JRE)]].


===== Installation =====

Téléchargez l'archive **dtek_1.25_portable.zip** multiplateforme [[http://dtekdev.tuxfamily.org/|à cette adresse]].

[[archivage|Décompressez l'archive]], vous obtiendrez un dossier contenant l'exécutable de l'application.

===== Configuration =====

Dans le dossier extrait, modifiez les [[:permissions]] du script **dtek_linux.sh** pour le rendre exécutable.

===== Utilisation =====

Lancez le script **dtek_linux.sh**.

===== Désinstallation =====

Pour supprimer ce logiciel, il suffit de supprimer son dossier.
