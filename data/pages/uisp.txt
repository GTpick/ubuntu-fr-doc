====== UISP ======

===== Introduction =====

UISP est un programme (equivalent à ponyprog sous windows) pour utiliser vos carte stk 200, 500,... donc pour programmer les microcontrôleur AVR (AT90S...)


===== Installation =====

<code>sudo apt-get install uisp binutils-avr</code>
