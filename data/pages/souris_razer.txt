====== Configurer sa souris Razer ======
**Razercfg** un outil qui permet la configuration de nombreuses souris de marque Razer sous Linux.

Il inclut seulement les options qui sont supportées par la souris elle-même, comme changer les DPI, scanner les fréquences, les couleurs de diode...\\
Il permet aussi de sauvegarder et changer facilement d'un profils à un autre, Bureautique, Jeux...

{{ :applications:razercfg_xenial.png?350 }}
===== Pré-requis =====
 
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation =====
**Razercfg** est disponible à travers un [[:ppa]]. Pour l'installer :
  * [[:ppa#ajout_d_un_ppa_a_vos_sources_logicielles|Ajoutez le PPA]] **ppa:nilarimogard/webupd8**((https://launchpad.net/~nilarimogard/webupd8/+archive/ppa)) dans vos sources de logiciels,
  * Puis [[:tutoriel:comment_modifier_sources_maj#recharger_la_liste_des_paquets|rechargez la liste des paquets]]
  * [[:tutoriel:comment_installer_un_paquet|Installez les paquets]] **[[apt>razercfg,qrazercfg]]** et si vous souhaitez un applet pour y accéder plus facilement **[[apt>qrazercfg-applet]]**.

===== Utilisation =====
 
Lancez l'application via le [[:unity#tableau_de_bord_dash|tableau de bord Unity]]  ou via le [[:terminal]] (toutes versions d'Ubuntu) avec la [[:commande_shell|commande]] suivante :
<code>qrazercfg</code>
 
===== Désinstallation =====
 
Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.

=====Compatibilité=====
**Razercfg** supporte les périphériques suivants :
  * Razer DeathAdder Classic
  * Razer DeathAdder 3500 DPI
  * Razer DeathAdder Black Edition
  * Razer DeathAdder 2013
  * Razer DeathAdder Chroma
  * Razer Krait
  * Razer Lachesis Classic
  * Razer Naga Classic mouse
  * Razer Naga 2012 mouse
  * Razer Naga 2014 mouse
  * Razer Naga Hex mouse
  * Razer Taipan mouse
  * Razer Mamba Tournament Edition 
Les souris qui suivent sont listées comme stable, mais il leur manque quelques petites fonctionnalités : 
  * Razer Boomslang CE
  * Razer Copperhead
  * Razer Lachesis Classic

Pour avoir la liste complète des fonctionnalités reportez-vous à la page [[http://bues.ch/cms/hacking/razercfg.html#supported_hardware_features|Rasercfg]].

Rasercfg comprend également un support basique pour les souris Razer Ouroboros avec ou sans fil.
