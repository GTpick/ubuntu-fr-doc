
====== CD label générator ======

===== Installation =====

==== Interface texte ====

FIXME trivial 

    apt-get install cdlabelgen

==== Interface graphique ====

=== GTK en python ===

Installez les paquets **[[apt>python-id3 python-cddb python-glade2]]**

Télechargez la dernière version de l'archive tar.gz et non du programme python [[http://sourceforge.net/projects/gtkcdlabel/files/gtkcdlabel/]].  Téléchargez l'archive et décompressez.  Vous pouvez executer usr/bin/gtkcdlabel.py .

Sous [[Mate]], remplacer le visionneur de [[gnome]] /usr/bin/evince par /usr/bin/atril pour pouvoir utiliser le bouton visionner, ou par tout autre visionneur de fichier pdf.

FIXME Pour une installation plus sécurisée, mais aussi plus longue, changer le propriétaire et le groupe de tous ces fichiers en root:root et placez-les dans la bonne place dans l'arborescence POSIX. (ça devrait marcher, je reviens dès que j'ai essayé!)

==== Convertir les images en format eps  ====

Contrairement à ce qui est marqué dans l'interface, vous pouvez utiliser n'importe quel fichier image pour créer vos pochettes.  Sinon, voici la commande magique...

    convert mon_image.jpg mon_image.jpg.eps

[[http://stackoverflow.com/questions/5350246/to-convert-jpg-to-eps-format]] (Imagemagick est un bon choix.)
