====== Antidote Rx™: la perfection grammaticale à tout prix… ======

**Antidote™** est un outil moderne, rapide et pratique pour la rédaction en français.\\
C'est un logiciel intégré d'aide à la rédaction et de correction du français écrit (orthographe, grammaire, etc.). Il regroupe dix grands dictionnaires et dix guides linguistiques. Il interagit intelligemment avec ces dictionnaires et les textes en rédactions pour explorer tous les aspects d'une question linguistique. Antidote est intégrable au traitement de texte Writer de[[LibreOffice]].

Antidote™ est un logiciel propriétaire commercial.

===== Pré-requis =====

  * Acheter la boîte du logiciel Antidote RX™.
  * S'enregistrer (par Internet, par téléphone ou par courrier postal). Voir les instructions [[http://www.druide.com/|ici]].
  * Avoir [[:tutoriel:comment_installer_un_paquet|installé le paquet]] **[[apt>gksu]].
===== Installation =====
Il ne vous reste plus qu'à installer le logiciel (pensez à déplacer le sript du cd à votre répertoire personnel et à le rendre exécutable).
<code>gksu bash Installe_Antidote.bash</code>'

Enfin, il ne vous reste plus que les mises-à-jour.





== Activation version 7 ==

Pour activer Antidote Rx après une mise à jour de la version 7 il faut lancer Antidote avec la commande sudo.
  sudo antidote
Sinon il vous sera impossible d'activer votre produit: erreur "15,15" 

===== Configuration =====

==== Installation dans Firefox ====

  - Appuyez sur les touches « Alt+F2 » pour ouvrir un dialogue de lancement de commande.
  - Dans la boîte de dialogue qui s'ouvre, inscrivez la commande suivante :
      * Pour Firefox : firefox /usr/local/Druide/Antidote/Texteurs/Firefox/3.0/Antidote_Mozilla_3.xpi
  - Firefox se lancera avec un dialogue demandant la confirmation de l'installation de l'extension.
  - Cliquez sur le bouton « Installer maintenant ».
  - Quittez Firefox complètement.

==== Installation dans Thunderbird ====

  - Ouvrez Thunderbird,
  - Dans le menu « Outils », choisissez « Extensions ». La fenêtre des extensions de Thunderbird apparait.
  - Cliquez sur le bouton « Installer ». La fenêtre d’ouverture de fichier apparait.
  - Furetez jusqu'au dossier d'Antidote, soit le chemin suivant : /usr/local/Druide/Antidote/Texteurs/Thunderbird/1.5/
  - Ouvrez le fichier « Antidote_Mozilla_3.xpi ».
  - Thunderbird demandera une confirmation de l’installation. Cliquez sur le bouton « Installer maintenant » dans la fenêtre de confirmation qui apparaît.
  - Quittez Thunderbird complètement.

==== Intégration dans LibreOffice ====

  - Ouvrez le traitement de texte de [[:LibreOffice]] Writer.
  - Dans le menu « Outils », sélectionnez l'option « Gestionnaire d'extensions… »
  - Dans le gestionnaire, cliquez sur le bouton « Ajouter… »
  - Dans le gestionnaire de fichiers qui apparaît, naviguez pour sélectionner le fichier « /usr/local/Druide/Antidote/Texteurs/OpenOffice.org/Antidote-OOo.uno.pkg ».
  - Cliquez sur le bouton « Ouvrir »
  - Après l’installation, fermez le Gestionnaire de packages en appuyant sur le bouton « Fermer »,
  - Redémarrez le traitement de texte Libreoffice Writer.

===== Problèmes et solutions =====


==== Messages d'erreurs ====

Si lors de l'utilisation les messages d'erreurs suivants apparaissent :

> Les définitions du Petit Druide ne sont pas disponibles. / Code 801

> Erreur d'initialisation. Impossible de poursuivre l'exécution de la commande.

> Erreur à la lecture du fichier. / Les données de l'index du Visuel n'ont pu être chargées.

Il s'agit d'un problème de permission d'accès aux fichiers locaux qui peut être résolu de la manière suivante :

  sudo chmod -R 777 /usr/local/Druide/Antidote
