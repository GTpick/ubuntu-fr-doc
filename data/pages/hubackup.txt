====== HUBackup ======

HUBackup est un petit logiciel (très simple/basique) de sauvegarde de données personnelles.

===== Installation =====

Il vous suffit, pour cela, d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt://hubackup|hubackup]]** disponible dans les [[:depots#universe_et_multiverse|dépôts Universe]].

<note important>Le paquet hubackup n'est plus disponible pour Jaunty, le paquet destiné à Intrepid fonctionne néanmoins sur cette version.
</note>

===== Utilisation =====

==== Sauvegarder ====
  - Lancer //Système → Administration → Home User Backup//
  - Sélectionner le répertoire que vous souhaitez sauvegarder et choisir la destination de la sauvegarde. Si vous avez un graveur de CD ou de DVD il devrait apparaître dans cette liste.
  - Appuyer sur **Backup** et suivre les instructions à l'écran.

|{{ :systeme:administration:hubackup_intrepid_03.png |}}|{{ :systeme:administration:hubackup_intrepid_04.png |}}|
==== Restaurer ====
  - Lancer //Système → Administration → Home User Restore//
  - Suivre les instructions à l'écran.
