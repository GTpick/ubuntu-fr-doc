====== Fotoxx =====

**Fotoxx** est un logiciel de gestion et retouches de photos.

[[http://www.kornelix.net/fotoxx/fotoxx.html|Aperçu en images des fonctionnalités de Fotoxx (en milieu de page)]]. \\

{{:fotoxx_13.10.1.png?600}}

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation =====

====via les dépôts officiels====

Pour installer Fotoxx, il suffit d'[[:tutoriel/comment_installer_un_paquet|installer]] le paquet **[[apt>fotoxx]]**.

====via les paquets DEB====

Le [[http://www.kornelix.net/downloads/downloads.html|site de Fotoxx]] met à disposition un DEB pour les PC 32 bits (paquet fotoxx-XX.XX-i686.deb), et un autre pour les 64 bits (fotoxx-XX.XX-x86_64.deb).

Avant de lancer le programme, il vous faudra aussi [[:tutoriel/comment_installer_un_paquet|installer]] ces paquets **[[apt>dcraw,ufraw]]**

=====Utilisation=====

Le [[raccourci-lanceur|lanceur]] de Fotoxx se trouve dans le menu Applications -> Graphisme.

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, le configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
