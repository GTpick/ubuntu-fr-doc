======= caca-utils =======

{{  http://caca.zoy.org/static/libcaca-logo.png?64}}

**caca-utils** est un ensemble d'outils en ligne de commande permettant de convertir et visualiser images et vidéos en jeu de caractères [[wpfr>ASCII]]. Le nom est intentionnel, les auteurs offrant un logiciel ludique sans prétention sous licence publique « Faites-en bien ce que vous voulez »((traduction libre de [[http://sam.zoy.org/wtfpl/|Do What The Fuck You Want To Public License]])).

{{http://caca.zoy.org/static/images/stitch-80x45.png}}

===== Pré-requis =====
 
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation =====

Pour installer cette application, il suffit d'[[:tutoriel:comment_installer_un_paquet|installez le paquet]] **[[apt>caca-utils]]**. Le paquet **[[apt>imagemagick]]** est également suggéré pour certaines opérations, telles que l'exemple présenté dans la rubrique [[#Utilisation]].

===== Utilisation =====

Après installation, vous avez alors à votre disposition un arsenal de commandes, vous permettant de visionner un contenu "standard" en ASCII.

==== Outils ====

^ Outil ^ Description |
| cacademo | Affiche des effets d'art ASCI avec transitions animées |
| cacaplay | Joue un fichier d'animation libcaca. La commande doit être suvi d'un nom de fichier sous forme fichier.caca |
| cacaserver | Joue des fichiers d'animation libcaca à travers le port réseau 51914 |
| cacaview | Visionneuse d'image en mode texte. |
| img2txt | convertit des images vers des fichiers en mode texte coloré |

Consultez les pages [[man]] de ces outils pour plus d'informations.

==== Exemple d'utilisation ====
<code>
wget http://webcreation.webege.com/wp-content/uploads/2010/03/Tux-G2.png
img2txt --format=svg Tux-G2.png > tux.svg
mogrify -format png tux.svg
</code>

De [[http://webcreation.webege.com/wp-content/uploads/2010/03/Tux-G2.png|ceci]] vous obtenez [[http://webcreation.webege.com/wp-content/uploads/2010/03/tux.png|cela]].

À noter que le paquet **[[apt>imagemagick]]** est requis dans cet exemple.

===== Désinstallation =====
 
Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
