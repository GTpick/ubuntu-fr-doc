====== Pencil ======

Pencil se présente comme un simple logiciel destiné à réaliser des animations 2D

Pencil est un logiciel Open Source multiplateforme.

{{http://static.sftcdn.net/fr/scrn/70000/70213/4_Pencil_Win__02.jpg|}}
<note tip>Ce logiciel ne doit pas être confondu avec le logiciel [[pencilevolus|Pencil Evolus]], logiciel de création de prototypage.</note>
===== Installation =====

Pour installer ce logiciel, il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>pencil|pencil]]**.

===== Utilisation=====

Lancez l'application depuis le menu //Applications -> Graphisme -> Pencil// ou //via// la [[:commande_shell|commande]] suivante :

  pencil

Par défaut deux pistes sont créés à l'ouverture, une pour les images bitmap et une vectorielle.

Vous pouvez ajouter deux autres types de piste, une piste caméra et une son.

Le plan de travail est un calque sans bord.

La taille de la piste caméra est paramétrable en double-cliquant sur le nom de celle-ci. Pour les déplacements, il faut utiliser l'outil déplacement (la main), ce qui créer automatiquement une nouvelle image clé et une interpolation entre les deux. Vous pouvez combiner un changement de taille et un déplacement.

Il est possible d'importer une série d'images (très utile pour peindre sur une vidéo) dans une piste bitmap par glisser-déposer.

Le logiciel gère l'onion skin ("pelure d'oignon" en français) d'une image en arrière.

Pour le réglage de la tablette graphique, il faut aller dans //Edit -> Preference -> General//.

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
