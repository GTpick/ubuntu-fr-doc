====== Piwigo ======

Piwigo est un logiciel de galerie photo pour le web. Le projet a démarré en 2002 sous le nom PhpWebGallery qui est devenu Piwigo le 15 février 2009. Piwigo est sous [[wpfr>Licence_publique_générale_GNU|licence GPL]].

L'une de ses particularités est de proposer un [[http://fr.piwigo.org/doc/doku.php?id=pwg2:apprendre:outils:ploader|logiciel multi-plateformes]] permettant de facilement créer et envoyer ses galeries photos sur l'hébergement. La liste complète des fonctionnalités est disponible [[http://fr.piwigo.org/basics/features|ici]].

{{:photo:piwigo_karmic_01.png?500|}}

===== Pré-requis =====

Le principal pré-requis pour Piwigo est de disposer d'un hébergement web avec le support de PHP et Mysql. Cet hébergement peut se faire sur votre machine personnelle ou chez un hébergeur, gratuit ou payant.
Pour vous aider dans votre choix, Piwigo vous propose [[http://fr.piwigo.org/doc/doku.php?id=pwg2:apprendre:install:avant|quelques explications]].

===== Installation =====

<note tip>
Depuis la version 2.3, piwigo intègre un 'Gestionnaire de mise à jour' ,il y a possibilité d'installer pour les versions précédentes  un plugin pour mettre à jour.
</note>


Cependant, deux méthodes d'installation sont possibles :

  * [[http://fr.piwigo.org/basics/installation_netinstall|NetInstall]] : processus d'installation automatique qui consiste à télécharger un unique fichier à déposer sur votre espace web, lequel se charge de récupérer tous les fichiers nécessaires pour faire fonctionner Piwigo.

  * [[http://fr.piwigo.org/basics/installation_manual|Installation manuelle]] : processus d'installation manuel qui consiste à télécharger et à déposer manuellement sur votre espace web, tous les fichiers nécessaires pour faire fonctionner Piwigo.

==== Utilisation de pLoader ====

pLoader permet d'uploader facilement les photos vers votre piwigo en ligne, en y ajoutant des commentaires et autres méta-données.
Cliquez sur **[[apt>ploader]]** pour l'installer.
 
==== Utilisation de Jiwigo ====

[[http://www.jiwigo.com/]]

Jiwigo permet d'administrer facilement sa galerie Piwigo : ajout de catégories, de photos, consultation des photos etc.

Pour installer Jiwigo par les dépôts, il faut ajouter le dépôt suivant : 
<code>deb http://www.jiwigo.com/depot/ ./</code>
Puis rafraichir la liste des paquets : 
<code>sudo apt-get update</code>
Et enfin, installer le logiciel : 
**[[apt>jiwigo|Jiwigo]]**

Pour que Jiwigo fonctionne, [[java|Java]] doit être installé sur l'ordinateur.
==== Utilisation de Shotwell ====

Shotwell, offre la possibilité de téléverser vers Piwigo, à condition d'avoir installé l'extension pwg.images.addSimple.
(pwg.images.addSimple est intégré à Piwigo depuis la version 2.3)

==== Utilisation de Digikam ====

Un plugin Piwigo est disponible pour [[:DigiKam]] (cf. [[http://www.digikam.org/drupal/node/503|annonce]]). Si vous avez [[apt://kipi-plugins]] d'[[:tutoriel:comment_installer_un_paquet|installé]], l'export piwigo est propsé par défaut dans DigiKam > Exportation > Piwigo.

==== Utilisation de PiwigoBook  ====

cf.  [[http://fr.piwigo.org/ext/extension_view.php?eid=500| la page de présentation sur la doc de piwigo]]

===== Désinstallation =====

Pour supprimer cette application, il suffit de supprimer sur votre hébergement le dossier contenant Piwigo et de supprimer également la base Mysql correspondante.
