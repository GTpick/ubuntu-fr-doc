====== X-Moto ======

{{ applications:jeux:xmoto.jpg?300}}
**X-Moto** est un jeu où vous devez conduire une motocross avec une vue de profil. Il y a des centaines de niveaux disponibles, qui garantissent de nombreuses heures de jeux. Pour finir un niveau, il faut récupérer toutes les fraises, et ensuite aller au point d'arrivée : une fleur.

===== Pré-requis =====
 
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  * Avoir activé l'[[accélération graphique]].((le jeu semblerait fonctionner sans l'accélération, mais de façon saccadée.))

===== Installation =====


==== Via les dépôts ====

Pour installer ce jeu, il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>xmoto]]**.


==== Via PlayDeb ====

Une version plus à jour est disponible sur le dépôt PlayDeb. Installez le dépôt [[:PlayDeb]], puis rendez-vous sur [[http://www.playdeb.net/software/xmoto|la page du jeu]] et cliquez sur le bouton « Install this now ».
