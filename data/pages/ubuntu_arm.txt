======Ubuntu pour processeur ARM======

ARM est une architecture de processeur utilisé dans une grande variété d'applications, telles que:

    * Tablettes (ex : Microsoft Surface, Apple iPad, Archos 101, Samsung galaxy tab 2, etc);
    * Téléphones portables (surtout les smartphones);
    * Appareils de réseau (Thecus série N, Linksys NSLU2, QNAP TS série, etc);
    * Machine de prototypage d'ingénierie, d’électronique embarquée ([[http://www.gumstix.com/|Gumstix]], [[http://beagleboard.org/|BeagleBoard]], certaines cartes Arduino, Raspberry Pi, etc);
    * Des notebooks (Sharp Netwalker, Toshiba AC100, etc).

Ubuntu vise le processeur ARM EABI, en respectant le type ARMv7 + VFP ISA et en prenant partiellement en charge les jeux d'instructions antérieurs (ARMv5t, ARMV6). Cette version était disponible dans les premières versions du port ARM (jaunty, karmic). Les objectifs d'ubuntu ARM visent généralement le [[https://wiki.ubuntu.com/ARM/Thumb2|jeu d'instructions Thumb2]].
