====== CamicriCube ======
{{:applications:cubelogo.png?100 }} \\ Le logiciel **CamicriCue** est un application sous licence libre ([[wpfr>Licence_publique_g%C3%A9n%C3%A9rale_GNU|GPL]]), écrite en [[:Mono]] qui permet d'installer des paquets sur un ordinateur hors-ligne, en les téléchargeant sur un ordinateur connecté. Il s'utilise sous WIndows comme sous Ubuntu et la famille debian.
\\

\\
<note important>La dernière version de Camicricube, s'appelle [[:camicricube-server]]. Elle est une réécriture complète de l'application. Donc si vous souhaitez travailler avec la version la plus récente consultez [[:camicricube-server|la page qui la concerne]].</note>
===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  * Si vous pensez exécuter Cube sous Windows vous aurez besoin d'installer [[http://download.xamarin.com/GTKforWindows/Windows/gtk-sharp-2.12.21.msi|GtkSharp]] et //Microsoft .Net 4.0// si ce n'est pas déjà le cas.
  
===== Installation =====

Pour installer ce logiciel, téléchargez l'archive **zip** correspondant à votre [[:architecture_materielle|architecture]] [[https://launchpad.net/camicricube|ici]].\\
[[:archivage|Décompressez l'archive]] sur une clé USB.


===== Utilisation =====

Déplacez-vous jusqu'au dossier **cube** précédemment décompressé. Sous Ubuntu double-cliquez sur l'exécutable **cube-linux**. Choisissez //Lancer//.
Sous Windows double-cliquez sur **cube.exe**.

====Sur l'ordinateur sans connexion====
Exécutez d'abord l'application sur l'ordinateur qui n'a pas d'accès à internet.
Lorsque cette fenêtre s'affiche, entrez le nom de votre projet et cliquez sur **Create**:

{{:applications:cube_trusty.png?350}}

Une nouvelle fenêtre s'ouvre qui vous propose de télécharger la liste des paquet, choisissez **Cancel** et à nouveau **No** puisque vous n'avez pas d'accès à internet.


====Sur l'ordinateur avec connexion====
Maintenant rendez-vous sur un ordinateur avec une connexion internet. Lancez à nouveau **Cube** et choisissez votre projet dans le champs **Open existing project**. Cette fois-ci si on vous propose de télécharger la liste des paquets, acceptez.

{{:applications:cube2_trusty.png?400}}

Cube va chercher les mise-à-jour pour tous vos logiciels installés sur votre ordinateur hors-ligne. Vous pouvez également faire un recherche pour télécharger de nouveaux paquets.
Il ne vous reste plus qu'à cliquer sur **Download All Marked**, pour télécharger les mise-à-jour et les paquets choisis.
Fermez Cube et retournez sur l'ordinateur hors-connexion.

====Sur l'ordinateur sans connexion====
Ouvrez à nouveau **Cube** répondez **Yes** aux diverses questions. Il ne vous reste plus qu'à cliquez sur **Downloaded** pour voir les paquets et mise-à-jour disponibles, puis sur **Mark All downloaded**.
Terminez en cliquant sur **Install All Marked**.
Votre système est à jour.

===== Désinstallation =====

Pour supprimer cette application, il suffit de supprimer le dossier Cube décompressé précédemment.
