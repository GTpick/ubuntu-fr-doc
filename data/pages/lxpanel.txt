====== LXpanel ======

LXPanel est un tableau de bord pour [[:LXDE]] basée sur GTK+ dérivée de [[:Fbpanel]]. Léger, configurable, c'est le tableau de bord par défaut de [[:LXDE]], mais elle peut être installée sur d'autres environnements.

Cette barre comporte :
  * Un menu généré à partir des fichiers *.desktop,
  * Une fonction lanceur de tâche accessible par Alt + F2,
  * Des lanceurs d'applications configurables,
  * Une liste des fenêtres ouvertes,
  * Des applets (horloge, zone de notification).

{{http://pix.toile-libre.org/upload/original/1252253069.png?650}}



===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.


===== Installation =====

Lxpanel est déjà installé par défaut sous [[:Lubuntu]] mais si vous souhaitez l'utiliser sur un autre [[:environnements|environnement]] [[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>lxpanel]]**. 


===== Configuration =====

<note warning>Bien que LXPanel soit dérivé de [[:fbpanel]], les fichiers de configuration de ces 2 tableaux de bord ne sont pas compatibles.</note>

Les fichiers de configuration sont situés dans le dossier caché **~/.config/lxpanel/LXDE** mais LXPanel peut être configuré sans passer par ceux-ci grâce à une interface graphique. Pour y accéder, il suffit d'un clic droit sur le panneau et de choisir **Paramètres du tableau de bord**.\\

Le premier onglet //Taille// permet de configurer l'apparence globale du tableau, la position, la taille et d'autres aspects du tableau :

{{:personnalisation:lxpanel:lxpanel1_trusty.png?350}} 

Le deuxième onglet permet de modifier l'apparence du tableau de bord, le thème, la couleur et la police :

{{:personnalisation:lxpanel:lxpanel2_trusty.png?350}}

 
Le troisième onglet //Applet du tableau de bord// permet de définir quels applets sont activés et s'ils doivent ou non s'étirer pour couvrir le tableau. En fait, chaque élément affichable sur la barre est considéré comme un applet (ou plugin), y compris le menu application et l'horloge.

{{:personnalisation:lxpanel:lxpanel3_trusty.png?350}} {{:personnalisation:lxpanel:lxpanel4_trusty.png?350}}

Le quatrième et dernier onglet permet entre autre de choisir le gestionnaire de fichiers et l'émulateur de terminal :

{{:personnalisation:lxpanel:lxpanel5_trusty.png?350}}


===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. La configuration de l'application sera conservée ou supprimée selon la méthode de désinstallation que vous choisirez.
