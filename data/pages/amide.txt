====== Amide ======

Amide (Amide's a Medical Imaging Data Examiner) est un logiciel complet de visualisation et d’analyse d’images médicales.

Cette application permet entre autres :

    * le support des formats d'image suivants : Analyse, DICOM 3.0, ECAT 6, ECAT 7, Gif, Interfile, NEMA, Raw ;
    * la navigation selon les 3 plans de coupes orthogonaux (axial, sagital, coronal) ;
    * de nombreux outils de visualisation : échelle de couleur, zoom , contraste, réorientation des volumes ;
    * le recalage et la fusion de plusieurs séries d'images ;
    * le rendu volumique ;
    * la segmentation ;
    * le tracé de régions d'intérêt.

{{ amide_lucid.png?500 }}

<note important>Ce logiciel ne disposant pas du marquage CE, il ne doit pas être utilisé à des fins de diagnostic médical.</note>
===== Pré-requis =====
 
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation =====

Pour installer ce logiciel, il suffit d'[[:tutoriel:comment_installer_un_paquet#paquet_present_dans_les_depots|installer le paquet]] **[[apt>amide]]**.

===== Utilisation =====
 
Lancez l'application via le [[:unity#tableau_de_bord_dash|tableau de bord]] (Unity) ou via le [[:terminal]] (toutes versions d'Ubuntu) avec la [[:commande_shell|commande]] suivante : <code>amide</code>

Pour importer des images au format DICOM, aller dans le menu File -> Import File(Guess) puis naviguer ensuite vers le dossier contenant les images et cliquer sur un des fichiers. Les données seront ensuite automatiquement importées.

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
