====== Ace Of Penguins ======

{{http://www.delorie.com/store/ace/as.gif}}

"Ace of penguins" est une suite de jeux de réflexion classiques mais néanmoins sympathiques, car à l'effigie de **Tux** .\\
Ils sont distribués sous licence GNU/GPL.\\

FIXME Ces modules semblent incompatibles avec Lucid (10.4) en architecture 64 bit.\\
FIXME Idem en architecture 32 bits !



===== Installation =====

[[:tutoriel:comment_installer_un_paquet|Installez le paquet]] **[[apt://ace-of-penguins|ace-of-penguins]]**.


Plusieurs jeux seront installés dans le répertoire /usr/games/:\\
*ace_canfield\\
*ace_mastermind\\
*ace_pegged\\

*ace_freecell\\

*ace_merlin\\
*ace_solitaire\\

*ace_thornq\\

===== utilisation =====

==== Lancement ====

Les jeux peuvent être lancés par des lignes de commande:

<code>ace_canfield     
ace_mastermind   
ace_pegged       
ace_taipei
ace_freecell     
ace_merlin       
ace_solitaire    
ace_thornq
ace_golf         
ace_minesweeper  
ace_taipedit</code>

Il est possible d'utiliser l'[[:alacarte|éditeur de menu gnome]] pour ajouter les icones dans le sous menu "Applications/Jeux". L'icone "ace of penguins" se trouve dans le répertoire: /usr/share/doc/ace-of-penguins/as.gif


==== Aide ====

plusieurs possibilités:\\
*consulter l'aide durant le jeux avec la touche F1\\
*consulter le site internet de l'auteur\\
*utiliser les pages manuelles dans la console, par exemple, dans une console taper:
<code>man ace_freecell</code>
