====== Decibel Audio Player ======

**Decibel Audio Player** est un [[lecteur audio]] très léger nécessitant peu de dépendances. 

{{decibelaudioplayer.png|}}

===== Installation =====
<note important>decibel_audio_player n'est plus dans les dépôts officiels d'Ubuntu depuis [[Xenial]]. </note>
==== via les dépôts officiels  ====
[[:tutoriel:comment_installer_un_paquet|Installez le paquet]] **[[apt>decibel-audio-player|decibel-audio-player]].** 

==== via le paquet deb du site officiel  ====

<note warning>Attention, cette méthode n'est pas recommandée par les développeurs.</note>

Téléchargez et [[:tutoriel:comment_installer_un_paquet|Installez le paquet DEB]] que vous trouverez ici : [[http://decibel.silent-blade.org/index.php?n=Main.Download|decibel.silent-blade.org]]. 
  

===== Modules =====

**Les différents modules permettent d'ajouter diverses fonctionnalités au lecteur.**

Pour y accéder il suffit de cliquer sur **Édition** / **Préférences** et de cocher la ou les cases en fonction du ou des modules que vous souhaitez activer.

<note>Certains modules peuvent nécessiter l'installation de dépendances. Auquel cas elles vous seront notifiées par celui-ci </note>

=== Modules présents dans la version 1.03. ( site officiel ) ===

   * **AudioScrobbler** : Met à jour votre profil last.FM.
   * **Bibliothèque** : Organise votre musique par tags plutôt que par fichiers.
   * **CD Audio** : Lit les CD audio.
   * **Explorateurs de fichiers ( activé par défaut. Possibilité de modifier les préférences )** : Vous permet d'explorer vos disques.
   * **Fichier de statut** : Génère un fichier texte selon ce que vous écoutez.
   * **Icône de statut** : Ajoute une icône dans la zone de notification.
   * **Messagerie instantanée** : Met à jour le message de statut de votre client de messagerie instantanée.
   * **Notification** : Affiche un avertissement à chaque changement de morceau.
   * **Pochettes** : Affiche les pochettes des albums.
   * **ReplayGain** : Normalise le volume.
   * **Twitter** : Met à jour votre compte Twitter.
   * **Égaliseur** : Ajuster le niveau des bandes de fréquences. 

===== interface =====
Plusieurs interfaces sont disponibles et accessibles depuis le menu **Mode** ou en utilisant les **[[raccourcis_clavier|raccourcis clavier]]**.

  * **Complet** : Maj-Ctrl-F
  * **Playlist** : Maj-Ctrl-P
  * **Mini** : Maj-Ctrl-M
  * **Nano** : Maj-Ctrl-N

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
