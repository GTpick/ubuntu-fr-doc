====== Ajouter une image dans votre document ======

===== Recommandations =====

  * Utiliser de préférence une image au format PNG ((PNG est idéal pour les captures d'écran)).
  * Si l'image est une [[:capture_d_ecran|capture d'écran]], alors le thème doit être le **thème par défaut** comme //Ambiance// pour Ubuntu. Idem pour Kubuntu et Xubuntu.
  * La nommer de la manière suivante : **''nomdelapage_uversion_xx.png''**, où :
    * ''nomdelapage'' est le nom de la page ;
    * ''uversion'' est la version d'Ubuntu selon le nom de code (hardy, jaunty, karmic, lucid, maverick, natty, ...) ;
    * ''xx'' est le numéro d'ordre de la capture sur 2 positions (facultatif si une seule capture, mais conseillé au cas où ; le logo peut ne pas avoir de numéro ou porter le 00).
  * Éviter les images trop grandes et/ou penser à réduire la taille pour un affichage correct.
  * Penser à donner une brève description à votre image après l'insertion

===== Comment procéder =====


  * Se connecter **(seul les utilisateurs enregistrés peuvent poster une image)**.
  * Éditer une page. 
  * Choisir l'icône image dans la barre d'outils (image ci-dessous) :
{{:playground:ajout-image_00.png?200|barre d'outil édition}}

  * Le **Mediamanager** du wiki s'ouvre dans une pop-up (image ci-dessous) :
{{:playground:ajout_image_01.png?450|Ajout d'image}}
  * Choisissez et ouvrez la catégorie (ou sous-catégorie) dans lequel vous souhaitez déposer votre image (arborescence partie gauche)
  * Cliquez sur "Sélection de fichiers..." pour sélectionner l'image à téléverser.
<note>Si vous souhaitez créer une sous-catégorie dans la catégorie active, il suffit de faire précéder le nom du fichier (une fois dans la case de téléchargement) par le nom de la future sous-catégorie séparé par un"** : **"</note>
  * Cliquez sur "télécharger"
  * Une fois téléchargée, un bouton "terminé" apparaîtra. Cliquez dessus, pour rendre effective votre image. 
  * Éditer les paramètres (Cible, alignement, taille) en cliquant dessus (image ci-dessous) : \\
{{:playground:ajout_image_02.png?200|Paramètres de l'image}}
  * Et pour terminer, cliquez sur le bouton "insérer" afin que DokuWiki fabrique le code de l'image. Elle sera disposée au point d'insertion de votre page en cours de modification.

<note warning>Si vous modifiez le nom (//wikiname//) de votre image, conservez l'extension sous peine de refus à l'envoi.</note>
<note tip>Si votre image est trop grande, vous pouvez la réduire via la syntaxe suivante :  

<file>{{:nomdelapage_uversion_xx.png?150}}</file>
 (ajustez la valeur (150) à votre besoin).</note>
===== Supprimer les images obsolètes =====

Suivant l'espace dans lequel vous avez enregistré votre image, vous pourrez aussi la supprimer. Il suffit d'afficher le formulaire d'ajout d'image, et de cliquer sur l'icône de poubelle.
Si votre nouvelle capture remplace une ancienne capture et que vous ne parvenez pas à l'enlever, merci d'ajouter le nom de l'image à supprimer sur [[:wiki:images_a_supprimer|cette liste]] afin qu'elle soit supprimée ultérieurement.
