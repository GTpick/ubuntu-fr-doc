<note importante>Ce modèle de page doit être utilisé comme base, lors de la création d'une page du Wiki consacrée à un modèle d'ordinateur portable, **si l'installation n'est pas immédiate ou est particulière**. Vous ne devez pas écrire votre page de documentation ici, mais créer une **nouvelle page**, dans laquelle vous **recopierez** ce modèle de page et qu'ensuite vous modifierez.

Les remarques générales suivantes s'appliquent toujours :
  * Nommer la page le plus simplement et le plus clairement possible : « //Dell Inspiron 1520// » devient « //dell_inspiron_1520// ».
  * La page doit être créée à la racine du Wiki :\\ [[:constructeur_modèle_série]]

Vous pouvez récupérer le code Wiki de ce modèle à l'aide du bouton « Modifier cette page », dans le menu de gauche. 

**N'oubliez pas de rajouter votre portable dans [[:portable|la liste des portables]], avec un lien vers votre page nouvellement créée.**</note>

==== Pavé tactile ====

Vous pouvez consulter la page consacrée au [[:touchpad|pavé tactile]] //(touchpad)// pour savoir comment utiliser toutes ses possibilités (ascenseur vertical, ascenseur horizontal, ...).
**Le Touchpad présente souvent le défaut suivant : lorsqu'on tape au clavier, on clique sans le vouloir avec la paume. Consultez [[:touchpad#problemes_de_contact_de_paume|ce paragraphe]] pour corriger ce défaut.**

<note>Cette optimisation ne dépend pas du portable utilisé. Inutile donc de détailler les manips, puisqu'elles sont décrites dans une page spécifique. Cela facilite de plus la maintenance des infos relatives au Touchpad.


**Si vous constatez que l'optimisation que vous proposez est classique, préférez en faire une page séparée que vous lierez depuis les pages portables concernées.**
</note>
