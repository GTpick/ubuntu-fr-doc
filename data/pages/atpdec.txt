
====== ATPDEC ======
Atpdec est un utilitaire en ligne de commande permettant de décoder les images envoyées par les satellites météo NOAA.

<note> Documentation en construction</note>

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation =====

  * Installer les librairies libsndfile-dev et libpng-dev nécessaires à la compilation du programme atpdec-1.7:
<code>sudo apt-get install libsndfile-dev 
sudo apt-get install libpng-dev
</code>
  *Pour installer Atpdec, télécharger les sources :
<code>wget http://freefr.dl.sourceforge.net/project/atpdec/atpdec%20sources/1.7/atpdec-1.7.tar.gz</code>
  *Compiler :
<code>tar -zxvf atpdec-1.7.tar.gz 
cd ./atpdec-1.7
make</code>
  *Placer le fichier obtenu avec vos exécutables :
<code>sudo cp atpdec /usr/local/bin/</code>

===== Utilisation =====
  *Pour obtenir les options du programme, rapper :
<code>atpdec -h</code>
===== Désinstallation =====

  *Supprimer simplement le fichier exécutable :
<code>sudo rm /usr/local/bin/atpdec</code>
