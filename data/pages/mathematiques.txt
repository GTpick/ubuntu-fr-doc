====== Mathématiques ======



Liste de logiciels de maths sous Ubuntu :


==== Logiciels de mathématiques pour les enfants ====

  * [[:DR Géo]] - Logiciel de géométrie dynamique.
  * [[:Kig]] - Logiciel de géométrie dynamique. Intégré à kdeedu.
  * [[:TuxMathscrabble]] - Petit quiz mathématique.
  * [[:WIMS]] - Serveur d'exercices mathématiques. Installable localement sur une machine ou en réseau.
  * [[kbrush]] - Apprentissage du calcul avec les fractions. Intégré à kdeedu.
  * [[:GeoGebra]] - Logiciel de géométrie dynamique. Utilise JAVA.
  * [[GeoLabo]] - Logiciel de géométrie dynamique. Utilise Java.
  * [[GeoNext]] - Logiciel de géométrie dynamique. Utilise Java.



==== Fonctions ====
  *[[Edugraphe]] : traceur de courbes, utilisation de la méthode d'Euler... 
  *[[geg]] : traceur de courbes très simple
  *[[GeoGebra]] : édition de courbe, de droite, de cercles, d'angles, de symétries... En langage Java.
  *[[Gnuplot]] :  représentations graphiques (2 ou 3 dimensions) de fonctions numériques ou de données. 
  *[[KmPlot]] : éditeur d'équations et traceur de courbes. 





==== Géométrie ====
  *[[Dr_Géo]] : logiciel de géométrie interactive. (Ressemble à Cabri Géomètre de Windows).
  *[[GeoGebra]] : programme de géométrie dynamique (qui permet aussi de tracer les courbes représentatives des fonctions)
  *[[GéoLabo]] : programme de géométrie dynamique
  *[[GeoNext]] : programme de géométrie dynamique
  *[[Kig]] : logiciel de géométrie dynamique
  *[[K3DSurf]] : modeleur de surface (peut être utilisé de façon plus ou moins anecdotique)
  *[[TeXgraph]] : logiciel de géométrie et de dessin pour LaTeX
  *[[CaRMetal]] : très bon logiciel de géométrie dynamique
  *[[giac-xcas]] : logiciel de calcul formel, programmation, tableur, géométrie, etc.
  * [[:Xaos]] : un utilitaire pour découvrir les fractales


==== Calcul formel ====
  *[[Maxima]] : très bon logiciel pour les algèbres.
  *[[giac-xcas]] : logiciel de calcul formel, programmation, tableur, géométrie...
  *[[Mathomatic]] : logiciel de calcul algébrique
  *[[sagemath]] : logiciel de calcul formel et numérique qui inclut Maxima et dispose d'une interface web




==== Calculs numériques ====
Voir aussi : [[calcul_numerique]]

  *[[scilab]] : logiciel de calcul numérique (orienté matrices), proche de Matlab, d'origine française.
  *[[Octave]] : logiciel de calcul numérique (orienté matrices), proche de Matlab.
  *[[R]] : logiciel de statistiques, similaire à S.
  *[[gdl]] : logiciel de calcul numérique, proche de IDL.


==== Calculatrices ====
  * [[bc]] et [[dc]] : calculatrices à utiliser dans la console ;
  * [[KCalc]] : une calculatrice très simple ;
  * [[SpeedCrunch]] : une calculatrice simple installée par défaut pour Kubuntu ;
  * [[Abakus]] : une calculatrice de niveau intermédiaire, permettant d'utiliser la notation polonaise inverse ;
  * [[Qalculate!]] : calculatrice avancée permettant de tracer des fonctions 2D ;
  * [[Kalgebra]] : calculatrice avancée permettant de tracer des fonctions 2D et 3D ;
  * [[Tiemu]] : simule une Texas Instrument Ti 89/Ti92/Ti92+ ;
  * [[Qalc]] : calculatrice avancée en ligne de commande.
  * [[Extcalc]] : calculatrice scientifique multifonction et graphique.


==== Communication avec des calculatrices externes ====

  * [[Casetta]] (pour calculatrice Casio) : permet de communiquer et d'échanger des informations entre une calculatrice Casio et un ordinateur. Voir aussi son interface graphique gasetta [[casio_programme_de_tranfert]] ;
  * [[Tilp]] [pour calculatrice Texas Instrument] : permet de communiquer et d'échanger des informations entre une calculatrice Texas Instrument et un ordinateur.
==== Divers mathématiques ====
  *[[Dmaths]] : permet d'ajouter des objets mathématiques à OpenOffice.org.
  *[[GraphThing]] : créer, manipuler et étudier des graphes.
  *[[http://pecesjocdetangr.sourceforge.net/ |Tangram]] : logiciel pour jouer au tangram
  * [[Pyromaths]] :  Générateur de fiches d'exercices de mathématiques au format pdf.
  * [[Geophar]] : Le couteau suisse des mathématiques. Dispo dans les depots depuis Vivid, [[http://wxgeo.free.fr/doc/html/help.html|Une page de manuel sur wxgeo]]
