====== XPDF : lecteur de fichiers PDF ======

//Il existe plusieurs outils pour lire des fichiers PDF sous Linux, voir [[PDF]].//

**XPDF** est un lecteur de fichiers PDF très rapide et léger, qui se lance quasi instantanément. L'interface est effectivement très simpliste, mais suffit pour une utilisation occasionnelle. Il possède toutes les fonctions basiques d'un lecteur PDF.

===== Pré-requis =====

Pour permettre l'installation, vous devez avoir activé l'accès aux [[:depots#universe_et_multiverse|dépôts Universe]].



===== Installation =====

L'installation est très simple : il vous suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt://xpdf|xpdf]]**.

Pour afficher des caractères asiatiques, vous pouvez [[:tutoriel:comment_installer_un_paquet|installer les paquets]] suivants : **[[apt://xpdf-korean|xpdf-korean]]** , **[[apt://xpdf-japanese|xpdf-japanese]]** , **[[apt://xpdf-chinese-simplified|xpdf-chinese-simplified]]** , **[[apt://xpdf-chinese-traditional|xpdf-chinese-traditional]]**. 

===== Utilisation =====
 
Lancez l'application via le [[:unity#tableau_de_bord_dash|dash]] (Unity) ou via le [[:terminal]] (toutes versions d'Ubuntu) avec la [[:commande_shell|commande]] suivante :
<code>xpdf</code>
 
===== Désinstallation =====
 
Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
