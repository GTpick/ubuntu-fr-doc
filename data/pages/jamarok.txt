====== Jamarok ======
<note warning>Cet article n'a plus d'utilité pour les versions récente d'Amarok, si tel est votre cas rendez vous sur la page d'[[Amarok]]. Néanmoins il comporte toujours un intéret pour xmms.</note>

Normalement conçu pour fonctionner avec Amarok, ce programme permet d'écouter de la musique libre ([[http://www.jamendo.com/fr/|Jamendo]], [[http://musique-libre.org|Musique-Libre]], [[http://archive.org/audio]]) en ligne, même si, comme moi, vous utilisez xmms.

===== Installation =====

  * Télécharger jamarok_0.4-1_i386.deb sur le site [[http://dogbreath.free.fr/JamaroK/]] : <code>wget http://dogbreath.free.fr/JamaroK/jamarok_0.4-1_i386.deb</code>
  * Installer le paquet : <code>sudo dpkg -i jamarok_0.4-1_i386.deb</code>
  * <code>sudo apt-get install -f</code> pour tenir compte des dépendances

Cela n'installe pas de raccourci dans le menu, vous pouvez le faire en utilisant Menu editor ou en faisant un raccourci sur votre bureau.

{{applications:jamarok.png }}

L'icône se trouve dans /usr/share/icons/hicolor/32x32/apps/jamarok.png

===== Utiliser xmms =====

Lors de l'ouverture du programme, cliquez sur l'onglet **Options** et cochez "**Lecture de la playlist avec Xmms**".
