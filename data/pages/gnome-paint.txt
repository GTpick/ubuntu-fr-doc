====== gnome-paint ======

**gnome-paint** est un éditeur graphique simple conçu pour l'environnement de bureau [[GNOME]]. Il s'inspire de Microsoft Paint.

{{:applications:gnome-paint_lucid_01.png?500}}

<note important>Sous Ubuntu 10.04 LTS 64 bits, l'application présente quelques problèmes de stabilité (certains menus ne s'affichent pas, le curseur de la brosse n'apparait pas dans la fenêtre, certains boutons sont masqués, l'outil Texte semble inopérant...</note>

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation =====

===== Sous Ubuntu Xenial 16.04 =====

Installer le logiciel depuis les dépôts : [[apt>gnome-paint]]

===== Versions plus anciennes d'Ubuntu =====


Ce logiciel n'est pas disponible dans les dépôts officiels. On peut l'installer à l'aide d'un paquet disponible sur le site du projet.

  - Téléchargez le paquet .deb pour votre [[versions|version d'Ubuntu]] sur la [[http://code.google.com/p/gnome-paint/downloads/list|page de téléchargement]]
    * Pour Ubuntu 32 bits : choisissez le fichier **gnome-paint_0.3-1_i386.deb**
    * Pour [[ubuntu_64bits|Ubuntu 64 bits]] : choisissez le fichier **gnome-paint_0.3-1_amd64.deb**
  - [[:tutoriel:comment_installer_un_paquet|Installez-le]].


laouedj ===== Utilisation=====

Lancez l'application depuis le menu //Applications -> Graphisme -> Éditeur de dessin Gnome Paint// ou //via// la [[:commande_shell|commande]] suivante :

  sudo gnome-paint

==== Formats de fichier ====

gnome-paint reconnait les formats suivants : BMP, ICO, JPG, PNG et TIFF.

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
