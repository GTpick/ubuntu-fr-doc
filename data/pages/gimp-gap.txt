====== Gap : GIMP Animation Package ======

Le GAP ou Gimp Animation Package se présente comme une collection de greffons qui étendent encore les capacités d’animation du [[:GIMP]] en permettant la réalisation d’animations plus sophistiquées.

Exemple de morphing réalisé avec Gimp-GAP:

{{http://fc03.deviantart.net/fs11/i/2006/188/5/a/Morphing_Images_in_GIMP_by_fence_post.gif|Exemple de morphing réalisé avec Gimp-GAP}} 

 
===== Pré-requis =====
 
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
===== Installation =====


Il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>gimp-gap]]**.

===== Utilisation =====

==== Morphing ====

<note help>Tutoriel repris de ce [[http://fence-post.deviantart.com/art/Morphing-Images-in-GIMP-35963918|tutoriel sous cc (by-nc-sa)]]

Vous pouvez [[http://www.youtube.com/watch?v=qaSz8WNKjD4|voir aussi un tutoriel vidéo en anglais]].
</note>


Les étapes :
  -  Ouvrir vos 2 images dans deux calques différents. ( si possible de dimension similaire ) 
  - Ajouter un canal alpha au calque de l'arrière plan. ( Clic droit sur le calque et 'Ajouter un canal alpha').
  - Maintenant vous pouvez aller dans **Vidéo -> morphing**
  - **facultatif** : Vous pouvez cliquer sur le bouton forme ce qui placera des points au contour (rectangulaire ) de votre image  : afin d'éviter certaines distorsions. C'est facultatif mais à savoir.
  - Placer des points à des positions similaires sur les deux images. ( ex : centre de l'oeil sur les deux, point du nez ... ) Il faut en mettre suffisamment pour dessiner les traits, en mettre des millions ne sert pas non plus. C'était la partie la plus difficile :-P.
  - Choisir le nombre de pas pour le morphing : (cad : le nombre d'image entre ces deux images).
  - **facultatif** : Vous pouvez cocher l'option qualité : mais ca prendra plus de temps. 
  - Cliquer sur valider. **VOILA**.

Vous pourrez sélectionner une image intermédiaire, ou enregistrer en animation (gif) ou vidéo.
