====== Mahjongg ======

**Mahjongg** est un jeu de patience utilisant les tuiles du jeu oriental [[wpfr>Mah-jong]] classique. Il fait partie de l'ensemble de jeux du projet [[:GNOME]].

{{jeux:mahjongg_xenial.png?500}}


Pour un "vrai" jeu de Mahjong à quatre joueurs (ou contre l'ordinateur), vous trouverez des informations sur cette  [[http://manpages.ubuntu.com/manpages/hardy/man6/xmj.6.html|page]] (en anglais).


===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.


===== Installation =====

Ce logiciel est déjà installé dans Ubuntu. Pour l'installer sous d'autres [[:environnements|environnements de bureau]], il suffit d'[[:tutoriel:comment_installer_un_paquet#paquet_present_dans_les_depots|installer le paquet]] **[[apt>gnome-mahjongg]]**.

<note tip>Sous [[:Kubuntu]], préférez-lui [[apt>Kmahjongg]], qui fait partie des jeux de l'environnement [[:KDE]].</note>

===== Utilisation =====
 
Lancez l'application via le [[:unity#tableau_de_bord_dash|tableau de bord]] (Unity) ou via le [[:terminal]] (toutes versions d'Ubuntu) avec la [[:commande_shell|commande]] suivante : <code>mahjongg</code

L'objectif est de retirer toutes les tuiles du jeu. Les tuiles sont retirées par paires assorties. Les tuiles ne peuvent pas être retirées si elles n'ont pas au moins un bord long de libre. Cliquez sur n'importe quelle tuile libre et sur son double pour les retirer du jeu. Répétez cela, si vous le pouvez, jusqu'à ce que toutes les tuiles soient retirées.

Il y a neufs agencements possibles du jeu (réglable dans le menu //Paramètres -> Préférences//) :
  * Facile (par défaut)
  * Difficile
  * Croix de diffusion
  * Les murs de la pyramide
  * Dragon rouge
  * Tic-Tac-Toe
  * Nuage
  * Quatre ponts
  * La Ziggurat

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, le configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
