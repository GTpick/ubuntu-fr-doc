====== Ktouch : Apprentissage de la dactylographie ======

Développé pour [[:KDE]], Ktouch fonctionne parfaitement sous [[:GNOME]].\\
**Ktouch** est doté d'une interface graphique représentant un clavier dont les couleurs indiquent la position des doigts.

L'interface est munie d'une série d'indicateur vous permettant de suivre votre évolution durant la leçon qui s'affiche sous la forme d'une ligne parme.\\ Votre frappe est reprise textuellement dans une autre ligne située sous l'exercice, ligne verte.

{{ktouch2016.png|Interface de Ktouch}}


Vous pouvez sélectionner le niveau de la leçon grâce à deux boutons, incrémenter et décrémenter.
===== Pré-requis =====
 
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation =====

[[:tutoriel:comment_installer_un_paquet|Installez le paquet]] **[[apt>ktouch]]** et pour avoir Ktouch en français, il faut installer [[apt>language-pack-kde-fr]].

<note>Il y a de nombreuses dépendances… l'installation peut peser près de 300 Mo</note>
 
===== Configuration =====

À la première utilisation de Ktouch, il vous faudra configurer le clavier français ainsi que les exercices réservés à notre langue.

Pour le clavier aller dans **Settings -> Keyboard layout** puis sélectionner le clavier français ( **French (fr)** ).

Pour les leçons en français aller à **Training -> Default Lectures -> French (auto generated)**.

La configuration du logiciel vous est proposée par l'intermédiaire du menu **Settings -> Configure Ktouch**

Une fois ceci fait, vous êtes prêt à travailler.

<note tip>Si Ktouch refuse de se lancer sous gnome, lancer la commande :
<code>sudo chmod -R 777 .kde </code></note>


===== Utilisation =====

Pour lancer Ktouch aller dans **Applications -> Education -> Ktouch**. Ou sous Unity lancez l'application via le [[:unity#tableau_de_bord_dash|tableau de bord]] avec le mot clé **ktouch**.

Placer votre index gauche sur la touche F, le droit sur la touche J, le reste des doigts se placeront automatiquement sur les touches voisines et en route...

Ktouch se souvient de vos leçons et s'ouvrira dorénavant sur l'exercice en cours.

Vous pouvez admirer votre performance en visionnant vos statistiques (icône représentant une feuille de papier).

Bon apprentissage.


===== Désinstallation =====
Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer le paquet ktouch]].
