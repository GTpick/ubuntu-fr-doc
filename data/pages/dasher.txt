====== Dasher ======

Dasher est une interface de saisie de texte par prédiction ([[http://web-ia.ac-poitiers.fr/ia17/Ais17/blub/flash/captDasher2.htm|démonstration vidéo]]), via une gestuelle continue et naturelle de pointage. Dasher est un système de saisie de texte compétitif lorsqu'un grand clavier ne peut être utilisé - par exemple :
  * Sur un ordinateur de poche ou un ordinateur portable ;
  * Quand on travaille sur ordinateur avec une main, par manette de jeux, écran tactile, balle pointeuse, ou une souris ;
  * Quand on travaille sur ordinateur sans les mains (par exemple : souris commandée par la tête ou pointeur oculaire). 

{{:applications:dasher_xenial.png}}

===== Pré-requis =====
 
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
=====Installation=====

Pour installer Dasher, il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>dasher]]**.

=====Configuration=====

Lancez l'application via le [[:unity#tableau_de_bord_dash|tableau de bord Unity]] avec le mot clé **Dasher**.
Dans les préférences, choisir **Français / French with numerals and punctuation**.

=====Utilisation=====

Cliquez sur **Nouveau fichier**, puis un clic à droite de la croix centrale pour commencer la saisie. Bougez le curseur pour choisir les caractères que vous souhaitez saisir. Un autre clic pour stopper la saisie.

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
