====== WinFF - interface graphique pour ffmpeg ou Avconv======

WinFF est une interface graphique à [[FFmpeg]] ou [[avconv]] très simple d'utilisation et personnalisable. cette application permet de réaliser rapidement des encodages vidéos.

{{winff.png?300}}

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation =====

**WinFF** est disponible dans les [[:dépôts]] d'Ubuntu. 

==== Via les dépôts officiels====

Pour installer cette application, il suffit d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>winff]]**.

==== Via le PPA de WinFF ====
<note important>Installer un paquet hors des [[:depots|dépôts officiels]] peut présenter un risque d'instabilité de votre système.</note>
Si pour [[:precise|Precise 12.04]] vous souhaitez une version plus récente que celle présente dans les dépôts, vous pouvez utiliser ce [[:PPA]]:
  * [[:PPA|Ajoutez le PPA]] **ppa:paul-climbing/ppa**((https://launchpad.net/~paul-climbing/+archive/ppa)) dans vos sources de logiciels, puis
  * [[:tutoriel:comment_modifier_sources_maj#recharger_la_liste_des_paquets|rechargez la liste des paquets]],
  * enfin, [[:tutoriel:comment_installer_un_paquet|installer le paquet]] **[[apt>winff]]**.

En ligne de [[:commande_shell|commande]] depuis un [[:terminal]]:
  * Pour [[:GNOME]], [[:Xfce]] et [[:Lxde]]: <code>sudo add-apt-repository ppa:paul-climbing/ppa && sudo apt-get update && sudo apt-get install winff</code>
  * Pour [[:KDE]]<code>sudo add-apt-repository ppa:paul-climbing/ppa && sudo apt-get update && sudo apt-get install winff-qt</code>

==== Les Pré-réglages (Presets) ====

Les pré-réglages (presets) fournis avec le paquet (que ce soit des [[:dépôts]] officiels ou du [[:PPA]]) sont parfois obsolètes; il faut télécharger les pré-réglages correspondants à la version de Libavcodec que l'on a installé (libavcodec53 dans le cas de Ubuntu 12.04), en se rendant sur [[http://code.google.com/p/winff/downloads/list|cette page]] .
Puis on installe le nouveau fichier de presets grâce au menu "fichier" ---> "importer un pré-réglage".\\
À noter: libavcodec53 n'est pas disponible pour [[trusty|Trusty 14.04]].

==== Utiliser Avconv au lieu de FFmpeg ====

À partir d'[[:trusty|Ubuntu Trutsy 14.04]], [[:FFmpeg]] n'est plus disponible et remplacé par [[:Avconv]]; les versions de Winff pour [[Trusty|Trusty 14.04]] et au delà sont configurées par défaut pour cela. 

Pour [[:precise|Ubuntu Precise 12.04]], [[:FFmpeg]] est toujours présent, mais FFplay, FFprobe et FFserver ne sont plus que des liens vers Avplay, Avprobe et Avserver.\\
Il est possible de demander à Winff d'utiliser directement [[:Avconv]] et Avplay; pour cela, aller dans les préférences (menu "éditer" ---> "préférences"), et, sous l'onglet "Linux", corriger les chemins vers les exécutables : on aura respectivement /usr/bin/avconv et /usr/bin/avplay.\\
Au niveau des presets, ça ne change rien : les options supportées par FFmpeg le sont aussi par Avconv; en revanche, si on se fait des presets personnalisés, il faut savoir qu'Avconv a des fonctions que FFmpeg n'avait pas (lire le [[:tutoriel/console_commandes_de_base#man|man]]).

===== Utilisation =====
Lancez l'application depuis le [[:unity#tableau_de_bord_dash|tableau de bord d'Unity]] ou pour les menus classiques, //Applications -> Multimedia - > WinFF//. \\
Ou dans un [[:terminal]] pour toutes [[:variantes]] en saisissant la [[ :commande_shell|commande]] suivante: <code>winff</code>
L'utilisation est simple et intuitive, on ajoute un fichier vidéo, on choisit le format de ré-encodage, on valide, l'application fait le travail.\\
=====Configuration=====

====Multithreading====
Vous pouvez activer le **[[:multithreading]]** dans le menu //Éditer -> Préférences//. Cochez la case **multithreading** (dans la version 1.5.3 utiliser plusieurs enfilades d'encodage) et tapez le nombre de processeur ou de threads que vous voulez activer.\\
8-) Vous trouverez les détails de l'utilisation sur le [[http://lprod.org/wiki/doku.php/video:winff#utilisation|site lprod.org, section winff]].

====Les réglages manuels====
**WinFF** offre de très nombreux préréglages forts utiles et très simple à utiliser. Voir [[#Les préréglages|chapitre suivant]]. Toutefois, si vous ne trouvez pas votre bonheur, vous pouvez utiliser vos propres réglages audio et vidéo. Pour y accéder, un clic sur "option" en haut à droite et les onglets utiles pour les réglages personnalisés vont apparaître.
