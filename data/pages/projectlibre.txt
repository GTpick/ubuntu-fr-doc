====== Project libre ======

**Project libre** est un logiciel gratuit et libre de [[wpfr>gestion de projet]]. Il a pour but de remplacer MS Project, d'où une interface utilisateur similaire et une possibilité d'ouvrir les fichiers MS Project. Project libre est interopérable Project, les diagrammes de Gantt et les tableaux PERT.

Project libre fonctionne sur la plate-forme [[:Java]], ce qui lui permet de fonctionner sur une variété importante de systèmes d'exploitation. Son interface est en français.

Project libre est une reprise améliorée (fork) du logiciel abandonné [[:openproj]].

{{ http://a.fsdn.com/con/app/proj/projectlibre/screenshots/ProjectLibre_Gantt.jpeg?500 }}
===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.



===== Installation =====
Project libre n'est pas disponible dans les dépôts officiels. Un paquet DEB est disponible sur le site du projet :

  * Téléchargez le paquet **projectlibre_1.5.8-1.deb** (prenez la dernière version en date) sur [[http://sourceforge.net/projects/projectlibre/files/ProjectLibre/|la page de téléchargement SourceForge]] ;
  * [[:/tutoriel/comment_installer_un_paquet#installer_un_paquet_hors_depots_non_recommande|Installez-le]].

===== Utilisation =====

Lancez l'application depuis le menu //Applications -> Bureautique -> Project Libre// ou //via// la [[:commande_shell|commande]] suivante :

  projectlibre

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, le configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
