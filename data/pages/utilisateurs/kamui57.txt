===== Typographie =====

==== Lettres accentuées ====

  * É : CapsLock + é
  * È : CapsLock + è
  * À : CapsLock + à
  * Ê : un appui sur ^, puis sur shift+E
  * Ë : un appui sur shift + ^, puis sur shift + E

==== Autres lettres ====

  * Ç : CapsLock + ç
  * æ : altgr + a
  * Æ : altgr + shift + a

==== Guillemets ====

  * « : altgr + z
  * » : altgr + x
  * “ : altgr + v
  * ” : altgr + b

==== Flèches ====

  * ← : algtr + y
  * ↓ : algtr + u
  * → : algtr + i
  * ↑ : altgr + shift + u

==== Degré/numéro ====

  * ° : altgr + shift + ^
  * º : altgr + shift + m

==== Maths ====

  * ø : altgr + shift + o
  * Ø : altgr + shift + o
  * × : altgr + shift + ;
  * ÷ : altgr + shift + :
  * · : altgr + :
  * ± : altgr + shift + ç

==== Monnaies ====

  * € : altgr + e
  * £ : altgr + shift + "
  * $ : altgr + shift + '
