====== Synthèse des pages jeux ======

Le but de cette page est de propose un tableau de synthèse de toutes les pages jeux disponible sur le wiki Ubuntu-fr.

===== Légende =====

Le tableau est classé par ordre alphabétique, selon la légende suivante :
  * //**Nom du jeu**// : pas besoin d'explication ...

  * //**Type**// : indique pour quelle plate-forme le jeu a été développé
    * {{:logo:cof.png?24.png|paquet dans les dépots}} Jeux natif pour Linux, et présent dans les dépôts Ubuntu activés par défaut((voir le portail "[[http://doc.ubuntu-fr.org/jeu|Les jeux GNU/Linux natifs]]"))
    * {{:logo_tux.jpg?24|jeu GNU/Linux natif}} Jeux natif pour Linux, mais non disponible dans les dépôts Ubuntu activés par défaut((voir le portail "[[http://doc.ubuntu-fr.org/jeu|Les jeux GNU/Linux natifs]]"))
    * {{:wine.png|jeu Windows natif}} Jeux natif Windows à installer avec WinE ou équivalent((voir également le portail "[[http://doc.ubuntu-fr.org/jeu|Les jeux Windows]]"))


  * //**Statut**// : indique si le jeu est jouable ou non sur Ubuntu
    * {{wiki:modeles:laptop_ok.png|Installer, jouer}} Le jeu est pleinement fonctionnel sur Ubuntu((cas de la plupart des jeux présents dans les dépôts))
    * {{wiki:modeles:laptop_work.png|Configuration nécessaire}} Le jeu fonctionne après configuration((cas de certains jeux nécessitant WinE))
    * {{icons:important.png|peu mieux faire}} Le jeu fonctionne en partie (voir les remarques)
    * {{wiki:modeles:laptop_notok.png|passez votre chemin}} Le jeu n'est pas fonctionnel sur Ubuntu


  * //**Genre**// : indique à quelle famille appartient le jeu((voir le portail "[[liste_des_pages_contenant_le_tag_jeu|liste des jeux classés par genre]]"))

  * //**Abréviations**// : 
     * RTS = Real-Time Strategy (jeu de stratégie temps réel)
     * FPS= First-Person Shooter (jeu de tir à vue)
     * TBS= Turn-Based Strategy (jeu de stratégie au tour par tour)
     * RPG= Role-Playing Game (jeu de rôle)
     * MMORPG= Massively Multiplayer Online Role-Playing Game (jeu de rôle massif multijoueur)

  * //**Remarque**// : un complément d'information si besoin
