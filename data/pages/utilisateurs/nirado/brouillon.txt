====== Cardapio ======

Cardapio est un applet qui se veut remplacer l'habituel menu "Applications, Raccourcis, Système" de Gnome. Souvent mis à jour, il intègre un système d'extensions et est doté d'une zone de recherche pour retrouver facilement vos applications.


===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation =====

Pour installer Cardapio, et faire en sorte qu'il soit mis à jour automatiquement, commencez par ajouter le dépôt suivant :   [[:ppa|Personal Package Archives (PPA)]] 
<code>
ppa:cardapio-team/unstable
</code>

Pour cela, rendez-vous dans la **Logithèque Ubuntu** (habituellement Applications > Logithèque Ubuntu) et faites : 
  * Dans la fenêtre Logithèque Ubuntu : Clic sur menu **Édition** > **Sources de logiciels**...
  * Saisissez votre mot de passe ;
  * Dans la fenêtre Sources de logiciels qui apparaît : Clic sur onglet **Autres logiciels** ;
  * Dans l'onglet Autres logiciels : Cliquez sur le bouton **Ajouter** ;
  * Dans la boîte de dialogue qui apparaît, saisir ''ppa:cardapio-team/unstable'', puis cliquer sur le bouton **Ajouter une source de mis à jour** ;
  * De retour dans la fenêtre Sources de logiciels, cliquer sur le bouton **Fermer** 

À partir de ce moment, le dépôt à été ajouté. Vous pouvez maintenant vous rendre dans la logithèque et saisir le nom du logiciel ''cardapio'' pour l'installer, ou tout simplement cliquer sur le lien suivant : [[apt://cardapio]].


===== Configuration =====

Pour voir apparaître Cardapio dans votre tableau de bord. Faites un clic droit sur un coin vierge du tableau de bord, puis cliquer sur **Ajouter au tableau de bord**. Recherchez cardapio et ajoutez le.


===== Utilisation=====

Un simple clic 

===== Désinstallation =====

Pour supprimer cette application, il suffit de [[:tutoriel:comment_supprimer_un_paquet|supprimer son paquet]]. Selon la méthode choisie, la configuration globale de l'application est conservée ou supprimée. Les journaux du système, et les fichiers de préférence des utilisateurs dans leurs dossiers personnels sont toujours conservés.
