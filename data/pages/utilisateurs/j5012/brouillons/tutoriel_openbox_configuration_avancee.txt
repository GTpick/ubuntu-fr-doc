====== Configuration avancée d'Openbox ======

Ce tutoriel décrit les différentes configurations possibles, d'abord avec une interface graphique, puis en modifiant directement les variables de configuration.

===== Pré-requis =====

  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.
  * Savoir [[:tutoriel:comment_installer_un_paquet|comment installer les paquets]]
  * Avoir lu et compris préalablement la page de doc concernant [[:Openbox]]
[...]
===== Applications de configuration =====

  * obconf
  * obmenu
  * [[http://sourceforge.net/projects/obapps/|obapps]]
  * ...

===== Fichiers et variables de configuration =====

  * Souris et clavier : ~/.config/openbox/rc.xml
  * Applications au démarrage de session : ~/.config/openbox/autostart
  * Comportement global : ~/.config/openbox/environment
  * Menu : ~/.config/openbox/menu.xml
  * Gestion et interactions : ~/.config/openbox/rc.xml
  * Rédaction des thèmes

==== Configuration des menus ====

  * syntaxe des menus statiques
  * scripting de menus dit dynamiques
  * menu Debian

===== Utilitaires et accessoires =====

  * gestion energie
  * mise en veille
  * hibernation
  * dock d'icons
  * modules d'extensions pour fenêtres (widgets)
  * modules d'extensions pour bureau (desklets, panels, pagers ...)
  * [[http://openbox.org/wiki/Help:Contents#Cool_programs_to_run_with_Openbox|divers et variés]]
  * [[http://openbox.org/wiki/Openbox:Community_portal|+++]]

===== Conclusion =====

Confectionnez votre Ubuntu avec Openbox !

===== Problèmes connus=====

Openbox s'installe et se configure seul à partir d'une Ubuntu Minimale.
