====== NO$GBA : émulateur nintendo ds======

===== Introduction =====
<note warning>Programme propriétaire uniquement disponible sous windows et qui n'est plus développé.</note>
NO$GBA est un émulateur nintendo DS pour windows. Il fonctionne, cela dit très bien via [[wine]] et est de base meilleur que les autres émuateurs de la nintendo DS
.
<note important>À noter que NO$GBA émule aussi très bien la game boy advance</note>

===== Installation =====

Pour le télécharger il suffit d'aller sur le site d'emunova : lien mort

Une fois le téléchargement terminé, placez l'archive dans le dossier où vous le souhaitez puis décompressez-le. Normalement le dossier no$gba-w devrait apparaître, allez-y. Ensuite placez les roms dans le dossier "SLOT" (Facultatif) qui devrait apparaître après avoir sélectionner le fichier NO$GBA, celà dit, vous pouvez toujours indiquer le repertoire de vos roms nintendo DS.


===== Erreurs avec OpenGL =====

Il se peut que lorsque vous lancez un jeu, soudainement apparaissent des boites d'erreur affichant "err 00000000". Cela est dû à OpenGL qui est mal reconnu par No$Gba sous Unix. La solution la plus rapide est de redémarrer l'émulateur et de modifier dans les options "Emulations option" puis à "3D Render" mettre "nocash" ou "none" (initialement "opengl").
